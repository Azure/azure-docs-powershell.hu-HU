# Revoke-AzSnapshotAccess

## Áttekintés
A pillanatképhez való hozzáférés visszavonása

## SZINTAXISA

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Visszavonás-AzSnapshotAccess** parancsmag visszavonja a pillanatképhez való hozzáférést.

## Példák

### Példa 1
```
PS C:\> Revoke-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

A "ResourceGroup01" nevű erőforráscsoport "Snapshot01" nevű pillanatképéhez való hozzáférés visszavonása
