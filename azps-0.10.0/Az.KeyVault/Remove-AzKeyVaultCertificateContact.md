# Remove-AzKeyVaultCertificateContact

## Áttekintés
Törli a tanúsítványok értesítéseihez regisztrált névjegykártyát.

## SZINTAXISA

```
Remove-AzKeyVaultCertificateContact [-VaultName] <String> [-EmailAddress] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzKeyVaultCertificateContact** parancsmag törli a tanúsítványok értesítéseihez regisztrált ügyfelet.

## Példák

### 1. példa: a tanúsítvány-névjegykártya eltávolítása
```
PS C:\>Remove-AzKeyVaultCertificateContact -VaultName "Contoso01" -EmailAddress "patti.fuller@contoso.com"
```

Ez a parancs eltávolítja a Patti Fuller-t a Contoso01-kulcs boltozatához.
