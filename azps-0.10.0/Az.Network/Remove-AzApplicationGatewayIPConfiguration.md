# Remove-AzApplicationGatewayIPConfiguration

## Áttekintés
IP-konfiguráció eltávolítása egy alkalmazás-átjáróból.

## SZINTAXISA

```
Remove-AzApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Remove-AzApplicationGatewayIPConfiguration** parancsmag az Azure Application Gateway-ből távolítja el az IP-konfigurációt.

## Példák

### Példa 1: IP-konfiguráció eltávolítása Azure Application Gateway-ből
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

Az első parancs beolvassa az Application Gatewayt, és a $AppGw változóban tárolja.

A második parancs eltávolítja a Subnet02 nevű IP-konfigurációt az $AppGw-ban tárolt alkalmazás-átjáróról.
