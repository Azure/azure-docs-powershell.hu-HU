# Remove-AzLocalNetworkGateway

## Áttekintés
Helyi hálózati átjáró törlése

## SZINTAXISA

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A helyi hálózat átjárója a helyszíni VPN-eszközt jelképező objektum.

A **Remove-AzLocalNetworkGateway** parancsmag a helyszíni átjáró név és erőforrás csoport neve alapján történő törlésére szolgáló objektumot törli.

## Példák

### 1: helyi hálózati átjáró törlése
```
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Törli a helyi hálózati átjáró objektumát "myLocalGW" néven a "myRG" erőforráscsoporthoz.

Megjegyzés: a **Remove-AzVirtualNetworkGatewayConnection** parancsmag segítségével először törölnie kell minden kapcsolatot a helyi hálózati átjáróval.
