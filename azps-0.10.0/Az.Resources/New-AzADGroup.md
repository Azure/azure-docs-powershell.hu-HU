# New-AzADGroup

## Áttekintés
Új Active Directory-csoport létrehozása

## SZINTAXISA

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Új Active Directory-csoport létrehozása

## Példák

### Példa 1 – új HIRDETÉSCSOPORT létrehozása

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Egy új HIRDETÉSCSOPORTOT hoz létre a "MyGroupDisplayName" névvel és a levelek becenevevel " myemail@domain.com ".
