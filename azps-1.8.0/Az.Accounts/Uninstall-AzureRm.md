# Uninstall-AzureRm

## Áttekintés
Eltávolítja az összes AzureRm-modult egy gépről.

## SZINTAXISA

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
Eltávolítja az összes AzureRm-modult egy gépről.

## Példák

### Példa 1
```
PS C:\> Uninstall-AzureRm
```

Ha futtatja ezt a parancsot, akkor az összes AzureRm-modult eltávolítja a számítógépről annak a PowerShell-verziónak a verziójára, amelyben a parancsmag fut.
