# Resume-AzAnalysisServicesServer

## Áttekintés
Az Analysis Services Server egy példányának folytatása

## SZINTAXISA

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az Resume-AzAnalysisServicesServer parancsmag az Analysis Services Server egy példányát folytatja

## Példák

### Példa 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Ez a parancs folytatja a TestServer nevű felfüggesztett kiszolgáló resourcegroup testgroup
