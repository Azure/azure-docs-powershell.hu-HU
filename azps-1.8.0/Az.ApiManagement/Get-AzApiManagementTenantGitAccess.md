# Get-AzApiManagementTenantGitAccess

## Áttekintés
A git Access konfigurációját kapja egy bérlő számára.

## SZINTAXISA

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzApiManagementTenantGitAccess** parancsmag a bérlői fiókhoz tartozó git Access-konfigurációt kapja.

## Példák

### Példa 1: a bérlői hozzáférési konfiguráció beszerzése
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```

Ez a parancs a megadott környezetben a git Access konfigurációját kapja meg.
