# Remove-AzApiManagementProperty

## Áttekintés
Egy API-kezelési tulajdonság eltávolítása.

## SZINTAXISA

```
Remove-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzApiManagementProperty** parancsmag eltávolítja az Azure API Management **tulajdonságot**.

## Példák

### 1. példa: tulajdonság eltávolítása
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -PassThru
```

Ez a parancs eltávolítja az ID Property11 tartalmazó tulajdonságot.
