# Remove-AzAutomationConnection

## Áttekintés
Automatizálási kapcsolat eltávolítása

## SZINTAXISA

```
Remove-AzAutomationConnection [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzAutomationConnection** parancsmag eltávolít egy kapcsolatot az Azure Automation szolgáltatásból.

## Példák

### 1. példa: kapcsolat eltávolítása
```
PS C:\>Remove-AzAutomationConnection -AutomationAccountName "Contoso17" -Name "ContosoConnection" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs eltávolítja a ContosoConnection nevű tanúsítványt a Contoso17 nevű automatizálási fiókban.
