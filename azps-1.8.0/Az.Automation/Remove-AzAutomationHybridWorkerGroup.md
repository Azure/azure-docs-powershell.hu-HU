# Remove-AzAutomationHybridWorkerGroup

## Áttekintés
Eltávolítja a hibrid munkás csoportot az automatizálásból.

## SZINTAXISA

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
Az Remove-AzAutomationHybridWorkerGroup parancsmag eltávolítja a hibrid munkacsoportot az automatizálásból.

## Példák

### Példa 1
Ez a parancs eltávolítja a hibrid munkatársat név szerint.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
