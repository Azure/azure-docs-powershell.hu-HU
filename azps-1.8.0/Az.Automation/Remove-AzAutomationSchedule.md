# Remove-AzAutomationSchedule

## Áttekintés
Automatizálási ütemterv törlése

## SZINTAXISA

```
Remove-AzAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzAutomationSchedule** parancsmag az Azure Automation szolgáltatásból törli az ütemtervet.

## Példák

### 1. példa: ütemterv eltávolítása
```
PS C:\>Remove-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs törli a Schedule01 az automatizálási fiók Contoso17 az erőforrás csoport ResourceGroup01.
