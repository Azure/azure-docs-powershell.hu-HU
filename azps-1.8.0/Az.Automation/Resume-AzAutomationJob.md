# Resume-AzAutomationJob

## Áttekintés
Befejezi a felfüggesztett automatizálási feladatot.

## SZINTAXISA

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **resume-AzAutomationJob** parancsmag felfüggesztette az Azure automatizálási feladatot.
Adja meg a felfüggesztett feladatot.
A feladat felfüggesztéséhez használja az Suspend-AzAutomationJob parancsmagot.

## Példák

### 1. példa: felfüggesztett feladat folytatása
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Ez a parancs folytatja a megadott azonosítót tartalmazó munkát.
