# Disable-AzVmssDiskEncryption

## Áttekintés
Letiltja a lemez titkosítását a VM-léptékű készleteken.

## SZINTAXISA

```
Disable-AzVmssDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [[-ExtensionName] <String>] [-VolumeType <String>] [-ForceUpdate] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Letiltja a lemez titkosítását a VM-léptékű készleteken.

## Példák

### Példa 1
```
PS C:\> Disable-AzVmssDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Letiltja a VMSS001 nevű, a Group001 nevű erőforráscsoport csoportjába tartozó "VM" méretarányt.
