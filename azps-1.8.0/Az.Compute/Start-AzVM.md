# Start-AzVM

## Áttekintés
Azure virtuális gépet indít el.

## SZINTAXISA

### ResourceGroupNameParameterSetName (alapértelmezett)
```
Start-AzVM [-Name] <String> [-ResourceGroupName] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [[-Name] <String>] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **Start-AzVM** parancsmag egy Azure virtuális gépet indít el.

## Példák

### Példa 1: virtuális gép indítása
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Ez a parancs elindítja a VirtualMachine07 nevű virtuális gépet a ResourceGroup11.
