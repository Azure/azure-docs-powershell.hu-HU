# Remove-AzDataLakeAnalyticsCatalogCredential

## Áttekintés
Az Azure Data Lake Analytics hitelesítő adatainak törlése

## SZINTAXISA

```
Remove-AzDataLakeAnalyticsCatalogCredential [-Account] <String> [-DatabaseName] <String> [-Name] <String>
 [[-Password] <PSCredential>] [-Recurse] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az Remove-AzDataLakeAnalyticsCatalogCredential parancsmag törli az Azure Data Lake Analytics katalógusának hitelesítő adatait.

## Példák

### 1. példa: hitelesítő adatok eltávolítása
```
PS C:\> Remove-AzDataLakeAnalyticsCatalogCredential -AccountName "ContosoAdla" `
                      -DatabaseName "DatabaseName" `
                      -Name "CredName"
```

Ez a parancs eltávolítja a megadott adattó-Analytics-katalógus hitelesítő adatait.
