# Stop-AzDataLakeAnalyticsJob

## Áttekintés
Feladat visszavonása

## SZINTAXISA

```
Stop-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzDataLakeAnalyticsJob** parancsmag visszamondja az Azure Data-tó Analytics-munkáját.

## Példák

### Példa 1: feladat visszavonása
```
PS C:\>Stop-AzDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Ez a parancs lemondja a feladatot a megadott AZONOSÍTÓval.
