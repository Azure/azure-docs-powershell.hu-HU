# Grant-AzHDInsightHttpServicesAccess

## Áttekintés
HTTP-hozzáférést biztosít a fürthöz.

## SZINTAXISA

```
Grant-AzHDInsightHttpServicesAccess [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Grant-AzHDInsightHttpServicesAccess** parancsmag az Azure HDInsight-fürtökhöz az ODBC, a Ambari, a Oozie és a webes szolgáltatások segítségével http-hozzáférést ad.

## Példák

### 1. példa: HTTP-hozzáférés engedélyezése Azure HDInsight-fürthöz
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzHDInsightHttpServicesAccess `
            -ClusterName $clusterName `
            -HttpCredential $newClusterCreds
```

Ez a parancs HTTP-hozzáférést ad a-Hadoop-001 nevű fürthöz.
