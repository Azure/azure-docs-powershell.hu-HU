# Revoke-AzHDInsightRdpServicesAccess

## Áttekintés
Letiltja a Windows-fürtök RDP-elérését.

## SZINTAXISA

```
Revoke-AzHDInsightRdpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Visszavonás-AzHDInsightRdpServicesAccess** parancsmag letiltja a Remote Desktop Protocol (RDP) protokoll elérését egy Windows-alapú Azure HDInsight-fürthöz.

## Példák

### 1. példa: az RDP-hozzáférés letiltása egy adott fürthöz
```
PS C:\>Revoke-AzHDInsightRdpServicesAccess -ClusterName "your-hadoop-001"
```

Ez a parancs visszavonja az RDP-hozzáférést a-Hadoop-001 nevű fürthöz.
