# Use-AzHDInsightCluster

## Áttekintés
Kijelöli az Invoke-RmAzureHDInsightHiveJob parancsmaggal használni kívánt fürtöt.

## SZINTAXISA

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **use-AzHDInsightCluster** parancsmag azt az Azure HDInsight-fürtöt adja meg, amely a kaptár-feladatok elvégzéséhez használható Invoke-AzHDInsightHiveJob parancsmagot jelöli.

## Példák

### 1. példa: válasszon egy fürtöt a kaptár lekérdezések küldéséhez
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Ez a parancs kiválaszt egy fürtöt a kaptár-lekérdezések küldéséhez.
