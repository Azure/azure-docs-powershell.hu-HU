# Remove-AzMediaService

## Áttekintés
Adathordozó-szolgáltatás eltávolítása.

## SZINTAXISA

```
Remove-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzMediaService** parancsmag eltávolítja a médiafájlt.

## Példák

### 1. példa: a médiaadatfolyam-szolgáltatás eltávolítása
```
PS C:\>Remove-AzMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Ez a parancs eltávolítja a MediaService0011 nevű médiafájlt a ResourceGroup001 nevű erőforráscsoport erőforrás csoportjában.
