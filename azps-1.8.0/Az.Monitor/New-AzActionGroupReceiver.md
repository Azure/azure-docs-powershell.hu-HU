# New-AzActionGroupReceiver

## Áttekintés
Új akciócsoport-vevőkészülék létrehozása.

## SZINTAXISA

### NewEmailReceiver (alapértelmezett)
```
New-AzActionGroupReceiver -Name <String> [-EmailReceiver] -EmailAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewSmsReceiver
```
New-AzActionGroupReceiver -Name <String> [-SmsReceiver] [-CountryCode <String>] -PhoneNumber <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewWebhookReceiver
```
New-AzActionGroupReceiver -Name <String> [-WebhookReceiver] -ServiceUri <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **New-AzActionGroupReceiver** parancsmag új műveleti csoportba foglalt vevőkészüléket hoz létre a memóriában.

## Példák

### Példa 1: új e-mail-vevőkészülék létrehozása a memóriában.
```
PS C:\>$emailReceiver = New-AzActionGroupReceiver -Name 'emailReceiver1' -EmailReceiver -EmailAddress 'user1@example.com'
```

A parancs új e-mail-vevőkészüléket hoz létre a memóriában.

### 2. példa: új SMS-vevőkészülék létrehozása a memóriában.
```
PS C:\>$smsReceiver = New-AzActionGroupReceiver -Name 'smsReceiver1' -SmsReceiver -CountryCode '1' -PhoneNumber '5555555555'
```

A parancs új SMS-vevőkészüléket hoz létre a memóriában.

### 3. példa: új webhook-vevőkészülék létrehozása a memóriában.
```
PS C:\>$webhookReceiver = New-AzActionGroupReceiver -Name 'webhookReceiver1' -WebhookReceiver -ServiceUri 'http://test.com'
```

A parancs új webhook-vevőkészüléket hoz létre a memóriában.
