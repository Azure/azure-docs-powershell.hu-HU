# New-AzApplicationGatewayAutoscaleConfiguration

## Áttekintés
Az alkalmazás-átjáró automatikus méretarányos konfigurációjának létrehozása.

## SZINTAXISA

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzApplicationGatewayAutoscaleConfiguration** parancsmag automatikus méretezési konfigurációt hoz létre az Azure Application Gateway rendszerhez.

## Példák

### Példa 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Az első parancs automatikusan létrehoz egy automatikus méretarányos konfigurációt, amelynek minimális kapacitása 3.
A második parancs az alkalmazás-átjárót az automatikus méretezési konfigurációval hozza létre.
