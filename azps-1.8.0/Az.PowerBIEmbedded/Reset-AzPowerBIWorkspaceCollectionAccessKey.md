# Reset-AzPowerBIWorkspaceCollectionAccessKey

## Áttekintés
A megadott Access-kulcs visszaállítása

## SZINTAXISA

```
Reset-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **reset-AzPowerBIWorkspaceCollectionAccessKey** parancsmag visszaállítja a megadott Access-kulcsot a Power bi-munkaterületi gyűjteményben.

## Példák

### 1. példa: az elsődleges hozzáférés kulcsának alaphelyzetbe állítása
```
PS C:\>Reset-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Ez a parancs visszaállítja a megadott erőforráscsoport WCN11 nevű munkaterületi gyűjtemény elsődleges hozzáférési kulcsát.
