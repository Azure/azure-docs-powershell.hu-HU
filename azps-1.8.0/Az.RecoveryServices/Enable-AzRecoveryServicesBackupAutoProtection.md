# Enable-AzRecoveryServicesBackupAutoProtection

## Áttekintés
Ez a parancs lehetővé teszi a felhasználóknak, hogy automatikusan védjék az összes meglévő védtelen DBs-t és egy olyan DB-t, amelyet később felvesznek a megadott házirenddel. Az Azure Backup szolgáltatás ezután rendszeresen ellenőrzi az automatikusan védett tárolókat az új DBs-ben, és automatikusan megvédi őket.

## SZINTAXISA

```
Enable-AzRecoveryServicesBackupAutoProtection [-InputItem] <ProtectableItemBase>
 [-BackupManagementType] <BackupManagementType> [-WorkloadType] <WorkloadType> [-Policy] <PolicyBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
Az **enable-AzRecoveryServicesBackupAutoProtection** parancsmag az Azure Auto Backup Protection házirendjét egy védett elemen állítja be.

## Példák

### Példa 1
```
PS C:\> $Pol = Get-AzRecoveryServicesBackupProtectionPolicy -Name "DefaultPolicy"
PS C:\> $container = Get-AzRecoveryServicesBackupContainer -ContainerType AzureVMAppContainer
PS C:\> Get-AzRecoveryServicesBackupProtectableItem -Container $container -WorkloadType "MSSQL" -ItemType "SQLInstance" | Enable-AzRecoveryServicesBackupAutoProtection -BackupManagementType "AzureWorkload" -WorkloadType "MSSQL" -Policy $Pol
```

Az első parancsmag alapértelmezett házirend-objektumot kap, majd a $Pol változóban tárolja.
A második parancsmag a $Pol házirendjét használva állítja be a AzureWorkload a biztonsági mentési házirendet.
