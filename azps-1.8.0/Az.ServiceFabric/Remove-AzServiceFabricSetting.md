# Remove-AzServiceFabricSetting

## Áttekintés
Egy vagy több szolgáltatási anyag beállítás eltávolítása a fürtből

## SZINTAXISA

### OneSetting
```
Remove-AzServiceFabricSetting [-ResourceGroupName] <String> [-Name] <String> -Section <String>
 -Parameter <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BatchSettings
```
Remove-AzServiceFabricSetting [-ResourceGroupName] <String> [-Name] <String>
 -SettingsSectionDescription <PSSettingsSectionDescription[]> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Távolítsa el a **AzServiceFabricSetting** a Service Fabric beállításaiból a fürtből.

## Példák

### Példa 1
```
PS c:> Remove-AzServiceFabricSetting -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Section 'EseStore' -Parameter 'MaxCursors'
```

Ez a parancs eltávolítja a beállítások "MaxCursors" nevű "EseStore" szakaszát.
