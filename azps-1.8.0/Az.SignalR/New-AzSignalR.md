# New-AzSignalR

## Áttekintés
Szignáló szolgáltatás létrehozása

## SZINTAXISA

```
New-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-Location <String>] [-Sku <String>]
 [-UnitCount <Int32>] [-Tag <System.Collections.Generic.IDictionary`2[System.String,System.String]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Szignáló szolgáltatás létrehozása
Ha nem adja meg a paramétereket, a program a következő értékeket fogja használni:
* `ResourceGroupName`: az alapértelmezett erőforráscsoport beállítása `Set-AzDefault -ResourceGroupName` .
* `Location`: az erőforráscsoport helye
* `Sku`: `Standard_S1`

## Példák

### Jelző szolgáltatás teljesítményszámláló létrehozása
```powershell
PS C:\> New-AzSignalR -ResourceGroupName myResourceGroup1 -Name mysignalr1 -Location eastus -Sku Standard_S1

HostName                                           Location       ServerPort PublicPort ProvisioningState Version
--------                                           --------       ---------- ---------- ----------------- -------
mysignalr1.service.signalr.net                     eastus         5002       5001       Succeeded         1.0-preview
```
