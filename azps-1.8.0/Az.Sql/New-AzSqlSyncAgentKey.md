# New-AzSqlSyncAgentKey

## Áttekintés
Azure SQL-szinkronizáló ügynök kulcs létrehozása.

## SZINTAXISA

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzSqlSyncAgentKey** parancsmag az Azure SQL Sync Agent kulcsát hozza létre.

## Példák

### 1. példa: a Szinkronizáló ügynök kulcsa az Azure SQL szinkronizáló ügynök számára.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Ez a parancs létrehoz egy szinkronizáló ügynököt egy Azure SQL szinkronizáló ügynökhöz.
