# Remove-AzSqlDatabase

## Áttekintés
Azure SQL-adatbázis eltávolítása.

## SZINTAXISA

```
Remove-AzSqlDatabase [-DatabaseName] <String> [-Force] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzSqlDatabase** parancsmag eltávolítja az Azure SQL-adatbázisát.
Ezt a parancsmagot az SQL Server stretch Database szolgáltatása is támogatja az Azure-ban.

## Példák

### 1. példa: adatbázis eltávolítása Azure SQL Server-kiszolgálóról
```
PS C:\>Remove-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Ez a parancs eltávolítja a Database01 nevű adatbázist a kiszolgáló Server01.
