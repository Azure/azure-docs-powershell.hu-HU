# Remove-AzStorageSyncService

## Áttekintés
Ezzel a paranccsal törlődik a megadott tárterület-szinkronizálási szolgáltatás.

## SZINTAXISA

### InputObjectParameterSet (alapértelmezett)
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StringParameterSet
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Ezzel a paranccsal törlődik a megadott tárterület-szinkronizálási szolgáltatás. A tárterület-szinkronizálási szolgáltatás csak akkor távolítható el, ha az összes tárolt szinkronizálási csoport és regisztrált kiszolgáló törlődik először.

## Példák

### Példa 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Ezzel a paranccsal törlődik a tárterület-szinkronizálási szolgáltatás.
