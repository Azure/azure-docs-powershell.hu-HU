# Remove-AzStreamAnalyticsJob

## Áttekintés
Adatfolyam-elemzési feladat eltávolítása.

## SZINTAXISA

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzStreamAnalyticsJob** parancsmag aszinkron módon töröl egy adott adatfolyam-elemzési feladatot az Azure-ban.

## Példák

### 1. példa: feladat eltávolítása
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Ez a parancs eltávolítja a projekt StreamingJob.
