# Remove-AzApiManagementAuthorizationServer

## Áttekintés
Egy engedélyezési kiszolgáló eltávolítása.

## SZINTAXISA

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzApiManagementAuthorizationServer** parancsmag eltávolítja az Azure API-kezelési engedélyezési kiszolgálóját.

## Példák

### 1. példa: engedélyezési kiszolgáló eltávolítása
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Ez a parancs eltávolítja a megadott API-kezelési engedélyezési kiszolgálót.
Mivel az *erő* paraméter van megadva, nincs szükség megerősítésre.
