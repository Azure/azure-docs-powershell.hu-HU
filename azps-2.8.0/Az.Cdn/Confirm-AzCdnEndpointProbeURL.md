# Confirm-AzCdnEndpointProbeURL

## Áttekintés
Ellenőrzi a szonda URL-címét.

## SZINTAXISA

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Confirm-AzCdnEndpointProbeURL** parancsmag igazolja, hogy a szonda URL-címe használható a dinamikus webhely-gyorsításhoz.

## Példák

### Példa 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

Hitelesíti a szonda URL-címét " http://www.bing.com/images "
