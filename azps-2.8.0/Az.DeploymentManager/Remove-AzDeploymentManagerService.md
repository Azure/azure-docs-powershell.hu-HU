# Remove-AzDeploymentManagerService

## Áttekintés
A szolgáltatás törlése.. A szolgáltatás törlése előtt minden, a szolgáltatás alatt létrehozott szolgáltatási egységet törölni kell.

## SZINTAXISA

### Interaktív (alapértelmezett)
```
Remove-AzDeploymentManagerService [-ResourceGroupName] <String> [-ServiceTopologyName] <String>
 [-Name] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByServiceTopologyObject
```
Remove-AzDeploymentManagerService [-Name] <String> [-ServiceTopologyObject] <PSServiceTopologyResource>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByServiceTopologyResourceId
```

 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```

```

## Leírás
A **Remove-AzDeploymentManagerService** parancsmag a szolgáltatás topológiája alatt töröl egy szolgáltatást.
Adja meg a szolgáltatást a nevével, a szolgáltatási topológiával, és az erőforráscsoport nevével. Másik lehetőségként megadhatja a szolgáltatás objektumát vagy a ResourceId.

## Példák

### Példa 1
```powershell
PS C:\> Remove-AzDeploymentManagerService -ResourceGroupName ContosoResourceGroup -ServiceTopologyName ContosoServiceTopology -Name ContosoService1
```

Ez a parancs törli a ContosoService1 nevű szolgáltatást a ContosoResourceGroup ContosoServiceTopology nevű szolgáltatási topológiában.

### 2. példa: szolgáltatás törlése az erőforrás-azonosítóval
```powershell
PS C:\> Remove-AzDeploymentManagerService -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/serviceTopologies/ContosoServiceTopology/services/ContosoService1"
```

Ez a parancs törli a ContosoService1 nevű szolgáltatást a ContosoResourceGroup ContosoServiceTopology nevű szolgáltatási topológiában.

### Példa 3: szolgáltatás törlése a Service objektummal.
```powershell
PS C:\> Remove-AzDeploymentManagerService -InputObject $serviceObject
```

Ez a parancs törli azt a szolgáltatást, amelynek a neve, a szolgáltatás topológiájának neve és a ResourceGroup megegyezik a $serviceObject neve, a ServiceTopologyName és a ResourceGroupName tulajdonságaival.
