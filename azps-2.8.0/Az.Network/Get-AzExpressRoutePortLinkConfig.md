# Get-AzExpressRoutePortLinkConfig

## Áttekintés
Beolvassa a ExpressRoutePort-kapcsolat konfigurációját.

## SZINTAXISA

### ResourceNameParameterSet (alapértelmezett)
```
Get-AzExpressRoutePortLinkConfig -ExpressRoutePort <PSExpressRoutePort> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePortLinkConfig -ResourceId <String> -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzExpressRoutePortLinkConfig** parancsmag kikeresi egy ExpressRoutePort hivatkozásának konfigurációját.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -Name Link1
```

A ExpressRoutePort $erport Link1-konfigurációjának beolvasása

### 2. példa
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -ResourceId $id
```

A ExpressRoutePort-ban a ResourceId $id a hivatkozás konfigurációját kapja $erport
