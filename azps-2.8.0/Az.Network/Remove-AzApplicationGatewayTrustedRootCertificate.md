# Remove-AzApplicationGatewayTrustedRootCertificate

## Áttekintés
Egy megbízható főtanúsítvány eltávolítása egy alkalmazás-átjáróból.

## SZINTAXISA

```
Remove-AzApplicationGatewayTrustedRootCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzApplicationGatewayTrustedRootCertificate** parancsmag egy meglévő alkalmazás-átjárótól eltávolítja a megbízható legfelső szintű tanúsítványokat.

## Példák

### Példa 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name "myRootCA"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Az első parancs beolvassa az Application Gatewayt, és a $gw változóban tárolja.
A második parancs eltávolítja a myRootCA nevű megbízható főtanúsítványt az $gw-ban tárolt alkalmazás-átjáróról.
A harmadik parancs az Azure rendszerhez frissíti az Application Gatewayt.
