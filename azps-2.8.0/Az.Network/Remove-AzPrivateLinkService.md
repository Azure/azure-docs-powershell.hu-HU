# Remove-AzPrivateLinkService

## Áttekintés
Privát link szolgáltatás eltávolítása

## SZINTAXISA

```
Remove-AzPrivateLinkService -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzPrivateLinkService** parancsmag eltávolítja a privát hivatkozás szolgáltatást

## Példák

### Példa 1
```
Remove-AzPrivateLinkService -ResourceGroupName TestResourceGroup -Name TestPrivateLinkService
```

Ez a példa eltávolítja a TestPrivateLinkService nevű privát hivatkozás szolgáltatást.
