# Remove-AzOperationalInsightsWorkspace

## Áttekintés
Munkaterület eltávolítása.

## SZINTAXISA

```
Remove-AzOperationalInsightsWorkspace [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzOperationalInsightsWorkspace** parancsmag töröl egy meglévő munkaterületet.
Ha a munkaterületet egy meglévő fiókhoz kapcsolta a *Vevőkód* paraméterrel a létrehozási időponton keresztül, az eredeti fiók nem törlődik az üzemeltetési statisztika portálon.

## Példák

### 1. példa: munkaterület eltávolítása név szerint
```
PS C:\>Remove-AzOperationalInsightsWorkspace -ResourceGroupName "ContosResourceGroup" -Name "MyWorkspace"
```

Ez a parancs eltávolítja a MyWorkspace nevű munkaterületet az ContosoResourceGroup nevű erőforráscsoport-csoportból.

### 2. példa: munkaterület eltávolítása a csővezeték használatával és megerősítés nélkül
```
PS C:\>Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosResourceGroup" -Name "MyWorkspace" | Remove-AzOperationalInsightsWorkspace -Force
```

Ez a parancs a Get-AzOperationalInsightsWorkspace parancsmagot használja a MyWorkspace nevű munkaterület beolvasásához, majd átadja azt a **Remove-AzOperationalInsightsWorkspace** parancsmagnak a pipeline operátorral az eltávolításához.
Az *erő* paraméter megadása óta a parancs nem kéri, mielőtt eltávolítja a munkaterületet.
