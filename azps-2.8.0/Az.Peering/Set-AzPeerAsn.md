# Set-AzPeerAsn

## Áttekintés
Elérhetőségi adatok frissítése

## SZINTAXISA

### ByName (alapértelmezett)
```
Set-AzPeerAsn [-Name] <String> [-Email <String[]>] [-Phone <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Alapértelmezett
```
Set-AzPeerAsn [-InputObject] <PSPeerAsn> [-Email <String[]>] [-Phone <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Lehetővé teszi, hogy az előfizetésben szereplő PeerAsn frissítse a partnerek adatait.

## Példák

### Példa 1
