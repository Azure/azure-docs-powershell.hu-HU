# Remove-AzRedisCache

## Áttekintés
Eltávolítja a Redis-gyorsítótárat.

## SZINTAXISA

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzRedisCache** parancsmag eltávolítja az Azure Redis-gyorsítótárát.

## Példák

### 1. példa: Redis-gyorsítótár eltávolítása és az eredmény visszaadása
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Ez a parancs eltávolítja a Redis-gyorsítótárat, és megjeleníti, hogy a művelet sikeres-e.

### 2. példa: Redis-gyorsítótár eltávolítása, és az eredmény nem jeleníthető meg
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Ez a parancs eltávolítja a Redis-gyorsítótárat.
Mivel a *PassThru* paraméter nincs megadva, a művelet eredménye nem jelenik meg.
