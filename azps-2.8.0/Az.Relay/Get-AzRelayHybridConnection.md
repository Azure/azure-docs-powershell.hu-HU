# Get-AzRelayHybridConnection

## Áttekintés
A továbbítási névtérben található megadott HybridConnection leírását kapja.

## SZINTAXISA

```
Get-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzRelayHybridConnection** parancsmag leírását kapja a megadott HybridConnection a továbbítási névtéren belül.

## Példák

### Példa 1
```
PS C:\>Get-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection

                              ybridConnections/TestHybridConnection
Name                        : TestHybridConnection

```

A HybridConnection leírását számítja ki.
