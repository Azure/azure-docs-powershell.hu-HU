# Add-AzServiceFabricNode

## Áttekintés
Csomópontok hozzáadása a fürt adott csomópont-típusához

## SZINTAXISA

```
Add-AzServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Add-AzServiceFabricNode** segítségével csomópontokat adhat hozzá az adott csomópont-típushoz. Csak meg kell adnia a csomópont típusához hozzáadni kívánt csomópontok számát.

## Példák

### Példa 1
```powershell
PS c:> Add-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Ez a parancs 2 csomópontot ad hozzá az "N1" csomópont típushoz.
