# Disable-AzSqlInstanceAdvancedDataSecurity

## Áttekintés
A továbbfejlesztett adatbiztonság letiltása egy felügyelt példányon.

## SZINTAXISA

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **disable-AzSqlInstanceAdvancedDataSecurity** parancsmag a továbbfejlesztett adatbiztonságot letiltja egy felügyelt példányon.

## Példák

### Példa 1 – a felügyelt példány speciális adatbiztonságának letiltása
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Példa 2 – a kiszolgáló speciális adatbiztonságának letiltása a felügyelt példányok erőforrásból
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
