# Disable-AzSqlServerAdvancedDataSecurity

## Áttekintés
Letiltja a kiszolgálón a speciális adatbiztonságot.

## SZINTAXISA

```
Disable-AzSqlServerAdvancedDataSecurity [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **disable-AzSqlServerAdvancedDataSecurity** parancsmag letiltja a kiszolgálón a speciális adatbiztonságot.

## Példák

### Példa 1
### Példa 1 – a Server Advanced Data Security letiltása
```powershell
PS C:\>  Disable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```

### Példa 2 – a Server Advanced Data Security letiltása a kiszolgálói erőforrásból
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Disable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```
