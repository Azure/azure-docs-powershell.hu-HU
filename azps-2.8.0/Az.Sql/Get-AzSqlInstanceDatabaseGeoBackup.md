# Get-AzSqlInstanceDatabaseGeoBackup

## Áttekintés
Információt ad az Azure SQL Managed instance Database redundáns biztonsági másolatról.

## SZINTAXISA

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzSqlInstanceDatabaseGeoBackup** parancsmag egy vagy több Azure SQL-adatbázisból redundáns biztonsági mentést kap egy Azure SQL-adatbázisból felügyelt példányról.
