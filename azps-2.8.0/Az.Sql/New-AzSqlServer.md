# New-AzSqlServer

## Áttekintés
SQL-adatbázis kiszolgálóját hozza létre.

## SZINTAXISA

```
New-AzSqlServer -ServerName <String> -SqlAdministratorCredentials <PSCredential> -Location <String>
 [-Tags <Hashtable>] [-ServerVersion <String>] [-AssignIdentity] [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzSqlServer** parancsmag egy Azure SQL adatbázis-kiszolgálót hoz létre.

## Példák

### Példa 1: új Azure SQL adatbázis-kiszolgáló létrehozása
```
PS C:\>New-AzSqlServer -ResourceGroupName "ResourceGroup01" -Location "Central US" -ServerName "server01" -ServerVersion "12.0" -SqlAdministratorCredentials (Get-Credential)
ResourceGroupName        : resourcegroup01
ServerName               : server01
Location                 : Central US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
```

Ez a parancs a 12-es verziójú Azure SQL-adatbázis kiszolgálóját hozza létre.
