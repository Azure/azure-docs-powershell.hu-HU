# Remove-AzStorageSyncCloudEndpoint

## Áttekintés
Ez a parancs a megadott Felhőbeli végpontot fogja törölni egy szinkronizálási csoportból. Ha nincs legalább egy felhő végpontja, a szinkronizálási csoportban nem lehet más kiszolgálói végpontokat szinkronizálni.

## SZINTAXISA

### StringParameterSet (alapértelmezett)
```
Remove-AzStorageSyncCloudEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-InputObject] <PSCloudEndpoint> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Ez a parancs a megadott Felhőbeli végpontot fogja törölni egy szinkronizálási csoportból. Az Azure-fájl megosztásakor a Felhőbeli végpontokra mutató hivatkozások érintetlenül maradnak a folyamat során. Ez a parancs csak a leszereléshez szolgál. A Felhőbeli végpontok eltávolítása romboló művelet. A kiszolgálói végpontok nem szinkronizálhatók legalább egy Felhőbeli végpont nélkül. Ezt a műveletet nem kell végrehajtani a szinkronizálási problémák megoldásához. Ha az Azure-fájlmegosztás ismét megjelenik ugyanahhoz a szinkronizálási csoporthoz, Felhőbeli végpontként, az ütközési fájlokat és más nem szándékolt következményeket okozhat.

## Példák

### Példa 1
```powershell
PS C:\> Remove-AzStorageSyncCloudEndpoint -Force -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -Name "myCloudEndpointName"
```

Ez a parancs eltávolítja a Felhőbeli végpontot.
