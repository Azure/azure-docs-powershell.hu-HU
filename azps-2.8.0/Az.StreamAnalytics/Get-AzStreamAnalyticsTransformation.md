# Get-AzStreamAnalyticsTransformation

## Áttekintés
Információkat kap az adatfolyam-elemzésről.

## SZINTAXISA

```
Get-AzStreamAnalyticsTransformation [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzStreamAnalyticsTransformation** parancsmag információkat kap az adatfolyam-elemzési feladatokon meghatározott átalakításról.

## Példák

### 1. példa: információk kérése az adatfolyam-elemzésről
```
PS C:\>Get-AzStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "StreamingJob"
```

Ez a parancs információt ad az StreamingJob nevű StreamingJob nevű átalakításról.
