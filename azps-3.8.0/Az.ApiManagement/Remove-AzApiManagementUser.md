# Remove-AzApiManagementUser

## Áttekintés
Egy meglévő felhasználó törlése

## SZINTAXISA

```
Remove-AzApiManagementUser -Context <PsApiManagementContext> -UserId <String> [-DeleteSubscriptions]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzApiManagementUser** parancsmag egy meglévő felhasználó törlésére.

## Példák

### Példa 1: felhasználó törlése
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementUser -Context $apimContext -UserId "0123456789" -Force
```

Ez a parancs egy meglévő felhasználót töröl.
