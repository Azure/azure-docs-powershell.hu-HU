# Set-AzApiManagementGroup

## Áttekintés
API-kezelési csoport beállítása.

## SZINTAXISA

```
Set-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **set-AzApiManagementGroup** parancsmag API-kezelési csoportot állít be.

## Példák

### Példa 1: felügyeleti csoport beállítása
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementGroup -Context $apimContext -GroupId "0001" -Description "Updated Management Group" -Name "Group0001"
```

Ez a parancs a Group0001 nevű felügyeleti csoportot állítja be.
