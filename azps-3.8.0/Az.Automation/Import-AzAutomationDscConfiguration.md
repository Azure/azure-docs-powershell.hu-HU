# Import-AzAutomationDscConfiguration

## Áttekintés
A DSC-konfiguráció importálása automatizálásra

## SZINTAXISA

```
Import-AzAutomationDscConfiguration -SourcePath <String> [-Tags <IDictionary>] [-Description <String>]
 [-Published] [-Force] [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az **import-AzAutomationDscConfiguration** PARANCSMAG az APS-alapú szükséges állapot-konfigurációs (DSC) konfigurációt importálja az Azure automatizálási szolgáltatásba.
Adja meg egy olyan APS-parancsfájl elérési útját, amely egyetlen DSC-konfigurációt tartalmaz.

## Példák

### 1. példa: DSC-konfiguráció importálása automatizálásra
```
PS C:\>Import-AzAutomationDscConfiguration -AutomationAccountName "Contoso17"-ResourceGroupName "ResourceGroup01" -SourcePath "C:\DSC\client.ps1" -Force
```

Ez a parancs importálja a DSC konfigurációt a client.ps1 nevű fájlban a Contoso17 nevű automatizálási fiókba. A parancs az *erő* paramétert adja meg. Ha van egy meglévő DSC-konfiguráció, az azt jelenti, hogy a parancs felülírja.
