# Remove-AzAutomationDscConfiguration

## Áttekintés
Az automatizálásból eltávolítja a DSC-konfigurációkat.

## SZINTAXISA

```
Remove-AzAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzAutomationDscConfiguration** parancsmag az Azure automatizálási beállításainak (DSC) megfelelő konfigurációját távolítja el.
