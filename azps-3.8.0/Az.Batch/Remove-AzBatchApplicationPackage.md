# Remove-AzBatchApplicationPackage

## Áttekintés
Egy alkalmazáscsomag-rekord és a bináris fájl törlése

## SZINTAXISA

```
Remove-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationName] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzBatchApplicationPackage** parancsmag alkalmazásspecifikus csomagot és bináris fájlt töröl egy Azure-fiókból.

## Példák

### 1. példa: alkalmazáscsomag törlése egy köteg fiókból
```
PS C:\>Remove-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "litware" -ApplicationVersion "1.0"
```

Ez a parancs törli a Bárdos Iparcikk alkalmazás 1,0-ös verzióját a ContosoBatchGroup-fiókból.
A parancs törli mind a csomagot, mind a bináris fájlt tartalmazó blobot.
