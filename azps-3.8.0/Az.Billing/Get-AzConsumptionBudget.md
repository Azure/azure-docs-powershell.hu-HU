# Get-AzConsumptionBudget

## Áttekintés
A költségvetések listájának beszerzése előfizetésben vagy erőforráscsoporthoz

## SZINTAXISA

```
Get-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## Leírás
A Get-AzConsumptionBudget parancsmag egy előfizetésben vagy egy erőforráscsoport listájában kapja meg a költségvetéseket.
