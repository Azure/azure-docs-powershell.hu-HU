# Publish-AzBlueprint

## Áttekintés
A tervezet új verziójának közzététele

## SZINTAXISA

```
Publish-AzBlueprint -Version <String> -ChangeNote <String> -Blueprint <PSBlueprint> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A Blueprint-definíciók új verziójának közzététele

## Példák

### Példa 1
```powershell
PS C:\> Publish-AzBlueprint -Blueprint $bp -Version 1.0
```
