# Get-AzCdnEndpointNameAvailability

## Áttekintés
A CDN-végpont elérhetőségi állapotát kapja.

## SZINTAXISA

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzCdnEndpointNameAvailability** parancsmag az Azure Content Delivery Network (CDN) végpontjának elérhetőségi állapotát kapja.
