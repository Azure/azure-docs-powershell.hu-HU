# Invoke-AzVMReimage

## Áttekintés
Egy Azure Virtual Machine Reimage-et.

## SZINTAXISA

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **meghívó-AzVMReimage** parancsmag egy Azure Virtual Machine-t ábrázol.

## Példák

### Példa 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Ez a parancs a VirtualMachine07 nevű virtuális gépet a ResourceGroup11-ban.
