# Remove-AzAvailabilitySet

## Áttekintés
Az Azure-ról származó elérhetőségi készlet eltávolítása.

## SZINTAXISA

```
Remove-AzAvailabilitySet [-ResourceGroupName] <String> [[-Name] <String>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzAvailabilitySet** parancsmag az Azure-ról származó elérhetőségi készletet távolítja el.

## Példák

### Példa 1: elérhetőségi készlet eltávolítása
```
PS C:\> Remove-AzAvailabilitySet -Name "AvailabilitySet03" -ResourceGroupName "ResourceGroup11"
```

Ez a parancs eltávolítja a ResourceGroup11 nevű erőforráscsoport AvailabilitySet03 nevű elérhetőségi készletét.
