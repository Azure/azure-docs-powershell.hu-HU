# New-AzCosmosDBSqlCompositePath

## Áttekintés
Új PSCompositePath típusú objektum létrehozása A set-AzCosmosDBSqlContainer paraméter értéke lehet.

## SZINTAXISA

```
New-AzCosmosDBSqlCompositePath [-Path <String>] [-Order <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
Az SQL API CompositePath megfelelő objektum.

## Példák

### Példa 1
```powershell
PS C:\> New-AzCosmosDBSqlCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
