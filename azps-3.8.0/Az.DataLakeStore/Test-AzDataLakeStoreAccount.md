# Test-AzDataLakeStoreAccount

## Áttekintés
Teszteli az adattó áruházbeli fiók létezését.

## SZINTAXISA

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **teszt-AzDataLakeStoreAccount** parancsmag az Adattó áruházbeli fiók létezését teszteli.

## Példák

### Példa 1: fiók tesztelése
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Ez a parancs azt teszteli, hogy a ContosoADL nevű fiók létezik-e.
