# Add-AzKeyVaultNetworkRule

## Áttekintés
Felvesz egy olyan szabályt, amely az ügyfél internetes címén alapuló fő boltozathoz való hozzáférés korlátozását jelenti.

## SZINTAXISA

### ByVaultName (alapértelmezett)
```
Add-AzKeyVaultNetworkRule [-VaultName] <String> [[-ResourceGroupName] <String>] [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Add-AzKeyVaultNetworkRule [-InputObject] <PSKeyVault> [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Add-AzKeyVaultNetworkRule [-ResourceId] <String> [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
Az **Add-AzKeyVaultNetworkRule** parancsmag hozzáférést ad vagy korlátoz a kulcsfájl eléréséhez az IP-címük vagy a virtuális hálózat által kijelölt hívók számára. A szabály lehetővé tette a hozzáférés korlátozását más felhasználók, alkalmazások vagy biztonsági csoportok számára, amelyekre a hozzáférési házirenden keresztül engedélyt kapott.
