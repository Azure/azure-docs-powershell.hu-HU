# Remove-AzKustoCluster

## Áttekintés
Töröl egy meglévő Kusto-fürtöt.

## SZINTAXISA

### ByNameAndResourceGroup (alapértelmezett)
```
Remove-AzKustoCluster [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Töröl egy meglévő Kusto-fürtöt.

## Példák

### Példa 1 – meglévő Kusto-fürt törlése név szerint

```
PS C:\> Remove-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

A fenti parancs törli a "mykustocluster" nevű Kusto-fürtöt az "testrg" erőforráscsoport nevében.

### 2. példa – meglévő Kusto-fürt törlése csővezetékről

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Remove-AzKustoCluster
```

A fenti parancs az "mykustocluster" nevű Kusto-fürtöt az "testrg" erőforráscsoport nevében a `Get-AzKustoCluster` parancsmag használatával kapja, majd az eredményt a `Remove-AzKustoCluster` törléséhez.
