# New-AzUserAssignedIdentity

## Áttekintés
Egy új felhasználó által kiosztott identitást hoz létre, vagy egy meglévő, felhasználó által kiosztott identitást frissít.

## SZINTAXISA

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzUserAssignedIdentity** parancsmag új felhasználó által kiosztott identitást hoz létre. Ha egy már meglévő identitással használja, frissítette az identitást.
Ha Azure Resource Manager címkét szeretne hozzáadni az identitáshoz, használja az Set-AzResource parancsmagot.
