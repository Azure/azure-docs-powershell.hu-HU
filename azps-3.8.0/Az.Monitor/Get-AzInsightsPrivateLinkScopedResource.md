# Get-AzInsightsPrivateLinkScopedResource

## Áttekintés
A magánjellegű kapcsolat hatókörű erőforrásának beszerzése

## SZINTAXISA

### ByScopeParameterSet (alapértelmezett)
```
Get-AzInsightsPrivateLinkScopedResource -ResourceGroupName <String> -ScopeName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObjectParameterSet
```
Get-AzInsightsPrivateLinkScopedResource [-Name <String>] -InputObject <PSMonitorPrivateLinkScope>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzInsightsPrivateLinkScopedResource -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A magánjellegű kapcsolat hatókörű erőforrásának beolvasása vagy listázása, a hatókörben lévő erőforrás lehet naplózási Analytics munkaterületet vagy alkalmazás-keresési összetevőt.

## Példák

### Példa 1
```powershell
Get-AzInsightsPrivateLinkScopedResource -ResourceGroupName "rg_name" -ScopeName "scope_name"
```

Lista hatókörű erőforrás a privát hivatkozás hatóköre "scope_name"

### 2. példa
```powershell
Get-AzInsightsPrivateLinkScopedResource -ResourceGroupName "rg_name" -ScopeName "scope_name" -Name "scoped_resource_name"
```

A "scope_name" név "scoped_resource_name" nevű "a magánjellegű hivatkozás hatóköre" területének beolvasása
