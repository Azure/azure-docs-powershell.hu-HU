# Get-AzServiceEndpointPolicyDefinition

## Áttekintés
A szolgáltatás végpont-házirendjének definícióját kapja meg.

## SZINTAXISA

```
Get-AzServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Get-AzServiceEndpointPolicyDefinition** parancsmag szolgáltatási végpont házirend-definícióját kapja meg.

## Példák

### Példa 1
```
$policydef= Get-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ServiceEndpointPolicy $Policy
```

Ez a parancs beolvassa a ServiceEndpointPolicyDefinition1-ban a ServiceEndpointPolicy $Policy nevű szolgáltatás végpont-házirendjének definícióját $policydef változóban tárolja.
