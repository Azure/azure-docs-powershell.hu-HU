# New-AzApplicationGatewayFirewallPolicy

## Áttekintés
Egy alkalmazás-átjáró tűzfal-házirendjét hozza létre.

## SZINTAXISA

```
New-AzApplicationGatewayFirewallPolicy -Name <String> -ResourceGroupName <String> -Location <String>
 [-CustomRule <PSApplicationGatewayFirewallCustomRule[]>]
 [-PolicySetting <PSApplicationGatewayFirewallPolicySettings>]
 [-ManagedRule <PSApplicationGatewayFirewallPolicyManagedRules>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzApplicationGatewayFirewallPolicy** parancsmag egy alkalmazás-átjáró tűzfal-házirendet hoz létre.

## Példák

### Példa 1
```powershell
PS C:\> $firewallPolicy = New-AzApplicationGatewayFirewallPolicy -Name wafResource1 -ResourceGroupName "rg1"  -Location  "westus" -CustomRules $customRule
```

Ezzel a paranccsal létrehoz egy új Azure Application gateway-tűzfal-házirendet "wafResource1" nevű erőforráscsoport "rg1" nevű erőforráscsoport "westus" nevű "" nevű "" nevű új, a $customRule változóban meghatározott egyéni szabályokkal.
