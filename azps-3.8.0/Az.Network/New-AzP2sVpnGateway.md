# New-AzP2sVpnGateway

## Leírás
A **New-AzP2sVpnGateway** parancsmag lehetővé teszi, hogy új P2SVpnGateway hozzon létre a VirtualHub területről a webhelyre való kapcsolódás pontra a webhely ügyfeleitől az Azure VirtualWan.
