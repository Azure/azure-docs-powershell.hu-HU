# New-AzPrivateLinkServiceIpConfig

## Áttekintés
Hozzon létre egy privát kapcsolat szolgáltatás IP-konfigurációját.

## SZINTAXISA

```
New-AzPrivateLinkServiceIpConfig -Name <String> [-PrivateIpAddressVersion <String>]
 [-PrivateIpAddress <String>] [-PublicIpAddress <PSPublicIpAddress>]
 [-Subnet <PSSubnet>] [-Primary]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **New-AzPrivateLinkServiceIpConfig** parancsmag létrehoz egy privát kapcsolati szolgáltatás IP-konfigurációját. Ezt a konfigurációt a program a privát végpontról származó bejövő forgalom forrás NAT-nek használja. 

## Példák

### Példa 1
```
New-AzPrivateLinkServiceIpConfig -Name $IpConfigurationName -PrivateIpAddress "10.0.0.5" -Primary
```

Ez a példa létrehoz egy privát hivatkozás szolgáltatás IP-konfigurációját a memóriában.
