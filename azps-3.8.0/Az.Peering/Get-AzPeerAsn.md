# Get-AzPeerAsn

## Áttekintés
A PeerAsn objektum lesz a KARJÁN.

## SZINTAXISA

```
Get-AzPeerAsn [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Az előfizetéshez tartozó PeerAsn kapja meg.
