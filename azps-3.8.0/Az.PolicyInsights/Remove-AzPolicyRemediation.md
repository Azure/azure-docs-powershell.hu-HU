# Remove-AzPolicyRemediation

## Áttekintés
Házirend-kármentesítés törlése

## SZINTAXISA

### ByName (alapértelmezett)
```
Remove-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AllowStop] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzPolicyRemediation -ResourceId <String> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzPolicyRemediation -InputObject <PSRemediation> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzPolicyRemediation** parancsmag törli a házirend-szervizelést.

## Példák

### Példa 1: házirend-szervizelés törlése az erőforráscsoport hatókörében
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

Ez a parancs törli a "remediation1" nevű kármentesítési tevékenységet a "myRG" erőforráscsoport nevében.

### 2. példa: a kezelési csoport törlése csővezetéken keresztüli szervizeléssel
```
PS C:\> $remediation = Get-AzPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Remove-AzPolicyRemediation -Confirm
```

Ez a parancs törli a "remediation1" nevű szervizelést a "mg1" felügyeleti csoportból. A megerősítést kérő üzenet jelenik meg az erőforrás törlése előtt.

### 3. példa: házirend-kármentesítés megszüntetése és törlése
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1" -AllowStop
```

Ez a parancs törli a "remediation1" nevű kármentesítési tevékenységet a "myRG" erőforráscsoport nevében. Ha a szervizelés folyamatban van, a program törli a törlés előtt.
