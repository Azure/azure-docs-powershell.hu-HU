# Stop-AzPolicyRemediation

## Áttekintés
Visszavonja a folyamatban lévő házirend-szervizelést.

## SZINTAXISA

### ByName (alapértelmezett)
```
Stop-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Stop-AzPolicyRemediation -ResourceId <String> [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Stop-AzPolicyRemediation -InputObject <PSRemediation> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzPolicyRemediation** parancsmag megszünteti a folyamatban lévő házirend-szervizelést. Az aktív üzembe állítások törlődnek, és a rendszer nem hoz létre új központi telepítőket.

## Példák

### Példa 1: házirend-szervizelés megszüntetése az erőforráscsoport hatókörében
```
PS C:\> Stop-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

Ez a parancs a "myRG" erőforráscsoport "remediation1" nevű szervizelését törli.

### 2. példa: a kezelési csoport lemondása csővezetéken keresztüli szervizelésének lemondása
```
PS C:\> $remediation = Get-AzPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Stop-AzPolicyRemediation
```

Ez a parancs lemondja a "remediation1" nevű "mg1" felügyeleti csoportban található "" nevű szervizelést.
