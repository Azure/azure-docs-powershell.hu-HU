# Get-AzRedisCacheFirewallRule

## Áttekintés
Tűzfal-szabályok beszerzése a Redis-gyorsítótárban.

## SZINTAXISA

```
Get-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> [-RuleName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Ha a **RuleName** paraméter, ha meg van adva, a **Get-AzRedisCacheFirewallRule** parancsmag részletesen ismerteti az Azure Redis gyorsítótárában lévő megadott tűzfalszabályok adatait. Ha csak a **név** érték van megadva, ez a művelet a Redis-gyorsítótárban elérhető összes tűzfal-szabályt beilleszti.

## Példák

### Példa 1: egyetlen tűzfalszabály beszerzése
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32
```

Ez a parancs a mycache nevű Redis-gyorsítótárból kapja meg a ruleone nevű tűzfal-szabályt.

### 2. példa: a tűzfalszabályok minden szabályának beolvasása
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruletwo
        RuleName          : ruletwo
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.33
        EndIP             : 10.0.0.64
```

Ez a parancs a mycache nevű Redis-gyorsítótárból kapja meg az összes tűzfal-szabályt.
