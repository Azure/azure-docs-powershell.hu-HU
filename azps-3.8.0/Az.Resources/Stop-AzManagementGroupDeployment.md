# Stop-AzManagementGroupDeployment

## Áttekintés
Futó telepített példány lemondása felügyeleti csoportban

## SZINTAXISA

### StopByDeploymentName (alapértelmezett)
```
Stop-AzManagementGroupDeployment [-ManagementGroupId] <String> [-Name] <String> [-PassThru]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzManagementGroupDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzManagementGroupDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzManagementGroupDeployment** parancsmag lemondja azokat a telepített példányokat, amelyek egy felügyeleti csoportban elkezdődtek, de nem fejeződött be.
Ha meg szeretné szüntetni egy központi üzembe állítást, akkor a bevezetésnek hiányos kitöltési állapottal kell rendelkeznie (például kiépítési vagy sikertelen).

Ha egy felügyeleti csoportban szeretne bevezetést létrehozni, használja az New-AzManagementGroupDeployment parancsmagot.

## Példák

### Példa 1
```
PS C:\>Stop-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01"
```

Ez a parancs lemondja a "deployment01" futó központi telepítőt a "myMG" felügyeleti csoportban.

### 2. példa
```
PS C:\>Get-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01" | Stop-AzManagementGroupDeployment
```

Ez a parancs a "deployment01" ("myMG") felügyeleti csoport "" üzembe állítását adja vissza, és lemondja azt.
