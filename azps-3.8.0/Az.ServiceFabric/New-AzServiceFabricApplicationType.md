# New-AzServiceFabricApplicationType

## Áttekintés
Új szolgáltatásból álló anyag típusú alkalmazás létrehozása a megadott erőforráscsoport és fürt csoportban

## SZINTAXISA

```
New-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A parancsmag a megadott erőforráscsoport és fürt területéhez létrehoz egy új szolgáltatás szövet-alkalmazási típust.

## Példák

### Példa 1
```powershell
PS C:\> $resourceGroupName = "testRG"
PS C:\> $clusterName = "testCluster"
PS C:\> $appTypeName = "testAppType"
PS C:\> $appType = New-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

Ebben a példában az új "testAppType" típusú alkalmazás jön létre az erőforráscsoport és a fürt mezőben.
