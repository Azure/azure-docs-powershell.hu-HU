# Remove-AzServiceFabricNode

## Áttekintés
Csomópontok eltávolítása a megadott csomópont-típusból egy fürtből

## SZINTAXISA

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzServiceFabricNode** használatával eltávolíthatja a csomópontokat egy adott csomópontból a fürtből. Az Eltávolítás csak akkor járjon le, ha az megfelel a fürtök állapotát jelző metrikának.

## Példák

### Példa 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1' -NumberOfNodesToRemove 2
```

Ez a parancs eltávolítja a 2 csomópontot a "NT1" NodeType.
