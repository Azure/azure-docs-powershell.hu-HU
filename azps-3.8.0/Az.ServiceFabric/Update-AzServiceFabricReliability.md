# Update-AzServiceFabricReliability

## Áttekintés
A fürt elsődleges csomópont-típusának megbízhatósági szintjeinek frissítése

## SZINTAXISA

```
Update-AzServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
**Frissítse a AzServiceFabricReliability** a fürt elsődleges csomópont-típusának megbízhatóságát.

## Példák

### Példa 1
```
PS c:> Update-AzServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

Ez a parancs az elsődleges csomópont típusának megbízhatósági rétegét ezüst értékre módosítja.
