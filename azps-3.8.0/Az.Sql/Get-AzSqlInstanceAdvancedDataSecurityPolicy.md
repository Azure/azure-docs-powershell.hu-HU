# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## Áttekintés
A felügyelt példány speciális adatbiztonsági házirendjének lekérdezése.

## SZINTAXISA

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzSqlInstanceAdvancedDataSecurityPolicy** parancsmag a felügyelt példány speciális adatbiztonsági házirendjének lekérdezését kéri.

## Példák

### Példa 1 – a felügyelt példány speciális adatbiztonsága
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Példa: a felügyelt példány speciális adatbiztonsága a felügyelt példányok erőforrásból
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
