# Remove-AzSqlInstanceKeyVaultKey

## Áttekintés
Kulcs-Vault-kulcs eltávolítása SQL-alapú felügyelt példányból

## SZINTAXISA

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (alapértelmezett)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A Remove-AzSqlInstanceKeyVaultKey parancsmag eltávolítja a megadott felügyelt példányból a kulcsfájl kulcsát. Ne feledje, hogy az SQL által felügyelt példánynak a kulcs boltozatához tartozó engedélyei nem változnak. Az engedélyek módosításához használja a set-AzKeyVaultAccessPolicy. Ne feledje, hogy ez a parancsmag nem változtatja meg a Key Vault-t. Ha el szeretne távolítani egy billentyűt a Key Vault-ról, használja a Remove-AzureKeyVaultKey.
