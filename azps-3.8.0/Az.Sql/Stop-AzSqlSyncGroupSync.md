# Stop-AzSqlSyncGroupSync

## Áttekintés
Szinkronizálási csoport szinkronizálásának leállítása

## SZINTAXISA

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **stop-AzSqlSyncGroupSync** parancsmag leállítja a szinkronizálási csoportok szinkronizálását.

## Példák

### Példa 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Ez a parancs leállítja a szinkronizálás csoport mysg zajló szinkronizálást.
