# Remove-AzStreamAnalyticsOutput

## Áttekintés
Az adatfolyam-elemzésből származó adatok kimenetének törlése.

## SZINTAXISA

```
Remove-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzStreamAnalyticsOutput** parancsmag aszinkron módon törli az Azure-ban az adatfolyam-elemzési feladatok kimenetét.

## Példák

### 1. példa: a kimenet eltávolítása egy projektből
```
PS C:\>Remove-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Ez a parancs eltávolítja a kimeneti kimenetet a projekt StreamingJob.
