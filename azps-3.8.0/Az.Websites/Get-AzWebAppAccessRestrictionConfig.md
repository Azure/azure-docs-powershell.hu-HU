# Get-AzWebAppAccessRestrictionConfig

## Áttekintés
Az Azure Web App elérési restiction-konfigurációját kapja meg.

## SZINTAXISA

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzWebAppAccessRestrictionConfig** parancsmag hozzáférést kap az Azure Web App alkalmazással kapcsolatos korlátozási konfigurációhoz.

## Példák

### Példa 1: webalkalmazás hozzáférés-korlátozási konfigurációjának beszerzése erőforráscsoporthoz
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Ez a parancs a ContosoSite nevű webalkalmazás hozzáférés korlátozási konfigurációját kapja meg, amely az erőforráscsoport alapértelmezett – webes WestUS – tartozik.
