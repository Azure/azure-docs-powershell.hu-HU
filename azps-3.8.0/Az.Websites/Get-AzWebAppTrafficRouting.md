# Get-AzWebAppTrafficRouting

## Áttekintés
A megadott bővítőhelyhez tartozó útválasztási szabályok beszerzése

## SZINTAXISA

```
Get-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RuleName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzWebAppTrafficRouting** parancsmag az Azure Web App-bővítőhelyről kapja meg az útválasztási szabályok konfigurációját.

## Példák

### Példa 1 az adott útválasztási szabály beolvasása a WebApp bővítőhelyről
```powershell
PS C:\> Get-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite"  -RuleName 'Stg'
```

Ez a parancs az adott WebApp-bővítőhelyhez tartozó útválasztási szabályt kapja meg.
