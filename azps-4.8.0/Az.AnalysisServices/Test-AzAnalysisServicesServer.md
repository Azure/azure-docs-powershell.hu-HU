# Test-AzAnalysisServicesServer

## Áttekintés
Az Analysis Services-kiszolgáló egy példányának tesztelése

## SZINTAXISA

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A Test-AzAnalysisServicesServer parancsmag teszteli az Analysis Services-kiszolgáló egy példányának létezését

## Példák

### Példa 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Ez a parancs azt ellenőrzi, hogy van-e TestServer nevű kiszolgáló a resourcegroup testgroup
