# Remove-AzApiManagementLogger

## Áttekintés
API-kezelési naplózó eltávolítása.

## SZINTAXISA

```
Remove-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzApiManagementLogger** parancsmag egy Azure API-kezelési **naplózó** eltávolítása.

## Példák

### 1. példa: naplózó eltávolítása
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123" -Force
```

Ez a parancs eltávolítja az azonosító Logger123 tartalmazó naplózó.
