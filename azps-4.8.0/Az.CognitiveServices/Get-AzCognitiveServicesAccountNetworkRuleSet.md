# Get-AzCognitiveServicesAccountNetworkRuleSet

## Áttekintés
A NetworkRule tulajdonság beszerzése a kognitív szolgáltatások fiókban

## SZINTAXISA

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzCognitiveServicesAccountNetworkRuleSet** parancsmag a NetworkRule tulajdonságát kapja meg a kognitív Services-fiókban.

## Példák

### Példa 1: a NetworkRule tulajdonság beszerzése egy megadott kognitív szolgáltatási fiókból
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Ez a parancs beolvassa a NetworkRule tulajdonságot egy megadott kognitív Services-fiókkal.
