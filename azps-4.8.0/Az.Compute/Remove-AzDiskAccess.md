# Remove-AzDiskAccess

## Áttekintés
A Disk Access-erőforrás eltávolítása.

## SZINTAXISA

### DefaultParameterSet (alapértelmezett)
```
Remove-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIDParameterSet
```
Remove-AzDiskAccess [-ResourceId] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzDiskAccess [-InputObject] <PSDiskAccess> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzDiskAccess** parancsmag eltávolítja a Disk Access-erőforrást.

## Példák

### 1. példa: a lemezvezérlő-hozzáférés eltávolítása alapértelmezett paraméterérték használatával
```
PS C:\> Remove-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01"
```

Ez a parancs eltávolítja a "DiskAccess01" nevű Disk Access nevű "ResourceGroup01" erőforráscsoportot

### 2. példa: a lemezvezérlő-elérés eltávolítása erőforrás-AZONOSÍTÓval
```
PS C:\> $myDiskAccess = Get-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01"
PS C:\> Remove-AzDiskAccess -ResourceId $myDiskAccess.id
```

A parancs erőforrás-AZONOSÍTÓval eltávolítja a lemez elérését.

### 3. példa: a beviteli objektummal való lemezellenőrzés eltávolítása
```
PS C:\> $myDiskAccess = Get-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01"
PS C:\> Remove-AzDiskAccess -InputObject $myDiskAccess
```

Ez a parancs eltávolítja a InputObject által biztosított lemezterületet

### 4. példa: a csővezeték beviteli objektuma általi hozzáférés eltávolítása
```
PS C:\> Get-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" | Remove-AzDiskAccess 
```

Ez a parancs eltávolítja a InputObject
