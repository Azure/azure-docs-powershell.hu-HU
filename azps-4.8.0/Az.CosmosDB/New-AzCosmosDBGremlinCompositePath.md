# New-AzCosmosDBGremlinCompositePath

## Áttekintés
Új PSCompositePath típusú objektum létrehozása A set-AzCosmosDBGremlinGraph paraméter értéke lehet.

## SZINTAXISA

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Az Gremlin API CompositePath megfelelő objektum.

## Példák

### Példa 1
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
