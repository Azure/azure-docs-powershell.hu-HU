# Suspend-AzDataFactoryPipeline

## Áttekintés
Az Azure Data Factory csővezetékének felfüggesztése

## SZINTAXISA

### ByFactoryName (alapértelmezett)
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **felfüggesztés – AzDataFactoryPipeline** parancsmag felfüggeszti a csővezetéket az Azure Data Factory-ban.
A csővezetéket az Resume-AzDataFactoryPipeline parancsmag használatával folytathatja újra.

## Példák

### 1. példa: a csővezeték felfüggesztése
```
PS C:\>Suspend-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Ez a parancs felfüggeszti a DPWikiSample nevű csővezetéket az WikiADF nevű adatgyárban.
A parancs a $True értékét számítja ki.
