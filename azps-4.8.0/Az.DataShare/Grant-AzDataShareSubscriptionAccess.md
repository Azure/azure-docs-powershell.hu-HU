# Grant-AzDataShareSubscriptionAccess

## Áttekintés
Visszavont megosztási előfizetéshez való hozzáférés támogatása a forrás megosztásához

## SZINTAXISA

### ByFieldsParameterSet (alapértelmezett)
```
Grant-AzDataShareSubscriptionAccess -ResourceGroupName <String> -AccountName <String> [-ShareName <String>]
 -ShareSubscriptionId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByProviderShareSubscriptionIdParameterSet
```
Grant-AzDataShareSubscriptionAccess -ShareSubscriptionId <String> -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Grant-AzDataShareSubscriptionAccess** parancsmag előfizetéses hozzáféréssel ruházza fel a források megosztását

```
Id                        : /subscriptions/1834da9b-787a-44f6-ae81-60707ab8c957/resourceGroups/ADS/providers/Microsoft.DataShare/accounts/WikiAdsAccount/shares/AdsShare/shareSubscriptions/8ee6e6fd-b4a1-49a4-bb66-f187f38e0e12
Name                      : AdsShare
Type                      : Microsoft.DataShare/ShareSubscriptions
```

Hozzáférést biztosít az azonosító 8ee6e6fd-b4a1-49a4-bb66-f187f38e0e12 által azonosított előfizetés megosztásához
