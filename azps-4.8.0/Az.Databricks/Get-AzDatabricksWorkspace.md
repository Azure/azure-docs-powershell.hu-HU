# Get-AzDatabricksWorkspace

## Áttekintés
Megkapja a munkaterületet.

## SZINTAXISA

### Lista1 (alapértelmezett)
```
Get-AzDatabricksWorkspace [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Beszerzése
```
Get-AzDatabricksWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzDatabricksWorkspace -InputObject <IDatabricksIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Lista
```
Get-AzDatabricksWorkspace -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Leírás
Megkapja a munkaterületet.

## Példák

### Példa 1: Databricks-munkaterület beszerzése névvel
```powershell
PS C:\> Get-AzDatabricksWorkspace -Name databricks-test -ResourceGroupName testgroup

Location Name            Type
-------- ----            ----
eastus   databricks-test Microsoft.Databricks/workspaces
```

Ez a parancs Databricks-munkaterületet kap egy erőforráscsoportben.

### 2. példa: az előfizetésben az összes Databricks-munkaterület listázása
```powershell
PS C:\> Get-AzDatabricksWorkspace

Location Name                           Type
-------- ----                           ----
eastus   databricks-test                Microsoft.Databricks/workspaces
eastus   databricks-test-with-custom-vn Microsoft.Databricks/workspaces
```

Ez a parancs felsorolja az előfizetésben lévő összes Databricks-munkaterületet.

### 3. példa: egy erőforráscsoport összes Databricks-munkaterületének listázása
```powershell
PS C:\> Get-AzDatabricksWorkspace -ResourceGroupName testgroup

Location Name                           Type
-------- ----                           ----
eastus   databricks-test                Microsoft.Databricks/workspaces
eastus   databricks-test-with-custom-vn Microsoft.Databricks/workspaces
```

Ez a parancs felsorolja az erőforráscsoport összes Databricks-munkaterületét.
