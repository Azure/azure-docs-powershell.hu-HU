# Get-AzWvdWorkspace

## Áttekintés
Munkaterületet kaphat.

## SZINTAXISA

### Lista1 (alapértelmezett)
```
Get-AzWvdWorkspace [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Beszerzése
```
Get-AzWvdWorkspace -Name <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzWvdWorkspace -InputObject <IDesktopVirtualizationIdentity> [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

### Lista
```
Get-AzWvdWorkspace -ResourceGroupName <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## Leírás
Munkaterületet kaphat.

## Példák

### Példa 1: a Windows virtuális asztali Worksapce beszerzése név szerint
```powershell
PS C:\> Get-AzWvdWorksapce -ResourceGroupName ResourceGroupName -Name WorkspaceName

Location   Name                 Type
--------   ----                 ----
eastus     WorkspaceName Microsoft.DesktopVirtualization/workspaces
```

Ez a parancs egy Windows virtuális asztali munkaterületet kap egy Erőforráscsoportben.

### 2. példa: Windows virtuális asztali munkaterületek listázása
```powershell
PS C:\> Get-AzWvdWorksapce -ResourceGroupName ResourceGroupName

Location   Name           Type
--------   ----           ----
eastus     WorkspaceName1 Microsoft.DesktopVirtualization/workspaces
eastus     WorkspaceName2 Microsoft.DesktopVirtualization/workspaces
```

Ez a parancs egy Windows virtuális asztali munkaterületet sorol fel egy Erőforráscsoportben.

INPUTOBJECT <IDesktopVirtualizationIdentity> : Identity paraméter
  - `[ApplicationGroupName <String>]`: Az alkalmazás csoportjának neve
  - `[ApplicationName <String>]`: Az alkalmazás neve a megadott alkalmazásobjektum-csoporton belül
  - `[DesktopName <String>]`: A megadott asztali csoportban lévő asztal neve

  - `[SessionHostName <String>]`: A megadott alkalmazáskészleten belüli Munkamenetcímtár neve
  - `[SubscriptionId <String>]`: A cél-előfizetés azonosítója.
  - `[UserSessionId <String>]`: A megadott Munkamenetcímtár felhasználói munkamenetének neve
