# Get-AzEventHubClustersAvailableRegion

## Áttekintés
Egy Eventhub-fürt vagy a szektorcsoportok listájának lekérdezése az adott erőforráscsoport számára

## SZINTAXISA

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Azoknak a régióknak a Get-AzEventHubClustersAvailableRegion parancsmag listája, amelyekhez dedikáltak.
