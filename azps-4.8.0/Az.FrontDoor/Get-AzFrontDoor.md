# Get-AzFrontDoor

## Áttekintés
A bejárati ajtó betöltője

## SZINTAXISA

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzFrontDoor** cmdletGet beilleszti az összes meglévő ajtót az aktuális előfizetésben, amely megfelel a megadott adatoknak.
