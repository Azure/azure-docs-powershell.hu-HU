# Get-AzHpcCacheSku

## Áttekintés
Az előfizetésben elérhető összes SKU-t megkapja.

## SZINTAXISA

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzHpcCacheSku** parancsmag az előfizetésben elérhető SKU-számokból álló listát ad eredményül.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
