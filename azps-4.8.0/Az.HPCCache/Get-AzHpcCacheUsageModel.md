# Get-AzHpcCacheUsageModel

## Áttekintés
Az NFS-tárterület minden usageModels megkapja.

## SZINTAXISA

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzHpcCacheUsageModel** PARANCSMAG az NFS-tárolókhoz használható használati modellek listáját jeleníti meg.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
