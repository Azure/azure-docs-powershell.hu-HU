# Get-AzMarketplacePrivateStoreOffer

## Áttekintés
Szerezzen be egy vagy több privát áruház ajánlatát.

## SZINTAXISA

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Szerezzen be egy vagy több privát áruház ajánlatát a bérlői hatókör csoportban hozzáadott nyilvános + privát csomagokkal. Ha az előfizetési azonosító be van írva, csak az előfizetések hatóköre szerint szerezzen be egy vagy több magánjellegű áruház ajánlatát.
