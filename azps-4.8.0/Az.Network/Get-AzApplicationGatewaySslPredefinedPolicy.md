# Get-AzApplicationGatewaySslPredefinedPolicy

## Áttekintés
Az alkalmazás-átjáró által megadott előre meghatározott SSL-házirendeket kapja.

## SZINTAXISA

```
Get-AzApplicationGatewaySslPredefinedPolicy [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzApplicationGatewaySslPredefinedPolicy** parancsmag az Application Gateway által biztosított előre meghatározott SSL-házirendeket kapja meg.
