# Get-AzAutoApprovedPrivateLinkService

## Áttekintés
A magánhálózati szolgáltatás-azonosítók egy olyan sorát kapja, amely egy privát végponthoz kapcsolható az automatikus jóváhagyással.

## SZINTAXISA

```
Get-AzAutoApprovedPrivateLinkService -Location <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzAutoApprovedPrivateLinkService** a magánhálózati szolgáltatás-azonosítók egy olyan sorát kapja, amely egy privát végponthoz kapcsolható az automatikus jóváhagyással.

## Példák

### Például
```
Get-AzAutoApprovedPrivateLinkService -Location westus -ResourceGroupName TestResourceGroup
```

Ez a példa a privát hivatkozás szolgáltatás-azonosító egy olyan sorát adja eredményül, amely egy privát végponthoz kapcsolható az automatikus jóváhagyással.
