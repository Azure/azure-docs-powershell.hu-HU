# Get-AzNetworkVirtualAppliance

## Áttekintés
Hálózati virtuális berendezések beszerzése vagy listázása

## SZINTAXISA

### ResourceNameParameterSet (alapértelmezett)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A Get-AzNetworkVirtualAppliance parancsok hálózati virtuális eszközökhöz tartozó erőforrásokat nyernek vagy sorol fel.
