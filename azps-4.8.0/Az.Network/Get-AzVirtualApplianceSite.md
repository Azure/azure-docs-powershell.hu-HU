# Get-AzVirtualApplianceSite

## Áttekintés
A hálózat virtuális készülékhez kapcsolódó erőforrás (oka) t tartalmazó webhelyek beszerzése vagy listázása

## SZINTAXISA

### ResourceNameParameterSet (alapértelmezett)
```
Get-AzVirtualApplianceSite [-Name <String>] [-NetworkVirtualApplianceId <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzVirtualApplianceSite -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A Get-AzVirtualApplianceSite olyan webhelyeket kap vagy sorol fel, amelyek egy vagy több hálózati virtuális készülék-erőforráshoz kapcsolódnak.
