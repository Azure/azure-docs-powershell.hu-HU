# New-AzApplicationGatewayRewriteRuleHeaderConfiguration

## Áttekintés
Egy új szabály fejléc-konfigurációját hozza létre egy alkalmazás-átjáróhoz.

## SZINTAXISA

```
New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName <String> [-HeaderValue <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
**Az AzApplicationGatewayRewriteRuleHeaderConfiguration** parancsmag létrehoz egy átírási szabályt az Azure Application átjáróhoz.

## Példák

### Példa 1
```powershell
PS C:\> $hc = New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName abc -HeaderValue def
```

Ez a parancs átírják a szabály fejlécének konfigurációját, és az eredményt az $hc nevű változóban tárolja.
