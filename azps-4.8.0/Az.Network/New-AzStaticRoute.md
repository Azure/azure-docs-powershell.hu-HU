# New-AzStaticRoute

## Áttekintés
Létrehoz egy StaticRoute objektumot, amelyet azután hozzáadhat egy RoutingConfiguration objektumhoz.

## SZINTAXISA

```powershell
New-AzStaticRoute -Name <String> -AddressPrefix <String[]> -NextHopIpAddress <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
StaticRoute objektumot hoz létre.

## Példák

### Példa 1
```powershell
PS C:\> New-AzStaticRoute -Name "route1" -AddressPrefix @("10.20.0.0/16", "10.30.0.0/16") -NextHopIpAddress "10.90.0.5"

Name   AddressPrefixes              NextHopIpAddress
----   ---------------              ----------------
route1 {10.20.0.0/16, 10.30.0.0/16} 10.90.0.5
```

A fenti parancs létrehoz egy StaticRoute objektumot, amelyet Ezután hozzáadhat a RoutingConfiguration-objektumokhoz.
