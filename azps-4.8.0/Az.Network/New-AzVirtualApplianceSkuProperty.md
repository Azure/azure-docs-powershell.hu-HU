# New-AzVirtualApplianceSkuProperty

## Áttekintés
Adja meg az erőforráshoz tartozó hálózati virtuális készülék SKU-t.

## SZINTAXISA

```
New-AzVirtualApplianceSkuProperty -VendorName <String> -BundledScaleUnit <String> -MarketPlaceVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A New-AzVirtualApplianceSkuProperties parancs a Network Virtual Appliance Resource (SKU) t határozza meg.

## Példák

### Példa 1
```powershell
PS C:\> $var=New-AzVirtualApplianceSkuProperty -VendorName "barracudasdwanrelease" -BundledScaleUnit 1 -MarketPlaceVersion 'latest'
```

Hozzon létre egy virtuális készülék SKU-tulajdonságú objektumát, amelyet New-AzNetworkVirtualAppliance paranccsal szeretne használni.
