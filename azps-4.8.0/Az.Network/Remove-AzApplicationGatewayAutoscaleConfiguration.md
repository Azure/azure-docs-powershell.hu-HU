# Remove-AzApplicationGatewayAutoscaleConfiguration

## Áttekintés
Az automatikus méretezési konfiguráció eltávolítása egy alkalmazás-átjáróról.

## SZINTAXISA

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzApplicationGatewayAutoscaleConfiguration** parancsmag egy meglévő alkalmazás-átjáróból távolítja el az automatikus méretezés konfigurációját.

## Példák

### Példa 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Az első parancs beolvassa az Application Gatewayt, és az $gw változóban tárolja.
A második parancs eltávolítja az automatikus méretezési konfigurációt az alkalmazás-átjáróról.
A harmadik parancs az Azure rendszerhez frissíti az Application Gatewayt.
