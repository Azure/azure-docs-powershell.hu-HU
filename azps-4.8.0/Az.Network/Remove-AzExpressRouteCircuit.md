# Remove-AzExpressRouteCircuit

## Áttekintés
ExpressRoute-áramkör eltávolítása.

## SZINTAXISA

```
Remove-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzExpressRouteCircuit** parancsmag eltávolítja a ExpressRoute-áramkört.

## Példák

### Példa 1: ExpressRoute-áramkör törlése
```
Remove-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### 2. példa: ExpressRoute-áramkör törlése a csővezeték használatával
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```
