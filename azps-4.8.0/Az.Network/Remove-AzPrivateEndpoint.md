# Remove-AzPrivateEndpoint

## Áttekintés
Privát végpont eltávolítása.

## SZINTAXISA

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzPrivateEndpoint** parancsmag eltávolítja a privát végpontot. 

## Példák

### Példa 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

Ez a példa eltávolítja a MyPrivateEndpoint1 nevű privát végpontot.
