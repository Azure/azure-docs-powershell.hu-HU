# Get-AzRecoveryServicesAsrAlertSetting

## Áttekintés
Beilleszti a beállított Azure-webhely helyreállítási értesítési beállításait a boltozathoz.

## SZINTAXISA

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzRecoveryServicesAsrAlertSetting** parancsmag a beállított Azure-webhely helyreállítási értesítési beállításait kapja meg a boltozathoz.

## Példák

### Példa 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Értesítési/értesítési beállítás kérése az Azure webhely-helyreállításhoz.
