# Remove-AzRelayHybridConnection

## Áttekintés
Eltávolítja a HybridConnection a megadott HybridConnection-névtérből.

## SZINTAXISA

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzRelayHybridConnection** parancsmag eltávolítja a HybridConnection a megadott továbbító névtérből.

## Példák

### Példa 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Eltávolítja a HybridConnection `TestHybridConnection` a névtérből `TestNameSpace-Relay1` .
