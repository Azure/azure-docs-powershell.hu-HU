# Save-AzResourceGroupDeploymentTemplate

## Áttekintés
Egy erőforráscsoport-telepítő sablont ment a fájlba.

## SZINTAXISA

```
Save-AzResourceGroupDeploymentTemplate -ResourceGroupName <String> -DeploymentName <String> [-Path <String>]
 [-Force] [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Save-AzResourceGroupDeploymentTemplate**  parancsmag egy erőforráscsoport-telepítő sablont JSON-fájlba menti.

## Példák

### Példa 1: központi telepítő sablon mentése
```powershell
PS C:\>Save-AzResourceGroupDeploymentTemplate -DeploymentName "TestDeployment" -ResourceGroupName "TestGroup"
```

Ez a parancs beolvassa a TestDeployment a telepítő sablonból, és egy JSON-fájlként menti az aktuális könyvtárat.

### 2. példa

Egy erőforráscsoport-telepítő sablont ment a fájlba. autogenerated
