# New-AzIotSecuritySolutionUserDefinedResourcesObject

## Áttekintés
Új felhasználó által definiált erőforrások létrehozása a IOT biztonsági megoldásához

## SZINTAXISA

```
New-AzIotSecuritySolutionUserDefinedResourcesObject -Query <String> -QuerySubscriptionList <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A AzIotSecuritySolutionUserDefinedResourcesObject parancsmag létrehoz egy új, felhasználó által definiált erőforrás-objektumot a IOT biztonsági megoldásához.

## Példák

### Példa 1
```powershell
PS C:\> New-AzIotSecuritySolutionUserDefinedResourcesObject -Query 'where type != "microsoft.devices/iothubs" | where name contains "v2"' 
-QuerySubscriptionList @("XXXXXXXX-XXXX-XXXXX-XXXX-XXXXXXXXXXXX")

Query: 'where type != "microsoft.devices/iothubs" | where name contains "v2"' 
QuerySubscriptions: ["XXXXXXXX-XXXX-XXXXX-XXXX-XXXXXXXXXXXX"]
```

Új felhasználó által definiált erőforrások létrehozása
