# Get-AzSqlServerTransparentDataEncryptionProtector

## Áttekintés
Az áttetsző adattitkosítás (TDE) oltalmazójának beolvasása

## SZINTAXISA

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az Get-AzSqlServerTransparentDataEncryptionProtector parancsmag információkat kap a TDE-oltalmazóról.

## Példák

### Példa 1: az átlátszó adattitkosítás (TDE) oltalmazó beolvasása
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Ez a parancs a TDE Protector nevű kiszolgálót kapja meg a ContosoServer nevű erőforráscsoport nevű ContosoResourceGroup.
ResourceGroupName kiszolgálónév – típus ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
