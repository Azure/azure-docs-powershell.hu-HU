# Invoke-AzSqlInstanceFailover

## Áttekintés
Azure SQL-alapú felügyelt példány feladatátvétele

## SZINTAXISA

```
Invoke-AzSqlInstanceFailover [-Name] <String> [-AsJob] [-PassThru] [-Force] [-ReadableSecondary]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **meghívó-AzSqlInstanceFailover** parancsmag egy Azure SQL-alapú felügyelt példány feladatátvételét hajtja végre.

## Példák

### Példa 1
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01"
```

Ez a parancs a "ManagedInstance01" nevű példány elsődleges replikáját fogja lecserélni.

### 2. példa
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01" -ReadableSecondary
```

Ez a parancs a "ManagedInstance01" olvasható másodlagos replikát fogja átirányítani.
