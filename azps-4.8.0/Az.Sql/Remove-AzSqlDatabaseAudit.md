# Remove-AzSqlDatabaseAudit

## Áttekintés
Egy Azure SQL-adatbázis naplózási beállításainak eltávolítása.

## SZINTAXISA

### DatabaseParameterSet (alapértelmezett)
```
Remove-AzSqlDatabaseAudit [-ResourceGroupName] <String> [-ServerName] <String> [-DatabaseName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DatabaseObjectParameterSet
```
Remove-AzSqlDatabaseAudit -DatabaseObject <AzureSqlDatabaseModel> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzSqlDatabaseAudit** parancsmag eltávolítja az Azure SQL-adatbázisok naplózási beállításait.
A parancsmag használatához a *ResourceGroupName* , a *kiszolgálónév* és a *databasename* paraméterrel keresse meg az adatbázist.

## Példák

### 1. példa: az Azure SQL-adatbázisok naplózási beállításainak eltávolítása
```
PS C:\>Remove-AzSqlDatabaseAudit -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

### 2. példa: az Azure SQL-adatbázisok naplózási beállításainak eltávolítása a csővezetéken keresztül
```
PS C:\> Get-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" | Remove-AzSqlDatabaseAudit
```
