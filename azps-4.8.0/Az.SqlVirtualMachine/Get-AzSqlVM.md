# Get-AzSqlVM

## Áttekintés
Egy vagy több SQL virtuális gépet kap.

## SZINTAXISA

### ResourceGroupOnly (alapértelmezett)
```
Get-AzSqlVM [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### név
```
Get-AzSqlVM [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceId
```
Get-AzSqlVM [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A Get-AzSqlVM parancsmag egy vagy több SQL virtuális gépet kap.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzSqlVM

Name ResourceGroupName  LicenseType Sku       Offer          SqlManagementType
---- -----------------  ----------- ---       -----          -----------------
vm   ResourceGroup01    PAYG        Developer SQL2017-WS2016 Full
vm2  ResourceGroup02    PAYG        Developer SQL2017-WS2016 Full
```

Ez a parancs az aktuális előfizetésben található összes Azure SQL-gépen információt kap.

### 2. példa
```powershell
PS C:\> Get-AzSqlVM -ResourceGroupName "ResourceGroup01"
Name ResourceGroupName  LicenseType Sku       Offer          SqlManagementType
---- -----------------  ----------- ---       -----          -----------------
vm   ResourceGroup01    PAYG        Developer SQL2017-WS2016 Full
```

Ez a parancs információt kap az összes Azure SQL-gépen az aktuális előfizetésben, amely az erőforráscsoport ResourceGroup01 van társítva.

### 3. példa
```powershell
PS C:\> Get-AzSqlVM -ResourceGroupName "ResourceGroup01" -Name "vm"
Name ResourceGroupName  LicenseType Sku       Offer          SqlManagementType
---- -----------------  ----------- ---       -----          -----------------
vm   ResourceGroup01    PAYG        Developer SQL2017-WS2016 Full
```

Ez a parancs információt kap az erőforráscsoport ResourceGroup01 rendelt virtuális virtuális gép "VM" rendszeréről.
