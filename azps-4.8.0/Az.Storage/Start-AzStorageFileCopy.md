# Start-AzStorageFileCopy

## Áttekintés

## Leírás
A **Start-AzStorageFileCopy** parancsmag a forrásfájl szövegfájlba való másolását kezdi.

## Példák

### 1. példa: a másolási művelet indítása fájlból fájlból fájlból a megosztás név és fájlnév használatával
```
PS C:\>Start-AzStorageFileCopy -SrcShareName "ContosoShare01" -SrcFilePath "FilePath01" -DestShareName "ContosoShare02" -DestFilePath "FilePath02"
```

Ez a parancs a fájlból fájlból másolt műveletet indítja el.
A parancs a megosztás neve és a fájlnév nevet adja meg

### 2. példa: a másolási művelet indítása blobról fájlra a tároló neve és a blob neve segítségével
```
PS C:\>Start-AzStorageFileCopy -SrcContainerName "ContosoContainer01" -SrcBlobName "ContosoBlob01" -DestShareName "ContosoShare" -DestFilePath "FilePath02"
```

Ez a parancs egy másolati műveletet indít el a blob-fájlból.
A parancs a tároló nevét és a blob nevét adja meg
