# Invoke-AzStorageSyncFileRecall

## Áttekintés
Ez a parancs visszahívja az összes többszintű fájlt a helyi merevlemezre.

## SZINTAXISA

### StringParameterSet (alapértelmezett)
```
Invoke-AzStorageSyncFileRecall [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Pattern <String>] [-RecallPath <String>] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Invoke-AzStorageSyncFileRecall [-ResourceId] <String> [-Pattern <String>] [-RecallPath <String>] [-PassThru]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameterSet
```
Invoke-AzStorageSyncFileRecall [-InputObject] <PSServerEndpoint> [-Pattern <String>] [-RecallPath <String>]
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Ha a felhőalapú réteg beállítás engedélyezve van egy kiszolgáló-végponton (egy regisztrált kiszolgálón lévő adott helyen), akkor ez a parancs segítségével visszahívhatja az összes többszintű fájlt a helyi merevlemezre. Javasoljuk, hogy a visszahívás megkezdése előtt tiltsa le a Felhőbeli rétegeket a kiszolgálón futó végponton. Ha a többszintű beállítás továbbra is fennáll, a visszahívás a többszintű fájlok elérését okozó egyéb fájlok elérését eredményezi, ami nem éri el a helyi lemezen tartózkodó összes tartalom kívánt célját.

## Példák

### Példa 1
```powershell
PS C:\> Invoke-AzStorageSyncFileRecall -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -ServerEndpointName "myServerEndpointName"
```

Ez a parancs rekurzívan visszahívja az összes olyan többszintű fájlt, amely a megadott kiszolgálói végpont gyökerének elérési útján található.
