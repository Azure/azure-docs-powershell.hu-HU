# Test-AzStreamAnalyticsInput

## Áttekintés
Egy bemenet kapcsolati állapotát teszteli.

## SZINTAXISA

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **AzStreamAnalyticsInput** parancsmag segítségével az adatfolyam-elemzés segítségével kapcsolódhat egy bemenethez.

## Példák

### Példa 1: egy bemeneti adatfolyam kapcsolati állapotának tesztelése
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Ez teszteli a StreamingJob bemeneti EntryStream kapcsolati állapotát.
