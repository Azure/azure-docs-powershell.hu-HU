# Get-AzSynapseNotebook

## Áttekintés
Információt kap a munkaterületen lévő jegyzetfüzetekről.

## SZINTAXISA

### GetByName (alapértelmezett)
```
Get-AzSynapseNotebook -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseNotebook -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzSynapseNotebook** parancsmag információkat kap a munkaterületen lévő jegyzetfüzetekről. Ha megad egy jegyzetfüzet nevét, a parancsmag információt kap a jegyzetfüzetről. Ha nem ad meg nevet, a parancsmag a munkaterületen lévő összes jegyzetfüzetre vonatkozóan információkat kap.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace | Format-Table

WorkspaceName    Properties                                         Name
-------------    ----------                                         --
ContosoWorkspace Microsoft.Azure.Commands.Synapse.Models.PSNotebook ContosoNotebook1
ContosoWorkspace Microsoft.Azure.Commands.Synapse.Models.PSNotebook ContosoNotebook2
```

Beolvassa az összes jegyzetfüzet listáját a munkaterület ContosoWorkspace.

### 2. példa
```powershell
PS C:\> Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook
```

Egy ContosoNotebook nevű jegyzetfüzetet kap a munkaterület ContosoWorkspace.

### 3. példa
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseNotebook -Name ContosoNotebook
```

Egy ContosoNotebook nevű jegyzetfüzetet kap a munkaterület ContosoWorkspace a csővezetéken keresztül.
