# Stop-AzSynapseTrigger

## Áttekintés
A munkaterületen leállít egy triggert.

## SZINTAXISA

### StopByName (alapértelmezett)
```
Stop-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByObject
```
Stop-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzSynapseTrigger** parancsmag leállítja a munkaterületen az indítót. Ha az indító az "Elindítva" állapotban van, a parancsmag leállítja az indítási fázist, és már nem indítja el a csővezetékeket. Ha az indító már a "leállt" állapotban van, ez a parancsmag nincs befolyással.

## Példák

### Példa 1
```powershell
PS C:\> Stop-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Leállít egy ContosoTrigger nevű triggert a munkaterület ContosoWorkspace.

### 2. példa
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Stop-AzSynapseTrigger -Name ContosoTrigger
```

Egy ContosoTrigger nevű triggert állít le a munkaterület ContosoWorkspace a csővezetéken keresztül.

### 3. példa
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Stop-AzSynapseTrigger
```

Állítsa le a ContosoTrigger nevű triggert a munkaterület ContosoWorkspace a csővezetéken keresztül.
