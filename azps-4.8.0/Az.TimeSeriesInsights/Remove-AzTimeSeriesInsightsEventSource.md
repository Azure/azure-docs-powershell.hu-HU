# Remove-AzTimeSeriesInsightsEventSource

## Áttekintés
A megadott nevű eseményforrás törlése a megadott előfizetésből, erőforrás csoportból és környezetből

## SZINTAXISA

### Delete (alapértelmezett)
```
Remove-AzTimeSeriesInsightsEventSource -EnvironmentName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzTimeSeriesInsightsEventSource -InputObject <ITimeSeriesInsightsIdentity> [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Leírás
A megadott nevű eseményforrás törlése a megadott előfizetésből, erőforrás csoportból és környezetből

## Példák

### 1. példa: megadott eseményforrás eltávolítása név szerint
```powershell
PS C:\> Remove-AzTimeSeriesInsightsEventSource -EnvironmentName tsitest001 -Name iots001 -ResourceGroupName testgroup

```

Ez egy adott eseményforrás eltávolítása.

### 2. példa: meghatározott eseményforrás eltávolítása objektumból
```powershell
PS C:\> $es = Get-AzTimeSeriesInsightsEventSource -EnvironmentName tsitest001 -ResourceGroupName testgroup -Name iots001
PS C:\> Remove-AzTimeSeriesInsightsEventSource -InputObject $es

```

Ez egy adott eseményforrás eltávolítása.
