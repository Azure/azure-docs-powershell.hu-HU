# Update-AzWebAppAccessRestrictionConfig

## Áttekintés
Itt frissítheti az Azure-alapú webalkalmazások restiction-konfigurációjának az SCM-webhelyre való öröklését.

## SZINTAXISA

### InputValuesParameterSet (alapértelmezett)
```
Update-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String>
 [-ScmSiteUseMainSiteRestrictionConfig] [-SlotName <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzWebAppAccessRestrictionConfig [-PassThru] -InputObject <PSAccessRestrictionConfig>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az **Update-AzWebAppAccessRestrictionConfig** parancsmag az Azure-alapú webalkalmazások hozzáférés-korlátozási konfigurációját frissíti.

## Példák

### Példa 1: Web App SCM-webhely frissítése a fő webhelyről való hozzáférési korlátozások használatához

Az alábbi példa frissíti a IpRule nevű, az erőforráscsoport MyResourceGroup tartozó webalkalmazást, amely az SCM-webhelyen található fő webhely hozzáférés-korlátozási konfigurációját használja.

```powershell <!-- Aladdin Generated Example --> 
Update-AzWebAppAccessRestrictionConfig -Name IpRule -ResourceGroupName MyResourceGroup -ScmSiteUseMainSiteRestrictionConfig
```
