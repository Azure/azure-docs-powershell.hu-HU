# Update-AzWebAppTrafficRouting

## Áttekintés
Útválasztási szabály frissítése a bővítőhelyre

## SZINTAXISA

```
Update-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az **Update-AzWebAppTrafficRouting** parancsmag frissíti az Azure Web App-bővítőhelyek útválasztási szabályának konfigurációját.

## Példák

### Példa: az útválasztási szabály frissítése a termelési forgalom 15%-át STG-bővítőhelyre
```powershell
PS C:\>Update-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
- RoutingRule @{AtionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

Ez a parancs egy útválasztási szabály frissítésével továbbítja a termelési forgalom 15%-át a STG-bővítőhelyre.

### 2. példa: útválasztási szabály frissítése, hogy a termelési forgalmat átviheti a 50%-ról 90%-ra növekményes STG.
```powershell
PS C:\>Update-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

Ez a parancs egy útválasztási szabály frissítésével átviszi a termelési forgalmat a 50%-ról 90%-ra növekvő STG.
