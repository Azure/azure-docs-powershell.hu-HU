# Get-AzAksVersion

## Áttekintés
Az elérhető verzió listája a felügyelt Kubernetes-fürtök létrehozásához.

## SZINTAXISA

```
Get-AzAksVersion -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Az elérhető verzió listája a felügyelt Kubernetes-fürtök létrehozásához.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzAksVersion -Location westus
```
