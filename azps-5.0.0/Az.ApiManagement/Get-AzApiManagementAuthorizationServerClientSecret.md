# Get-AzApiManagementAuthorizationServerClientSecret

## Áttekintés
Megkapja az API-kezelés engedélyezési kiszolgálójának az ügyfél titkát.

## SZINTAXISA

### ContextParameterSet (alapértelmezett)
```
Get-AzApiManagementAuthorizationServerClientSecret -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServerClientSecret [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzApiManagementAuthorizationServerClientSecret** parancsmag az Azure API-kezelés engedélyezési kiszolgálójának az ügyfél titkát kapja.

## Példák

### Példa 1: meghatározott engedélyezési kiszolgáló-ügyfél titkos azonosítójának beszerzése
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServerClientSecret -Context $ApiMgmtContext -ServerId "0123456789"
```

Ez a parancs a megadott engedélyezési kiszolgáló titkát kapja.
