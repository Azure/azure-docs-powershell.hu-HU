# Get-AzApiManagementDiagnostic

## Áttekintés
A szolgáltatás szintjén vagy az API szintjén konfigurált diagnosztikai adatok részletes ismertetése. A diagnosztika az API-kezelési átjárótól érkező kérések és válaszok naplózására szolgál.

## SZINTAXISA

### ContextParameterSet (alapértelmezett)
```
Get-AzApiManagementDiagnostic -Context <PsApiManagementContext> [-DiagnosticId <String>] [-ApiId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementDiagnostic -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzApiManagementDiagnostic** az API-kezelési szolgáltatásban megadott keresési diagnosztikai szolgáltatás részleteit kapja meg.

Ez a parancs az API-hatókörben konfigurált összes diagnosztikát bekerül `echo-api`
