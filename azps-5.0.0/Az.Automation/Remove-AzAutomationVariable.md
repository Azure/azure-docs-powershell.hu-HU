# Remove-AzAutomationVariable

## Áttekintés
Automatizálási változó eltávolítása.

## SZINTAXISA

```
Remove-AzAutomationVariable [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzAutomationVariable** parancsmag egy változót távolít el az Azure automatizálásból.

## Példák

### Példa 1: változó eltávolítása
```
PS C:\>Remove-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -Force -ResourceGroupName "ResourceGroup01"
```

Ez a parancs eltávolítja a StringVariable22 nevű változót a Contoso17 nevű automatizálási fiókban.
Ez a parancs az *erő* paramétert adja meg.
Ezért nem kéri a megerősítést.
