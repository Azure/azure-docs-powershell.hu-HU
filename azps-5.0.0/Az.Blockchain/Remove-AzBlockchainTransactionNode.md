# Remove-AzBlockchainTransactionNode

## Áttekintés
Törölje a tranzakció csomópontot.

## SZINTAXISA

### Delete (alapértelmezett)
```
Remove-AzBlockchainTransactionNode -BlockchainMemberName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzBlockchainTransactionNode -InputObject <IBlockchainIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Leírás
Törölje a tranzakció csomópontot.

## Példák

### 1. példa: a tranzakciós csomópont eltávolítása
```powershell
PS C:\> Remove-AzBlockchainTransactionNode -Name transacnode002 -BlockchainMemberName dolauli002 -ResourceGroupName testgroup

```

Ez a parancs eltávolítja a tranzakció csomópontot.

### 2. példa: a tranzakciós csomópont eltávolítása
```powershell
PS C:\> $node = Get-AzBlockchainTransactionNode -BlockchainMemberName dolauli002 -Name transacnode003 -ResourceGroupName $env.resourceGroup 
PS C:\> Remove-AzBlockchainTransactionNode -InputObject $node
```

Ez a parancs eltávolítja a tranzakció csomópontot.
