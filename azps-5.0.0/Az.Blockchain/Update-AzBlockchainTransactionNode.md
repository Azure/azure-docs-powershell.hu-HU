# Update-AzBlockchainTransactionNode

## Áttekintés
Frissítse a tranzakció csomópontot.

## SZINTAXISA

### UpdateExpanded (alapértelmezett)
```
Update-AzBlockchainTransactionNode -BlockchainMemberName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-FirewallRule <IFirewallRule[]>] [-Password <SecureString>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzBlockchainTransactionNode -InputObject <IBlockchainIdentity> [-FirewallRule <IFirewallRule[]>]
 [-Password <SecureString>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Leírás
Frissítse a tranzakció csomópontot.

## Példák

### Példa 1: transcation csomópont frissítése
```powershell
PS C:\> $tag = @{'key1'='update'}
PS C:\> Update-AzBlockchainTransactionNode -BlockchainMemberName dolauli002 -Name transacnode002 -ResourceGroupName testgroup -Tag $tag

Name           Type                                                    Location
----           ----                                                    --------
transacnode002 Microsoft.Blockchain/blockchainMembers/transactionNodes eastus
```

Ez a parancs frissíti a tranzakció csomópontot.

### 2. példa: transcation-csomópont frissítése
```powershell
PS C:\> $tag = @{'key2'='update'}
PS C:\> $tNode = Get-AzBlockchainMember -BlockchainMemberName dolauli002 -ResourceGroupName testgroup -Name transacnode002
PS C:\> Update-AzBlockchainTransactionNode -InputObject $tNode -Tag $tag

Name           Type                                                    Location
----           ----                                                    --------
transacnode002 Microsoft.Blockchain/blockchainMembers/transactionNodes eastus
```

Ez a parancs frissíti a tranzakció csomópontot.

FIREWALLRULE <IFirewallRule [] >: beolvassa vagy beállítja a tűzfalszabályok szabályait.
  - `[EndIPAddress <String>]`: A tűzfalszabály-tartományok végpontjának IP-címét adja meg vagy állítja be.
  - `[RuleName <String>]`: A tűzfalszabályok nevének beírása vagy bekapcsolása.
  - `[StartIPAddress <String>]`: Beilleszti vagy beállítja a tűzfalszabály-tartományok kezdő IP-címét.

INPUTOBJECT <IBlockchainIdentity> : Identity paraméter
  - `[BlockchainMemberName <String>]`: Blockchain tag neve.
  - `[Id <String>]`: Erőforrás-identitás elérési útja
  - `[Location <String>]`: Hely neve.
  - `[OperationId <String>]`: Műveleti azonosító.
  - `[ResourceGroupName <String>]`: Az erőforrást tartalmazó erőforráscsoport neve. Ezt az értéket az Azure Resource Manager API-ból vagy a portálról kaphatja meg.
  - `[SubscriptionId <String>]`: A Microsoft Azure-előfizetést egyedileg azonosító előfizetési azonosítót kapja. Az előfizetési azonosító az összes szolgáltatási hívás URI azonosítójának része.
  - `[TransactionNodeName <String>]`: A tranzakció csomópont neve.
