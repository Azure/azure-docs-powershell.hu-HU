# Publish-AzBlueprint

## Áttekintés
A tervezet új verziójának közzététele

## SZINTAXISA

```
Publish-AzBlueprint -Version <String> [-ChangeNote <String>] -Blueprint <PSBlueprint>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A Blueprint-definíciók új verziójának közzététele

## Példák

### Példa 1
