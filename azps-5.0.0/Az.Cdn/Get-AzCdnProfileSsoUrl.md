# Get-AzCdnProfileSsoUrl

## Áttekintés
Beilleszti a CDN-profil egyszeri bejelentkezést tartalmazó URL-címét.

## SZINTAXISA

### ByFieldsParameterSet (alapértelmezett)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzCdnProfileSsoUrl** parancsmag az Azure Content Delivery Network (CDN) profiljának egyszeri bejelentkezés URL-címét kapja meg.
Ez az URL-cím lehetővé teszi a felhasználóknak, hogy csatlakozzanak egy további portálhoz, és használják a CDN további funkcióit.
