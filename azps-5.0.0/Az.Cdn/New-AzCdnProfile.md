# New-AzCdnProfile

## Áttekintés
Egy CDN-profilt hoz létre.

## SZINTAXISA

```
New-AzCdnProfile -ProfileName <String> -Location <String> -Sku <PSSkuName> -ResourceGroupName <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzCdnProfile** parancsmag az Azure Content Delivery Network (CDN) profilját hozza létre.

## Példák

### Példa 1

Egy CDN-profilt hoz létre. autogenerated

```powershell

```
