# New-AzDiskAccess

## Áttekintés
Disk Access-erőforrás létrehozása

## SZINTAXISA

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzDiskAccess** parancsmag létrehoz egy Disk Access-erőforrást

## Példák

### Példa 1
```
PS C:\> New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

Ez a parancs a megadott tulajdonságokkal rendelkező lemez-hozzáférést hoz létre.
