# New-AzCosmosDBCassandraClusterKey

## Áttekintés
Új CosmosDB Cassandra cluster billentyűt hoz létre.

## SZINTAXISA

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **New-AzCosmosDBCassandraClusterKey** létrehoz egy új CosmosDB Cassandra cluster kulcsot.

## Példák

### Példa 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
