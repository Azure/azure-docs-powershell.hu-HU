# Invoke-AzDataBoxEdgeDevice

## Áttekintés
Meghívja az eszköz adott műveleteit.

## SZINTAXISA

## Leírás
A **meghívó AzDataBoxEdgeDevice** parancsmag az adatmező szélén lévő eszközön elindítja a frissítéseket, letölti és telepíti a frissítéseket. Az automatikus vizsgálat naponta fut, a következő parancsmag futtatásával explicit módon aktiválhatja a vizsgálatot.

## Példák

### Példa 1
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -ScanForUpdate
```

### 2. példa
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -FetchUpdate
```

### 3. példa
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -InstallUpdate
```
