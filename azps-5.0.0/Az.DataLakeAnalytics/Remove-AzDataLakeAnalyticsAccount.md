# Remove-AzDataLakeAnalyticsAccount

## Áttekintés
Az adattó-Analytics-fiók törlése

## SZINTAXISA

```
Remove-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzDataLakeAnalyticsAccount** parancsmag véglegesen törli az Azure Data Lake Analytics-fiókját.

## Példák

### 1. példa: fiók eltávolítása
```
PS C:\>Remove-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Ez a parancs eltávolítja a megadott Data Lake Analytics-fiókot.
