# Restore-AzDataLakeStoreDeletedItem

## Áttekintés
A törölt fájlok vagy mappák visszaállítása az Azure Data Lakeben.

## SZINTAXISA

### Alapértelmezett (alapértelmezett)
```
Restore-AzDataLakeStoreDeletedItem [-Account] <String> [-Path] <String> [-Destination] <String>
 [-Type] <String> [-RestoreAction <String>] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### InputObject
```
Restore-AzDataLakeStoreDeletedItem [-Account] <String> [-DeletedItem] <DataLakeStoreDeletedItem>
 [-RestoreAction <String>] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Restore-AzDataLakeStoreDeletedItem** parancsmag visszaállítja a törölt fájlokat vagy mappákat az Adattó áruházban. A AzDataLakeStoreDeletedItem a törölt elemek elérési útját igényli a kukában.
Figyelmeztetés: a törlés visszavonása ajánlott művelet. Nincs garancia arra, hogy a fájl törlés után visszaállíthatók. Az API használatát a whitelisten keresztül engedélyezheti. Ha az ADL-fiókja nem whitelist, akkor ezzel az API-val nem fog bevezetni a kivételt. További információért és segítségért forduljon a Microsoft ügyfélszolgálatához.

## Példák

### 1. példa: fájl visszaállítása az adattó áruházból a-Force beállítás használatával
```
PS > Restore-AzDataLakeStoreDeletedItem -Account ml1ptrashtest -Path 927e8fb1-a287-4353-b50e-3b4a39ae4088 -Destination adl://ml1ptrashtest.azuredatalake.com/test0/file_1230 -Type "file" -Force
PS >
```
