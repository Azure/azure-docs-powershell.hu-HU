# Get-AzDtlAutoStartPolicy

## Áttekintés
A DevTest Labs automatikus indítási házirendjét kapja meg.

## SZINTAXISA

```
Get-AzDtlAutoStartPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzDtlAutoStartPolicy** parancsmag egy olyan Lab automatikus kezdési házirendjét kapja meg, amely a laboratóriumi virtuális gépeket ütemezi automatikus indításra.
A parancsmag az engedélyezett vagy letiltott állapotot adja meg, valamint a hét napjait és a nap azon időpontját, amelyet beállított, hogy az automatikus indításhoz az Lab virtuális gépeket ütemezni lehessen.

## Példák

### Példa 1

A DevTest Labs automatikus indítási házirendjét kapja meg. autogenerated

```powershell <!-- Aladdin Generated Example --> 
Get-AzDtlAutoStartPolicy -LabName <String> -ResourceGroupName MyResourceGroup
```
