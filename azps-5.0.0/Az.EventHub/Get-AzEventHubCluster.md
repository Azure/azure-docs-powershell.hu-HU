# Get-AzEventHubCluster

## Áttekintés
Egyetlen esemény központi fürtjén kapja meg a részleteket, vagy az esemény-hub-fürtök listáját kapja.

## SZINTAXISA

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A Get-AzEventHubCluster parancsmag az esemény központi fürtjének részleteit vagy az adott erőforráscsoport összes esemény-központi fürtjének listáját adja eredményül.
Ha a fürt neve meg van határozva, akkor egy egyetlen fürt adatait adja vissza a rendszer.
Ha a fürt neve nincs megadva, a megadott erőforráscsoport összes szektorcsoportjának listáját a program visszaadja.
