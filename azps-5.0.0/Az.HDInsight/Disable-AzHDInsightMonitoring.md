# Disable-AzHDInsightMonitoring

## Áttekintés
Letiltja a figyelést egy HDInsight-fürtben, és a releváns naplók az engedélyezés során meghatározott figyelési munkaterületre áramlanak.

## SZINTAXISA

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **disable-AzHDInsightMonitoring** parancsmag letiltja a figyelést az Azure HDInsight-fürtökben.

## Példák

### Példa 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

A figyelés le lesz tiltva a HDInsight-fürtben, és a fontos naplók a figyelési munkaterületre áramlanak.

### 2. példa
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

A figyelés le lesz tiltva a HDInsight-fürtben, és a fontos naplók a figyelési munkaterületre áramlanak.
