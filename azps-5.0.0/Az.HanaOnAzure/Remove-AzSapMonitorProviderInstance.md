# Remove-AzSapMonitorProviderInstance

## Áttekintés
A megadott előfizetéshez, erőforráscsoporthez, SapMonitor és erőforrás nevéhez tartozó szolgáltató példány törlése.

## SZINTAXISA

### Delete (alapértelmezett)
```
Remove-AzSapMonitorProviderInstance -Name <String> -ResourceGroupName <String> -SapMonitorName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzSapMonitorProviderInstance -InputObject <IHanaOnAzureIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Leírás
A megadott előfizetéshez, erőforráscsoporthez, SapMonitor és erőforrás nevéhez tartozó szolgáltató példány törlése.

## Példák

### 1. példa: az SAP-monitor példányának eltávolítása név szerint
```powershell
PS C:\> Remove-AzSapMonitorProviderInstance -ResourceGroupName nancyc-hn1 -SapMonitorName ps-spamonitor-t01 -Name ps-sapmonitorins-t02

```

Ez a parancs eltávolítja az SAP-figyelés példányát név szerint.

### 2. példa: az SAP-monitor példányának eltávolítása objektumból
```powershell
PS C:\> $sapIns = Get-AzSapMonitorProviderInstance -ResourceGroupName nancyc-hn1 -SapMonitorName ps-spamonitor-t01 -Name ps-sapmonitorins-t01
PS C:\> Remove-AzSapMonitorProviderInstance -InputObject $sapIns

```

Ez a parancs eltávolítja az objektum által létrehozott SAP-monitor példányát.
