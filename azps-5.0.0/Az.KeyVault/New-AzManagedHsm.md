# New-AzManagedHsm

## Áttekintés
Felügyelt HSM-t hoz létre.

## SZINTAXISA

```
New-AzManagedHsm [-Name] <String> [-ResourceGroupName] <String> [-Location] <String>
 [-Administrator] <String[]> [-Sku <String>] [-Tag <Hashtable>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzManagedHsm** parancsmag létrehoz egy felügyelt HSM-t a megadott erőforráscsoport számára. Ha a felügyelt HSM alkalmazásban szeretne hozzáadni, eltávolítani vagy listázni a kulcsokat, akkor a felhasználónak felhasználói azonosító hozzáadásával kell engedélyt adni a rendszergazdának.

## Példák

### Példa 1: StandardB1 felügyelt HSM létrehozása
```powershell
PS C:\> New-AzManagedHsm -Name 'myhsm' -ResourceGroupName 'myrg1' -Location 'eastus2euap' -Administrator "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"

Name  Resource Group Name Location    SKU
----  ------------------- --------    ---
myhsm myrg1               eastus2euap StandardB1
```

A parancs létrehoz egy myhsm nevű felügyelt HSM-nevet a hely eastus2euap. A parancs hozzáadja a felügyelt HSM-t a myrg1 nevű erőforráscsoport csoportjához. Mivel a parancs nem ad meg értéket a *SKU* paraméterhez, létrehoz egy Standard_B1 felügyelt HSM-t.

### 2. példa: CustomB32 felügyelt HSM létrehozása
```powershell
PS C:\>New-AzManagedHsm -Name 'myhsm' -ResourceGroupName 'myrg1' -Location 'eastus2euap' -Administrator "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx" -Sku 'CustomB32'
Name  Resource Group Name Location    SKU

----  ------------------- --------    ---
myhsm myrg1               eastus2euap CustomB32
```

                      

Ez a parancs a felügyelt HSM-t az előző példában szereplőhöz hasonlóan hozza létre. A CustomB32 felügyelt HSM létrehozásakor azonban a CustomB32 értékét adja meg a *SKU* paraméterhez.
