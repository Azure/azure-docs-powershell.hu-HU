# Remove-AzManagedHsm

## Áttekintés
A felügyelt HSM törlése

## SZINTAXISA

### RemoveManagedHsmByName (alapértelmezett)
```
Remove-AzManagedHsm [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByInputObject
```
Remove-AzManagedHsm [-InputObject] <PSManagedHsm> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedHsmByResourceId
```
Remove-AzManagedHsm [-ResourceId] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzManagedHsm** parancsmag törli a megadott felügyelt HSM-t.
Az adott példányban található összes kulcsot is törli.
Fontos tudni, hogy az erőforráscsoport megadása esetén a parancsmag nem kötelező.

## Példák

### 1. példa: felügyelt HSM eltávolítása
```powershell
PS C:\> Remove-AzManagedHsm -HsmName 'myhsm' -Force

True
```

Ez a parancs eltávolítja a myhsm nevű felügyelt HSM-t a jelenlegi előfizetésből.

### 2. példa: a felügyelt HSM eltávolítása egy megadott erőforráscsoport-csoportból
```powershell
PS C:\> Remove-AzManagedHsm -HsmName 'myhsm' -ResourceGroupName "myrg1" -PassThru

True
```

Ez a parancs eltávolítja a myhsm nevű felügyelt HSM nevet a myrg1 nevű erőforráscsoport nevében.
Ha nem adja meg az erőforráscsoport nevét, a parancsmag a megnevezett felügyelt HSM-t keresi a jelenlegi előfizetésben való törléshez.
