# Remove-AzManagedHsmKey

## Áttekintés
Egy felügyelt HSM kulcsának törlése

## SZINTAXISA

### RemoveByKeyNameParameterSet (alapértelmezett)
```
Remove-AzManagedHsmKey [-HsmName] <String> [-Name] <String> [-Force] [-PassThru] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObjectParameterSet
```
Remove-AzManagedHsmKey [-InputObject] <PSKeyVaultKeyIdentityItem> [-Force] [-PassThru] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A Remove-AzManagedHsmKey parancsmag a felügyelt HSM-beli kulcsokat törli.
Ha a kulcsot véletlenül töröltük, a kulcs visszaállítható a Undo-AzManagedHsmKeyRemoval felhasználó által speciális "helyreállítási" engedélyekkel.
A parancsmag értéke magas a **ConfirmImpact** tulajdonság számára.

## Példák

### 1. példa: kulcs eltávolítása a felügyelt HSM-ből
```powershell
PS C:\> Remove-AzManagedHsmKey -HsmName testmhsm -Name testkey -PassThru

Vault/HSM Name       : testmhsm
Name                 : testkey
Id                   : https://testmhsm.managedhsm.azure.net:443/keys/testkey/9a9de2bcec540c3b160cd54cbae71339
Deleted Date         : 10/14/2020 9:35:06 AM
Scheduled Purge Date : 1/12/2021 9:35:06 AM
Enabled              : False
Expires              : 10/14/2022 8:13:29 AM
Not Before           : 10/14/2020 8:13:33 AM
Created              : 10/14/2020 8:14:01 AM
Updated              : 10/14/2020 8:14:01 AM
Recovery Level       : Recoverable+Purgeable
Tags                 :
```

Ez a parancs eltávolítja a testkey nevű kulcsot a testmhsm nevű felügyelt HSM-ből.

### 2. példa: kulcs eltávolítása a felhasználó jóváhagyása nélkül
```powershell
PS C:\> Remove-AzManagedHsmKey -HsmName testmhsm -Name testkey -Force
```

Ez a parancs eltávolítja a testkey nevű kulcsot a testmhsm nevű felügyelt HSM-ből.
A parancs az *erő* paramétert adja meg, ezért a parancsmag nem kér megerősítést.

### 3. példa: törölt kulcsok végleges törlése a felügyelt HSM-ből
```powershell
PS C:\> Remove-AzManagedHsmKey -HsmName testmhsm -Name testkey -InRemovedState
```

Ez a parancs eltávolítja a testkey nevű kulcsot véglegesen a testmhsm nevű felügyelt HSM-ből.
A parancsmag végrehajtásához a "Purge" engedély szükséges, amelyet előzőleg és kifejezetten meg kell adni a felhasználónak a felügyelt HSM számára.

### 4. példa: kulcsok eltávolítása a csővezeték-kezelő használatával
```powershell
PS C:\> Get-AzManagedHsmKey -HsmName testmhsm | Where-Object {$_.Attributes.Enabled -eq $False} | Remove-AzManagedHsmKey
```

Ez a parancs a testmhsm nevű felügyelt HSM összes kulcsát beolvassa, és a csővezeték-kezelő segítségével átadja őket a **Where-Object** parancsmagnak.
Ez a parancsmag átadja azokat a kulcsokat, amelyek $False értékkel rendelkeznek az **engedélyezett** attribútumhoz az aktuális parancsmaghoz.
A parancsmag eltávolítja ezeket a billentyűket.
