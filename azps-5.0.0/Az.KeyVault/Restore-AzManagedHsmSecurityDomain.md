# Restore-AzManagedHsmSecurityDomain

## Áttekintés
Az előző biztonsági másolat biztonsági mentése a felügyelt HSM szolgáltatáshoz.

## SZINTAXISA

### ByName (alapértelmezett)
```
Restore-AzManagedHsmSecurityDomain -Keys <KeyPath[]> -SecurityDomainPath <String> [-PassThru] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Restore-AzManagedHsmSecurityDomain -Keys <KeyPath[]> -SecurityDomainPath <String> [-PassThru]
 -InputObject <PSKeyVaultIdentityItem> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
Ez a parancsmag visszaállítja az előző biztonsági tartomány adatainak biztonsági másolatát a felügyelt HSM-nek.

## Példák

### Példa 1
```powershell
PS C:\> $keys = @{PublicKey = "sd1.cer"; PrivateKey = "sd1.key"}, @{PublicKey = "sd2.cer"; PrivateKey = "sd2.key"}, @{PublicKey = "sd3.cer"; PrivateKey = "sd3.key"}
PS C:\> Restore-AzManagedHsmSecurityDomain -Name testmhsm -Keys $keys -SecurityDomainPath {pathOfBackup}\sd.ps.json
```

Először a biztonsági tartomány adatainak visszafejtéséhez szükséges kulcsokat kell megadnia. Ezután a **Restore-AzManagedHsmSecurityDomain** parancs visszaállítja az előző biztonsági tartomány adatainak biztonsági biztonsági másolatát a kezelt HSM-nek a következő billentyűk használatával.
