# Remove-AzMariaDbVirtualNetworkRule

## Áttekintés
A virtuális hálózati szabály törlése a megadott névvel

## SZINTAXISA

### Delete (alapértelmezett)
```
Remove-AzMariaDbVirtualNetworkRule -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzMariaDbVirtualNetworkRule -InputObject <IMariaDbIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Leírás
A virtuális hálózati szabály törlése a megadott névvel

## Példák

### Példa 1: virtuális hálózati szabály eltávolítása
```powershell
PS C:\> Remove-AzMariaDbVirtualNetworkRule -Name vnet-001 -ResourceGroupName mariadb-test-qu5ov0 -ServerName mariadb-test-9pebvn

```

Ez a parancs eltávolítja a virtuális hálózati szabályt.
