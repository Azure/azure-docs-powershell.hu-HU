# New-AzApplicationGatewayFirewallCondition

## Áttekintés
Egyeztető feltételt hoz létre az egyéni szabályokhoz.

## SZINTAXISA

```
New-AzApplicationGatewayFirewallCondition -MatchVariable <PSApplicationGatewayFirewallMatchVariable[]>
 -Operator <String> [-NegationCondition <Boolean>] -MatchValue <String[]> [-Transform <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **New-AzApplicationGatewayFirewallCondition** létrehoz egy egyeztetési feltételt a tűzfal egyéni szabályához.

## Példák

### Példa 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallCondition -MatchVariable $variable -Operator Contains -NegationCondition false -Transforms Lowercase, Trim -MatchValue abc, cde
```

A parancs új egyeztetési feltételt hoz létre a $variableban definiált egyezési változóval, az operátor tartalmaz és tagadási feltételt tartalmaz, a Transfroms, például a kisbetűs és a kimetszet, a hol. van érték az ABC és a CDE. A program az új egyezési feltételt $condition menti.
