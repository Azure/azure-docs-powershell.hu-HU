# New-AzFirewallHubIpAddress

## Áttekintés
Az IP-címek telefonszámok a virtuális központi tűzfalon

## SZINTAXISA

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az IP-címek telefonszámok a virtuális központi tűzfalon. Ezek lehetnek nyilvános és privát címek is

## Példák

### Példa 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

Ez a példa egy központi IP-cím objektumot hoz létre, amelyben 2 nyilvános IP-cím szerepel. A HubIPAddress objektum a virtuális ssociated lévő tűzfalon van.
