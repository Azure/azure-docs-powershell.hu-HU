# Set-AzApplicationGatewayTrustedRootCertificate

## Áttekintés
Egy alkalmazás-átjáró megbízható főtanúsítványát frissíti.

## SZINTAXISA

```
Set-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **set-AzApplicationGatewayTrustedRootCertificate** parancsmag módosította az alkalmazás-átjárók meglévő megbízható főtanúsítványát.

## Példák

### Példa 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\rootCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

A fenti példa szerint az esetek azt mutatják be, hogy miként frissíthet egy meglévő megbízható főtanúsítványt a főtanúsítvány bedobásakor.
Az első parancs beolvassa az Application Gatewayt, és a $gw változóban tárolja.
A második parancs módosította a meglévő megbízható főtanúsítványot egy új főtanúsítványsal.
A harmadik parancs az Azure rendszerhez frissíti az Application Gatewayt.
