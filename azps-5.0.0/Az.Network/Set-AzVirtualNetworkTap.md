# Set-AzVirtualNetworkTap

## Áttekintés
Virtuális hálózat frissítése koppintással.

## SZINTAXISA

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **set-AzVirtualNetworkTap** parancsmag a virtuális hálózatokra koppintva frissíti a bővítményt.

## Példák

### Példa 1: virtuális hálózat beállítása koppintás
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

A parancs frissíti a célhely IpConfiguration, és frissíti a virtuális hálózatot.
Ha vannak olyan koppintásos konfigurációk, amelyek hivatkoznak rá, akkor az összes forrás-forgalom nem fog megjelenni az új IP-címek konfigurációjának frissítése után.
