# Get-AzOperationalInsightsCluster

## Áttekintés
Fürtök beolvasása és listázása

## SZINTAXISA

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Csoportosíthatja vagy listázhatja a fürtöket, az erőforráscsoport csoportban felsorolhatja az "-fürtnév" kifejezést, ha a "-fürtnév" és a "ResourceGroupName" nem volt megadva.

## Példák

### Példa 1
```powershell
Get-AzOperationalInsightsCluster -ResourceGroupName {rg-name} -ClusterName {cluster-name}
```
