# Get-AzLegacyPeering

## Áttekintés
Segítségével örökölt társközi erőforrásokat konvertálhat az Azure Resource Management (ARM) erőforrásaira. 

## SZINTAXISA

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A parancs segítségével megtekintheti azokat az örökölt társközi erőforrásokat, amelyek mindegyike az Azure erőforrás-kezelési (ARM) erőforrásokra konvertálja őket.
