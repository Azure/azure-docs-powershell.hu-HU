# New-AzPeerAsnContactDetail

## Áttekintés
A PeerAsn-ban a memória-kapcsolat részleteit hozza létre. 

## SZINTAXISA

```
New-AzPeerAsnContactDetail -Role <String> -Email <String> [-Phone <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Hozzon létre egy a Memory PeerAsn-adatait.

## Példák

### A partnerek részleteinek létrehozása és hozzáadása a PeerAsn-hoz
```powershell
PS C:\> $nocContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> $customerContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> New-AzPeerAsn -Name $name -PeerName "Contoso Networks Limited" -PeerAsn 65000 -ContactDetail $nocContact,$customerContact
```

Szerepkör és e-mail-cím szükséges. A telefon nem kötelező. A telefon támogatja a +-() vagy a szóközöket. A PeerAsn tartalmaznia kell legalább egy, a "Noc" típusú adatot.
