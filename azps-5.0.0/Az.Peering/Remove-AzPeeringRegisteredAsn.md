# Remove-AzPeeringRegisteredAsn

## Áttekintés
A fölérendelt társközi erőforrásból törölheti vagy távolíthatja el a regisztrált ASN-t.

## SZINTAXISA

### ByName (alapértelmezett)
```
Remove-AzPeeringRegisteredAsn [-ResourceGroupName] <String> [-PeeringName] <String> [-Name] <String> [-Force]
 [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzPeeringRegisteredAsn -InputObject <PSPeeringServicePrefix> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzPeeringRegisteredAsn [-ResourceId] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Lehetővé teszi a Registered ASN-nek a fölérendelt társközi erőforrásból való eltávolítását.

## Példák

### Példa 1
```powershell
PS C:\> Remove-AzPeeringRegisteredAsn -ResourceId $resourceId
```

Az ASN-es regisztráció erőforrás-azonosítóval való eltávolítása
