# Get-AzPowerBIEmbeddedCapacity

## Áttekintés
A PowerBI beágyazott kapacitásainak részleteit kapja meg.

## SZINTAXISA

### ByCapacityOrResourceGroupOrSubscription (alapértelmezett)
```
Get-AzPowerBIEmbeddedCapacity [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzPowerBIEmbeddedCapacity -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A Get-AzPowerBIEmbeddedCapacity parancsmag a PowerBI beágyazott kapacitásának részleteit kapja meg.
