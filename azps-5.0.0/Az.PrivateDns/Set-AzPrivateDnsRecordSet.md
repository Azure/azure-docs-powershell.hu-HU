# Set-AzPrivateDnsRecordSet

## Áttekintés
Frissítések/rekordok beállítása privát DNS-zónában.

## SZINTAXISA

```
Set-AzPrivateDnsRecordSet -RecordSet <PSPrivateDnsRecordSet> [-Overwrite]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az Set-AzPrivateDnsRecordSet parancsmag egy helyi RecordSet objektumból frissíti az Azure Private DNS szolgáltatásban lévő rekordot. A rekordhalmaz-objektum paraméterként vagy a pipeline operátor segítségével átadható. A Confirm paramétert és $ConfirmPreference Windows PowerShell-változót használva beállíthatja, hogy a parancsmag kér-e megerősítést. A Record set nem frissül, ha az Azure Private DNS-ben módosította a helyi RecordSet objektum beolvasását. Ez védelmet nyújt az egyidejű módosítások számára. Ezt a működést a felülírás paraméterrel szüntetheti meg, amely az egyidejű módosításoktól függetlenül frissíti a rekordot.

## Példák

### Példa 1: rekordazonosító frissítése
```powershell
PS C:\> $RecordSet = Get-AzPrivateDnsRecordSet -ResourceGroupName MyResourceGroup -ZoneName myzone.com -Name www -RecordType A
PS C:\> Add-AzPrivateDnsRecordConfig -RecordSet $RecordSet -Ipv4Address 172.16.0.0
PS C:\> Add-AzPrivateDnsRecordConfig -RecordSet $RecordSet -Ipv4Address 172.31.255.255
PS C:\> Set-AzPrivateDnsRecordSet -RecordSet $RecordSet

# These cmdlets can also be piped:

PS C:\> Get-AzPrivateDnsRecordSet -ResourceGroupName MyResourceGroup -ZoneName myzone.com -Name www -RecordType A | Add-AzPrivateDnsRecordConfig -Ipv4Address 172.16.0.0 | Add-AzPrivateDnsRecordConfig -Ipv4Address 172.31.255.255 | Set-AzPrivateDnsRecordSet

                    rk/privateDnsZones/myzone.com/A/www
Name              : www
ZoneName          : myzone.com
ResourceGroupName : MyResourceGroup
Ttl               : 3600
Etag              : xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
RecordType        : A

Metadata          :
```

Az első parancs az Get-AzPrivateDnsRecordSet parancsmagot használja a megadott rekordtípus beolvasásához, majd a $RecordSet változóban tárolja. A második és a harmadik parancs az off-line műveletekkel két rekordot hozhat létre a rekordhoz. A végleges parancs a Set-AzPrivateDnsRecordSet parancsmagot használja a frissítés véglegesítéséhez.
