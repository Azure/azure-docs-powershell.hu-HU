# New-AzRecoveryServicesVault

## Áttekintés
Új helyreállítási szolgáltatások boltozatának létrehozása

## SZINTAXISA

```
New-AzRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzRecoveryServicesVault** parancsmag új helyreállítási szolgáltatások boltozatát hozza létre.

## Példák

### Példa 1
```
PS C:\> New-AzRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Helyreállítási szolgáltatás boltozatának létrehozása az erőforrás csoportban és az adott helyen.
