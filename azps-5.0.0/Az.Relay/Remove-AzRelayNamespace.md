# Remove-AzRelayNamespace

## Áttekintés
Eltávolítja a névteret a megadott erőforráscsoport-csoportból. 

## SZINTAXISA

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzRelayNamespace** parancsmag eltávolítja a névteret a megadott erőforrás-csoportból.

## Példák

### Példa 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

A továbbítási névtér eltávolítása `TestNameSpace-Relay1` a megadott erőforrás-csoportból `Default-ServiceBus-WestUS` .
