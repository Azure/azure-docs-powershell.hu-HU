# New-AzADGroup

## Áttekintés
Új Active Directory-csoport létrehozása

## SZINTAXISA

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Új Active Directory-csoport létrehozása Az alábbi engedélyek szükségesek:

- Azure Active Directory-diagram
  - Directory. ReadWrite. All
- Microsoft Graph
  - Directory. ReadWrite. All
  - PrivilegedAccess. ReadWrite. AzureADGroup

## Példák

### Példa 1: új HIRDETÉSCSOPORT létrehozása

```powershell
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "MyGroupNick"
```

Létrehoz egy új HIRDETÉSCSOPORTOT a "MyGroupDisplayName" névvel és a "MyGroupNick" levelezési felhasználónévvel.
