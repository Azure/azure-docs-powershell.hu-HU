# Register-AzProviderFeature

## Áttekintés
A fiók Azure Provider szolgáltatását regisztrálja.

## SZINTAXISA

```
Register-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Register-AzProviderFeature** parancsmag a fiók Azure Provider szolgáltatását regisztrálja.

## Példák

### 1. példa: funkció regisztrálása
```
PS C:\>Register-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Ezzel felveszi a Microsoft. Network AllowApplicationSecurityGroups funkcióját a fiókjába.
