# Remove-AzManagementGroupSubscription

## Áttekintés
Új előfizetés eltávolítása egy felügyeleti csoportból.

## SZINTAXISA

```
Remove-AzManagementGroupSubscription [-GroupName] <String> [-SubscriptionId] <Guid> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzManagementGroupSubscription** parancsmag felügyeleti csoportból távolítja el az előfizetést.

## Példák

### Példa 1: előfizetés eltávolítása egy felügyeleti csoportból
```powershell
PS C:\> Remove-AzManagementGroupSubscription -GroupName "TestGroup" -SubscriptionId 2120692d-35c3-44c8-81f5-631fa7351726
```
