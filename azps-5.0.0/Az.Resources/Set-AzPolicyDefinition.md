## Leírás
A **set-AzPolicyDefinition** parancsmag módosította a házirend-definíciót.

## Példák

### Példa 1: házirend-definíció leírásának frissítése
```
PS C:\> $PolicyDefinition = Get-AzPolicyDefinition -Name 'VMPolicyDefinition'
PS C:\> Set-AzPolicyDefinition -Id $PolicyDefinition.ResourceId -Description 'Updated policy to not allow virtual machine creation'
```

Az első parancs az Get-AzPolicyDefinition parancsmag használatával megkapja a VMPolicyDefinition nevű házirend-definíciót.
A parancs az objektumot a $PolicyDefinition változóban tárolja.
A második parancs frissíti a $PolicyDefinition **ResourceId** tulajdonsága által azonosított házirend-definíció leírását.

### 2. példa: házirend-definíciós mód frissítése
```
PS C:\> Set-AzPolicyDefinition -Name 'VMPolicyDefinition' -Mode 'All'
```

Ez a parancs frissíti a VMPolicyDefinition nevű házirend-definíciót a Set-AzPolicyDefinition parancsmagot használva az "all" (mód) tulajdonság beállításához.

### 3. példa: egy házirend-definíció metaadatait frissíti
```
PS C:\> Set-AzPolicyDefinition -Name 'VMPolicyDefinition' -Metadata '{"category":"Virtual Machine"}'


Name               : VMPolicyDefinition

```

Ez a parancs frissíti a VMPolicyDefinition nevű házirend-definíció metaadatait, így jelezve, hogy a kategóriája "virtuális gép".
