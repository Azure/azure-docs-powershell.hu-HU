# Az. biztonsági modul
## Leírás
Az Azure Security Center az Azure-előfizetések és az Azureon kívüli többi számítógép biztonsága felett nyújt irányítást.

## Az. biztonsági parancsmagok

Letiltja a speciális veszélyforrások elleni védelemre vonatkozó házirendet egy tárterület-vagy cosmosDB-fiókhoz.

### [Enable-AzSecurityAdvancedThreatProtection](Enable-AzSecurityAdvancedThreatProtection.md)
Lehetővé teszi a speciális veszélyforrások védelmének házirendjét egy tárterület-vagy cosmosDB-fiókhoz.
