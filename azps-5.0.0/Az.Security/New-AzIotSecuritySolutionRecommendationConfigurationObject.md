# New-AzIotSecuritySolutionRecommendationConfigurationObject

## Áttekintés
Új ajánlási konfiguráció létrehozása a IOT biztonsági megoldásához

## SZINTAXISA

```
New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType <String> -Enabled <Boolean>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A AzIotSecuritySolutionRecommendationConfigurationObject létrehoz egy új ajánlási konfigurációs objektumot a IOT biztonsági megoldásához.
Így az ajánlás állapota úgy van konfigurálva, hogy be van-e kapcsolva vagy le van tiltva.

## Példák

### Példa 1
```powershell
PS C:\> New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType "IoT_ACRAuthentication" -Enabled $false

RecommendationType: "IoT_ACRAuthentication"
Name: "Service prinicpal not used with ACR repository"
Status: "Disabled"
```

Új ajánlási konfiguráció létrehozása az "IoT_ACRAuthentication" vagy a letiltási beállítás típusához
