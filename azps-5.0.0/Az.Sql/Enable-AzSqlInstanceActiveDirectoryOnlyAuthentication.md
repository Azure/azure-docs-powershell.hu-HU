# Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication

## Áttekintés
Engedélyezi az Azure AD only hitelesítést egy bizonyos SQL-alapú felügyelt példányhoz.

## SZINTAXISA

### UseResourceGroupAndInstanceNameParameterSet (alapértelmezett)
```
Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication [-ResourceGroupName] <String> [-InstanceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UseInputObjectParameterSet
```
Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication -InputObject <AzureSqlManagedInstanceModel>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UserResourceIdParameterSet
```
Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az **enable-AzSqlInstanceActiveDirectoryOnlyAuthentication** parancsmag csak az Azure Active Directory (Azure ad) hitelesítési követelményét engedélyezi az aktuális előfizetésben a AzureSQL kezelt példányok esetében.

## Példák

### Példa 1
```powershell
PS C:\>Enable-AzSqlInstanceActiveDirectoryOnlyAuthentication -ResourceGroupName "ResourceGroup01" -InstanceName "ManagedInstance01"
ResourceGroupName InstanceName        AzureADOnlyAuthentication
----------------- ---------- ----------- -------- -----------
ResourceGroup01   ManagedInstance01   True
```

Ez a parancs lehetővé teszi az Azure Active Directory (Azure AD) hitelesítést csak a ResourceGroup01 nevű ManagedInstance01 társított AzureSQL-beli felügyelt példányok hitelesítési követelményeihez.
