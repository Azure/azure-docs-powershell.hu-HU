# Get-AzSqlElasticPoolActivity

## Áttekintés
Egy rugalmas készleten kapja meg a műveletek állapotát.

## SZINTAXISA

```
Get-AzSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String> [-OperationId <Guid>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Get-AzSqlElasticPoolActivity** parancsmag a műveletek állapotát az Azure SQL-adatbázisok rugalmas készletében kapja.
A Pool-készítési és a konfigurációs frissítések állapotát is láthatja.

## Példák

### Példa 1: műveletek állapotának megszerzése rugalmas készlet esetén
```
PS C:\>Get-AzSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Ez a parancs a ElasticPool01 nevű rugalmas készlet műveletének állapotát kapja meg.
