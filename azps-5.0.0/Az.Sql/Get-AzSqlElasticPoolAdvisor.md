# Get-AzSqlElasticPoolAdvisor

## Áttekintés
Egy vagy több tanácsadót kap egy Azure SQL elasztikus poolban.

## SZINTAXISA

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzSqlElasticPoolAdvisor** parancsmag egy vagy több Azure SQL elasztikus Pool-tanácsadót kap az Azure SQL rugalmas készletéhez.
