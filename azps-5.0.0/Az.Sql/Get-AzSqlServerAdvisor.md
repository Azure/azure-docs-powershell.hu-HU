# Get-AzSqlServerAdvisor

## Áttekintés
Egy vagy több tanácsadót kap egy Azure SQL Server-kiszolgálón.

## SZINTAXISA

```
Get-AzSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzSqlServerAdvisor** parancsmag egy vagy több Azure SQL Server-tanácsadót kap az Azure SQL Server rendszerhez.
