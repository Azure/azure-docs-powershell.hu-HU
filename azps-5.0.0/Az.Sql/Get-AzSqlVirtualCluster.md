# Get-AzSqlVirtualCluster

## Áttekintés
Információt ad az Azure SQL virtuális fürtökről.

## SZINTAXISA

### GetVirtualClusterByResourceGroup (alapértelmezett)
```
Get-AzSqlVirtualCluster [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetVirtualClusterByNameAndResourceGroup
```
Get-AzSqlVirtualCluster [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetVirtualClusterByResourceId
```
Get-AzSqlVirtualCluster [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzSqlVirtualCluster** parancsmag információt ad egy vagy több Azure SQL-szektorcsoportról.
Adjon meg egy virtuális fürtöt, ha csak a fürt adatait szeretné látni.

Ez a parancs információkat kap az erőforráscsoport ResourceGroup01 társított összes virtuális fürtökről.

Ez a parancs információt kap a virtuális fürtök VirtualCluster1 az erőforráscsoport ResourceGroup01.
