# Remove-AzSqlDatabaseFailoverGroup

## Áttekintés
Egy Azure SQL-adatbázis feladatátvételi csoportjának eltávolítása.

## SZINTAXISA

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
Ez a parancs eltávolítja a megadott nevű feladatátvételi csoportot, így az összes adatbázis és replikációs kapcsolat érintetlen marad. A figyelő végpontját a rendszer nem regisztrálja a DNS-ből.
A feladatátvevő csoport elsődleges kiszolgálóját kell használni a parancs végrehajtásához.

## Példák

### Példa 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Feladatátvételi csoport eltávolítása
