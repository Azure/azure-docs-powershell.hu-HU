# Stop-AzSqlElasticPoolActivity

## Áttekintés
Egy rugalmas készleten lemondja az aszinkron frissítési műveletet.

## SZINTAXISA

```
Stop-AzSqlElasticPoolActivity [-PassThru] [-ServerName] <String> [-ElasticPoolName] <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzSqlElasticPoolActivity** parancsmag egy rugalmas készleten megszünteti az aszinkron frissítési műveletet.

## Példák

### 1. példa: az aszinkron frissítési művelet megszakítása rugalmas készleten
