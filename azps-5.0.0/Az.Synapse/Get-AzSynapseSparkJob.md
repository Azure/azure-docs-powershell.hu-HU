# Get-AzSynapseSparkJob

## Áttekintés
Egy szinapszis-elemző szikra munkát kap.

## SZINTAXISA

### GetSparkJobsByIdParameterSet (alapértelmezett)
```
Get-AzSynapseSparkJob -WorkspaceName <String> -SparkPoolName <String> [-LivyId <Int32>] [-Name <String>]
 [-ApplicationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetSparkJobsByIdFromParentObjectParameterSet
```
Get-AzSynapseSparkJob -SparkPoolObject <PSSynapseSparkPool> [-LivyId <Int32>] [-Name <String>]
 [-ApplicationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzSynapseSparkJob** parancsmag az Azure szinapszis Analytics Spark-feladatot kapja.
Ha nem ad meg feladatot, ez a parancsmag minden feladatot beilleszt.

## Példák

### Példa 1
```powershell
PS C:\> Get-AzSynapseSparkJob -WorkspaceName ContosoWorkspace -SparkPoolName ContosoSparkPool
```

Ez a parancs egy Spark-medence alatti minden munkahelyhez jut.

### 2. példa
```powershell
PS C:\> Get-AzSynapseSparkJob -WorkspaceName ContosoWorkspace -SparkPoolName ContosoSparkPool -LivyId 119
```

Ez a parancs a megadott AZONOSÍTÓval kapja a feladatot.

### 3. példa
```powershell
PS C:\> Get-AzSynapseSparkJob -WorkspaceName ContosoWorkspace -SparkPoolName ContosoSparkPool -ApplicationId application_1585023543211_0004
```

Ez a parancs a megadott alkalmazásspecifikus AZONOSÍTÓval kapja a feladatot.

### 4. példa
```powershell
PS C:\> $pool = Get-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -SparkPoolName ContosoSparkPool
PS C:\> $pool | Get-AzSynapseSparkJob
```

Ez a parancs a szikra alatt a csővezetéken keresztül minden munkahelyhez bekerül.
