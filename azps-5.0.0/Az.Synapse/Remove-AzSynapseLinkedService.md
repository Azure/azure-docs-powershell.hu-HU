# Remove-AzSynapseLinkedService

## Áttekintés
Egy kapcsolt szolgáltatás eltávolítása a munkaterületről.

## SZINTAXISA

### RemoveByName (alapértelmezett)
```
Remove-AzSynapseLinkedService -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByObject
```
Remove-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzSynapseLinkedService -InputObject <PSLinkedServiceResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzSynapseLinkedService** parancsmag eltávolított egy kapcsolt szolgáltatást a munkaterületről.

## Példák

### Példa 1
```powershell
PS C:\> Remove-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
```

Ez a parancs eltávolítja a ContosoLinkedService nevű kapcsolt szolgáltatást a ContosoWorkspace nevű munkaterületről.

### 2. példa
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Remove-AzSynapseLinkedService -Name ContosoLinkedService
```

Ez a parancs eltávolítja a ContosoLinkedService nevű kapcsolt szolgáltatást a ContosoWorkspace – csővezeték nevű munkaterületről.

### 3. példa
```powershell
PS C:\> $linkedService = Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
PS C:\> $linkedService | Remove-AzSynapseLinkedService
```

Ez a parancs eltávolítja a ContosoLinkedService nevű kapcsolt szolgáltatást a ContosoWorkspace – csővezeték nevű munkaterületről.
