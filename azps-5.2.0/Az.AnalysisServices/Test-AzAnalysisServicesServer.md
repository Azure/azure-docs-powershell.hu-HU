# Test-AzAnalysisServicesServer

## SYNOPSIS
Annak vizsgálata, hogy az Analysis Services-kiszolgáló példánya létezik-e

## SZINTAXIS

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Test-AzAnalysisServicesServer parancsmag teszteli az Analysis Services-kiszolgáló egy példányának meglétét

## PÉLDÁK

### 1. példa
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Ez a parancs teszteli, hogy van-e testserver nevű kiszolgáló az erőforráscsoport tesztcsoportban
