# Get-AzApiManagementOpenIdConnectProvider

## SYNOPSIS
OpenID Connect-szolgáltatókat kap.

## SZINTAXIS

### GetAllOpenIdConnectProviders (alapértelmezett)
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByOpenIdConnectProviderId
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-OpenIdConnectProviderId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByName
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApiManagementOpenIdConnectProvider** parancsmag openID Connect-szolgáltatókat kap az Azure API Management szolgáltatásban.
A ClientSecret nem fog szerepelni az eredmény részletei között. Az ügyfél titkosítására használja a **Get-AzApiManagementOpenIdConnectProviderClientSecretet.**

## PÉLDÁK

### 1. példa: Az összes szolgáltató lekérte
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext
```

Ez a parancs az összes OpenID Connect-szolgáltatót beveszi a megadott környezetben.

### 2. példa: Szolgáltató lekérte azonosító használatával
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01"
```

Ez a parancs az OICProvider01 azonosítóval rendelkezik.

### 3. példa: Szolgáltató lekérte egy név használatával
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -Name "Contoso OpenID Connect Provider"
```

Ez a parancs a Contoso OpenID Connect Provider nevű szolgáltatót kapja meg.
