# Remove-AzApiManagementApiRevision

## SYNOPSIS
Egy adott API-változat eltávolítása

## SZINTAXIS

### ByApiId (alapértelmezett)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApiManagementApiRevision** parancsmag eltávolít egy adott API-változatot.

## PÉLDÁK

### 1. példa: API-változat eltávolítása
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

Ez a parancs eltávolítja az API felülvizsgálatát `2` az `echo-api` API felügyeleti szolgáltatásból.

### 2. példa

Ez a parancs eltávolítja az API echo-api 2 változatát az API Management szolgáltatásból. (automatikusan generált)

```powershell
<!-- Aladdin Generated Example --> 
Remove-AzApiManagementApiRevision -ApiId '0001' -ApiRevision 6 -Context <PsApiManagementContext>
```
