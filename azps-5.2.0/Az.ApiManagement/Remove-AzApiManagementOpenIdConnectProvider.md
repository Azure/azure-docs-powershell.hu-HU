# Remove-AzApiManagementOpenIdConnectProvider

## SYNOPSIS
Eltávolít egy OpenID Connect-szolgáltatót.

## SZINTAXIS

```
Remove-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> -OpenIdConnectProviderId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApiManagementOpenIdConnectProvider** parancsmag eltávolít egy OpenID Connect-szolgáltatót az Azure API-kezeléshez.

## PÉLDÁK

### 1. példa: Szolgáltató eltávolítása
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01" -PassThru
```

Ez a parancs eltávolít egy OICProvider01 azonosítójú szolgáltatót.
