# Save-AzApiManagementTenantGitConfiguration

## SYNOPSIS
Módosítások mentése véglegesítés létrehozásával az aktuális konfigurációhoz.

## SZINTAXIS

```
Save-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Save-AzApiManagementTenantGitConfiguration** parancsmag a módosításokat úgy menti, hogy létrehoz egy véglegesítést, amely tartalmazza az aktuális konfigurációs pillanatképet a tárban lévő ágba.

## PÉLDÁK

### 1. példa: A konfiguráció módosításainak mentése
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Save-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

Ez a parancs úgy menti a módosításokat, hogy létrehoz egy véglegesítést az aktuális konfigurációs pillanatkép alapján a tár adott ágában.
