# Get-AzAutomationModule

## SYNOPSIS
Metaadatokat kap a modulokhoz az Automatizálásból.

## SZINTAXIS

### ByAll (alapértelmezett)
```
Get-AzAutomationModule [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationModule [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzAutomationModule** parancsmag metaadatokat kap a modulokhoz az Azure Automationtől.

## PÉLDÁK

### 1. példa: Az összes modul lekérte
```
PS C:\>Get-AzAutomationModule -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs a Contoso17 nevű automatizálási fiók összes modulját beveszi.

### 2. példa: Modul lekérte
```
PS C:\>Get-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs kap egy ContosoModule nevű modult a Contoso17 nevű automatizálási fiókban.
