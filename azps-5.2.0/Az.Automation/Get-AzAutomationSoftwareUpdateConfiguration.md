# Get-AzAutomationSoftwareUpdateConfiguration

## SYNOPSIS
Az Azure Automatizálási szoftverfrissítések konfigurációjának listáját tartalmazza.

## SZINTAXIS

### ByAll (alapértelmezett)
```
Get-AzAutomationSoftwareUpdateConfiguration [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSoftwareUpdateConfiguration -Name <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByVMId
```
Get-AzAutomationSoftwareUpdateConfiguration -AzureVMResourceId <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Get-AzAutomationSoftwareUpdateConfiguration a szoftverfrissítések konfigurációjának listáját adja vissza. Adott szoftverfrissítés-konfiguráció megadásához adja meg a név paraméterét. A virtuális gép azure-erőforrás-azonosítójának megadásával listába is sorolhatja az egyes Azure virtuális gépekre vonatkozó szoftverfrissítés-konfigurációkat.

## PÉLDÁK

### 1. példa
Azure Automation-szoftverfrissítések név szerint való beállítása.

```powershell
PS C:\> Get-AzAutomationSoftwareUpdateConfiguration -ResourceGroupName "mygroup" -AutomationAccountName "myaccount" -Name "MyWeeklySchedule"
```
