# New-AzAutomationKey

## SYNOPSIS
Az automatizálási fiók regisztrációs kulcsait újragenerálja.

## SZINTAXIS

```
New-AzAutomationKey [-KeyType] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzAutomationKey** parancsmag újragenerálja az Azure Automation-fiók regisztrációs kulcsait.

## PÉLDÁK

### 1. példa: Automatizálási fiók kulcsának újragenerálása
```
PS C:\>New-AzAutomationKey -KeyType Primary -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

Ez a parancs az ResourceGroup01 nevű erőforráscsoport AutomationAccount01 nevű Azure Automation-fiókjának elsődleges kulcsát újragenerálja.
