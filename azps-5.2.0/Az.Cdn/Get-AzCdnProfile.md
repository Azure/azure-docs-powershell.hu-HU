# Get-AzCdnProfile

## SYNOPSIS
CDN-profilt kap.

## SZINTAXIS

```
Get-AzCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

A **Get-AzCdnProfile** parancsmag kap egy Azure Content Delivery Network -profilt (CDN) és annak kapcsolódó adatait.
