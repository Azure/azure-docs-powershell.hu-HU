# Unpublish-AzCdnEndpointContent

## SYNOPSIS
CDN-végpont végleges véglegesen való végleges véglegesen ürét.

## SZINTAXIS

### ByFieldsParameterSet (alapértelmezett)
```
Unpublish-AzCdnEndpointContent -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 -PurgeContent <String[]> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByObjectParameterSet
```
Unpublish-AzCdnEndpointContent -CdnEndpoint <PSEndpoint> -PurgeContent <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Unpublish-AzCdnEndpointContent parancsmag** véglegesen véglegesen elvégz egy Azure Content Delivery Network (CDN) végpont tartalmát.
