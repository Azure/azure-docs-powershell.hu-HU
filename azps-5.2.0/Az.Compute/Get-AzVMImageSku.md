# Get-AzVMImageSku

## SYNOPSIS
VMImage-SKUs-t kap.

## SZINTAXIS

```
Get-AzVMImageSku -Location <String> -PublisherName <String> -Offer <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzVMImageSku** parancsmag VMImage termékváltozatokat kap.

## PÉLDÁK

### 1. példa: VMImage-SKUs-k lekérte
```
PS C:\> Get-AzVMImageSku -Location "Central US" -PublisherName "Fabrikam" -Offer "LinuxServer"
```

Ez a parancs a megadott közzétevő és ajánlat SKUs-ját kapja meg.
