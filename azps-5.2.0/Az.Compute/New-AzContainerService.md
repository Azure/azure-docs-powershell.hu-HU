# New-AzContainerService

## SYNOPSIS
Tárolószolgáltatást hoz létre.

## SZINTAXIS

```
New-AzContainerService [-ResourceGroupName] <String> [-Name] <String> [-ContainerService] <PSContainerService>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzContainerService** parancsmag létrehoz egy tárolószolgáltatást.
Adjon meg egy tárolószolgáltatás-objektumot, amelyet létrehozhat a New-AzContainerServiceConfig parancsmag használatával.

## PÉLDÁK

### 1. példa: Tárolószolgáltatás létrehozása
```
PS C:\> New-AzResourceGroup -Name "ResourceGroup17" -Location "East US" -Force
PS C:\> $Container = New-AzContainerServiceConfig -Location "East US" -OrchestratorType "DCOS" -MasterDnsPrefix "MasterResourceGroup17" -AdminUsername "acslinuxadmin" -SshPublicKey "<ssh-key>" | Add-AzContainerServiceAgentPoolProfile -Name "AgentPool01" -VmSize "Standard_A1" -DnsPrefix "APResourceGroup17" -Count 2
PS C:\> New-AzContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17" -ContainerService $Container
```

Az első parancs létrehoz egy ResourceGroup17 nevű erőforráscsoportot a megadott helyen.
További információért lásd a New-AzResourceGroup parancsmagot.
A második parancs létrehoz egy tárolót, majd a $Container tárolja.
További információért lásd a New-AzContainerServiceConfig parancsmagot.
Az utolsó parancs létrehoz egy tárolószolgáltatást a $Container.
A szolgáltatás neve csResourceGroup17.
