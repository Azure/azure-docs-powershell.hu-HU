# Test-AzContainerRegistryNameAvailability

## SYNOPSIS
Egy tároló beállításjegyzék-nevének elérhetőségét ellenőrzi.

## SZINTAXIS

```
Test-AzContainerRegistryNameAvailability [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A Test-AzContainerRegistryNameAvailability parancsmag ellenőrzi, hogy a tároló beállításjegyzékének neve érvényes-e és használható-e.

## PÉLDÁK

### 1. példa: A tároló beállításjegyzék-nevének elérhetőségét ellenőrzi
```powershell
PS C:\>Test-AzContainerRegistryNameAvailability -Name 'SomeRegistryName'

NameAvailable Reason Message
------------- ------ -------
         True
```

Ez a parancs ellenőrzi, hogy rendelkezésre áll-e a \` SomeRegistryName nevű tárolóadatbázisnév. \`
