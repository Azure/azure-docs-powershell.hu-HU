# Get-AzDataFactoryGatewayAuthKey

## SYNOPSIS
Átjáró-hitelesítési kulcsot kap egy Azure Data Factoryhoz.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Get-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDataFactoryGatewayAuthKey** parancsmag átjárókulcsot kap egy megadott Azure Data Factory-átjáróhoz.
Ezzel a hitelesítési kulcssal regisztrálhatja az átjárót egy felhőszolgáltatással.

## PÉLDÁK

       gGHqUg1THLqA=
Key2 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@kFXxBdFCEBeL7LPB3hA3LqLd1uNFbyv
       YmWxtV4WD3JQ=

Ez a parancs átjáró-hitelesítési kulcsot kap a MyGateway nevű adat gyári átjáróhoz.
