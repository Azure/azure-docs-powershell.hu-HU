# Test-AzDataLakeStoreAccount

## SYNOPSIS
Teszteli egy Data Lake Store-fiók meglétét.

## SZINTAXIS

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Test-AzDataLakeStoreAccount** parancsmag teszteli a Data Lake Store-fiók meglétét.

## PÉLDÁK

### 1. példa: Fiók tesztelése
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Ez a parancs teszteli, hogy létezik-e a ContosoADL nevű fiók.
