# New-AzDataMigrationService

## SYNOPSIS
Létrehozza az Azure Database Migration Service új példányát.

## SZINTAXIS

```
New-AzDataMigrationService -ResourceGroupName <String> -Name <String> -Location <String> -Sku <String>
 -VirtualSubnetId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A New-AzDataMigrationService parancsmag létrehoz egy új példányt az Azure Adatbázis-áttelepítési szolgáltatásból. Ez a parancsmag a meglévő Azure Resource Group nevet veszi fel, a létrehozni szükséges Azure Adatbázis-áttelepítési szolgáltatás új példányának egyedi nevét, a példány kiépítésének régióját, a DMS Worker termékváltozat nevét, valamint annak az Azure Virtuális alhálózatnak a nevét, amelyen a szolgáltatás található. Az előfizetés nevéhez nincs paraméter, mert a felhasználónak meg kell adnia az Azure bejelentkezési munkamenet alapértelmezett előfizetését, vagy végre kell hajtania a Get-AzSubscription -SubscriptionName "MySubscription" | Select-AzSubscription másik előfizetést.

## PÉLDÁK

### 1. példa
```
PS C:\> New-AzDataMigrationService -ResourceGroupName myResourceGroup -Name TestService -Location "Central US" -Sku Basic_2vCores -VirtualSubnetId $virtualSubNetId
```

A fenti példa bemutatja, hogy miként hozhat létre egy TestService nevű Azure Database Migration Service-példányt a közép-amerikai régióban.
