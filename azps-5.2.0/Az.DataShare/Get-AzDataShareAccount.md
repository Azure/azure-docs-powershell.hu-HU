# Get-AzDataShareAccount

## SYNOPSIS
Információkat kap a DataShare-fiókokról

## SZINTAXIS

### ByFieldsParameterSet (alapértelmezett)
```
Get-AzDataShareAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroupParameterSet
```
Get-AzDataShareAccount -ResourceGroupName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareAccount -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDataShareAccount** parancsmag információkat kap egy Azure-előfizetés vagy -erőforráscsoport adatmegosztási fiókjairól.
Ha megadja egy fiók nevét, ez a parancsmag információt kap a datshare-fiókról.
Ha nem ad meg nevet, ez a parancsmag információt kap egy Azure-előfizetés/erőforráscsoport összes megosztott fiókáról.
