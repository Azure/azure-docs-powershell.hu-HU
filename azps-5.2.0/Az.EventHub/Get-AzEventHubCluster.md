# Get-AzEventHubCluster

## SYNOPSIS
Lekérte egyetlen eseményközpont-fürt adatait, vagy lekérte az Eseményközpont-fürtök listáját.

## SZINTAXIS

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A Get-AzEventHubCluster parancsmag vagy az Eseményközpont-fürt adatait, vagy az eseményközpont-fürtök listáját adja vissza az adott erőforráscsoportban.
Ha meg van téve a fürt neve, a visszaadott érték egyetlen fürt részletei.
Ha nem adja meg a fürt nevét, a megadott erőforráscsoport összes fürtének listáját adja vissza a visszaadott érték.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzEventHubCluster -ResourceGroupName RSG-Cluster27651 -Name Eventhub-Cluster-5557

            ntHub/clusters/Eventhub-Cluster-5557
Name      : Eventhub-Cluster-5557
Location  : southcentralus
CreatedAt : 09/10/2020 22:09:57
UpdatedAt : 09/10/2020 23:02:48
MetricId  :
Status    :
Sku       : Microsoft.Azure.Commands.EventHub.Models.PSEventHubsClusterSkuAttributes
Tags      : {[ClusterTag2, Tag4], [ClusterTag1, Tag3]}

```

Az "RSG-Cluster27651" erőforráscsoport "Eventhub-Cluster-5557" fürtének detialsát adja eredményül.
