# Get-AzEventHubNamespace

## SYNOPSIS
Lekérte egy Eseményközpont névterének adatait, vagy lekérte az aktuális Azure-előfizetés összes Event Hubs névterét.

## SZINTAXIS

```
Get-AzEventHubNamespace [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Get-AzEventHubNamespace parancsmag vagy egy megadott Event Hubs névtér adatait, vagy az aktuális Azure-előfizetés összes Eseményközpont névterének listáját kapja.
Ha meg van biztosítanak egy névteret, a visszaadott érték egyetlen Eseményközpont névterének adatait tartalmazza.
Ha a névtér neve nem szerepel a listában, a visszaadott névterek listája fog eredményül adni.
