# Get-AzIotHub

## SYNOPSIS
Információkat kap az IotHubsról egy előfizetésben.

## SZINTAXIS

### ListIotHubsByResourceGroup (alapértelmezett)
```
Get-AzIotHub [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetIotHubByName
```
Get-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
Információkat kap az IotHubsról egy előfizetésben.
Megtekintheti egy előfizetés összes IotHub-példányát, vagy szűrheti az eredményeket egy erőforráscsoport vagy egy adott IotHub-név szerint.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzIotHub
```

Az előfizetésben található összes IotHubot beveszi.

### 2. példa
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup"
```

Beveszi az előfizetés összes IotHubját, amely a "myresourcegroup" nevű erőforráscsoporthoz tartozik.

### 3. példa
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Információkat kap a "myiothub" nevű IotHubról.
