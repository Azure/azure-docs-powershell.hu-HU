# Get-AzKeyVaultCertificateContact

## SYNOPSIS
Egy kulcstároló tanúsítványértesítéseihez regisztrált névjegyeket kap.

## SZINTAXIS

### VaultName (alapértelmezett)
```
Get-AzKeyVaultCertificateContact [-VaultName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByInputObject
```
Get-AzKeyVaultCertificateContact [-InputObject] <PSKeyVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceId
```
Get-AzKeyVaultCertificateContact [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzKeyVaultCertificateContact parancsmag** olyan névjegyeket kap, amelyek egy kulcstároló tanúsítványértesítéseihez vannak regisztrálva az Azure Key Vaultban.

## PÉLDÁK

### 1. példa: Az összes tanúsítvány kapcsolattartója
```powershell
PS C:\> $Contacts = Get-AzKeyVaultCertificateContact -VaultName "Contoso"

Email                   VaultName
-----                   ---------
username@microsoft.com  Contoso
username1@microsoft.com Contoso
```

Ez a parancs a Contoso kulcstárolóban lévő tanúsítványobjektumok összes névjegyét bekéri, majd a $Contacts tárolja.
