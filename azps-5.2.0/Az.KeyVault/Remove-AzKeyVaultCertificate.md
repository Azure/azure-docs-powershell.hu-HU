# Remove-AzKeyVaultCertificate

## SYNOPSIS
Eltávolít egy tanúsítványt egy kulcstárolóból.

## SZINTAXIS

### ByVaultNameAndName (alapértelmezett)
```
Remove-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String> [-InRemovedState] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObject
```
Remove-AzKeyVaultCertificate [-InputObject] <PSKeyVaultCertificateIdentityItem> [-InRemovedState] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzKeyVaultCertificate** parancsmag eltávolítja a tanúsítványt a kulcstárolóból.

## PÉLDÁK

### 1. példa: Tanúsítvány eltávolítása

Ez a parancs eltávolítja a SelfSigned01 nevű tanúsítványt a ContosoKV01 nevű kulcstárolóból.

### 2. példa: A törölt tanúsítvány végleges törlése a kulcstárolóból
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName 'Contoso' -Name 'MyCert' -InRemovedState
```

Ez a parancs véglegesen eltávolítja a "MyCert" nevű tanúsítványt a "Contoso" kulcstárolóból.
A parancsmag végrehajtatása "végleges végleges" engedélyt igényel, amelyet korábban és kifejezetten a kulcstárban adott meg a felhasználónak.
