# Restore-AzKeyVaultCertificate

## SYNOPSIS
Visszaállít egy tanúsítványt egy biztonsági másolatból egy kulcstárolóban.

## SZINTAXIS

### ByVaultName (alapértelmezett)
```
Restore-AzKeyVaultCertificate [-VaultName] <String> [-InputFile] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Restore-AzKeyVaultCertificate [-InputObject] <PSKeyVault> [-InputFile] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Restore-AzKeyVaultCertificate [-ResourceId] <String> [-InputFile] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Restore-AzKeyVaultCertificate** parancsmag egy biztonsági másolatból létrehoz egy tanúsítványt a megadott kulcstárolóban.
Ez a tanúsítvány a bemeneti fájlban található biztonsági másolat másolata, és a neve megegyezik az eredeti tanúsítvánnyal.
Ha a kulcstároló már tartalmaz egy azonos nevű tanúsítványt, a parancsmag nem tudja felülírni az eredeti tanúsítványt.
Ha a biztonsági másolat egy tanúsítvány több verzióját tartalmazza, a rendszer az összes verziót visszaállítja.
A kulcstároló, amelybe visszaállítja a tanúsítványt, eltérhet attól a kulcstárolótól, amelyről a tanúsítványt biztonsági másolatban tárolta.
A kulcstárnak azonban ugyanazt az előfizetést kell használnia, és ugyanabban a földrajzi régióban (például Észak-Amerikában) kell lennie egy Azure-régióban.
Az Azure-régiók földrajzokkal való megfeleltetését a Microsoft Azure Trust Center https://azure.microsoft.com/support/trust-center/) webhelyén tudja meg.
