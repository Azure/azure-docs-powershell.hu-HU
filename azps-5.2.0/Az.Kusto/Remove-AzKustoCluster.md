# Remove-AzKustoCluster

## SYNOPSIS
Egy Kusto-fürt törlése.

## SZINTAXIS

### Törlés (alapértelmezett)
```
Remove-AzKustoCluster -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzKustoCluster -InputObject <IKustoIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## LEÍRÁS
Egy Kusto-fürt törlése.

## PÉLDÁK

### 1. példa: Meglévő Kusto-fürt törlése név szerint
```powershell
PS C:\> Remove-AzKustoCluster -ResourceGroupName testrg -Name testnewkustocluster
```

A fenti parancs törli a "testrg" erőforráscsoportban a "testnewkustocluster" nevű Kusto-fürtöt.
