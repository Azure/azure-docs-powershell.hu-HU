# Remove-AzIntegrationAccountSchema

## SYNOPSIS
Eltávolít egy integrációs fióksémát.

## SZINTAXIS

```
Remove-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzIntegrationAccountSchema** parancsmag eltávolítja az integrációs fióksémát egy erőforráscsoportból.
Megadhatja az integrációs fiók nevét, az erőforráscsoport nevét és a séma nevét.
Ez a modul támogatja a dinamikus paramétereket.
Ha dinamikus paramétert használ, írja be a parancsba.
A dinamikus paraméterek nevének felfedezéséhez írjon be egy kötőjelet (-) a parancsmag neve után, majd a Tab billentyű többszöri lenyomása után lépkedhet a rendelkezésre álló paraméterek között.
Ha kihagy egy kötelező sablonparamétert, a parancsmag rákérdez az értékre.

## PÉLDÁK

### 1. példa: Integrációs fióksémának eltávolítása
```
PS C:\>Remove-AzIntegrationAccountSchema -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -SchemaName "IntegrationAccountSchema43"
```

Ez a parancs eltávolítja az IntegrationAccountSchema43 nevű integrációs fióksémát.
