# Export-AzMlWebService

## SYNOPSIS
A webszolgáltatás-definíciós objektumot JSON formátumú karakterláncként exportálja.

## SZINTAXIS

### ExportToFile
```
Export-AzMlWebService -WebService <WebService> -OutputFile <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ExportToJSON
```
Export-AzMlWebService -WebService <WebService> [-ToJsonString] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A megadott webszolgáltatás definíciós objektumát JSON formátumú karakterláncként exportálja.
A karakterláncot azonnal vissza is használhatja, vagy mentheti egy fájlba.

## PÉLDÁK

### 1. példa: Exportálás karakterláncként
```
Export-AzMlWebService -WebService $svc -ToJsonString
```

### 2. példa: Exportálás fájlba
```
Export-AzMlWebService -WebService $svc -OutputFile "C:\mlservice.json"
```
