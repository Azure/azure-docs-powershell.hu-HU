# New-AzUserAssignedIdentity

## SYNOPSIS
Létrehoz egy új felhasználóhoz rendelt identitást, vagy frissíti a meglévő felhasználóhoz rendelt identitást.

## SZINTAXIS

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzUserAssignedIdentity** parancsmag létrehoz egy új felhasználóhoz rendelt identitást. Ha egy már meglévő identitással használja, frissítette az identitást.
Ha Azure Resource Manager-címkéket szeretne hozzáadni az identitáshoz, használja a Set-AzResource parancsmagot.
