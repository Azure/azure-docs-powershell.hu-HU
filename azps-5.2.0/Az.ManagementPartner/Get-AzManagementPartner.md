# Get-AzManagementPartner

## SYNOPSIS
Az aktuális hitelesített felhasználó vagy egyszerű szolgáltatásnév Microsoft Partner Network(MPN)-azonosítóját kapja meg.

## SZINTAXIS

```
Get-AzManagementPartner [[-PartnerId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Az aktuális hitelesített felhasználó vagy egyszerű szolgáltatásnév Microsoft Partner Network(MPN)-azonosítóját kapja meg.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzManagementPartner
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

A jelenlegi felügyeleti partnerazonosító lekérte

### 2. példa
```powershell
PS C:\> Get-AzManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Szerezze be a jelenlegi felügyeleti partnerazonosítót egy partnerazonosító használatával, ha nem egyeznek, az nem fog sikerülni.
