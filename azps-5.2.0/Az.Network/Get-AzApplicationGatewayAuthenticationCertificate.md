# Get-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Hitelesítési tanúsítványt kap egy alkalmazás-átjáróhoz.

## SZINTAXIS

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayAuthenticationCertificate parancsmag** kap egy hitelesítési tanúsítványt egy Azure-alkalmazás-átjáróhoz.

## PÉLDÁK

### 1. példa: Megadott hitelesítési tanúsítvány lekérve
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $pool = Get-AzApplicationGatewayBackendAddressPool -Name "pool01" -ApplicationGateway $appgw
```

Az első parancs lekérte az appGwName nevű alkalmazás-átjárót, és a $appgw tárolja.
A második parancs lekéri a pool01 nevű hitelesítési tanúsítványt, és a $pool tárolja.
