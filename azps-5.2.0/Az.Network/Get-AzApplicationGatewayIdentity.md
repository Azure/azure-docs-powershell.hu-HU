# Get-AzApplicationGatewayIdentity

## SYNOPSIS
Identitás hozzárendelése az alkalmazás-átjáróhoz.

## SZINTAXIS

```
Get-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayIdentity** parancsmag identitást kap az alkalmazás-átjáróhoz rendelve.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzApplicationGatewayIdentity -ApplicationGateway $gw
```

Az alábbi példák azt mutatják be, hogy miként szerez be alkalmazás-átjáró identitást az Alkalmazás átjáróból.
