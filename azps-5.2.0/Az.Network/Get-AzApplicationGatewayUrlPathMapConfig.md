# Get-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
URL-útvonal-megfeleltetések tömbjét kapja vissza egy háttérkiszolgáló-készlethez.

## SZINTAXIS

```
Get-AzApplicationGatewayUrlPathMapConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayURLPathMapConfig** parancsmag URL-útvonal-megfeleltetéseket tartalmazó tömböt kap egy háttérkiszolgáló-készlethez.

## PÉLDÁK

### 1. példa: URL-útvonal-leképezés beállítása
```
PS C:\>Get-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway Gateway
```

Ez a parancs az URL-útvonal-leképezés beállításait az Átjáró nevű alkalmazás-átjárón található háttérkiszolgálóról kapja meg.
