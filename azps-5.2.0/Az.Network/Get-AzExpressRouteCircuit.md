# Get-AzExpressRouteCircuit

## SYNOPSIS
Azure ExpressRoute-áramkört kap az Azure-tól.

## SZINTAXIS

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzExpressRouteCircuit** parancsmaggal lekérhető egy ExpressRoute-áramköri objektum az előfizetéséből. A visszaadott áramköri objektum használható bemenetként más, ExpressRoute-áramkörön működő parancsmagok számára.
