# Get-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Háttércímkészlet-konfigurációt kap a terheléselosztáshoz.

## SZINTAXIS

```
Get-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzLoadBalancerBackendAddressPoolConfig** parancsmag egyetlen háttércímkészletet vagy a háttércímkészletek listáját kapja egy terheléselegyenítőn belül.

## PÉLDÁK

### 1. példa: A háttércímkészlet lekérte
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" -LoadBalancer $loadbalancer
```

Az első parancs egy MyLoadBalancer nevű meglévő terhelésegyenleg-kezelőt kap a MyResourceGroup nevű erőforráscsoportban, majd tárolja azt a $loadbalancer változóban.
A második parancs a backend-címkészlet BackendAddressPool02 nevű konfigurációját kapja meg a $loadbalancer.
