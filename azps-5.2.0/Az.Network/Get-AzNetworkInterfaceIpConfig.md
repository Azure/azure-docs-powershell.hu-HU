# Get-AzNetworkInterfaceIpConfig

## SYNOPSIS
Hálózati felület IP-konfigurációját használja a hálózati felülethez.

## SZINTAXIS

```
Get-AzNetworkInterfaceIpConfig [-Name <String>] -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzNetworkInterfaceIPConfig** parancsmag egy Hálózati felület IP-konfigurációt kap egy Azure hálózati felületről.

## PÉLDÁK

### 1: Hálózati felület IP-konfigurációjának beállítása
```
$nic1 = Get-AzNetworkInterface -Name mynic -ResourceGroupName $myrg
Get-AzNetworkInterfaceIpConfig -Name ipconfig1 -NetworkInterface $nic1
```

Az első parancs egy mynic nevű meglévő hálózati felületet kap, és a $nic 1 változóban tárolja. A második parancs a hálózati felület ipconfig1 ipconfig1 nevű IP-konfigurációját kapja meg.
