# New-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Automatikus méretezésű konfigurációt hoz létre az alkalmazás-átjáróhoz.

## SZINTAXIS

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzApplicationGatewayAutoscaleConfiguration parancsmag** létrehoz egy automatikus skálás konfigurációt egy Azure-alkalmazás-átjáróhoz.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Az első parancs létrehoz egy 3-as kapacitású automatikus skálázási konfigurációt.
A második parancs létrehoz egy automatikus skálázási konfigurációval konfigurált alkalmazás-átjárót.
