# New-AzApplicationGatewayFirewallDisabledRuleGroupConfig

## SYNOPSIS
Új letiltott szabálycsoport-konfigurációt hoz létre.

## SZINTAXIS

```
New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName <String> [-Rules <Int32[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzApplicationGatewayFirewallDisabledRuleGroupConfig** parancsmag létrehoz egy új letiltott szabálycsoport-konfigurációt.

## PÉLDÁK

### 1. példa
```
PS C:\> $disabledRuleGroup1 = New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName "REQUEST-942-APPLICATION-ATTACK-SQLI" -Rules 942130,942140
```

A parancs létrehoz egy új letiltott szabálycsoport-konfigurációt a REQUEST-942-APPLICATION-ATTACK-SQLI nevű szabálycsoporthoz, a 942130-as és a 942140-es szabály letiltva. Az új letiltott szabálycsoport-konfigurációt a $disabledRuleGroup 1.
