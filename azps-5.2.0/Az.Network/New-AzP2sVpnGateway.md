# New-AzP2sVpnGateway

## SYNOPSIS
Hozzon létre egy új P2SVpnGatewayt a VirtualHub alatt, és mutasson a webhelyre.

## LEÍRÁS
A **New-AzP2sVpnGateway** parancsmag lehetővé teszi, hogy új P2SVpnGatewayt hozzon létre a VirtualHub alatt, hogy a Point to site connectivity-ból a site clients to Azure VirtualWan alkalmazásba tudjanak áttérni.
