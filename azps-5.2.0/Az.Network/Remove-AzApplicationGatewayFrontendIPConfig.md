# Remove-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Eltávolítja az előlapi IP-konfigurációt egy alkalmazás-átjáróból.

## SZINTAXIS

```
Remove-AzApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApplicationGatewayFrontendIPConfig** parancsmag eltávolítja a frontend IP-címet egy Azure-alkalmazás átjáróból.

## PÉLDÁK

### 1. példa: Előlapi IP-konfiguráció eltávolítása
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

Az első parancs egy ApplicationGateway01 nevű alkalmazás-átjárót kap, és azt a $AppGw tárolja.
A második parancs eltávolítja a FrontEndIP02 nevű előlapi IP-konfigurációt a $AppGw.
