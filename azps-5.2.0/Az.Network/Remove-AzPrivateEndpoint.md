# Remove-AzPrivateEndpoint

## SYNOPSIS
Eltávolít egy privát végpontot.

## SZINTAXIS

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzPrivateEndpoint** parancsmag eltávolít egy privát végpontot. 

## PÉLDÁK

### 1. példa
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

Ebben a példában eltávolítunk egy MyPrivateEndpoint1 nevű privát végpontot.
