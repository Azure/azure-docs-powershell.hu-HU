# Get-AzLegacyPeering

## SYNOPSIS
Régi társviszony-kezelő erőforrások azure-erőforrás-kezelési (ARM) típusú erőforrásokká alakítása. 

## SZINTAXIS

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A parancs a régi társviszony-létesítő erőforrások megtekintésére szolgál, amelyeket Ön mindössze Azure Resource Management (ARM) erőforrásokká konvertál.
