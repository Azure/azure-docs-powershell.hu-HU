# Get-AzRedisCache

## SYNOPSIS
Redis gyorsítótárat kap.

## SZINTAXIS

```
Get-AzRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzRedisCache** parancsmag megkapja a megadott Azure Redis-gyorsítótárat.
Ha nem ad meg paramétereket, ez a művelet az aktuális előfizetés összes redis-gyorsítótárát megkapja.

## PÉLDÁK

### 1. példa: Redis Cache lekért név szerint

```
        Name               : myexists
        Type               : Microsoft.Cache/Redis
        HostName           : myexists.redis.cache.windows.net
        Port               : 6379
        ProvisioningState  : succeeded
        SslPort            : 6380
        RedisConfiguration : {}
        EnableNonSslPort   : False
        RedisVersion       : 2.8
        Size               : 1GB
        Sku                : Basic
        Tag                : {}
        Zone               : []
```

Ez a parancs a Myexists nevű Redis Cache-t kapja meg.
