# Reset-AzRedisCache

## SYNOPSIS
Egy gyorsítótár csomópontjainak újraindítása.

## SZINTAXIS

```
Reset-AzRedisCache [-ResourceGroupName <String>] -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Reset-AzRedisCache** parancsmag újraindítja egy Azure Redis Cache-példány csomópontját.

## PÉLDÁK

### 1. példa: Mindkét csomópont újraindítása
```
PS C:\>Reset-AzRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

Ez a parancs újraindítja mindkét csomópontot a RedisCache06 nevű gyorsítótárban.
