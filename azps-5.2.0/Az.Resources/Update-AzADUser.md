# Update-AzADUser

## SYNOPSIS
Egy meglévő Active Directory-felhasználó frissítése.

## SZINTAXIS

### UPNOrObjectIdParameterSet (alapértelmezett)
```
Update-AzADUser -UPNOrObjectId <String> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UPNParameterSet
```
Update-AzADUser -UserPrincipalName <String> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Update-AzADUser -ObjectId <String> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzADUser -InputObject <PSADUser> [-DisplayName <String>] [-EnableAccount <Boolean>]
 [-Password <SecureString>] [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Egy meglévő Active Directory-felhasználó (munkahelyi/iskolai fiók, más néven szervezeti azonosító) frissítése.
További információ: https://msdn.microsoft.com/en-us/library/azure/ad/graph/api/users-operations#UpdateUser

## PÉLDÁK

### 1. példa: Felhasználó megjelenítendő nevének frissítése objektumazonosító használatával

```powershell
PS C:\> Update-AzADUser -ObjectId 155a5c10-93a9-4941-a0df-96d83ab5ab24 -DisplayName MyNewDisplayName
```

Frissíti a felhasználó megjelenítendő nevét a (155a5c10-93a9-4941-a0df-96d83ab5ab24) azonosítóval "MyNewDisplayName" azonosítóra.

### 2. példa: Felhasználó megjelenített nevének frissítése egyszerű felhasználónévvel

```powershell
PS C:\> Update-AzADUser -UserPrincipalName foo@domain.com -DisplayName MyNewDisplayName
```

Frissíti a felhasználó megjelenítendő nevét egyszerű felhasználónévvel ' ' a foo@domain.com "MyNewDisplayName" névre.

### 3. példa: Felhasználó megjelenítendő nevének frissítése pipázás használatával

```powershell
PS C:\> Get-AzADUser -ObjectId 155a5c10-93a9-4941-a0df-96d83ab5ab24 | Update-AzADUser -DisplayName MyNewDisplayName
```

A (155a5c10-93a9-4941-a0df-96d83ab5ab24) objektumazonosítóval és a Update-AzADUser-parancsmaghoz a felhasználó megjelenítendő nevének "MyNewDisplayName" névre való frissítéséhez bevezető vezetékeket kapja.
