# Az.Search Module
## Leírás
Ez a témakör az Azure Search parancsmagok súgótémakörét mutatja be.

## Az.Search parancsmagok
### [Get-AzSearchAdminKeyPair](Get-AzSearchAdminKeyPair.md)
Az Azure Search szolgáltatás rendszergazdai kulcspárt kap.

### [Get-AzSearchQueryKey](Get-AzSearchQueryKey.md)
Lekérdezi az Azure Search szolgáltatás lekérdezéskulcsát.

### [Get-AzSearchService](Get-AzSearchService.md)
Azure Search szolgáltatást kap.

### [New-AzSearchAdminKey](New-AzSearchAdminKey.md)
Az Azure Search szolgáltatás rendszergazdai kulcsának újragenerálása.

### [New-AzSearchQueryKey](New-AzSearchQueryKey.md)
Hozzon létre egy új lekérdezéskulcsot az Azure Search szolgáltatáshoz.

### [New-AzSearchService](New-AzSearchService.md)
Létrehoz egy Azure Search-szolgáltatást.

### [Remove-AzSearchQueryKey](Remove-AzSearchQueryKey.md)
Távolítsa el a lekérdezéskulcsot az Azure Search szolgáltatásból.

### [Remove-AzSearchService](Remove-AzSearchService.md)
Távolítson el egy Azure Search-szolgáltatást.

### [Set-AzSearchService](Set-AzSearchService.md)
Frissítsen egy Azure Search-szolgáltatást.
