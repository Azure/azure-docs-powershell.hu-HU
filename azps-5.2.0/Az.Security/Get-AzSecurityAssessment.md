# Get-AzSecurityAssessment

## SYNOPSIS
Biztonsági felméréseket és azok eredményeit kapja meg egy előfizetésben

## SZINTAXIS

### SubscriptionScope (alapértelmezett)
```
Get-AzSecurityAssessment [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzSecurityAssessment -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdScope
```
Get-AzSecurityAssessment -Name <String> -AssessedResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceId
```
Get-AzSecurityAssessment -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Biztonsági felmérést és az előfizetéssel kapcsolatos eredményeket kapja. A biztonsági felmérésekből tudható, hogy mely ajánlott eljárásokat ajánlotta újra az Azure Biztonsági központ, hogy az Azure-előfizetését mérsékeljük.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzSecurityAssessment
```

Az előfizetés összes biztonsági felmérésének lekérte
