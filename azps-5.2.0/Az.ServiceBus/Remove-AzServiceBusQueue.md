# Remove-AzServiceBusQueue

## SYNOPSIS
Eltávolítja a várólistát a szolgáltatásbusz megadott névterében.

## SZINTAXIS

### QueuePropertiesSet (alapértelmezett)
```
Remove-AzServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String> [-PassThru]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### QueueInputObjectSet
```
Remove-AzServiceBusQueue [-InputObject] <PSQueueAttributes> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### QueueResourceIdSet
```
Remove-AzServiceBusQueue [-ResourceId] <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzServiceBusQueue** parancsmag eltávolítja a várólistát a szolgáltatásbusz megadott névterében.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Remove-AzServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -QueueName SB-Queue_exampl1
```

Eltávolítja a Service Bus `SB-Queue_exampl1` várólistát a névtérből. `SB-Example1`

### 2. példa: InputObject – Változó használata:
```powershell
PS C:\> $inputobject = Get-AzServiceBusQueue <params>
PS C:\> Remove-AzServiceBusQueue -InputObject $inputobject
```

Az -InputObject paraméterhez megadott $inputobject eltávolítása

### 3. példa: InputObject – Piping használata:
```powershell
PS C:\>  Get-AzServiceBusQueue <params> | Remove-AzServiceBusQueue
```

### 4. példa: ResourceId – Változó használata:
```powershell
PS c:\> $resourceid = Get-AzServiceBusQueue <params>
PS C:\> Remove-AzServiceBusQueue -ResourceId $resourceid.Id
```

Eltávolítja a service bus queue provided in the ARM id in $resourceid/string for -ResourceId parameter

### 5. példa: ResourceId – karakterláncként halad:
```powershell
PS C:\> Remove-AzServiceBusQueue -ResourceId "/subscriptions/xxxx-xxxxx-xxxxx-xxxxxx-xxxxxxx/resourceGroups/ResourceGroupName/providers/Microsoft.ServiceBus/namespaces/NamespaceName/queues/QueueName"
```
