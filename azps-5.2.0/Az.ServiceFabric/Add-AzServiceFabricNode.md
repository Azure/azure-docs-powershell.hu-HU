# Add-AzServiceFabricNode

## SYNOPSIS
Csomópontok hozzáadása a fürt adott csomóponttípushoz.

## SZINTAXIS

```
Add-AzServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Add-AzServiceFabricNode** segítségével csomópontokat adhat az adott csomóponttípushoz. Csak meg kell adnia a csomóponttípushoz hozzáadni kívánt csomópontok számát.

## PÉLDÁK

### 1. példa
```powershell
PS c:> Add-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Ez a parancs 2 csomópontot ad az "n1" csomóponttípushoz.
