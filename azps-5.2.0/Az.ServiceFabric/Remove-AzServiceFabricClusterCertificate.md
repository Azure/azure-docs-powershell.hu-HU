# Remove-AzServiceFabricClusterCertificate

## SYNOPSIS
Távolítsa el a fürt tanúsítványát a fürtbiztonság érdekében.

## SZINTAXIS

```
Remove-AzServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzServiceFabricClusterCertificate** segítségével eltávolíthat egy fürttanúsítványt a fürtből, ha van egy másik érvényes tanúsítvány, amely már használatban van a fürtben.

## PÉLDÁK

### 1. példa
```
PS C:\> Remove-AzServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Ez a parancs eltávolítja az 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A tanúsítványt a fürtbiztonság érdekében.
