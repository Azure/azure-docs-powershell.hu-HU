# Get-AzSqlElasticPool

## SYNOPSIS
Rugalmas készleteket és azok tulajdonságértékeket kap egy Azure SQL-adatbázisban.

## SZINTAXIS

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlElasticPool** parancsmag rugalmas készleteket és azok tulajdonságértékét kapja meg.
Adja meg egy meglévő rugalmas készlet nevét, hogy csak az adott készlet tulajdonságértékét lássa.
