# Get-AzSqlInstanceDatabaseGeoBackup

## SYNOPSIS
Az Azure SQL Felügyelt példány adatbázis redundáns biztonsági másolatának adatait adja eredményül.

## SZINTAXIS

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlInstanceDatabaseGeoBackup** parancsmag egy vagy több Azure SQL-adatbázisból redundáns biztonsági másolatot kap egy Azure SQL-adatbázis felügyelt példányából.
