# Remove-AzSqlSyncAgent

## SYNOPSIS
Eltávolít egy Azure SQL Sync-ügynököt.

## SZINTAXIS

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzSqlSyncAgent** parancsmag eltávolít egy Azure SQL Sync-ügynököt.

## PÉLDÁK

### 1. példa: Szinkronizálási ügynök eltávolítása
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Ez a parancs eltávolítja a syncAgent01 nevű Azure SQL Sync Agentt.
