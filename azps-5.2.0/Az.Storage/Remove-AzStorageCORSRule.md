# Remove-AzStorageCORSRule

## SYNOPSIS
Eltávolítja egy társzolgáltatás CORS-ját.

## SZINTAXIS

```
Remove-AzStorageCORSRule [-ServiceType] <StorageServiceType> [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStorageCORSRule** parancsmag eltávolítja a több forrásból való erőforrás-megosztást (CORS) egy Azure Storage-szolgáltatásban.
Ez a parancsmag törli a társzolgáltatás-típus összes CORS-szabályát.
A parancsmag társzolgáltatásának típusai a blob, a tábla, a várólista és a fájl.

## PÉLDÁK

### 1. példa: A BLOB-szolgáltatás CORS-szabályainak eltávolítása
```
PS C:\>Remove-AzStorageCORSRule -ServiceType Blob
```

Ez a parancs eltávolítja a Blob szolgáltatástípus CORS-szabályait.
