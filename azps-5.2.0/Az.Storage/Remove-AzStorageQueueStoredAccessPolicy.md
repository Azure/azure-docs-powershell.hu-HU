# Remove-AzStorageQueueStoredAccessPolicy

## SYNOPSIS
Egy tárolt hozzáférési házirend eltávolítása egy Azure-tárterület-várólistáról.

## SZINTAXIS

```
Remove-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStorageQueueStoredAccessPolicy** parancsmag eltávolít egy tárolt hozzáférési házirendet egy Azure-tárterület-várólistáról.

## PÉLDÁK

### 1. példa: Tárolt hozzáférési házirend eltávolítása egy társorból
```
PS C:\>Remove-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy04"
```

Ez a parancs eltávolítja a Policy04 nevű hozzáférési házirendet a MyQueue nevű társorból.
