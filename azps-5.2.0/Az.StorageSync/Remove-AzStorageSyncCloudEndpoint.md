# Remove-AzStorageSyncCloudEndpoint

## SYNOPSIS
Ez a parancs törli a megadott felhőbeli végpontot egy szinkronizálási csoportból. Legalább egy felhőbeli végpont nélkül a szinkronizálási csoport egyetlen másik kiszolgálói végpontja sem tud szinkronizálni.

## SZINTAXIS

### StringParameterSet (alapértelmezett)
```
Remove-AzStorageSyncCloudEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-InputObject] <PSCloudEndpoint> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncCloudEndpoint [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Ez a parancs törli a megadott felhőbeli végpontot egy szinkronizálási csoportból. Az Azure-fájl megosztásakor a felhőbeli végponthivatkozások érintetlenek maradnak a folyamat során. Ez a parancs csak leszerelésre szolgál. A felhőbeli végpontok eltávolítása ártalmas művelet. A kiszolgálói végpontok csak akkor szinkronizálódnak, ha legalább egy felhőbeli végpont jelen van. Ez a művelet nem végezhető el a szinkronizálási problémák megoldásához. Ha ezt az Azure-fájlmegosztást ismét hozzáadják ugyanannak a szinkronizálási csoportnak , mint felhőbeli végpontnak, az ütközési fájlokhoz és más nem kívánt következményekhez vezethet.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Remove-AzStorageSyncCloudEndpoint -Force -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -Name "myCloudEndpointName"
```

Ez a parancs eltávolítja a felhőbeli végpontot.
