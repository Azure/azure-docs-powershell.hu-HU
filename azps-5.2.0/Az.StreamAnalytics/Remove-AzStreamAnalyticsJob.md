# Remove-AzStreamAnalyticsJob

## SYNOPSIS
Eltávolítja a Stream Analytics feladatot.

## SZINTAXIS

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStreamAnalyticsSync** parancsmag aszinkron módon töröl egy adott Stream Analytics-feladatot az Azure-ban.

## PÉLDÁK

### 1. PÉLDA: Feladat eltávolítása
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Ez a parancs eltávolítja a Streaming Streaming Streaming feladatot.
