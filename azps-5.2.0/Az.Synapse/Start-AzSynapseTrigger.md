# Start-AzSynapseTrigger

## SYNOPSIS
Elindít egy eseményindítót egy munkaterületen.

## SZINTAXIS

### StartByName (alapértelmezett)
```
Start-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StartByObject
```
Start-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StartByInputObject
```
Start-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Start-AzSynapseTrigger** parancsmag elindít egy eseményindítót egy munkaterületen. Ha az eseményindító "Leállítva" állapotban van, a parancsmag elindítja az eseményindítót, és végül a definíciója alapján meghívja a prognózisokat. Ha az eseményindító már "Elindítva" állapotban van, ennek a parancsmagnak nincs hatása.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Start-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Elindít egy ContosoTrigger nevű eseményindítót a ContosoWorkspace munkaterületen.

### 2. példa
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Start-AzSynapseTrigger -Name ContosoTrigger
```

Elindít egy ContosoTrigger nevű eseményindítót a contosoworkspace-munkaterületen egy folyamaton keresztül.

### 3. példa
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Start-AzSynapseTrigger
```

Elindít egy ContosoTrigger nevű eseményindítót a contosoworkspace-munkaterületen egy folyamaton keresztül.
