# Remove-AzAppServicePlan

## SYNOPSIS
Eltávolít egy Azure App Service-előfizetést.

## SZINTAXIS

### S1
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-AppServicePlan] <PSAppServicePlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzAppServicePlan** parancsmag eltávolítja az Azure App Service-csomagokat.

## PÉLDÁK

### 1. példa: Appszolgáltatás-csomag eltávolítása
```
PS C:\>Remove-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Ez a parancs eltávolítja a ContosoASP nevű Azure App Service-tervet, amely a Default-Web-WestUS nevű erőforráscsoporthoz tartozik.
