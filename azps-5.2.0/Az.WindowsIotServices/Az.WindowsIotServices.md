# Az.WindowsIotServices modul
## Leírás
Microsoft Azure PowerShell: WindowsIotServices parancsmagok

## Az.WindowsIotServices parancsmagok
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
A Windows IoT-eszközszolgáltatás nem biztonsággal kapcsolatos metaadatainak lekérte.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Létrehozhatja vagy frissítheti egy Windows IoT-eszközszolgáltatás metaadatait.
A tulajdonság módosításának szokásos sémája az, hogy beolvassa a Windows IoT eszközszolgáltatás metaadatait és biztonsági metaadatait, majd kombinálja őket a módosított értékekkel egy új törzsben a Windows IoT eszközszolgáltatás frissítéséhez.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Windows IoT-eszközszolgáltatás törlése

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Frissíti egy Windows IoT-eszközszolgáltatás metaadatait.
A tulajdonság módosításának szokásos sémája az, hogy beolvassa a Windows IoT eszközszolgáltatás metaadatait és biztonsági metaadatait, majd kombinálja őket a módosított értékekkel egy új törzsben a Windows IoT eszközszolgáltatás frissítéséhez.
