# Get-AzEnvironment

## SYNOPSIS
Végpontok és metaadatok lekérte az Azure-szolgáltatások egy példányát.

## SZINTAXIS

```
Get-AzEnvironment [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Get-AzEnvironment parancsmag végpontokat és metaadatokat kap az Azure-szolgáltatások egy példányához.

## PÉLDÁK

### 1. példa: Az AzureCloud környezet beszerzése
```
PS C:\> Get-AzEnvironment AzureCloud

Name       Resource Manager Url          ActiveDirectory Authority
----       --------------------          -------------------------
AzureCloud https://management.azure.com/ https://login.microsoftonline.com/
```

Az alábbi példa bemutatja, hogyan lehet lekértékérték az AzureCloud -környezet végpontját és metaadatait.
