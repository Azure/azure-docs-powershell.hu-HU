# Get-AzSmartGroup

## SYNOPSIS
Intelligens csoportokra vonatkozó információk lekérte

## SZINTAXIS

### SmartGroupsListByFilter (alapértelmezett)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
**A Get-AzSmartGroup** parancsmag intelligens csoportokkal kapcsolatos információkat tartalmaz.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

List all smart groups formed in last 1 hour. A Format-List a lista minden intelligens csoportjának részletes adatait.

### 2. példa
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Intelligens csoport részleteinek lekérte azonosító (GUID) vagy erőforrás-azonosító (Complete ARM Id) szerint
