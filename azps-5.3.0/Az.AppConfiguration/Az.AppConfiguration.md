# Az.AppConfiguration Module
## Leírás
Microsoft Azure PowerShell: AppConfiguration parancsmagok

## Az.AppConfiguration parancsmagok
### [Get-AzAppConfigurationStore](Get-AzAppConfigurationStore.md)
Alkalmazáskonfigurációs áruházak letöltése vagy listába sorolja.

### [Get-AzAppConfigurationStoreKey](Get-AzAppConfigurationStoreKey.md)
A megadott konfigurációs tároló hozzáférési kulcsát sorolja fel.

### [New-AzAppConfigurationStore](New-AzAppConfigurationStore.md)
Létrehoz egy konfigurációs tárolót a megadott paraméterekkel.

### [New-AzAppConfigurationStoreKey](New-AzAppConfigurationStoreKey.md)
A megadott konfigurációs tároló hozzáférési kulcsának újragenerálása.

### [Remove-AzAppConfigurationStore](Remove-AzAppConfigurationStore.md)
Törli a konfigurációs tárolót.

### [Test-AzAppConfigurationStoreNameAvailability](Test-AzAppConfigurationStoreNameAvailability.md)
Annak ellenőrzése, hogy használható-e a konfigurációs tároló neve.

### [Update-AzAppConfigurationStore](Update-AzAppConfigurationStore.md)
A megadott paraméterekkel frissíti a konfigurációs tárolót.
