# Reset-AzAttestationPolicy

## SYNOPSIS
Alaphelyzetbe állítja a házirendet egy bérlői fiókból az Azure Attestationnben.}

## SZINTAXIS

### NameParameterSet
```
Reset-AzAttestationPolicy [-Name] <String> [-ResourceGroupName] <String> -Tee <String> [-Policy <String>]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Reset-AzAttestationPolicy [-ResourceId] <String> -Tee <String> [-Policy <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Reset-AzAttestationPolicy parancsmag alaphelyzetbe állítja a felhasználó által definiált előfizetési házirendet egy bérlői fiókból az Azure Attestation szolgáltatásban.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Reset-AzAttestationPolicy -Name pshtest -ResourceGroupName psh-test-rg -Tee SgxEnclave
```

Állítsa vissza a házirendet az alapértelmezettre az Attestation Provider *pshtest* for Tee type *SgxEnclave* esetén.

### 2. példa
```powershell
PS C:\> $resetJwt = Get-Content -Path .\reset.policy.txt.signed.txt
PS C:\> Reset-AzAttestationPolicy -Name pshtest -ResourceGroupName psh-test-rg -Tee SgxEnclave -Policy $resetJwt
```

Ha az Attestation Provider *pshtest* az különálló megbízhatósági modell használatára van konfigurálva, állítsa vissza a házirendet az alapértelmezettre a Tee típusú *SgxEnclave* esetén egy aláírt házirendet megjelölve.
