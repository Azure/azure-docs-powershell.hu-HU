# Remove-AzAutomationWebhook

## SYNOPSIS
Eltávolít egy webhookot egy automatizálási runbookból.

## SZINTAXIS

```
Remove-AzAutomationWebhook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzAutomationWebhook** parancsmag eltávolítja a webhookot az Azure Automation-runbookból.
A webhook törlődik.

## PÉLDÁK

### 1. példa: Webhook eltávolítása
```
PS C:\>Remove-AzAutomationWebhook -Name "Webhook11" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Force
```

Ez a parancs eltávolítja a Webhook11 nevű webhookot az AutomationAccount01 nevű Automatizálási fiókból.
A parancs a Force *paramétert* adja meg.
Ezért nem kér megerősítést.
