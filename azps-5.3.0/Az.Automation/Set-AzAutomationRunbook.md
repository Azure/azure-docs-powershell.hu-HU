# Set-AzAutomationRunbook

## SYNOPSIS
Módosít egy runbookot.

## SZINTAXIS

```
Set-AzAutomationRunbook [-Name] <String> [-Description <String>] [-Tag <IDictionary>] [-LogProgress <Boolean>]
 [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzAutomationRunbook** parancsmag módosítja egy Azure Automation-runbook konfigurációját az APS-ben.

## PÉLDÁK

### 1. példa: Részletes naplózás engedélyezése a runbookhoz
```
PS C:\>Set-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook02" -LogVerbose $True -ResourceGroupName "ResourceGroup01"
```

Ez a parancs részletes naplózást tesz lehetővé a Contoso17 nevű Azure Automation-fiók megadott runbookján.
