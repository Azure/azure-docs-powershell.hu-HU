# Remove-AzBatchApplication

## SYNOPSIS
Egy alkalmazást töröl egy Batch-fiókból.

## SZINTAXIS

```
Remove-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzBatchApplication** parancsmag töröl egy alkalmazást egy Azure Batch-fiókból.

## PÉLDÁK

### 1. példa: Alkalmazás törlése egy Batch-fiókból
```
PS C:\>Remove-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "Litware"
```

Ez a parancs törli a Litware alkalmazást a ContosoBatch-fiókból.
A parancs nem sikerül, ha az alkalmazás csomagokat tartalmaz.
