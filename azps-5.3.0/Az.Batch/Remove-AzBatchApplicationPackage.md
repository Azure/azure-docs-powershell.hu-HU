# Remove-AzBatchApplicationPackage

## SYNOPSIS
Egy alkalmazáscsomagrekord és a bináris fájl törlése.

## SZINTAXIS

```
Remove-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationName] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzBatchApplicationPackage** parancsmag egy alkalmazáscsomagrekordot és a bináris fájlt töröl egy Azure Batch-fiókból.

## PÉLDÁK

### 1. példa: Alkalmazáscsomag törlése Egy Batch-fiókból
```
PS C:\>Remove-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "litware" -ApplicationVersion "1.0"
```

Ez a parancs törli a Litware alkalmazás 1.0-s verzióját a ContosoBatchGroup-fiókból.
A parancs a csomag bináris fájlját tartalmazó csomagrekordot és blobot is törli.
