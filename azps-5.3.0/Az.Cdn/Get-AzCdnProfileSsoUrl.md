# Get-AzCdnProfileSsoUrl

## SYNOPSIS
Egy CDN-profil egyszeri bejelentkezési URL-címét kapja meg.

## SZINTAXIS

### ByFieldsParameterSet (alapértelmezett)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzCdnProfileSsoUrl** parancsmag megkapja az Azure Content Delivery Network (CDN) profil egyszeri bejelentkezési URL-címét.
Ez az URL lehetővé teszi a felhasználóknak, hogy egy kiegészítő portálhoz csatlakozzon, és a CDN további funkcióit használják.
