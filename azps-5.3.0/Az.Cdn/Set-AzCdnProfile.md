# Set-AzCdnProfile

## SYNOPSIS
Frissíti a CDN-profilt.

## SZINTAXIS

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzCdnProfile** parancsmag frissíti az Azure Content Delivery Network (CDN) profilt.
