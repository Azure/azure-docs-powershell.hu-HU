# Test-AzCdnCustomDomain

## SYNOPSIS
Annak ellenőrzése, hogy hozzáadható-e egy egyéni tartomány egy végponthoz.

## SZINTAXIS

### ByFieldsParameterSet (alapértelmezett)
```
Test-AzCdnCustomDomain -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 -CustomDomainHostName <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Test-AzCdnCustomDomain -CdnEndpoint <PSEndpoint> -CustomDomainHostName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Test-AzCdnCustomDomain** parancsmag ellenőrzi, hogy hozzáadható-e egy egyéni tartomány egy végponthoz a CName leképezés érvényességének igazolásával.
