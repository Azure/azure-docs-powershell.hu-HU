# Update-AzCognitiveServicesAccountNetworkRuleSet

## SYNOPSIS
Kognitív szolgáltatások fiók NetworkRule tulajdonságának frissítése

## SZINTAXIS

```
Update-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultAction <PSNetWorkRuleDefaultActionEnum>] [-IpRule <PSIpRule[]>]
 [-VirtualNetworkRule <PSVirtualNetworkRule[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
Az **Update-AzCognitiveServicesAccountNetworkRuleSet** parancsmag frissíti a Kognitív szolgáltatások fiók NetworkRule tulajdonságát

## PÉLDÁK

### 1. példa: A NetworkRule összes tulajdonságának frissítése, beviteli szabályok a JSON-nal
```
PS C:\> Update-AzCognitiveServicesAccountNetworkRuleSet -ResourceGroupName "myResourceGroup" -Name "myaccount" -DefaultAction Allow -IpRule (@{IpAddressOrRange="200.0.0.0/24"},@{IpAddressOrRange="28.2.0.0/16"})
    -VirtualNetworkRule (@{VirtualNetworkResourceId="/subscriptions/s1/resourceGroups/g1/providers/Microsoft.Network/virtualNetworks/vnet1/subnets/subnet1"},@{VirtualNetworkResourceId="/subscriptions/s1/resourceGroups/g1/providers/Microsoft.Network/virtualN
    etworks/vnet2/subnets/subnet2"})
```

Ez a parancs frissíti a NetworkRule összes tulajdonságát, a beviteli szabályokat pedig JSON-nal.

### 2. példa: A NetworkRule megkerülő tulajdonságának frissítése
```
PS C:\> Update-AzCognitiveServicesAccountNetworkRuleSet -ResourceGroupName "myResourceGroup" -Name "myaccount"
```

A NetworkRule ezen parancsfrissítésének Bypass tulajdonsága (más tulajdonságok nem változnak).

### 3. példa: A Kognitív szolgáltatások fiók networkrule-szabályainak megtisztíttatása
```
PS C:\> Update-AzCognitiveServicesAccountNetworkRuleSet -ResourceGroupName "myResourceGroup" -Name "myaccount" -IpRule @() -VirtualNetworkRule @()
```

Ez a parancs megtisztítja a Kognitív Szolgáltatások-fiók NetworkRule szabályát (más tulajdonságok nem változnak).
