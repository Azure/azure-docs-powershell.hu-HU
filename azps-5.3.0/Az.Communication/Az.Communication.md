# Az.Communication Module
## Leírás
Microsoft Azure PowerShell: Kommunikációs parancsmagok

## Az.Communication parancsmagok
### [Get-AzCommunicationService](Get-AzCommunicationService.md)
Szerezze be a CommunicationService szolgáltatást és tulajdonságait.

### [Get-AzCommunicationServiceKey](Get-AzCommunicationServiceKey.md)
Szerezze be a CommunicationService erőforrás hozzáférési kulcsait.

### [New-AzCommunicationService](New-AzCommunicationService.md)
Hozzon létre egy új CommunicationService szolgáltatást, vagy frissítsen egy meglévő CommunicationService szolgáltatást.

### [New-AzCommunicationServiceKey](New-AzCommunicationServiceKey.md)
A CommunicationService hozzáférési kulcs újragenerálása.
A PrimaryKey és a SecondaryKey nem lehet egyszerre újragenerálni.

### [Remove-AzCommunicationService](Remove-AzCommunicationService.md)
Művelet a CommunicationService törléséhez.

### [Set-AzCommunicationServiceNotificationHub](Set-AzCommunicationServiceNotificationHub.md)
Egy Azure értesítési központot erre a kommunikációs szolgáltatásra mutató hivatkozásokkal hoz létre.

### [Update-AzCommunicationService](Update-AzCommunicationService.md)
Művelet egy meglévő CommunicationService frissítéséhez.
