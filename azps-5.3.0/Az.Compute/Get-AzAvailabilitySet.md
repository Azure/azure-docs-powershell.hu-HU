# Get-AzAvailabilitySet

## SYNOPSIS
Az Azure rendelkezésre állási készleteket kap egy erőforráscsoportban.

## SZINTAXIS

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzAvailabilitySet** parancsmag az Azure rendelkezésre állási készletét kapja meg egy erőforráscsoportban.
Megadhatja egy lekért elérhetőségi készlet nevét.
