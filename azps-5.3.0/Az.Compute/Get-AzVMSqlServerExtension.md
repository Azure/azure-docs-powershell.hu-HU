# Get-AzVMSqlServerExtension

## SYNOPSIS
Egy SQL Server-bővítmény beállításait kapja meg egy virtuális gépen.

## SZINTAXIS

```
Get-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzVMSqlServerExtension** parancsmag az SQL Server-infrastruktúra IaaS-ügynökként való beállításait kapja meg egy virtuális gépen.

## PÉLDÁK

### 1. példa: SQL Server-bővítmény beállításainak lekérte virtuális gépen
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Ez a parancs az SQL Server-bővítmény beállításait egy ContosoVM07 nevű virtuális gépen kapja meg.

### 2. példa: A beállítások be szereznie a folyamat használatával
```
PS C:\> Get-AzVM -ServiceName "Service08" -Name "ContosoVM22" | Get-AzVMSqlServerExtension
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Ez a parancs a ContosoVM22 nevű virtuális gépet a Service08 szolgáltatásban a Get-AzVM parancsmag használatával kapja meg.
A parancs a folyamat műveleti operátorával átadja az eredményeket az aktuális parancsmagnak.
Az aktuális parancs az SQL Server IaaS-ügynök beállításait kapja meg a virtuális gépen.

### 3. példa: Adott SQL Server-verzió beállításainak lekérte
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07" -Version "1.0"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Ez a parancs az SQL Server-bővítmény 1.0-s verziójának beállításait kapja meg egy ContosoVM07 nevű virtuális gépen.
