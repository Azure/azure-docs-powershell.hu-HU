# Remove-AzContainerService

## SYNOPSIS
Eltávolít egy tárolószolgáltatást.

## SZINTAXIS

```
Remove-AzContainerService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzContainerService** parancsmag eltávolít egy tárolószolgáltatást az Azure-fiókjából.

## PÉLDÁK

### 1. példa: Tárolószolgáltatás eltávolítása
```
PS C:\> Remove-AzContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17"
```

Ez a parancs eltávolítja a CSResourceGroup17 nevű tárolószolgáltatást.
