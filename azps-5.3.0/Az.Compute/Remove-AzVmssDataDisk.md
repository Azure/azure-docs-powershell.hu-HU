# Remove-AzVmssDataDisk

## SYNOPSIS
Eltávolít egy adatlemezt a VMSS-ről.

## SZINTAXIS

### NameParameterSet
```
Remove-AzVmssDataDisk [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### LunParameterSet
```
Remove-AzVmssDataDisk [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Lun] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzVmssDataDisk** parancsmag eltávolít egy adatlemezt a VIRTUÁLISgép-mérethalmaz (VMSS) példányból.

## PÉLDÁK

### 1. példa
```
PS C:\> Remove-AzVmssDataDisk -VirtualMachineScaleSet $vmss -Name 'DataDisk1'
```

Ez a parancs eltávolítja a "DataDisk1" adatlemezt a VMSS-objektumból.

### 2. példa
```
PS C:\> Remove-AzVmssDataDisk -VirtualMachineScaleSet $vmss -Lun 0
```

Ez a parancs eltávolítja a LUN 0 adatlemezét a VMSS-objektumból.
