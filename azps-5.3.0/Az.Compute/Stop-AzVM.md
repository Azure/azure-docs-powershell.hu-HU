# Stop-AzVM

## SYNOPSIS
Leállítja az Azure virtuális gépet.

## SZINTAXIS

### ResourceGroupNameParameterSetName (alapértelmezett)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-ResourceGroupName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Stop-AzVM parancsmag** leállít egy Azure virtuális gépet.

## PÉLDÁK

### 1. példa: Virtuális gép leállítása
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Ez a parancs leállítja a VirtualMachine07 nevű virtuális gépet az ResourceGroup11-ben.
