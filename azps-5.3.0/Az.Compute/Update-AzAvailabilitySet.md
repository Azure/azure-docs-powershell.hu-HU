# Update-AzAvailabilitySet

## SYNOPSIS
Frissíti az elérhetőségi készletet.

## SZINTAXIS

```
Update-AzAvailabilitySet [-AvailabilitySet] <PSAvailabilitySet> [[-Sku] <String>]
 [-ProximityPlacementGroupId <String>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Update-AzAvailabilitySet** parancsmag frissíti az elérhetőségi készletet.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzAvailabilitySet -ResourceGroupName 'ResourceGroup01' -Name 'AvSet01' | Update-AzAvailabilitySet -Managed;
```

Ez a parancs egy felügyelt rendelkezésre állási készletre frissíti az "AvSet01" nevű elérhetőségi készletet az "ResourceGroup01" erőforráscsoportban.
