# Get-AzDataBoxJob

## SYNOPSIS
Információkat kap az adatdoboz-feladatokról

## SZINTAXIS

### ListParameterSet (alapértelmezett)
```
Get-AzDataBoxJob [-ResourceGroupName <String>] [-Completed] [-CompletedWithError] [-Cancelled] [-Aborted]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzDataBoxJob -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzDataBoxJob -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDataBox Jobss** parancsmag információkat kap az Azure-előfizetésben található adatkapcsolatú feladatokról.
Ha megadja az Erőforráscsoportot, ez a parancsmag az adott erőforráscsoport összes adatdoboz-feladatát megkapja. Ha megadja a feladat nevét az erőforráscsoport nevével együtt, ez a parancsmag információt kap az adott adatdoboz-feladatról.
Ha nem ad meg semmit, csak az előfizetés azonosítóját, ez a parancsmag információt kap az adott előfizetésben található összes adatdoboz-feladatról.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzDataBoxJob

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1
cleanbox                DataBox              Aborted             04-12-2018 16:07:41   westus               TestRg2
cleanbox-Clone          DataBox              Cancelled           25-04-2019 11:31:36   westus               TestRg2
.
.
.
```

Get-AzDataBoxJob nélküli lekérdezés az előfizetés összes adatbox-feladatát lehívja

### 2. példa
```powershell
PS C:\> Get-AzDataBoxJob -ResourceGroupName TestRg1

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1
.
.
.
```

Get-AzDataBoxJob ResourceGroupName paraméterrel való hozzárendelés a megadott erőforráscsoport összes adatbox-feladatát lekéri

### 3. példa
```powershell
PS C:\> Get-AzDataBoxJob -ResourceGroupName TestRg1 -Name testtip2

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1

```

Get-AzDataBoxJob ResourceGroupName és Name érték megadva esetén az adott adatdoboz-feladat lekéri

### 4. példa
```powershell
PS C:\> Get-AzDataBoxJob -ResourceId "/subscriptions/05b5dd1c-793d-41de-be9f-6f9ed142f695/resourceGroups/TestRg1/providers/Microsoft.DataBox/jobs/testtip2"

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1

```

Get-AzDataBoxJob ResourceId érték van megadva, az adott adatkészlet-feladat lehívása
