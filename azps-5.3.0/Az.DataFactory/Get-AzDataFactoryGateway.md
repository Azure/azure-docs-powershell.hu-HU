# Get-AzDataFactoryGateway

## SYNOPSIS
Információkat kap az Azure Data Factory logikai átjáróiról.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Get-AzDataFactoryGateway [-DataFactoryName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGateway [-DataFactory] <PSDataFactory> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDataFactoryGateway** parancsmag információt kap az Azure Data Factory logikai átjáróiról.
Ha megadja egy átjáró nevét, ez a parancsmag információt kap az átjáróról.
Ha nem ad meg nevet, ez a parancsmag információt kap egy adatüzem minden átjáróról.
Ha egy helyszíni Microsoft SQL Servert csatolt szolgáltatásként szeretne hozzáadni egy adatüzemhez, telepítenie kell egy átjárót a helyszíni számítógépén.

Ez a parancs információkat kap az ADF nevű erőforráscsoport WikiADF nevű adatkezelő üzemének összes logikai átjárójára.
