# Invoke-AzDataFactoryV2IntegrationRuntimeUpgrade

## SYNOPSIS
Frissíti az önkiszolgáló integrációs futásiidőt.

## SZINTAXIS

### ByIntegrationRuntimeName (alapértelmezett)
```
Invoke-AzDataFactoryV2IntegrationRuntimeUpgrade [-Name] <String> [-ResourceGroupName] <String>
 [-DataFactoryName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByResourceId
```
Invoke-AzDataFactoryV2IntegrationRuntimeUpgrade [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByIntegrationRuntimeObject
```
Invoke-AzDataFactoryV2IntegrationRuntimeUpgrade [-InputObject] <PSIntegrationRuntime>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Invoke-AzDataFactoryV2IntegrationRuntimeUpgrade** parancsmag frissíti az önkiszolgáló integrációs futásidejét, ha az új verzió elérhető.

## PÉLDÁK

### 1. példa: Önkiszolgáló integrációs futási idő frissítése
```
PS C:\> Invoke-AzDataFactoryV2IntegrationRuntimeUpgrade -ResourceGroupName 'rg-test-dfv2' -DataFactoryName 'test-df-eu2' -Name 'test-selfhost-ir'
```

A parancsmag frissíti a "test-selfhost-ir" nevű önkiszolgáló integrációs futásiidőt a "test-df-eu2" data factoryban.
