# Set-AzDataFactoryPipelineActivePeriod

## SYNOPSIS
Az adatszeletek aktív időszakának beállítása.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Set-AzDataFactoryPipelineActivePeriod [-PipelineName] <String> [-DataFactoryName] <String>
 [-StartDateTime] <DateTime> [[-EndDateTime] <DateTime>] [-AutoResolve] [-ForceRecalculate]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
Set-AzDataFactoryPipelineActivePeriod [-PipelineName] <String> [-DataFactory] <PSDataFactory>
 [-StartDateTime] <DateTime> [[-EndDateTime] <DateTime>] [-AutoResolve] [-ForceRecalculate]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzDataFactoryPipelineActivePeriod** parancsmag konfigurálja az Azure Data Factory-folyamat által feldolgozott adatszeletek aktív időszakát.
Ha a Set-AzDataFactorySliceStatus-parancsmag használatával módosítja egy adatkészlet szeletének állapotát, ellenőrizze, hogy a szelet kezdési és záró ideje a folyamat aktív időszakában van-e.
A folyamat létrehozása után megadhatja az adatok feldolgozásának időszakát.
Egy folyamat aktív időszakának megadása meghatározza az adatszeletek feldolgozásának időtartamát  az egyes Data Factory-adatkészletek elérhetőségi tulajdonságai alapján.

## PÉLDÁK

### 1. példa: Az aktív időszak konfigurálása
```
PS C:\>Set-AzDataFactoryPipelineActivePeriod -ResourceGroupName "ADF" -PipelineName "DPWikisample" -DataFactoryName "WikiADF" -StartDateTime 2014-05-21T16:00:00Z -EndDateTime 2014-05-22T16:00:00Z
Confirm
Are you sure you want to set pipeline 'DPWikisample' active period from '05/21/2014 16:00:00' to
'05/22/2014 16:00:00'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
True
```

Ez a parancs beállítja a DPWikisample nevű folyamat adatszeletei aktív időszakát.
A parancs értékként biztosítja az adatszeletek kezdő és záró pontjait.
A parancs egy számértéket ad $True.
