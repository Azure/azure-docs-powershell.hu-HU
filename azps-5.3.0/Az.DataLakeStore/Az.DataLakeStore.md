# Az.DataLakeStore modul
## Leírás
A szakasz témakörei az Azure Resource Manager (ARM) keretrendszer azure PowerShell-parancsmagokat tartalmaznak az Azure Data Lake Store-hoz. A parancsmagok a Microsoft.Azure.Commands.DataLakeStore névterében találhatóak. Ezek a parancsmagok csak Azure Data Lake Storage Gen1-fiókokkal működnek.

## Az.DataLakeStore parancsmagok
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Tűzfalszabály hozzáadása a megadott Data Lake Store-fiókhoz.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Tartalmat ad hozzá egy adattavatárban található elemhez.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Hozzáad egy megbízható identitásszolgáltatót a megadott Data Lake Store-fiókhoz.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Virtuális hálózati szabályt ad hozzá a megadott Data Lake Store-fiókhoz.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Megkísérli engedélyezni egy felhasználó által felügyelt kulcstárolónak a megadott Data Lake Store-fiók titkosítását.

### [Export-AzDataLakeStoreStoreDataItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Exportálja a teljes fa tulajdonságait (lemezhasználat és Acl) a megadott elérési útból a kimeneti útvonalra.

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Letölt egy fájlt a Data Lake Store-ból.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Részleteket kap egy Data Lake Store-fiókról.

### [Get-AzDataLakeStoreDataItem](Get-AzDataLakeStoreChildItem.md)
A Data Lake Store-ban található mappában lévő elemek listáját kapja meg.

### [Get-AzDataLakeStoreDataItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
A megadott elérési útban szereplő összes méret, fájl és könyvtár összesítését adja vissza.

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
A megadott tűzfalszabályt kapja meg a data lake store-ban.
Ha nincs megadva tűzfalszabály, akkor felsorolja a fiók összes tűzfalszabályát.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)
A Data Lake Store-ban található fájl vagy mappa részleteinek lekérte.

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)
Bejegyzést kap a Data Lake Store-ban tárolt fájl vagy mappa ACL-ében.

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
A Data Lake Store-ban tárolt fájl tartalmának be- és lekérte.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
A Data Lake Store-ban található fájl vagy mappa tulajdonosának lekérte.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
A Data Lake Store-ban tárolt fájl vagy mappa oktális engedélyét kapja meg.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Lekérte a megadott megbízható identitásszolgáltatót a megadott Data Lake Store-ban.
Ha nincs megadva szolgáltató, akkor felsorolja a fiók összes szolgáltatóját.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
A megadott virtuális hálózati szabályokat kapja meg a megadott Data Lake Store-ban.
Ha nincs megadva virtuális hálózati szabály, akkor felsorolja a fiók összes virtuális hálózati szabályát.

### [Import-AzDataLakeStoreItem](Import-AzDataLakeStoreItem.md)
Feltölt egy helyi fájlt vagy címtárat egy Data Lake Store-beli tárolóba.

### [Join-AzDataLakeStoreItem](Join-AzDataLakeStoreItem.md)
Egy vagy több fájlhoz csatlakozva létrehoz egy fájlt a Data Lake Store-ban.

### [Move-AzDataLakeStoreItem](Move-AzDataLakeStoreItem.md)

### [New-AzDataLakeStoreAccount](New-AzDataLakeStoreAccount.md)

### [New-AzDataLakeStoreItem](New-AzDataLakeStoreItem.md)
