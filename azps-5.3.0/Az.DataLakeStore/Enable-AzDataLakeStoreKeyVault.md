# Enable-AzDataLakeStoreKeyVault

## SYNOPSIS
Megkísérli engedélyezni egy felhasználó által felügyelt kulcstárolónak a megadott Data Lake Store-fiók titkosítását.

## SZINTAXIS

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzDataLakeStoreKeyVault** parancsmag megkísérli engedélyezni a felhasználó által felügyelt kulcstárolónak a megadott Data Lake Store-fiók titkosítását.

## PÉLDÁK

### 1. példa: A ContosoADLS-fiók kulcstárának engedélyezése
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

Ez a parancs megkísérli engedélyezni a felhasználó által felügyelt kulcstárolót a ContosoADLS nevű Data Lake Store-fiókhoz.
