# Remove-AzDataLakeStoreItem

## SYNOPSIS
Töröl egy fájlt vagy mappát a Data Lake Store-ban.

## SZINTAXIS

```
Remove-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]> [-Recurse] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataLakeStoreItem** parancsmag töröl egy fájlt vagy mappát a Data Lake Store-ban.

## PÉLDÁK

### 1. példa: Több elem eltávolítása
```
PS C:\>Remove-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/File01.txt","/MyFiles/File.csv"
```

Ez a parancs eltávolítja a fájlokat File01.txt és File.csv a Data Lake Store-ból.
