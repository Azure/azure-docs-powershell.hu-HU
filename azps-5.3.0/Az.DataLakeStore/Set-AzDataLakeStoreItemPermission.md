# Set-AzDataLakeStoreItemPermission

## SYNOPSIS
Módosítja a Data Lake Store-ban tárolt fájlok vagy mappák engedély oktális engedélyét.

## SZINTAXIS

```
Set-AzDataLakeStoreItemPermission [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Permission] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzDataLakeStoreItemPermission** parancsmag módosítja a Data Lake Store-ban található fájlok vagy mappák engedélyeinek oktális engedélyét.

## PÉLDÁK

### 1. példa: Elem engedélyeinek oktális beállítása
```
PS C:\>Set-AzDataLakeStoreItemPermission -AccountName "ContosoADL" -Path "/file.txt" -Permission 0770
```

Ez a parancs a fájl oktális engedélyét 0770-re állítja, azaz törli a berakott bitet, olvasási/írási/végrehajtási engedélyeket állít be a fájl tulajdonosának, olvasási/írási/végrehajtási engedélyeket állít be a fájl tulajdonosa csoportjához, valamint törli az olvasási/írási/végrehajtási engedélyeket más fájlokhoz.
