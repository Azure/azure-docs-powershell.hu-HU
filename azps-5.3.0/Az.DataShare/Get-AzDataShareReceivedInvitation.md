# Get-AzDataShareReceivedInvitation

## SYNOPSIS
Információkat kap a fogyasztói meghívókról.

## SZINTAXIS

```
Get-AzDataShareReceivedInvitation [-Location <String>] [-InvitationId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDataShareReceivedInvitation** parancsmag információt nyújt a felhasználó által visszavont összes meghívásról. Ha hely- vagy meghívóazonosítót ad meg, ez a parancsmag adott helyen szolgáltat információkat a meghívásról vagy a meghívóazonosítóról. Ellenkező esetben a felhasználónak küldött meghívók listáját tartalmazza.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzDataShareReceivedInvitation -location "westus"

InvitationStatus  : Pending
Location          : westus
RespondedAt       :

SenderCompanyName : ADS Test
SentAt            : 7/8/2019 10:47:10 PM
ShareName         : AdsShare
Description       : Some description

Name              : AdsInvitation
Type              : Microsoft.DataShare/consumerInvitations
```

Ez a commant információkat nyújt a fogyasztói meghívókról.
