# Get-AzDataShareSubscription

## SYNOPSIS
Információkat kap az előfizetés megosztásáról az adat-megosztási fiókban.

## SZINTAXIS

### ByFieldsParameterSet (alapértelmezett)
```
Get-AzDataShareSubscription -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareSubscription -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDataShareSubscription** parancsmag információkat nyújt az adatmegosztási fiókban való előfizetés megosztásáról. Ha meg van biztosítanak egy megosztási előfizetés nevét, a parancsmag információkat ad az adott megosztási előfizetésről. Ellenkező esetben a parancsmag felsorolja a megosztási előfizetéseket az adat-megosztási fiókban.
