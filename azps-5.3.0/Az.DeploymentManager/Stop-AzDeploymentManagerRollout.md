# Stop-AzDeploymentManagerRollout

## SYNOPSIS
Leállítja a bevezetést.

## SZINTAXIS

### Interaktív (alapértelmezett)
```
Stop-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Stop-AzDeploymentManagerRollout [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Stop-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Stop-AzDeploymentManagerRollout** parancsmag leállítja a folyamatban lévő bevezetést, és visszaad egy objektumot, amely a bevezetés aktuális állapotát képviseli.
Adja meg a bevezetést a neve és az erőforráscsoport neve szerint. Másik módon meg is használhatja a bevezetési objektumot vagy az Erőforrásazonosítót.

Felhívjuk a figyelmét arra, hogy miután leállt a bevezetés, az nem folytatható és nem indítható újra. Csak új bevezetést hozhat létre.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Stop-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout -SkipSucceeded
```

Ez a parancs leállítja a ContosoRollout nevű bevezetést a ContosoResourceGroup csoportban. 

### 2. példa: Bevezetés leállítása az erőforrás-azonosító használatával
```powershell
PS C:\> Restart-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Ez a parancs leállítja a ContosoRollout nevű bevezetést a ContosoResourceGroup csoportban.

### 3. példa: Bevezetés leállítása a bevezetési objektum használatával.
```powershell
PS C:\> Get-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Ez a parancs leállítja az olyan bevezetést, amelynek neve és ResourceGroup tulajdonsága megegyezik a $rolloutObject ResourceGroupName tulajdonságaival.
