# New-AzFrontDoorWafManagedRuleOverrideObject

## SYNOPSIS
Felügyelt szabály felülbírálási objektumának létrehozása

## SZINTAXIS

```
New-AzFrontDoorWafManagedRuleOverrideObject -RuleId <String> [-Action <String>] [-Disabled]
 [-Exclusion <PSManagedRuleExclusion[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
PSAzureManagedRuleOverride objektum létrehozása felügyelt WAF-szabálycsoport felülbírálása objektum létrehozásához.

## PÉLDÁK

### 1. példa
Felügyelt szabály felülbírálási objektumának létrehozása a 942250-es szabályhoz (amely az SQLI-csoportban van).

```powershell
PS C:\> New-AzFrontDoorWafManagedRuleOverrideObject -RuleId "942250" -Action Log

RuleId EnabledState Action
------ ------------ ------
942250      Enabled    Log
```
