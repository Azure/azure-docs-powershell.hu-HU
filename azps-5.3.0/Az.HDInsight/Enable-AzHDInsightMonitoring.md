# Enable-AzHDInsightMonitoring

## SYNOPSIS
A HDInsight-fürtben való figyelés engedélyezése és a kapcsolódó naplók az engedélyezés során megadott figyelési munkaterületre lesznek küldve.

## SZINTAXIS

```
Enable-AzHDInsightMonitoring [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzHDInsightMonitoring** parancsmag lehetővé teszi a figyelést az Azure HDInsight-fürtben.

## PÉLDÁK

### 1. példa
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

A HDInsight-fürtben engedélyezve lesz a figyelés, és a kapcsolódó naplókat a figyelési munkaterületre küldjük az 1d364e89-bb71-4503-aa3d-a23535aea7bd azonosítóval

### 2. példa
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

A HDInsight-fürtben engedélyezve lesz a figyelés, és a kapcsolódó naplókat a figyelési munkaterületre küldjük az 1d364e89-bb71-4503-aa3d-a23535aea7bd azonosítóval
