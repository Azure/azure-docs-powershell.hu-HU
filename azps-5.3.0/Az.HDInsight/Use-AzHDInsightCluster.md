# Use-AzHDInsightCluster

## SYNOPSIS
A parancsmaggal együtt használni kívánt Invoke-RmAzureHDInsightHiveJob kijelölése.

## SZINTAXIS

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Use-AzHDInsightCluster** parancsmag kiválasztja az Azure HDInsight-fürtöt a Invoke-AzHDInsightHiveJob-parancsmag számára a Hive-feladatok beküldése érdekében.

## PÉLDÁK

### 1. példa: Fürt kiválasztása a Hive-lekérdezések beküldéséhez
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Ez a parancs kijelöl egy fürtöt a Hive-lekérdezések beküldéséhez.
