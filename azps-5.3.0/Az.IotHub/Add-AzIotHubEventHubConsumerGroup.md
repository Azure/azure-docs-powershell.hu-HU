# Add-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Létrehoz egy eventhub fogyasztói csoportot.

## SZINTAXIS

```
Add-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
Létrehoz egy fogyasztói csoportot a megadott IotHubbal társított Eventhubban.

## PÉLDÁK

### 1. példa: Fogyasztói csoport hozzáadása a telemetriai eventhubhoz
```
PS C:\> Add-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName "myconsumergroup"
```

Hozzáad egy "myconsumergroup" nevű új fogyasztói csoportot az eventhubhoz telemetriai eseményekhez a "myiothub" nevű iothubban
