# Get-AzIntegrationAccount

## SYNOPSIS
Integrációs fiókokat kap.

## SZINTAXIS

```
Get-AzIntegrationAccount [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzIntegrationAccount** parancsmag integrációs fiókokat kap egy erőforráscsoporttól. Adja meg az integrációs fiók és az erőforráscsoport nevét.
Ez a modul támogatja a dinamikus paramétereket.
Ha dinamikus paramétert használ, írja be a parancsba.
A dinamikus paraméterek nevének felfedezéséhez írjon be egy kötőjelet (-) a parancsmag neve után, majd a Tab billentyű többszöri lenyomása után lépkedhet a rendelkezésre álló paraméterek között.
Ha kihagy egy kötelező sablonparamétert, a parancsmag rákérdez az értékre.

```
PS C:\>Get-AzIntegrationAccount -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31"

Name        : IntegrationAccount31
Type        : Microsoft.Logic/integrationAccounts
Location    : brazilsouth
Sku         : 
CreatedTime : 3/26/2016 4:26:07 PM
ChangedTime : 3/26/2016 4:26:07 PM
```

Ez a parancs egy IntegrationAccount31 nevű integrációs fiókot kap a megadott erőforráscsoportból.

### 2. példa: Integrációs fiókok lekérte egy erőforráscsoportba
```
PS C:\>Get-AzIntegrationAccount -ResourceGroupName "ResourceGroup11"

Ez a parancs integrációs fiókokat kap egy ResourceGroup11 nevű erőforráscsoportból.

### 3. példa: Az összes integrációs fiók lekérte
```
PS C:\>Get-AzIntegrationAccount

Ez a parancs az Azure-előfizetés összes integrációs fiókot beveszi.
