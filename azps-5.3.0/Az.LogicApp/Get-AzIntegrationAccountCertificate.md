# Get-AzIntegrationAccountCertificate

## SYNOPSIS
Integrációs fióktanúsítványokat kap egy erőforráscsoporttól.

## SZINTAXIS

```
Get-AzIntegrationAccountCertificate [-ResourceGroupName <String>] [-Name <String>] [-CertificateName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzIntegrationAccountCertificate parancsmag** integrációs fióktanúsítványokat kap egy erőforráscsoporttól.
Adja meg az integrációs fiók nevét, az erőforráscsoport nevét és a tanúsítvány nevét.
Ez a modul támogatja a dinamikus paramétereket.
Ha dinamikus paramétert használ, írja be a parancsba.
A dinamikus paraméterek nevének felfedezéséhez írjon be egy kötőjelet (-) a parancsmag neve után, majd a Tab billentyű többszöri lenyomása után lépkedhet a rendelkezésre álló paraméterek között.
Ha kihagy egy kötelező sablonparamétert, a parancsmag rákérdez az értékre.
