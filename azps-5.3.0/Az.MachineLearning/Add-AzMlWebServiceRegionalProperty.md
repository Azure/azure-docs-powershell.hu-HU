# Add-AzMlWebServiceRegionalProperty

## SYNOPSIS
Területi webszolgáltatás-tulajdonságokat hoz létre.

## SZINTAXIS

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName <String> -Name <String> -Region <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Azure Machine Learning regionális tulajdonságokat hoz létre egy meglévő webszolgáltatáshoz.

## PÉLDÁK

### 1. példa: Új regionális tulajdonságok hozzáadása az EGYESÜLT Egyesült Régió régióhoz

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName "myresourcegroup" -Name "mywebservicename" -Region westcentralus
```

Ez a példaparancs egy webszolgáltatás regionális tulajdonságát hozza létre az "Egyesült Államok nyugati részén"
