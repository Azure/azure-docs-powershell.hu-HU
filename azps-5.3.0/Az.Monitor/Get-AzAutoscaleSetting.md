# Get-AzAutoscaleSetting

## SYNOPSIS
Automatikus skálázási beállításokat kap.

## SZINTAXIS

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```
