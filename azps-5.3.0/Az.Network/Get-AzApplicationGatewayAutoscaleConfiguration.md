# Get-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Az alkalmazás átjárójának automatikus méretezésű konfigurációját kapja meg.

## SZINTAXIS

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayAutoscaleConfiguration parancsmag** az alkalmazás-átjáró automatikus méretezésű konfigurációját kapja meg.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

Az első parancs lekérte az alkalmazás átjáróját, és egy változóban $gw tárolja.
A második parancs kinyeri az automatikus skálás konfigurációt az alkalmazás-átjáróból.
