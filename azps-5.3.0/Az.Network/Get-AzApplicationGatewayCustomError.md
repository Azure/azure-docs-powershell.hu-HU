# Get-AzApplicationGatewayCustomError

## SYNOPSIS
Egyéni hiba(oka)t kap egy alkalmazás átjárótól.

## SZINTAXIS

```
Get-AzApplicationGatewayCustomError [-StatusCode <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayCustomError** parancsmag egyéni hiba(oka)t kap egy alkalmazás-átjárótól.

## PÉLDÁK

### 1. példa: Egyéni hibát kap egy alkalmazás-átjáróban
```powershell
PS C:\> $ce = Get-AzApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

Ez a parancs az 502-es http-állapotkód egyéni hibaüzenetét kapja vissza az alkalmazás-átjárótól $appgw.

### 2. példa: Egy alkalmazás-átjáró összes egyéni hibalistát beveszi
```powershell
PS C:\> $ces = Get-AzApplicationGatewayCustomError -ApplicationGateway $appgw
```

Ez a parancs be- és visszaadja az összes egyéni hiba listáját az $appgw.
