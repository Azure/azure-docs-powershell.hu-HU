# Get-AzApplicationGatewaySslProfilePolicy

## SYNOPSIS
Egy alkalmazás-átjáró SSL-profiljának SSL-házirendét használja.

## SZINTAXIS

```
Get-AzApplicationGatewaySslProfilePolicy -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewaySslProfilePolicy** parancsmag egy alkalmazás-átjáró SSL-profiljának SSL-házirendjére érvényes.

## PÉLDÁK

### 1. példa
```powershell
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SslProfile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
PS C:\> $sslpolicy = Get-AzApplicationGatewaySslProfilePolicy -SslProfile $SslProfile
```

Az első parancs az ApplicationGateway01 nevű alkalmazás-átjárót az ResourceGroup01 nevű erőforráscsoportban tárolja a $AppGw változóban. A második parancs az SslProfile01 nevű SSL-profilt kapja meg a $AppGw, és azt $SslProfile tárolja. Az utolsó parancs az SSL-profilból kapja meg az SSL$SslProfile és tárolja azt a $sslpolicy változóban.
