# Get-AzEffectiveRouteTable

## SYNOPSIS
A hálózati felület hatékony útvonaltábláját használja.

## SZINTAXIS

```
Get-AzEffectiveRouteTable -NetworkInterfaceName <String> [-ResourceGroupName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzEffectiveRouteTable** parancsmag a hálózati felületen alkalmazott tényleges útvonaltáblát adja eredményül.

## PÉLDÁK

### 1. példa: A hatékony útvonaltábla lekérte egy hálózati felületen
```
PS C:\>Get-AzEffectiveRouteTable -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "MyResourceGroup"
```

Ez a parancs a MyResourceGroup nevű erőforráscsoport MyNetworkInterface nevű hálózati felületével társított hatékony útvonaltáblát kapja meg.
