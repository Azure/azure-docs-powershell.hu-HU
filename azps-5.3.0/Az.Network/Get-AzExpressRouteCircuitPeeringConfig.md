# Get-AzExpressRouteCircuitPeeringConfig

## SYNOPSIS
ExpressRoute-kapcsolati társviszony-konfigurációt kap.

## SZINTAXIS

```
Get-AzExpressRouteCircuitPeeringConfig [-Name <String>] -ExpressRouteCircuit <PSExpressRouteCircuit>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzExpressRouteCircuitPeeringConfig parancsmag** beolvassa egy társviszony konfigurációját egy ExpressRoute-áramkörben.

## PÉLDÁK

### 1. példa: Az ExpressRoute-áramkör társviszony-konfigurációjának megjelenítése
```
$ckt = Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $RG
Get-AzExpressRouteCircuitPeeringConfig -Name "AzurePrivatePeering" -ExpressRouteCircuit $ckt
```
