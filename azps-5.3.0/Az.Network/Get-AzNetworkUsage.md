# Get-AzNetworkUsage

## SYNOPSIS
Az előfizetés hálózati használatának felsorolása

## SZINTAXIS

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Get-AzNetworkUsage parancsmag korlátozza és aktuálisan kihasználja a hálózati erőforrásokat.
