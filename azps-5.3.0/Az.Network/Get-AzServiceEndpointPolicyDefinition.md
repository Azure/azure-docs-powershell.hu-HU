# Get-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Szolgáltatásvégpont-házirend-definíciót kap.

## SZINTAXIS

```
Get-AzServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzServiceEndpointPolicyDefinition parancsmag** szolgáltatásvégpont-házirend-definíciót kap.

## PÉLDÁK

### 1. példa
```
$policydef= Get-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ServiceEndpointPolicy $Policy
```

Ez a parancs beszerzi a ServiceEndpointPolicyDefinition1 nevű szolgáltatásvégpont-házirend-definíciót a ServiceEndpointPolicy $Policy a $policydef változóban.
