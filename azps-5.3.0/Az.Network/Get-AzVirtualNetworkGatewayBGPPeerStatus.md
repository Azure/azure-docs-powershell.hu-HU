# Get-AzVirtualNetworkGatewayBGPPeerStatus

## SYNOPSIS
Egy Azure virtuális hálózati átjáró BGP-társlistáit sorolja fel

## SZINTAXIS

```
Get-AzVirtualNetworkGatewayBGPPeerStatus -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-Peer <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Ez a parancs felsorolja a BGP-társokat, amelyekhez az Azure virtuális hálózati átjáró be van állítva társviszonyként. Az egyes partnerek állapota is meg van adva.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzVirtualNetworkGatewayBgpPeerStatus -ResourceGroupName resourceGroup -VirtualNetworkGatewayName gatewayName

Asn               : 65515
ConnectedDuration : 9.01:04:53.5768637
LocalAddress      : 10.1.0.254
MessagesReceived  : 14893
MessagesSent      : 14900
Neighbor          : 10.0.0.254
RoutesReceived    : 1
State             : Connected
```

Beolvassa a BGP-társokat az ÁtjáróNév nevű Azure virtuális hálózati átjáróhoz az erőforráscsoport erőforráscsoportban.
Ebben a példában egy csatlakoztatott BGP-társ látható 10.0.0.254-es IP-címmel.
