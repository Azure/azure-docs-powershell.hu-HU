# Get-AzVirtualNetworkGatewayLearnedRoute

## SYNOPSIS
Az Azure virtuális hálózati átjáró által tanult útvonalak listája

## SZINTAXIS

```
Get-AzVirtualNetworkGatewayLearnedRoute -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Felsorolja az Azure virtuális hálózati átjáró által a különböző forrásokból megtanult útvonalakat. Ide tartoznak a BGP-kapcsolaton keresztül tanult útvonalak, valamint a statikus útvonalak is. 

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzVirtualNetworkGatewayLearnedRoute -ResourceGroupName resourceGroup -VirtualNetworkGatewayname gatewayName

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.1.0.0/16
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.0.0.254/32
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       : 65515
LocalAddress : 10.1.0.254
Network      : 10.0.0.0/16
NextHop      : 10.0.0.254
Origin       : EBgp
SourcePeer   : 10.0.0.254
Weight       : 32768
```

Az Erőforráscsoport erőforráscsoportban az átjárónév nevű Azure virtuális hálózati átjáró esetében beolvassa az Azure-átjáró által ismert útvonalakat. Ebben az esetben az Azure virtuális hálózati átjárónak két statikus útvonala van (10.1.0.0/16 és 10.0.0.254/32), valamint egy BGP-en keresztül tanult útvonal (10.0.0.0/16).
