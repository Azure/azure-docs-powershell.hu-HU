# New-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Létrehoz egy háttércímkészletet egy alkalmazás-átjáróhoz.

## SZINTAXIS

```
New-AzApplicationGatewayBackendAddressPool -Name <String> [-BackendIPAddresses <String[]>]
 [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzApplicationGatewayBackendAddressPool** parancsmag létrehoz egy háttércímkészletet egy Azure-alkalmazás átjáróhoz.
A háttércímek IP-címként, teljes tartománynévként vagy IP-konfigurációs azonosítóként is megadva.

## PÉLDÁK

### 1. példa: Háttércímkészlet létrehozása a háttérkiszolgáló teljes tartományának használatával
```
PS C:\>$Pool = New-AzApplicationGatewayBackendAddressPool -Name "Pool01" -BackendFqdns "contoso1.com", "contoso2.com"
```

Ez a parancs létrehoz egy Pool01 nevű háttércímkészletet a háttérkiszolgálók teljes tartománynevének használatával, és azt egy másik $Pool tárolja.

### 2. példa: Háttércímkészlet létrehozása a háttérkiszolgáló IP-címével
```
PS C:\>$Pool = New-AzApplicationGatewayBackendAddressPool -Name "Pool02" -BackendFqdns "10.10.10.10", "10.10.10.11"
```

Ez a parancs létrehoz egy Pool02 nevű háttércímkészletet a háttérkiszolgálók IP-címeinek használatával, és tárolja azt a $Pool változóban.
