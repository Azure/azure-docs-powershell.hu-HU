# New-AzFirewallThreatIntelWhitelist

## SYNOPSIS
Új veszélyforrás-felderítési engedélyezési lista létrehozása az Azure tűzfalhoz

## SZINTAXIS

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzFirewallThreatIntelWhitelist** parancsmag létrehoz egy veszélyforrás-intel whitelist objektumot, amely azure tűzfal létrehozásakor és beállításakor használható.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

Ebben a példában egy veszélyforrásokkal kapcsolatos intel whitelistát hoz létre, amely tartalmazza a két bejegyzésből és egy IP-cím whitelistát két bejegyzésből.
