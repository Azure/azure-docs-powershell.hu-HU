# New-AzRouteFilterRuleConfig

## SYNOPSIS
Útvonalszűrő szabályt hoz létre az útvonalszűrők számára.

## SZINTAXIS

```
New-AzRouteFilterRuleConfig [-Force] -Name <String> -Access <String> -RouteFilterRuleType <String>
 -CommunityList <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A New-AzRouteFilterRuleConfig parancsmag létrehoz egy útvonalszűrő szabályt egy Azure-útvonalszűrő számára.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $rule1 = New-AzRouteFilterRuleConfig -Name "Rule01" -Access "Allow" -RouteFilterRuleType "Community" -CommunityList "12076:5040"
```

A parancs létrehoz egy új útvonalszűrő szabályt, és azt egy változó $rule 1.
