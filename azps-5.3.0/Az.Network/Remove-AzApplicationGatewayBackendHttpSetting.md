# Remove-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
Eltávolítja a háttér-HTTP-beállításokat egy alkalmazás-átjáróból.

## SZINTAXIS

```
Remove-AzApplicationGatewayBackendHttpSetting -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Remove-AzApplicationGatewayBackendHttpSetting parancsmag eltávolítja a háttér-hypertext Transfer Protocol (HTTP) beállításait egy Azure-alkalmazás átjáróból.

## PÉLDÁK

### 1. példa: Háttér-HTTP-beállítások eltávolítása egy alkalmazás-átjáróból
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw -Name "BackEndSetting02"
```

Az első parancs egy ApplicationGateway01 nevű alkalmazás-átjárót kap, amely az ResourceGroup01 nevű erőforráscsoporthoz tartozik, és a $AppGw tárolja.
A második parancs eltávolítja a BackEndSetting02 nevű háttér-HTTP-beállítást a $AppGw.
