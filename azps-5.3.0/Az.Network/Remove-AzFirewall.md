# Remove-AzFirewall

## SYNOPSIS
Tűzfal eltávolítása

## SZINTAXIS

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzFirewall** parancsmag eltávolítja az Azure tűzfalat.

## PÉLDÁK

### 1. példa: Tűzfal létrehozása és törlése
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Ez a példa létrehoz egy tűzfalat, majd törli azt. A tűzfal törlésekor megjelenő üzenet elrejtését a -Force jelölővel használhatja.
