# Remove-AzLocalNetworkGateway

## SYNOPSIS
Helyi hálózati átjáró törlése

## SZINTAXIS

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A helyi hálózati átjáró a virtuális magánhálózati eszköz helyszíni objektumának megfelelő objektum.
A **Remove-AzLocalNetworkGateway** parancsmag a név és az erőforráscsoport neve alapján törli a helyi átjárót képviselő objektumot.

## PÉLDÁK

### 1. példa: Helyi hálózati átjáró törlése
```powershell
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Törli a helyi hálózati átjáró "myLocalGW" nevű objektumát a "myRG" erőforráscsoportban. A **Remove-AzVirtualNetworkGatewayConnection** parancsmaggal először törölnie kell a helyi hálózati átjáróval létesített összes kapcsolatot.
