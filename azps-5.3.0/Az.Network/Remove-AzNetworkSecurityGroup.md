# Remove-AzNetworkSecurityGroup

## SYNOPSIS
Eltávolít egy hálózati biztonsági csoportot.

## SZINTAXIS

```
Remove-AzNetworkSecurityGroup -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzNetworkSecurityGroup** parancsmag eltávolít egy Azure-hálózati biztonsági csoportot.

## PÉLDÁK

### 1. példa: Hálózati biztonsági csoport eltávolítása
```
PS C:\>Remove-AzNetworkSecurityGroup -Name "NSG-FrontEnd" -ResourceGroupName "TestRG"
```

Ez a parancs eltávolítja a TestRG nevű NSG-FrontEnd nevű erőforráscsoportból.
