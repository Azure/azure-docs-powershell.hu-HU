# Remove-AzPrivateLinkService

## SYNOPSIS
Privát hivatkozásszolgáltatás eltávolítása

## SZINTAXIS

```
Remove-AzPrivateLinkService -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzPrivateLinkService** parancsmag eltávolít egy privát hivatkozásszolgáltatást

## PÉLDÁK

### 1. példa
```
Remove-AzPrivateLinkService -ResourceGroupName TestResourceGroup -Name TestPrivateLinkService
```

Ez a példa eltávolít egy TestPrivateLinkService nevű privát hivatkozásszolgáltatást.
