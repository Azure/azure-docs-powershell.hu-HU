# Remove-AzRouteFilterRuleConfig

## SYNOPSIS
Eltávolít egy útvonalszűrő szabályt egy útvonalszűrőből.

## SZINTAXIS

```
Remove-AzRouteFilterRuleConfig -Name <String> -RouteFilter <PSRouteFilter> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzRouteFilterRuleConfig** parancsmag eltávolít egy útvonalszűrő szabályt az útvonalszűrőkből.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
```

Az első parancs egy RouteFilter01 nevű útvonalszűrőt kap, amely az ResourceGroup01 nevű erőforráscsoporthoz tartozik, és egy másik $rf tárolja.
A második parancs eltávolítja a Szabály01 nevű útvonalszűrő-szabályt a $rf.
