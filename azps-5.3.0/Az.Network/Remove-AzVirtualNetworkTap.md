# Remove-AzVirtualNetworkTap

## SYNOPSIS
Virtuális hálózati koppintás eltávolítása.

## SZINTAXIS

### RemoveByNameParameterSet (alapértelmezett)
```
Remove-AzVirtualNetworkTap -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzVirtualNetworkTap -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzVirtualNetworkTap -InputObject <PSVirtualNetworkTap> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzVirtualNetworkTap** parancsmag eltávolítja az Azure virtuális hálózatra való rákoppintást.

## PÉLDÁK

### 1. példa: Virtuális hálózatra való koppintás eltávolítása
```
PS C:\>Remove-AzNetworkInterface -Name "VirtualNetworkTap1" -ResourceGroup "ResourceGroup1"
```

Ez a parancs eltávolítja a VirtualNetworkTap1-et az Erőforráscsoport1 erőforráscsoportban.
Mivel a *Force* paramétert nem használja, a rendszer kérni fogja a felhasználótól a művelet megerősítését.
