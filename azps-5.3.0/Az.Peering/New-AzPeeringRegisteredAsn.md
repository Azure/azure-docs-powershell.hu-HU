# New-AzPeeringRegisteredAsn

## SYNOPSIS
Regisztrált ASN létrehozása társviszonyhoz

## SZINTAXIS

### ByResourceGroupAndName (alapértelmezett)
```
New-AzPeeringRegisteredAsn [-ResourceGroupName] <String> [-PeeringName] <String> [-Name] <String> -Asn <Int32>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
New-AzPeeringRegisteredAsn -InputObject <PSPeering> [-Name] <String> -Asn <Int32> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
New-AzPeeringRegisteredAsn [-ResourceId] <String> [-Name] <String> -Asn <Int32> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Regisztrált ASN-eket hozhat létre társviszony-objektumokhoz.

## PÉLDÁK

### Társviszony-létesítés és regisztrált asn-hálózat létrehozása
```powershell
PS C:\>$peering = Get-AzPeering -ResourceGroupName $resourceGroupName -Name $name
PS C:\>$peering | New-AzPeeringRegisteredAsn -Name $asnName -Asn $asn
```

Szerezze be azt a társviszonyt, amelybe regisztrált asn-t szeretne felvenni. Ezután adja át a parancsmagnak.

### Regisztrált asn létrehozása társviszony-erőforrásazonosítóval
```powershell
PS C:\>New-AzPeeringRegisteredAsn -ResourceId $resourceId -Name $asnName -Asn $asn
```
