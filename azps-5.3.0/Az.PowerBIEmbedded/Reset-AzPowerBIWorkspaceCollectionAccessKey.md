# Reset-AzPowerBIWorkspaceCollectionAccessKey

## SYNOPSIS
A megadott hozzáférési kulcs alaphelyzetbe állítása.

## SZINTAXIS

```
Reset-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Reset-AzPowerBIWorkspaceCollectionAccessKey** parancsmag alaphelyzetbe állítja a megadott hozzáférési kulcsot a Power BI-munkaterületi gyűjteményben.

## PÉLDÁK

### 1. példa: Az elsődleges hozzáférési kulcs alaphelyzetbe állítása
```
PS C:\>Reset-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Ez a parancs alaphelyzetbe állítja a WCN11 nevű munkaterület-gyűjtemény elsődleges hozzáférési kulcsát a megadott erőforráscsoportban.
