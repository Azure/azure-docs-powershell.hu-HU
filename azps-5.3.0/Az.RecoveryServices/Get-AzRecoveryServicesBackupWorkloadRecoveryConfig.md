# Get-AzRecoveryServicesBackupWorkloadRecoveryConfig

## SYNOPSIS
Ez a parancs egy biztonságiolt elem, például az SQL DB helyreállítási konfigurációját építi fel. A konfigurációs objektum minden részletet tárol, például a helyreállítási módot, a visszaállítási és alkalmazásspecifikus paraméterek célhelyét, például az SQL cél fizikai útvonalát.

## SZINTAXIS

### RpParameterSet (alapértelmezett)
```
Get-AzRecoveryServicesBackupWorkloadRecoveryConfig [[-RecoveryPoint] <RecoveryPointBase>]
 [[-TargetItem] <ProtectableItemBase>] [[-Item] <ItemBase>] [-OriginalWorkloadRestore]
 [-AlternateWorkloadRestore] [-TargetContainer <ContainerBase>] [-RestoreAsFiles]
 [-FromFull <RecoveryPointBase>] [-FilePath <String>] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### LogChainParameterSet
```
Get-AzRecoveryServicesBackupWorkloadRecoveryConfig [[-PointInTime] <DateTime>]
 [[-TargetItem] <ProtectableItemBase>] [[-Item] <ItemBase>] [-OriginalWorkloadRestore]
 [-AlternateWorkloadRestore] [-TargetContainer <ContainerBase>] [-RestoreAsFiles]
 [-FromFull <RecoveryPointBase>] [-FilePath <String>] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A parancs visszaadja a visszaállítási parancsmagnak átadott AzureWorkload-elemek helyreállítási konfigurációját.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $SQLRecoveryObject = Get-AzRecoveryServicesBackupRecoveryPoint -Item $SQLBkpItem $startdate $enddate | Get-AzRecoveryServicesWorkloadRecoveryConfig -OriginalWorkloadRestore
PS C:\> $SQLRecoveryObject = Get-AzRecoveryServicesBackupRecoveryPoint -Item $SQLBkpItem $startdate $enddate | Get-AzRecoveryServicesWorkloadRecoveryConfig -AlternateWorkloadRestore -TargetItem $SQLProtItem
```

A rendszer az első parancsmagot használja a Helyreállítási pont objektum beszerzése során.
A második parancsmag létrehoz egy helyreállítási tervet az eredeti hely-visszaállításhoz.
A harmadik parancsmag létrehoz egy helyreállítási tervet egy másik hely-visszaállításhoz.

### 2. példa

Ez a parancs egy biztonságiolt elem, például az SQL DB helyreállítási konfigurációját építi fel. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
Get-AzRecoveryServicesBackupWorkloadRecoveryConfig -AlternateWorkloadRestore -RecoveryPoint $rp[0] -TargetItem <ProtectableItemBase> -VaultId $vault.ID
```
