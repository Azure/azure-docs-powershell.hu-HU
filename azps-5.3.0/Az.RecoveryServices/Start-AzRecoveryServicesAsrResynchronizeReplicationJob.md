# Start-AzRecoveryServicesAsrResynchronizeReplicationJob

## SYNOPSIS
Replikációs újraszinkronizálás indítása.

## SZINTAXIS

### Alapértelmezett (alapértelmezett)
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Start-AzRecoveryServicesAsrResynchronizeReplication Parancsmag** elindítja a megadott védett elem replikációs újraszinkronizálását, ha a védett elem újraszinkronizálási szükséges állapotban van.

## PÉLDÁK

### 1. példa
```
PS C:\> Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem $rpi
```

Elindítja a replikációs újraszinkronizálási feladatot a több replikációval védett elemen.
