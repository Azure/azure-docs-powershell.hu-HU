# New-AzRedisCacheKey

## SYNOPSIS
A redis-gyorsítótár hozzáférési kulcsának újragenerálása.

## SZINTAXIS

```
New-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> -KeyType <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzRedisCacheKey** parancsmag újragenerálja az Azure Redis-gyorsítótár hozzáférési kulcsát.

## PÉLDÁK

### 1. példa: Az elsődleges kulcs újragenerálása
```
PS C:\>New-AzRedisCacheKey -ResourceGroupName "ResourceGroup03" -Name "myCache" -KeyType "Primary" -Force

          PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=

          SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Ez a parancs újra létrehozza a redis-gyorsítótár elsődleges kulcsát.

### 2. példa: Másodlagos kulcs újragenerálása
```
PS C:\>New-AzRedisCacheKey -ResourceGroupName "ResourceGroup03" -Name "myCache" -KeyType "Secondary" -Force
          PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=

          SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Ez a parancs a redis-gyorsítótár másodlagos kulcsát újra létrehozza.
