# Remove-AzRedisCache

## SYNOPSIS
A redis-gyorsítótár eltávolítása.

## SZINTAXIS

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzRedisCache** parancsmag eltávolítja az Azure Redis-gyorsítótárat.

## PÉLDÁK

### 1. példa: A redis cache eltávolítása és az eredmény visszaadése
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Ez a parancs eltávolítja a redis cache-t, és megjeleníti, hogy a művelet sikeres-e.

### 2. példa: A redis cache eltávolítása, és az eredmény nem jelenik meg
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Ez a parancs eltávolítja a redis-gyorsítótárat.
Mivel a *PassThru* paraméter nincs megadva, a művelet eredménye nem jelenik meg.
