# Remove-AzRedisCacheFirewallRule

## SYNOPSIS
Tűzfalszabály eltávolítása a redis-gyorsítótárból

## SZINTAXIS

### NormalParameterSet (alapértelmezett)
```
Remove-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> -RuleName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PSRedisFirewallRuleObject
```
Remove-AzRedisCacheFirewallRule -InputObject <PSRedisFirewallRule> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Tűzfalszabály eltávolítása a redis-gyorsítótárból

## PÉLDÁK

### 1. példa: Egyetlen tűzfalszabály eltávolítása
```
PS C:\>Remove-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone" -PassThru
True
```

Ez a parancs eltávolítja a ruleone nevű tűzfalszabályt a mycache nevű redis gyorsítótárból.
