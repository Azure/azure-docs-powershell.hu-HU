# Remove-AzWcfRelay

## SYNOPSIS
Eltávolítja a WcfRelay tartományt a megadott Továbbítás névterből.

## SZINTAXIS

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzWcfRelay** parancsmag eltávolítja a WcfRelay értéket a megadott Továbbítás névtérből.

## PÉLDÁK

### 1. példa
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

A WcfRelay eltávolítása `TestWCFRelay1` a névtérből. `TestNameSpace-Relay1`
