# Get-AzADGroup

## SYNOPSIS
Az Active Directory-csoportok szűrése.

## SZINTAXIS

### EmptyParameterSet (alapértelmezett)
```
Get-AzADGroup [-ObjectId <Guid>] [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### SearchStringParameterSet
```
Get-AzADGroup -DisplayNameStartsWith <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzADGroup -DisplayName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Get-AzADGroup -ObjectId <Guid> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount] [-Skip <UInt64>]
 [-First <UInt64>] [<CommonParameters>]
```

## LEÍRÁS
Az Active Directory-csoportok szűrése.

## PÉLDÁK

### 1. példa: Az összes AD-csoport felsorolása
```powershell
PS C:\> Get-AzADGroup
```

Egy bérlő összes AD-csoportját felsorolja.

### 2. példa: Az összes AD-csoport felsorolása lapozás használatával

```powershell
PS C:\> Get-AzADGroup -First 100
```

A bérlői webhely első 100 AD-csoportjait sorolja fel.

### 3. példa: Az AD-csoport lekérte objektumazonosító szerint

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

A (85F89C90-780E-4AA6-9F4F-6F268D322EEE) objektumazonosítójú AD-csoportot kapja.

### 4. példa: Csoportok felsorolása keresési karakterlánc alapján

```powershell
PS C:\> Get-AzADGroup -SearchString Joe
```

Azokat az AD-csoportokat sorolja fel, amelyeknek a megjelenítendő neve "József" névvel kezdődik.
