# Get-AzManagedApplication

## SYNOPSIS
Felügyelt alkalmazások lekérte

## SZINTAXIS

### GetBySubscription (alapértelmezett)
```
Get-AzManagedApplication [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByNameAndResourceGroup
```
Get-AzManagedApplication [-Name <String>] -ResourceGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzManagedApplication -Id <String> [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzManagedApplication** parancsmag felügyelt alkalmazásokat kap

## PÉLDÁK

### 1. példa: Az összes felügyelt alkalmazás lekérte egy erőforráscsoportba
```
PS C:\>Get-AzManagedApplication -ResourceGroupName "MyRG"
```

Ez a parancs a felügyelt alkalmazásokat a "MyRG" erőforráscsoportban kapja meg.

### 2. példa: Az összes felügyelt alkalmazás lekérte
```
PS C:\>Get-AzManagedApplication
```

Ez a parancs az aktuális előfizetés alatt lévő összes felügyelt alkalmazást lekérte
