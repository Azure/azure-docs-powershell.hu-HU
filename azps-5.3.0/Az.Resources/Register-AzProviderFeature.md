# Register-AzProviderFeature

## SYNOPSIS
Regisztrál egy Azure-szolgáltató funkciót a fiókjában.

## SZINTAXIS

```
Register-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Register-AzProviderFeature parancsmag** regisztrál egy Azure-szolgáltató funkciót a fiókjában.

## PÉLDÁK

### 1. példa: Funkció regisztrálása
```
PS C:\>Register-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Ez hozzáadja a Microsoft.Network AllowApplicationSecurityGroups funkcióját a fiókjához.
