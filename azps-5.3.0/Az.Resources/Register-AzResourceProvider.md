# Register-AzResourceProvider

## SYNOPSIS
Regisztrál egy erőforrás-szolgáltatót.

## SZINTAXIS

```
Register-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Register-AzResourceProvider** parancsmag regisztrál egy Azure-erőforrásszolgáltatót.

## PÉLDÁK

### 1. példa: Szolgáltató regisztrálása
```
PS C:\>Register-AzResourceProvider -ProviderNamespace Microsoft.Network
```

Ezzel regisztrálja a Microsoft.Network szolgáltatót a fiókjához.
