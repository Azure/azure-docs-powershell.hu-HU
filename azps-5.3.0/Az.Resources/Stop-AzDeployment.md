# Stop-AzDeployment

## SYNOPSIS
Futó telepítés megszakítása

## SZINTAXIS

### StopByDeploymentName (alapértelmezett)
```
Stop-AzDeployment [-Name] <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzDeployment -Id <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzDeployment -InputObject <PSDeployment> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Stop-AzDeployment** parancsmag megszakítja a telepítést az előfizetés hatókörében, amely elkezdődött, de nem fejeződött be.
Az üzembe helyezés befejezéséhez a központi telepítésnek hiányos kiépítési állapotnak (például Provisioning) kell lennie, nem pedig befejezettnek (például Kiépítve vagy Sikertelen).

Ha az előfizetés hatókörében létre kell hoznia egy központi telepítést, használja a New-AzDeployment parancsmagot.

Ez a parancsmag csak egy futó telepítést leállít. A Name *paraméter* használatával leállíthatja az adott telepítést.

## PÉLDÁK

### 1. példa
```
PS C:\>Stop-AzDeployment -Name "deployment01"
```

Ez a parancs megszakítja a futó telepítés "deployment01" futását az aktuális előfizetési hatókörben.

### 2. példa
```
PS C:\>Get-AzDeployment -Name "deployment01" | Stop-AzDeployment
```

Ez a parancs a központi telepítést a jelenlegi előfizetési hatókörben "deployment01" (központi telepítés) kapja meg, és lemondja azt.
