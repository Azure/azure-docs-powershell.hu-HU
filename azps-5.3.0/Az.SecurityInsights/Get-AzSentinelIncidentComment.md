# Get-AzSentinelIncidentComment

## SYNOPSIS
Incidens megjegyzésének bekérte.

## SZINTAXIS

### IncidentId (default)
```
Get-AzSentinelIncidentComment -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IncidentCommentId
```
Get-AzSentinelIncidentComment -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String>
 -IncidentCommentId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSentinelIncidentComment -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSentinelIncidentComment parancsmag** incidensmegsértő megjegyzést kap a megadott munkaterületről.
Ha megadja az *IncidentCommentId* és *az IncidentId* paramétert, egyetlen **IncidentComment** objektumot ad vissza.
Ha nem adja meg az *IncidentCommentId* paramétert, a megadott munkaterületen visszaadja a megadott Incidens megjegyzéseit tartalmazó tömböt.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $IncidentComments = Get-AzSentinelIncidentComment -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
```

Ebben a példában a megadott incidensre vonatkozó összes **Incidenscomments** adatokat megkapja a megadott munkaterületen, majd a $IncidentComments tárolja.

### 2. példa
```powershell
PS C:\> $IncidentComment = Get-AzSentinelIncidentComment -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId" -IncidentCommentId "MyIncidentCommentId"
```

Ez a példa egy **Incidenscommentet kap** a megadott incidenshez a megadott munkaterületen, majd tárolja azt a $IncidentComment változóban.
