# Remove-AzSentinelIncident

## SYNOPSIS
Incidens törlése.

## SZINTAXIS

### IncidentId (default)
```
Remove-AzSentinelIncident -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelIncident -InputObject <PSSentinelIncident> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzSentinelIncident** parancsmag véglegesen töröl egy incidenst egy adott munkaterületről.
Az Incidens **objektumokat** átadhatja a folyamat műveleti jelével, vagy megadhatja a szükséges paramétereket.
A Confirm paraméter és a Windows PowerShell $ConfirmPreference szabályozhatja, hogy a parancsmag megerősítést kér-e.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Remove-AzSentinelIncident -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
```

Ez a parancs eltávolítja az Incidenst a munkaterületről.
