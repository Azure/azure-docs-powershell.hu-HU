# Get-AzServiceBusQueue

## SYNOPSIS
A megadott szolgáltatásbusz-várólistához ad leírást.

## SZINTAXIS

```
Get-AzServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A megadott szolgáltatásbusz-várólistához ad leírást.


A várólista leírását adja eredményül.

### 2. példa
```
PS C:\> Get-AzServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Egy adott névtér várólistáinak listáját adja vissza. Alapértelmezés szerint 100 várólistát ad vissza, ha több mint 100 várólistát kell visszaadni, használja a -MaxCount Parameter paramétert.

### 3. példa
```
PS C:\> Get-AzServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Egy adott névtér első 150 várólistájának listáját adja eredményül.
