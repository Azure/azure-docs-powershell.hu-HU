# Get-AzServiceBusTopic

## SYNOPSIS
A megadott Service Bus-témakör leírását adja eredményül.

## SZINTAXIS

```
Get-AzServiceBusTopic [-ResourceGroupName <String>] [-Namespace <String>] [-Name <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzServiceBusTopic** parancsmag visszaadja a szolgáltatásbusz megadott névterének leírását.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -TopicName SB-Topic_exampl1

AutoDeleteOnIdle                    : P10675199DT2H48M5.4775807S

```

Az adott Service Bus névtérhez megadott témakör leírását adja eredményül.

### 2. példa
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Visszaadja az adott Service Bus névterhez kapcsolódó témakörök listáját. Alapértelmezés szerint 100 témakört ad vissza, ha több mint 100 témakört ad vissza, használja a -MaxCount Parameter paramétert.

### 3. példa
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Visszaadja az adott Service Bus névtér első 150 témakörének listáját.
