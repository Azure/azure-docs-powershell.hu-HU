# Enable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Speciális adatbiztonságot tesz lehetővé felügyelt példányon.

## SZINTAXIS

```
Enable-AzSqlInstanceAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzSqlInstanceAdvancedDataSecurity** parancsmag lehetővé teszi a speciális adatbiztonságot egy felügyelt példányon. A Speciális adatbiztonság egy egyesített biztonsági csomag, amely tartalmazza a felügyelt példány adatbesorolását, a biztonsági rés felmérését és a komplex veszélyforrások elleni védelmet. (A rendszer automatikusan létrehoz egy új tárfiókot a biztonsági rések felmérésére. Ha korábban erre a célra hoztak létre tárfiókot, azt használja a rendszer.

## PÉLDÁK

### 1. példa: Felügyelt példány speciális adatbiztonságának engedélyezése
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### 2. példa: Felügyelt példány speciális adatbiztonságának engedélyezése kiszolgálóerőforrásból
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### 3. példa

Speciális adatbiztonságot tesz lehetővé felügyelt példányon. (automatikusan generált)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlInstanceAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -InstanceName 'ContosoManagedInstanceName' -ResourceGroupName MyResourceGroup
```
