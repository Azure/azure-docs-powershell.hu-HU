# Enable-AzSqlServerActiveDirectoryOnlyAuthentication

## SYNOPSIS
Az Azure AD-hitelesítést csak egy adott SQL Serverhez engedélyezi.

## SZINTAXIS

### UseResourceGroupAndServerNameParameterSet (alapértelmezett)
```
Enable-AzSqlServerActiveDirectoryOnlyAuthentication [-ResourceGroupName] <String> [-ServerName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UseInputObjectParameterSet
```
Enable-AzSqlServerActiveDirectoryOnlyAuthentication -InputObject <AzureSqlServerModel>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UserResourceIdParameterSet
```
Enable-AzSqlServerActiveDirectoryOnlyAuthentication [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzSqlServerActiveDirectoryOnlyAuthentication** parancsmag csak az aktuális előfizetésben lévő AzureSQL-kiszolgálóhoz engedélyezi az Azure Active Directory (Azure AD) hitelesítését.

## PÉLDÁK

### 1. példa
```powershell
PS C:\>Enable-AzSqlServerActiveDirectoryOnlyAuthentication -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ResourceGroupName ServerName AzureADOnlyAuthentication
----------------- ---------- ----------- -------- -----------
ResourceGroup01   Server01   True
```

Ez a parancs lehetővé teszi, hogy az Azure Active Directory (Azure AD) csak hitelesítést követel meg egy Server01 nevű AzureSQL-kiszolgálóhoz, amely egy ResourceGroup01 nevű erőforráscsoporthoz van társítva.
