# Enable-AzSqlServerAdvancedDataSecurity

## SYNOPSIS
Engedélyezi a speciális adatbiztonságot a kiszolgálón.

## SZINTAXIS

```
Enable-AzSqlServerAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzSqlServerAdvancedDataSecurity** parancsmag lehetővé teszi a speciális adatbiztonságot a kiszolgálón. A Speciális adatbiztonság egy egységes biztonsági csomag, amely magában foglalja a kiszolgáló adatbesorolását, a biztonsági rés felmérését és a komplex veszélyforrások elleni védelmet. (Automatikusan létrejön egy új tárfiók a biztonsági rések felmérésére. Ha korábban erre a célra hoztak létre tárfiókot, azt használja a rendszer.

## PÉLDÁK

### 1. példa: A kiszolgáló speciális adatbiztonságának engedélyezése
```powershell
PS C:\>  Enable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### 2. példa: A kiszolgáló speciális adatbiztonságának engedélyezése kiszolgálóerőforrásból
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### 3. példa

Engedélyezi a speciális adatbiztonságot a kiszolgálón. (automatikusan generált)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlServerAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -ResourceGroupName MyResourceGroup -ServerName s1
```
