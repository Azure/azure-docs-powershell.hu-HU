# Get-AzSqlDatabaseTransparentDataEncryptionActivity

## SYNOPSIS
Beolvassa egy adatbázis TDE-vizsgálatának előrehaladását.

## SZINTAXIS

```
Get-AzSqlDatabaseTransparentDataEncryptionActivity [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlDatabaseTransparentDataEncryptionActivity** parancsmag egy Azure SQL-adatbázis TDE-vizsgálatának folyamatán megy keresztül.
Ha nem fut titkosítási időtartam, ez a parancsmag egy üres listát ad vissza.
További információt az Átlátszó adattitkosítás az Azure SQL-adatbázissal https://msdn.microsoft.com/library/dn948096 ( a Microsoft Developer Network Library https://msdn.microsoft.com/library/dn948096) webhelyén) című cikk tartalmaz.
Ezt a parancsmagot az Azure SQL Server Stretch Database szolgáltatása is támogatja.

## PÉLDÁK

### 1. példa: TDE-tevékenység lekésése adatbázishoz
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryptionActivity -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName : resourcegroup01
ServerName        : server01
DatabaseName      : database01
Status            : Encrypting
PercentComplete   : 3.662109
```

Ez a parancs a database01 nevű adatbázis TDE-tevékenységét kapja meg a kiszolgáló01 nevű kiszolgálón.
