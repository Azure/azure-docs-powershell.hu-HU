# Get-AzSqlElasticPoolAdvisor

## SYNOPSIS
Egy vagy több tanácsadót kap egy Azure SQL-rugalmassági készlethez.

## SZINTAXIS

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlElasticPoolAdvisor** parancsmag egy vagy több Azure SQL rugalmassági készlet tanácsadót kap egy Azure SQL rugalmassági készlethez.
