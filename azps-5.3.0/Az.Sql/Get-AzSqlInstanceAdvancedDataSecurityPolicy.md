# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## SYNOPSIS
Felügyelt példány speciális adatbiztonsági házirendet kap.

## SZINTAXIS

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlInstanceAdvancedDataSecurityPolicy** parancsmag beolvassa egy felügyelt példány Speciális adatbiztonsági házirendet.

## PÉLDÁK

### 1. példa: Felügyelt példány speciális adatbiztonságának lekérte
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### 2. példa: Felügyelt példány speciális adatbiztonságát kapja a felügyelt példány erőforrástól
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
