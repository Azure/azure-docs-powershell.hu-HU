# Remove-AzSqlDatabase

## SYNOPSIS
Eltávolít egy Azure SQL-adatbázist.

## SZINTAXIS

```
Remove-AzSqlDatabase [-DatabaseName] <String> [-Force] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzSqlDatabase** parancsmag eltávolít egy Azure SQL-adatbázist.
Ezt a parancsmagot az Azure SQL Server Stretch Database szolgáltatása is támogatja.

## PÉLDÁK

### 1. példa: Adatbázis eltávolítása Azure SQL-kiszolgálóról
```
PS C:\>Remove-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Ez a parancs eltávolítja az Adatbázis01 nevű adatbázist a kiszolgáló Kiszolgáló01 kiszolgálóról.
