# Get-AzSqlVMGroup

## SYNOPSIS
Egy vagy több sql virtuális gépcsoportot kap.

## SZINTAXIS

### ResourceGroupOnly (alapértelmezett)
```
Get-AzSqlVMGroup [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### név
```
Get-AzSqlVMGroup [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceId
```
Get-AzSqlVMGroup [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Get-AzSqlVMGroup parancsmag egy vagy több sql virtuális gépcsoportot kap.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzSqlVMGroup

Name       ResourceGroupName  Sku       Offer
----       -----------------  ---       -----
test-group ResourceGroup01    Developer SQL2017-WS2016
group1     ResourceGroup02    Developer SQL2017-WS2016
```

Ez a parancs információkat kap az aktuális előfizetés összes Azure SQL virtuális gépcsoportról.

### 2. példa
```powershell
PS C:\> Get-AzSqlVMGroup -ResourceGroupName "ResourceGroup01"
Name       ResourceGroupName  Sku       Offer
----       -----------------  ---       -----
test-group ResourceGroup01    Developer SQL2017-WS2016
```

Ez a parancs információkat kap az Aktuális előfizetés Azure SQL virtuális gépcsoportjairól az Erőforráscsoport01 erőforráscsoporthoz hozzárendelve.

### 3. példa
```powershell
PS C:\> Get-AzSqlVMGroup -ResourceGroupName "ResourceGroup01" -Name "test-group"
Name       ResourceGroupName  Sku       Offer
----       -----------------  ---       -----
test-group ResourceGroup01    Developer SQL2017-WS2016
```

Ez a parancs információkat kap az ResourceGroup01 erőforráscsoporthoz hozzárendelt SQL virtuális gépcsoport "tesztcsoport" adatairól.
