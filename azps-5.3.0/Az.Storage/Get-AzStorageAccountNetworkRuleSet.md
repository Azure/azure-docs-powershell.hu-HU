# Get-AzStorageAccountNetworkRuleSet

## SYNOPSIS
Tárfiók NetWorkRule tulajdonságának lekérte

## SZINTAXIS

```
Get-AzStorageAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzStorageAccountNetworkRuleSet** parancsmag egy tárfiók NetworkRule tulajdonságát kapja meg

## PÉLDÁK

### 1. példa: Egy adott tárfiók NetworkRule tulajdonságának be szereznie
```
PS C:\> Get-AzStorageAccountNetworkRuleSet  -ResourceGroupName "rg1" -AccountName "mystorageaccount"
```

Ez a parancs egy adott tárfiók NetworkRule tulajdonságát kapja meg
