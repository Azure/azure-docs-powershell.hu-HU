# Get-AzStorageSyncCloudEndpoint

## SYNOPSIS
Ez a parancs felsorolja egy adott szinkronizálási csoport összes felhőbeli végpontját.

## SZINTAXIS

### StringParameterSet (alapértelmezett)
```
Get-AzStorageSyncCloudEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ObjectParameterSet
```
Get-AzStorageSyncCloudEndpoint [-ParentObject] <PSSyncGroup> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentStringParameterSet
```
Get-AzStorageSyncCloudEndpoint [-ParentResourceId] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Ez a parancs felsorolja egy adott szinkronizálási csoport összes felhőbeli végpontját. Az egyes felhőbeli végpontok attribútumainak felsorolásához is használható.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzStorageSyncCloudEndpoint -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName"
```

Ez a parancs a megadott szinkronizálási csoportban található összes felhőbeli végpontot lekérte. A -CloudEndpointName érték megadásával adott értéket ad vissza.
