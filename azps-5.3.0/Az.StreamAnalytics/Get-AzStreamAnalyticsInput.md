# Get-AzStreamAnalyticsInput

## SYNOPSIS
Stream Analytics-feladatokkal bevitt adatokat kap.

## SZINTAXIS

```
Get-AzStreamAnalyticsInput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzStreamAnalyticsInput** parancsmag felsorolja a Stream Analytics-feladatban definiált összes bemenetet, vagy információt kap egy adott bemenetről.

## PÉLDÁK

### 1. példa: Információk lekérte a feladathoz definiált bemeneteket
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Ez a parancs információkat ad vissza a Streaming Streaming Streaming feladaton definiált összes bemenetről.

### 2. példa: Információ lekérte egy adott bemenetről egy feladathoz
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Ez a parancs információkat ad vissza a Stream Streaming Streaming feladaton definiált EntryStream bemenetről.
