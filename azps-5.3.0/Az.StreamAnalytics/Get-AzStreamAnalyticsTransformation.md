# Get-AzStreamAnalyticsTransformation

## SYNOPSIS
Információkat kap a Stream Analytics-feladatátalakításról.

## SZINTAXIS

```
Get-AzStreamAnalyticsTransformation [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzStreamAnalyticsTransformation** parancsmag információt kap a Stream Analytics-feladaton definiált átalakításról.

## PÉLDÁK

### 1. PÉLDA: Információ a Stream Analytics átalakításáról
```
PS C:\>Get-AzStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "StreamingJob"
```

Ez a parancs információkat ad vissza a Streaming Streaming Streaming nevű átalakításról a Streaming Streaming Nevű feladaton.
