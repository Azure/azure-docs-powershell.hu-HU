# Test-AzStreamAnalyticsInput

## SYNOPSIS
Egy bevitel kapcsolati állapotát teszteli.

## SZINTAXIS

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Test-AzStreamAnalyticsInput** parancsmag teszteli, hogy a Stream Analytics képes-e csatlakozni egy bemenethez.

## PÉLDÁK

### 1. példa: Bemeneti adatfolyam kapcsolati állapotának tesztelése
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Ez teszteli az input EntryStream kapcsolati állapotát a StreamingStreamben.
