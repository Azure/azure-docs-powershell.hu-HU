# Get-AzWebAppBackup

## SYNOPSIS

## SZINTAXIS

### FromResourceName
```
Get-AzWebAppBackup [-BackupId] <String> [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FromWebApp
```
Get-AzWebAppBackup [-BackupId] <String> [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzWebAppBackup** parancsmag lekérte az Azure Web App megadott biztonsági másolatát.

## PÉLDÁK

### 1. példa
```powershell
PS C:\>Get-AzWebAppBackup -ResourceGroupName "Default-Web-WestUS" -Name "WebAppStandard" -BackupId "12345"
```

Ez a parancs "12345" azonosítójú biztonsági másolatot kap a Default-Web-WestUS erőforráscsoporthoz tartozó WebAppStandard nevű webappból.
