# Remove-AzWebApp

## SYNOPSIS
Eltávolít egy Azure Web Appot.

## SZINTAXIS

### S1
```
Remove-AzWebApp [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebApp [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzWebApp** parancsmag eltávolít egy Azure Web Appot, feltéve hogy az erőforráscsoport és a Web App neve meg van téve.
Ez a parancsmag alapértelmezés szerint az összes résidőt és metrikát is eltávolítja.

## PÉLDÁK

### 1. példa: Webalkalmazás eltávolítása
```
PS C:\>Remove-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Ez a parancs eltávolítja a ContosoSite nevű Azure Web Appot, amely a Default-Web-WestUS nevű erőforráscsoporthoz tartozik.
