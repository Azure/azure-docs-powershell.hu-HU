# Start-AzWebApp

## SYNOPSIS
Elindít egy Azure Web Appot.

## SZINTAXIS

### S1
```
Start-AzWebApp [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Start-AzWebApp [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Start-AzWebApp** parancsmag elindít egy Azure Web Appot.

## PÉLDÁK

### 1. példa: Webalkalmazás kezdése
```
PS C:\>Start-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp"
```

Ez a parancs elindítja a ContosoWebApp nevű webappot, amely a Default-Web-WestUS nevű erőforráscsoporthoz tartozik.
