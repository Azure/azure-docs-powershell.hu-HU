# Get-AzApiManagementUser

## SYNOPSIS
Felhasználót vagy felhasználókat kap.

## SZINTAXIS

### GeAllUsers (alapértelmezett)
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByUserId
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-UserId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByUser
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-FirstName <String>] [-LastName <String>]
 [-State <PsApiManagementUserState>] [-Email <String>] [-GroupId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApiManagementUser** parancsmag egy megadott felhasználót vagy az összes felhasználót megkapja, ha nincs megadva felhasználó.

## PÉLDÁK

### 1. példa: Az összes felhasználó lekérte
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext
```

Ez a parancs az összes felhasználót beveszi.

### 2. példa: Felhasználó lekérte azonosító alapján
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -UserId "0123456789"
```

Ez a parancs azonosító alapján kap egy felhasználót.

### 3. példa: Felhasználók be szerezni vezetéknév szerint
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -LastName "Fuller"
```

Ez a parancs a megadott vezetéknévvel (Fuller) használó felhasználókat kapja meg.

### 4. példa: Felhasználó lekérte e-mail-cím alapján
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -Email "user@contoso.com"
```

Ez a parancs a megadott e-mail-címmel rendelkezik.

### 5. példa: Egy csoport összes felhasználója
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -GroupId "0001"
```

Ez a parancs a megadott csoport összes felhasználóját beveszi.
