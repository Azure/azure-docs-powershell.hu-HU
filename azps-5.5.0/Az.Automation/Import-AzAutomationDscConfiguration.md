# Import-AzAutomationDscConfiguration

## SYNOPSIS
DSC-konfigurációt importál az automatizálásba.

## SZINTAXIS

```
Import-AzAutomationDscConfiguration -SourcePath <String> [-Tags <IDictionary>] [-Description <String>]
 [-Published] [-Force] [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Import-AzAutomationDscConfiguration** parancsmag az APS kívánt államkonfigurációs (DSC) konfigurációját importálja az Azure Automationbe.
Adja meg egy APS-parancsfájl elérési útját, amely egyetlen DSC-konfigurációt tartalmaz.

## PÉLDÁK

### 1. példa: DSC-konfiguráció importálása automatizálásba
```powershell
PS C:\>Import-AzAutomationDscConfiguration -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -SourcePath "C:\DSC\client.ps1" -Force
```

Ez a parancs importálja a DSC-konfigurációt a client.ps1 Contoso17 nevű automatizálási fiókba. A parancs a Force *paramétert* adja meg. Ha van meglévő DSC-konfiguráció, ez a parancs lecseréli azt.

### 2. példa

DSC-konfigurációt importál az automatizálásba. (automatikusan generált)

<!-- Aladdin Generated Example -->
```powershell
Import-AzAutomationDscConfiguration -AutomationAccountName 'Contoso17' -Published -ResourceGroupName 'ResourceGroup01' -SourcePath 'C:\DSC\client.ps1'
```
