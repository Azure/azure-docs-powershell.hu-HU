# New-AzAutomationAccount

## SYNOPSIS
Létrehoz egy automatizálási fiókot.

## SZINTAXIS

```
New-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-Plan <String>]
 [-Tags <IDictionary>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzAutomationAccount** parancsmag létrehoz egy Azure Automation-fiókot egy erőforráscsoportban.
Az automatizálási fiók az automatizálási erőforrások tárolója, amely elkülönül a többi automatizálási fiók erőforrásaitól. Az automatizálási erőforrások közé tartoznak a runbookok, a kívánt állapotkonfigurációk, feladatok és eszközök.

## PÉLDÁK

### 1. példa: Automatizálási fiók létrehozása
```powershell
PS C:\> New-AzAutomationAccount -Name "ContosoAutomationAccount" -Location "East US" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs létrehoz egy ContosoAutomationAccount nevű új automatizálási fiókot az Egyesült Államok keleti régiójában.

### 2. példa

Létrehoz egy automatizálási fiókot. (automatikusan generált)

<!-- Aladdin Generated Example -->
```powershell
New-AzAutomationAccount -Location 'East US' -Name 'ContosoAutomationAccount' -ResourceGroupName 'ResourceGroup01' -Tags <IDictionary>
```
