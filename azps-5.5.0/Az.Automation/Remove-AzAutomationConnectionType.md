# Remove-AzAutomationConnectionType

## SYNOPSIS
Eltávolít egy automatizálási kapcsolattípust.

## SZINTAXIS

```
Remove-AzAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzAutomationConnectionType** parancsmag eltávolítja a kapcsolattípusokat az Azure Automationből.
A törölt kapcsolattípussal társított összes kapcsolat használhatatlanná válik.
Távolítsa el őket, hacsak nem hoz létre az alábbi feltételeknek megfelelő új kapcsolattípust: 
- A típus neve megegyezik az eredeti kapcsolattípus nevével. 
- A típusnak ugyanazok a meződefiníciói vannak, mint az eredeti kapcsolattípusnak.
További mezőket is tartalmazhat.

## PÉLDÁK

### 1. példa: Kapcsolattípus eltávolítása
```
PS C:\>Remove-AzAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs eltávolítja a ContosoConnectionType nevű kapcsolattípust a Contoso17 nevű automatizálási fiókból.
