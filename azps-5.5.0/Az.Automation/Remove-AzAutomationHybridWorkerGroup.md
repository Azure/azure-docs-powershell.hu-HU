# Remove-AzAutomationHybridWorkerGroup

## SYNOPSIS
Eltávolítja a hibrid dolgozók csoportját az automatizálásból.

## SZINTAXIS

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A Remove-AzAutomationHybridWorkerGroup parancsmag eltávolítja a hibrid dolgozók csoportját az automatizálásból.

## PÉLDÁK

### 1. példa
Ez a parancs név szerint eltávolítja a hibrid dolgozókat.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
