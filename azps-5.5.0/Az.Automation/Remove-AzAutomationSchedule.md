# Remove-AzAutomationSchedule

## SYNOPSIS
Törli az automatizálási ütemezést.

## SZINTAXIS

```
Remove-AzAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzAutomationSchedule** parancsmag törli az ütemezést az Azure Automation szolgáltatásból.

## PÉLDÁK

### 1. példa: Ütemezés eltávolítása
```
PS C:\>Remove-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs törli az Erőforráscsoport01 erőforráscsoport Contoso17 automatizálási fiókjában az Schedule01 nevű ütemezést.
