# Set-AzAutomationCertificate

## SYNOPSIS
Módosítja egy automatizálási tanúsítvány konfigurációját.

## SZINTAXIS

```
Set-AzAutomationCertificate [-Name] <String> [-Description <String>] [-Password <SecureString>]
 [-Path <String>] [-Exportable <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzAutomationCertificate** parancsmag módosítja egy tanúsítvány konfigurációját az Azure Automationben.

## PÉLDÁK

### 1. példa: Tanúsítvány módosítása
```
PS C:\>$Password = ConvertTo-SecureString -String "Password" -AsPlainText -Force
PS C:\> Set-AzAutomationCertificate -AutomationAccountName "Contos17" -Name "ContosoCertificate" -Path "./cert.pfx" -Password $Password -ResourceGroupName "ResourceGroup01"
```

Az első parancs egy egyszerű szöveges jelszót biztonságos karakterláncsá alakít át a ConvertTo-SecureString parancsmag használatával.
A parancs az objektumot a $Password tárolja.
A második parancs módosítja a ContosoCertificate nevű tanúsítványt.
A parancs a $Password.
A parancs megadja a fiók nevét és a feltöltött fájl elérési útját.
