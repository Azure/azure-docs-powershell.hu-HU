# Set-AzCdnEndpoint

## SYNOPSIS
Frissíti a CDN-végpontot.

## SZINTAXIS

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzCdnEndpoint** parancsmag frissíti az Azure Content Delivery Network (CDN) végpontot.
