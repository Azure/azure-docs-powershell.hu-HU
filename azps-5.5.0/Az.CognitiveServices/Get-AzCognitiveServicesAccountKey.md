# Get-AzCognitiveServicesAccountKey

## SYNOPSIS
Egy fiók API-kulcsait kapja meg.

## SZINTAXIS

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzCognitiveServicesAccountKey** parancsmag egy kiépített Kognitív Szolgáltatások-fiók API-kulcsait kapja meg.
A kognitív szolgáltatások fiókja két API-kulccsal rendelkezik: Key1 és Key2.
A kulcsok lehetővé teszik a kognitív szolgáltatások fiók végpontjának interakcióját.
A New-AzCognitiveServicesAccountKey kulcs újragenerálva használható.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
