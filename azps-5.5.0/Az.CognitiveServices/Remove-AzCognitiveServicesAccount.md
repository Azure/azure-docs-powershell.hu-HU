# Remove-AzCognitiveServicesAccount

## SYNOPSIS
A Kognitív szolgáltatások fiókjának törlése.

## SZINTAXIS

```
Remove-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzCognitiveServicesAccount** parancsmag törli a megadott Kognitív szolgáltatások-fiókot.

## PÉLDÁK

### 1. példa
Ez a parancs nem ad vissza semmit.

```powershell
PS C:\> Remove-AzCognitiveServicesAccount -ResourceGroupName cognitive-services-resource-group -name myluis
PS C:\>
```
