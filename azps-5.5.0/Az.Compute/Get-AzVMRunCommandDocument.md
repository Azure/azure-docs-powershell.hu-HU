# Get-AzVMRunCommandDocument

## SYNOPSIS
Futtatás parancsdokumentum létrehozása

## SZINTAXIS

```
Get-AzVMRunCommandDocument [-Location] <String> [[-CommandId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Futtatás parancsdokumentum létrehozása

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzVMRunCommandDocument -Location 'westus' -CommandId 'RunPowerShellScript'
```

Egy adott futtatott parancsdokumentumot kap a "RunPowerShellScript" parancshoz a "westus" nyelven.

### 2. példa
```
PS C:\> Get-AzVMRunCommandDocument -Location 'westus'
```

A "westus"-ban elérhető összes futtatás parancs.
