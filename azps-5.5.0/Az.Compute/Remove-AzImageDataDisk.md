# Remove-AzImageDataDisk

## SYNOPSIS
Eltávolít egy adatlemezt egy képobjektumból.

## SZINTAXIS

```
Remove-AzImageDataDisk [-Image] <PSImage> [-Lun] <Int32> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzImageDataDisk** parancsmag eltávolít egy adatlemezt egy képobjektumból.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' | Remove-AzImageDataDisk -Lun 1 | Update-AzImage;
```

Ez a parancs eltávolítja az 1. logikai egység adatlemezét az "ResourceGroup01" erőforráscsoport meglévő "Image01" képe alapján.
