# Remove-AzVMAccessExtension

## SYNOPSIS
Eltávolítja a VMAccess bővítményt egy virtuális gépről.

## SZINTAXIS

```
Remove-AzVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzVMAccessExtension** parancsmag eltávolítja a virtuális gépről a VIRTUÁLIS gép access (VMAccess) virtuális gép bővítményt.

## PÉLDÁK

### 1. példa

Eltávolítja a VMAccess bővítményt egy virtuális gépről. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzVMAccessExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
