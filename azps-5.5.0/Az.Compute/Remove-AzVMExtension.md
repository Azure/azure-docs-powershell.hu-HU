# Remove-AzVMExtension

## SYNOPSIS
Eltávolít egy bővítményt egy virtuális gépről.

## SZINTAXIS

```
Remove-AzVMExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzVMExtension** parancsmag eltávolít egy bővítményt a virtuális gép virtuális gép bővítményeiből.

## PÉLDÁK

### 1. példa: Bővítmény eltávolítása virtuális gépről
```
PS C:\> Remove-AzVMExtension -ResourceGroupName "ResourceGroup11" -Name "ContosoTest" -VMName "VirtualMachine22"
```

Ez a parancs eltávolítja a ContosoTest kiterjesztést a VirtualMachine22 nevű virtuális gépről az ResourceGroup11-ben.
