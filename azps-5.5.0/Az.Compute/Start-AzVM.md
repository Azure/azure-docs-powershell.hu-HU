# Start-AzVM

## SYNOPSIS
Elindít egy Azure virtuális gépet.

## SZINTAXIS

### ResourceGroupNameParameterSetName (alapértelmezett)
```
Start-AzVM [-Name] <String> [-NoWait] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-NoWait] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Start-AzVM parancsmag** elindít egy Azure virtuális gépet.

## PÉLDÁK

### 1. példa: Virtuális gép létrehozása
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Ez a parancs elindítja a VirtualMachine07 nevű virtuális gépet az ResourceGroup11-ben.
