# New-AzCosmosDBGremlinIncludedPathIndex

## SYNOPSIS
Új PSIndexes típusú objektumot hoz létre. Paraméterértékként a Set-AzCosmosDBGremlinIncludedPath paraméterértékeként is átadott.

## SZINTAXIS

```
New-AzCosmosDBGremlinIncludedPathIndex -DataType <String> [-Precision <Int32>] -Kind <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Gremlin API IncludedPath-indexének megfelelő objektum.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> New-AzCosmosDBGremlinIncludedPathIndex -DataType String -Precision -1 -Kind Hash

DataType Precision Kind
-------- --------- ----
String          -1 Hash
```
