# Test-AzDataLakeAnalyticsAccount

## SYNOPSIS
Ellenőrzi, hogy létezik-e Data Lake Analytics-fiók.

## SZINTAXIS

```
Test-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Test-AzDataLakeAnalyticsAccount** parancsmag ellenőrzi, hogy létezik-e Data Lake Analytics-fiók.

## PÉLDÁK

### 1. példa: Fiók létezik-e?
```
PS C:\>Test-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Ez a parancs teszteli, hogy létezik-e a ContosoAdlAccount nevű fiók.
