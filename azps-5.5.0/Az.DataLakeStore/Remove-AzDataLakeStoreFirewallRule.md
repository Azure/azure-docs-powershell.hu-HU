# Remove-AzDataLakeStoreFirewallRule

## SYNOPSIS
A megadott tűzfalszabály eltávolítása a megadott Data Lake Store-ból.

## SZINTAXIS

```
Remove-AzDataLakeStoreFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataLakeStoreFirewallRule** parancsmag eltávolítja a megadott tűzfalszabályt a megadott Data Lake Store-ból.

## PÉLDÁK

### 1. példa: Tűzfalszabály eltávolítása fiókból
```
PS C:\> Remove-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule
```

Eltávolítja a "MyFirewallRule" tűzfalszabályt a "ContosoADL" fiókból
