# Set-AzDataLakeStoreTrustedIdProvider

## SYNOPSIS
Módosítja a megadott megbízható identitásszolgáltatót a megadott Data Lake Store-ban.

## SZINTAXIS

```
Set-AzDataLakeStoreTrustedIdProvider [-Account] <String> [-Name] <String> [-ProviderEndpoint] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzDataLakeStoreTrustedIdProvider** parancsmag módosítja a megadott megbízható identitásszolgáltatót a megadott Data Lake Store-ban.

## PÉLDÁK

### 1. példa: Megbízható identitásszolgáltató végpontjának frissítése
```
PS C:\> Set-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider -ProviderEndpoint "https://sts.windows.net/6b04908c-b91f-40ce-8024-7ee8a4fd6150"
```

Ez a "ContosoADL" fiók "MyProvider" tűzfalszabályának szolgáltatói végpontját a következőre https://sts.windows.net/6b04908c-b91f-40ce-8024-7ee8a4fd6150 frissíti: "
