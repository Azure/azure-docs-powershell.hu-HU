# Start-AzHDInsightJob

## SYNOPSIS
Egy megadott Azure HDInsight-feladatot kezd egy adott fürtön.

## SZINTAXIS

```
Start-AzHDInsightJob [-ClusterName] <String> [-JobDefinition] <AzureHDInsightJobDefinition>
 [-HttpCredential] <PSCredential> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Start-AzHDInsight Job** parancsmag elindít egy definiált Azure HDInsight-feladatot egy adott fürtön.
Ez lehet MapReduce-feladat, Streaming MapReduce-feladat, Kaptár- vagy Vírusfólejátszási feladat.

## PÉLDÁK

### 1. példa: Feladat kezdése a megadott fürtön
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

# Hive job details
PS C:\> $statusFolder = "tempStatusFolder/"
PS C:\> $query = "SHOW TABLES"

PS C:\> New-AzHDInsightHiveJobDefinition -StatusFolder $statusFolder `
            -Query $query `
        | Start-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Ez a parancs elindít egy feladatot a "-hadoop-001" nevű fürtben.
