# Add-AzIotHubConfiguration

## SYNOPSIS
IoT automatikus eszközkezelési konfiguráció hozzáadása egy cél IoT-központban.

## SZINTAXIS

### ResourceSet (alapértelmezett)
```
Add-AzIotHubConfiguration [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 [-DeviceContent <Hashtable>] [-Priority <Int32>] [-TargetCondition <String>] [-Metric <Hashtable>]
 [-Label <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Add-AzIotHubConfiguration [-InputObject] <PSIotHub> -Name <String> [-DeviceContent <Hashtable>]
 [-Priority <Int32>] [-TargetCondition <String>] [-Metric <Hashtable>] [-Label <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Add-AzIotHubConfiguration [-ResourceId] <String> -Name <String> [-DeviceContent <Hashtable>]
 [-Priority <Int32>] [-TargetCondition <String>] [-Metric <Hashtable>] [-Label <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A konfigurációs tartalom json és kissé eltér az eszköz vagy a modul szándéka szerint. Az eszközkonfigurációk a következő alakban állnak: {"deviceContent":{...}}
A modulkonfigurációk a következő alakban állnak: {"moduleContent":{...}}
A konfigurációk a felhasználó által megadott, igény szerinti értékeléshez megadott metrikák alapján definiálhatóak.
A felhasználói metrikák: json és {"queries":{...}} vagy {"metrikák":{"lekérdezések":{...}}}.

Megjegyzés: A modulok célszabad feltételének "az devices.modules where" kezdettől kell kezdődnie. További https://docs.microsoft.com/azure/iot-hub/iot-hub-automatic-device-management információt itt is láthat.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Add-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1"
```

Eszközkonfiguráció létrehozása alapértelmezett metaadatokkal.

### 2. példa
```powershell
PS C:\> Add-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1" -Priority 3 -TargetCondition "tags.building=9 and tags.environment='test'"
```

Hozzon létre egy 3-as prioritású eszközkonfigurációt, amely akkor alkalmazható feltétellel, ha egy eszközt címkéznek a 9-es épületben, és a környezet "teszt".

### 3. példa
```powershell
PS C:\> $metrics = @{}
PS C:\> $metrics.add("query1", "select deviceId from devices where tags.location='US'")
PS C:\> Add-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1" -Metric $metrics
```

Felhasználói metrikákat használó eszközkonfiguráció létrehozása.

### 4. példa
```powershell
PS C:\> $labels = @{}
PS C:\> $labels.add("key0","value0")
PS C:\> $labels.add("key1","value1")
PS C:\> Add-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1" -Label $labels
```

Hozzon létre címkével egy eszközkonfigurációt.

### 5. példa
```powershell
PS C:\> $prop = @{}
PS C:\> $prop.add("Location", "US")
PS C:\> $content = @{}
PS C:\> $content.add("properties.desired.Region", $prop)
PS C:\> Add-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1" -DeviceContent $content
```

Hozzon létre tartalommal egy eszközkonfigurációt.
