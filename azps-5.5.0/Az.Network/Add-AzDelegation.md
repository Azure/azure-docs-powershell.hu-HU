# Add-AzDelegation

## SYNOPSIS
Delegálás hozzáadása alhálózathoz.

## SZINTAXIS

```
Add-AzDelegation -Name <String> -ServiceName <String> -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Az **Add-AzDelegation** parancsmag hozzáad egy szolgáltatásdelegálást egy Azure-alhálózathoz.

## PÉLDÁK

### 1. példa: Meghatalmazás hozzáadása
```powershell
PS C:\> $vnet = Get-AzVirtualNetwork -Name "myVNet" -ResourceGroupName "myResourceGroup"
PS C:\> $subnet = Get-AzVirtualNetworkSubnetConfig -Name "mySubnet" -VirtualNetwork $vnet
PS C:\> $subnet = Add-AzDelegation -Name "myDelegation" -ServiceName "Microsoft.Sql/servers" -Subnet $subnet
PS C:\> Set-AzVirtualNetwork $vnet
```

Az első parancs beolvassa azt a virtuális hálózatot, amelyen az alhálózat található. A második parancs elkülöníti az érdekes alhálózatot – azt, amelyiket delegálni szeretné. A harmadik parancs delegálást ad az alhálózathoz. Ez a példa akkor lehet hasznos, ha engedélyezni szeretné az SQL-nek, hogy felületi végpontokat hozzon létre ebben az alhálózatban. Az utolsó parancs elküldi a frissített alhálózatot a kiszolgálónak, hogy ténylegesen frissítse az alhálózatot.
