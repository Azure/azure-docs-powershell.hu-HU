# Get-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
Egy alkalmazás-átjáró háttér-HTTP-beállításait kapja meg.

## SZINTAXIS

```
Get-AzApplicationGatewayBackendHttpSetting [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Get-AzApplicationGatewayBackendHttpSetting parancsmag egy alkalmazás-átjáró háttér-HTTP-beállításait kapja meg.

## PÉLDÁK

### 1. példa: Háttér-HTTP-beállítások kérése név szerint
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Settings  = Get-AzApplicationGatewayBackendHttpSetting -Name "Settings01" -ApplicationGateway $AppGw
```

Az első parancs az ApplicationGateway01 nevű alkalmazás-átjárót az ResourceGroup01 nevű erőforráscsoportban tárolja a $AppGw változóban. A második parancs a Beállítások01 NEVŰ HTTP-beállításokat kapja $AppGw a beállításokat a $Settings változóban.

### 2. példa: Háttér-HTTP-beállítások gyűjteménye
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SettingsList  = Get-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw
```

Az első parancs az ApplicationGateway01 nevű alkalmazás-átjárót az ResourceGroup01 nevű erőforráscsoportban tárolja a $AppGw változóban. A második parancs a http-beállítások gyűjteményét $AppGw és a beállításokat a $SettingsList tárolja.
