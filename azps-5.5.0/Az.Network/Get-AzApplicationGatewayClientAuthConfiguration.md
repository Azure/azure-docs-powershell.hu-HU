# Get-AzApplicationGatewayClientAuthConfiguration

## SYNOPSIS
Ssl-profilobjektum ügyfél-hitelesítési konfigurációját kapja meg.

## SZINTAXIS

```
Get-AzApplicationGatewayClientAuthConfiguration -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayClientAuthConfiguration parancsmag** egy SSL-profilobjektum ügyfél-hitelesítési konfigurációját kapja meg.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SslProfile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
PS C:\> $ClientAuthConfig = Get-AzApplicationGatewayClientAuthConfiguration -SslProfile $SslProfile
```

Az első parancs az ApplicationGateway01 nevű alkalmazás-átjárót az ResourceGroup01 nevű erőforráscsoportban tárolja a $AppGw változóban. A második parancs az SslProfile01 nevű SSL-profilt kapja meg a $AppGw, és azt $SslProfile tárolja. Az utolsó parancs az SSL-profilból kapja meg az ügyfélhitelesítési $SslProfile és tárolja azt a $ClientAuthConfig változóban.
