# Get-AzApplicationGatewayFrontendPort

## SYNOPSIS
Egy alkalmazás átjárójának előlapi portját kapja meg.

## SZINTAXIS

```
Get-AzApplicationGatewayFrontendPort [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayFrontendPort** parancsmag egy alkalmazás-átjáró előtér-portját kapja meg.

## PÉLDÁK

### 1. példa: Adott előlapi port lekérte
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndPort = Get-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -ApplicationGateway $AppGw
```

Az első parancs egy ApplicationGateway01 nevű alkalmazás-átjárót kap az ResourceGroup01 nevű erőforráscsoportból, és a helyi $AppGw tárolja.
A második parancs a FrontEndPort01 nevű előlapi portot $AppGw a $FrontEndPort tárolja.

### 2. példa: Előlapi portok listájának lekérte
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndPorts = Get-AzApplicationGatewayFrontendPort  -ApplicationGateway $AppGw
```

Az első parancs egy ApplicationGateway01 nevű alkalmazás-átjárót kap az ResourceGroup01 nevű erőforráscsoportból, és a helyi $AppGw tárolja.
A második parancs lekérte az előlapi portok listáját a $AppGw és tárolja a $FrontEndPorts változóban.
