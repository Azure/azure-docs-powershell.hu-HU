# Get-AzApplicationGatewayIPConfiguration

## SYNOPSIS
Egy alkalmazás-átjáró IP-konfigurációját kapja meg.

## SZINTAXIS

```
Get-AzApplicationGatewayIPConfiguration [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayIPConfiguration** parancsmag egy alkalmazás-átjáró IP-konfigurációját kapja meg.
Az IP-konfiguráció tartalmazza azt az alhálózatot, amelyen az alkalmazás-átjáró telepítve van.

## PÉLDÁK

### 1. példa: Adott IP-konfiguráció beállítása
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $GatewaySubnet = Get-AzApplicationGatewayIPConfiguration -Name "GatewaySubnet01" -ApplicationGateway $AppGw
```

Az első parancs egy alkalmazás-átjárót kap, és a $AppGw tárolja. A második parancs egy GateSubnet01 nevű IP-konfigurációt kap a $AppGw.

### 2. példa: IP-konfigurációk listájának lekérte
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $GatewaySubnets = Get-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw
```

Az első parancs egy alkalmazás-átjárót kap, és a $AppGw tárolja. A második parancs az összes IP-konfiguráció listáját lekérte.
