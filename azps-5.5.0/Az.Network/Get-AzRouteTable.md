# Get-AzRouteTable

## SYNOPSIS
Útvonaltáblákat kap.

## SZINTAXIS

### NoExpand (alapértelmezett)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Kibontás
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzRouteTable** parancsmag azure-útválasztási táblákat kap.
Kaphat egyetlen útvonaltáblát, vagy lekértheti az összes útvonaltáblát egy erőforráscsoportban vagy az előfizetésében.
