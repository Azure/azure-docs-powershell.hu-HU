# Get-AzVirtualNetworkGatewaySupportedVpnDevice

## SYNOPSIS
Ez a parancsmag a támogatott VPN-eszközök márkáinak, modelljeinek és belső vezérlőprogram-verzióinak listáját adja vissza.

## SZINTAXIS

```
Get-AzVirtualNetworkGatewaySupportedVpnDevice -Name <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Ez a parancsmag a támogatott VPN-eszközök márkáinak, modelljeinek és belső vezérlőprogram-verzióinak listáját adja vissza.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzVirtualNetworkGatewaySupportedVpnDevice -ResourceGroupName TestRG -Name TestGateway 
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
```

A támogatott VPN-eszközök márkáinak, modelljeinek és belső vezérlőprogram-verzióinak listáját adja vissza:
<?xml version="1.0" encoding="utf-8"?>
