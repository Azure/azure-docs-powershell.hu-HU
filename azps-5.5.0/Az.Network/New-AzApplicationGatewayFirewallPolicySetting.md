# New-AzApplicationGatewayFirewallPolicySetting

## SYNOPSIS
Házirendbeállítás létrehozása a tűzfal házirendhez

## SZINTAXIS

```
New-AzApplicationGatewayFirewallPolicySetting [-Mode <String>] [-State <String>] [-DisableRequestBodyCheck]
 [-MaxRequestBodySizeInKb <Int32>] [-MaxFileUploadInMb <Int32>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzApplicationGatewayFirewallPolicySetting létrehoz** egy házirend-beállítást egy tűzfal házirendhez.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallPolicySetting -State $enabledState -Mode $enabledMode -DisableRequestBodyCheck -MaxFileUploadInMb $fileUploadLimitInMb -MaxRequestBodySizeInKb $maxRequestBodySizeInKb
```

A parancs létrehoz egy házirend-beállítást $enabledState állammal, $enabledMode móddal, a RequestCheck mint false, a FileUploadLimitInMb mint $fileUploadLimitInMb és a MaxRequestQuestSizeInKb értékeket $$maxRequestBodySizeInKb.
Az új házirendBeállítók tárolása a következő $condition.
