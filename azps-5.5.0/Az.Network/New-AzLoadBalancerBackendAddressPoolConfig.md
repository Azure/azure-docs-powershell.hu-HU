# New-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Háttércímkészlet-konfigurációt hoz létre a terheléselosztáshoz.

## SZINTAXIS

```
New-AzLoadBalancerBackendAddressPoolConfig -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzLoadBalancerBackendAddressPoolConfig** parancsmag létrehoz egy háttércímkészlet-konfigurációt egy Azure terheléselegyenítő számára.

## PÉLDÁK

### 1. példa: Háttércímkészlet-konfiguráció létrehozása terheléselosztáshoz
```
PS C:\>New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02"
```

Ez a parancs létrehoz egy BackendAddressPool02 nevű háttércímkészlet-konfigurációt a terheléselosztáshoz.
