# Remove-AzApplicationGateway

## SYNOPSIS
Eltávolít egy alkalmazás-átjárót.

## SZINTAXIS

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApplicationGateway** parancsmag eltávolít egy alkalmazás-átjárót.

## PÉLDÁK

### 1. példa: Adott alkalmazás átjáró eltávolítása
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs eltávolítja az ApplicationGateway01 nevű alkalmazás-átjárót az ResourceGroup01 nevű erőforráscsoportból.
