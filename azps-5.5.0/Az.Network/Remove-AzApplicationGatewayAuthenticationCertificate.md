# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Eltávolít egy hitelesítési tanúsítványt egy alkalmazás-átjáróból.

## SZINTAXIS

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApplicationGatewayAuthenticationCertificate parancsmag** eltávolít egy hitelesítési tanúsítványt egy Azure-alkalmazás-átjáróból.

## PÉLDÁK

### 1. példa: Hitelesítési tanúsítvány eltávolítása egy alkalmazás-átjáróból
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Az első parancs az appGwName nevű alkalmazás-átjárót kapja meg, és az eredményt a $appgw tárolja.
A második parancs eltávolítja a cert01 nevű hitelesítési tanúsítványt az alkalmazás-átjáróból.
A harmadik parancs frissíti az alkalmazás átjáróját.
