# Remove-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
A kimenő szabálykonfiguráció eltávolítása a terheléselosztásból.

## SZINTAXIS

```
Remove-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzLoadBalancerOutboundRuleConfig** parancsmag eltávolít egy kimenő szabálykonfigurációt egy Azure-terheléselegyenlőből.

## PÉLDÁK

### 1. példa: Kimenő szabály törlése az Azure terheléselosztási szolgáltatásból
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Remove-AzLoadBalancerOutboundRuleConfig -Name "RuleName" -LoadBalancer $slb
PS C:\>Set-AzLoadBalancer -LoadBalancer $slb
```

Az első parancs lekérte az eltávolítani kívánt kimenő szabálykonfigurációhoz társított terhelésegyenleg-elegyet, majd tárolja azt a $slb változóban.
A második parancs eltávolítja a kapcsolódó kimenő szabálykonfigurációt a terheléselosztásból.
A harmadik parancs frissíti a terheléselosztást.
