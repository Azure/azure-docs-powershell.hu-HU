# Remove-AzPublicIpAddress

## SYNOPSIS
Nyilvános IP-cím eltávolítása.

## SZINTAXIS

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzPublicIpAddress** parancsmag eltávolítja az Azure nyilvános IP-címét.

## PÉLDÁK

### 1: Nyilvános IP-cím típusú erőforrás eltávolítása
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Ez a parancs eltávolítja a nyilvános IP-cím $publicIpName az erőforráscsoportban $rgName.
