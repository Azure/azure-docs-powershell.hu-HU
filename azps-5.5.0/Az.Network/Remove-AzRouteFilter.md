# Remove-AzRouteFilter

## SYNOPSIS
Eltávolít egy útvonalszűrőt.

## SZINTAXIS

```
Remove-AzRouteFilter -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Remove-AzRouteFilter** parancsmag eltávolít egy útvonalszűrőt.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Remove-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
```

A parancs eltávolítja a RouteFilter01 nevű útvonalszűrőt az ResourceGroup01 nevű erőforráscsoportban.
