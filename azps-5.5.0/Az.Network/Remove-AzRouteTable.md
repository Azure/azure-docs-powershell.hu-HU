# Remove-AzRouteTable

## SYNOPSIS
Eltávolít egy útvonaltáblát.

## SZINTAXIS

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Remove-AzRouteTable** parancsmag eltávolít egy Azure-útválasztási táblát.

## PÉLDÁK

### 1. példa: Útvonaltábla eltávolítása
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Ez a parancs eltávolítja a RouteTable01 nevű útválasztási táblát az Erőforráscsoport11 nevű erőforráscsoportból.
A parancsmag megerősítést kér, mielőtt eltávolítja a táblát.
