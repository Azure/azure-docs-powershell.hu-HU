# Remove-AzVpnClientIpsecParameter

## SYNOPSIS
Eltávolítja a Virtuális hálózati átjáró erőforráshoz beállított egyéni IPsec-házirendet.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Remove-AzVpnClientIpsecParameter -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzVpnClientIpsecParameter -InputObject <PSVirtualNetworkGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzVpnClientIpsecParameter -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A virtuális hálózati átjáró az azure-beli átjárót képviselő objektum.
A **Remove-AzVpnClientIpsecParameter** parancsmag eltávolítja a virtuális hálózati átjárón beállított egyéni IPsec-paramétereket, amelyek a VirtualNetworkGateway neve és az Erőforráscsoport neve alapján beállítja az alapértelmezett vpn ipsec-házirendet VPN-átjárón.

## PÉLDÁK

### 1. példa: A virtuális hálózati átjárón beállított VPN ipsec-paraméterek törlése
```powershell
PS C:\> $delete = Remove-AzVpnClientIpsecParameter -VirtualNetworkGatewayName myGateway -ResourceGroupName myRG
```

Törli a virtuális hálózati átjáróban a "myGateway" nevű vpn egyéni ipsec-paramétereket a "myRG" erőforráscsoporton belül. Ez a parancs olyan bool objektumot ad vissza, amely azt mutatja, hogy az eltávolítás sikeres vagy sikertelen volt-e.
Megjegyzés: Ez azt eredményezi, hogy a virtuális hálózat átjárója alapértelmezett vpn ipsec-házirendet ad meg.
