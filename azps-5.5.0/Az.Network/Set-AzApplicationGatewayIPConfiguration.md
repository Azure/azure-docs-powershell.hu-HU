# Set-AzApplicationGatewayIPConfiguration

## SYNOPSIS
Módosítja egy alkalmazás-átjáró IP-konfigurációját.

## SZINTAXIS

### SetByResourceId
```
Set-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SubnetId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
Set-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-Subnet <PSSubnet>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzApplicationGatewayIPConfiguration** parancsmag módosítja az IP-konfigurációt.
Az IP-konfiguráció tartalmazza azt az alhálózatot, amelyen az alkalmazás-átjárót üzembe kell helyezni.

## PÉLDÁK

### 1. példa: Alkalmazás-átjáró IP-konfigurációjának frissítése
```
PS C:\>$VNet = Get-AzVirtualNetwork -Name "VNet01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Subnet = Get-AzVirtualNetworkSubnetConfig -Name "Subnet01" -VirtualNetwork $VNet 
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "AppgwSubnet01" -Subnet $Subnets
```

Az első parancs a VNet01 nevű virtuális hálózatot kapja meg, amely az ResourceGroup01 nevű erőforráscsoporthoz tartozik, és a $VNet tárolja.
A második parancs az Alhálózat01 nevű alhálózati konfigurációt kapja $VNet alhálózati változóban $Subnet tárolja.
A harmadik parancs egy ApplicationGateway01 nevű alkalmazás-átjárót kap, amely az ResourceGroup01 nevű erőforráscsoporthoz tartozik, és a $AppGw tárolja.
A "oda" parancs a $AppGw alhálózati konfigurációra állítja be az $Subnet.
