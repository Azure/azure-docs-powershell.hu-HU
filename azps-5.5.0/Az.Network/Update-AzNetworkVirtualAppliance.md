# Update-AzNetworkVirtualAppliance

## SYNOPSIS
Hálózati virtuális eszköz típusú erőforrás frissítése vagy módosítása.

## SZINTAXIS

```
Update-AzNetworkVirtualAppliance -Name <String> -ResourceGroupName <String>
 [-Sku <PSVirtualApplianceSkuProperties>] [-VirtualApplianceAsn <Int32>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Update-AzNetworkVirtualAppliance módosítja a hálózati virtuális eszköz erőforrást.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Update-AzNetworkVirtualAppliance -ResourceGroupName testrg -Name nva -VirtualApplianceAsn 1234
```

Módosítsa a virtuális berendezés ASN-számát.

### 2. példa

Hálózati virtuális eszköz típusú erőforrás frissítése vagy módosítása. (automatikusan generált)

<!-- Aladdin Generated Example -->
