# Disable-AzOperationalInsightsLinuxPerformanceCollection

## SYNOPSIS
Leállítja a teljesítményszámlálók linuxos számítógépekről való gyűjtését.

## SZINTAXIS

### ByWorkspaceName (alapértelmezett)
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Disable-AzOperationalInsightsLinuxPerformanceCollection parancsmag** leállítja a teljesítményszámlálók gyűjtését a csatlakoztatott Linux számítógépekről egy munkaterületen.
