# Enable-AzOperationalInsightsIISLogCollection

## SYNOPSIS
Elindítja az IIS-naplók gyűjteményét egy munkaterület számítógépein.

## SZINTAXIS

### ByWorkspaceName (alapértelmezett)
```
Enable-AzOperationalInsightsIISLogCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Enable-AzOperationalInsightsIISLogCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzOperationalInsightsIISLogCollection** parancsmag elindítja az IIS-naplók gyűjteményét egy munkaterület csatlakoztatott számítógépein.
