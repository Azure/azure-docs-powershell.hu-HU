# Get-AzOperationalInsightsStorageInsight

## SYNOPSIS
Információkat kap a Tárterület-betekintésről.

## SZINTAXIS

### ByWorkspaceName (alapértelmezett)
```
Get-AzOperationalInsightsStorageInsight [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByWorkspaceObject
```
Get-AzOperationalInsightsStorageInsight [-Workspace] <PSWorkspace> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzOperationalInsightsStorageInsight** parancsmag információt kap egy meglévő Tárolási információról.
Ha a Tárterület-információ neve meg van adva, ez a parancsmag információt kap a Tárterület-információról.
Ha nem ad meg nevet, ez a parancsmag információt kap a munkaterületen található összes tárterület-elemzésről.

## PÉLDÁK

### 1. példa: Tárterület-betekintés betekintés név szerint
```
PS C:\>Get-AzOperationalInsightsStorageInsight -Name "MyStorageInsight" -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Ez a parancs a ContosoWorkspace nevű munkaterületről bepillantást enged a MyStorageInsight nevű tárterület-információba.

### 2. példa: Tárterület-betekintés munkaterületi objektum használatával
```
PS C:\>$Workspace = Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"
PS C:\>Get-AzOperationalInsightsStorageInsight -Workspace $Workspace -Name "MyStorageInsight"
```

Az első parancs a **Get-AzOperationalInsightsWorkspace** parancsmagot használva behoz egy Operational Insights-munkaterületet, majd a $Workspace tárolja.
A második parancs bepillantást enged a MyStorageInsight nevű tárterület-$Workspace.
