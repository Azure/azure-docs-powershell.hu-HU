# Get-AzPeeringServiceCountry

## SYNOPSIS
A társviszony-létesítéshez elérhető országok listája.

## SZINTAXIS

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A társviszony-szolgáltatásban elérhető országok listája.
