# New-AzPostgreSqlFlexibleServer

## SYNOPSIS
Létrehoz egy új kiszolgálót.

## SZINTAXIS

```
New-AzPostgreSqlFlexibleServer [-Name <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-AdministratorLoginPassword <SecureString>] [-AdministratorUserName <String>] [-BackupRetentionDay <Int32>]
 [-Location <String>] [-PublicAccess <String>] [-Sku <String>] [-SkuTier <String>] [-StorageInMb <Int32>]
 [-Subnet <String>] [-SubnetPrefix <String>] [-Tag <Hashtable>] [-Version <ServerVersion>] [-Vnet <String>]
 [-VnetPrefix <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## LEÍRÁS
Létrehoz egy új kiszolgálót.

## PÉLDÁK

### 1. példa: Új rugalmas PostgreSql-kiszolgáló létrehozása argumentumokkal
```powershell

-Location eastus -AdministratorUserName postgresqltest -AdministratorLoginPassword $password -Sku Standard_B1ms -SkuTier Burstable -Version 12 -StorageInMb 10240 -PublicAccess none
```

Ez a parancsmag létrehozza a PostgreSql rugalmas kiszolgálóját alapértelmezett paraméterértékekkel, és létrehozza a kiszolgálót egy új virtuális hálózaton belül, és egy alhálózatot delegál a kiszolgálóra.
A hely alapértelmezett értékei a 2. nyugat-amerikai régió, a termékváltozat Standard_D2s_v3, a termékváltozatréteg a GeneralPurpose, a tárterület mérete pedig 128GiB.


Ha meg szeretné találni a kiszolgáló automatikusan létrehozott jelszavát, a ConvertFrom-SecureString konvertálja a "SecuredPassword" tulajdonságot egyszerű szöveggé.
(Például: $server. SecuredPassword | ConvertFrom-SecureString -AsPtext)
