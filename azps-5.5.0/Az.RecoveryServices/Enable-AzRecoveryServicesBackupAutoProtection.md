# Enable-AzRecoveryServicesBackupAutoProtection

## SYNOPSIS
Az **Enable-AzRecoveryServicesBackupAutoProtection** parancsmag a megadott házirendnek megfelelő aktuális és jövőbeli SQL-DBs automatikus védelmét állítja be a megadott példányon belül.

## SZINTAXIS

```
Enable-AzRecoveryServicesBackupAutoProtection [-InputItem] <ProtectableItemBase>
 [-BackupManagementType] <BackupManagementType> [-WorkloadType] <WorkloadType> [-Policy] <PolicyBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
Ez a parancs lehetővé teszi a felhasználóknak, hogy automatikusan védjék az összes meglévő nem védett SQL-DBs-t és minden olyan adatbázis-adatbázist, amelyet később hozzáad a megadott házirendhez.  Mivel az utasítás az összes jövőbeli DBs biztonsági mentése, a művelet SQLInstance szinten történik, az Azure biztonsági mentési szolgáltatás ezután rendszeresen ellenőrzi az automatikusan védett tárolókat az új DB-k után, és automatikusan védi őket.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $Pol = Get-AzRecoveryServicesBackupProtectionPolicy -Name "DefaultSQLPolicy"
PS C:\> $SQLInstance = Get-AzRecoveryServicesBackupProtectableItem -workloadType MSSQL -ItemType SQLInstance -VaultId $targetVault.ID -Name "MSSQLInstance" -ServerName "TestSQLServer" 
PS C:\> Enable-AzRecoveryServicesBackupAutoProtection -InputItem $SQLInstance -BackupManagementType AzureWorkload -WorkloadType MSSQL -Policy $Pol -VaultId $targetvault.ID 
```

Az első parancsmag kap egy alapértelmezett házirendobjektumot, majd tárolja azt a $Pol változóban.
A második parancsmag lekéri a megfelelő SQLInstance függvényt, amely egy védett elem. A 3. parancs ezután beállítja az automatikus védelmet ehhez a példányhoz a $Pol.

### 2. példa

Ez a parancs lehetővé teszi a felhasználóknak, hogy automatikusan védjék az összes meglévő nem védett DBs-t és minden olyan adatbázis-adatbázist, amelyet később hozzáadnak az adott házirendhez. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
Enable-AzRecoveryServicesBackupAutoProtection -BackupManagementType AzureVM -InputItem <ProtectableItemBase> -Policy $Pol -VaultId $vault.ID -WorkloadType AzureVM
```
