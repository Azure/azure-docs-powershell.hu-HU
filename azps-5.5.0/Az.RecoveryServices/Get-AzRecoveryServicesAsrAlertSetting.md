# Get-AzRecoveryServicesAsrAlertSetting

## SYNOPSIS
Beállítja az Azure Site Recovery értesítési beállításait a tárolóhoz.

## SZINTAXIS

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzRecoveryServicesAsrAlertSetting parancsmag** megkapja a tárolóhoz konfigurált Azure-webhely-helyreállítási értesítési beállításokat.

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Értesítési/értesítési beállítás az Azure-webhely-helyreállításhoz.
