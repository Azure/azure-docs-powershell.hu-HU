# Remove-AzRecoveryServicesAsrStorageClassificationMapping

## SYNOPSIS
Törli a megadott ASR-tárterületbesorolás megfeleltetését.

## SZINTAXIS

```
Remove-AzRecoveryServicesAsrStorageClassificationMapping -InputObject <ASRStorageClassificationMapping>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzRecoveryServicesAsrStorageClassificationMapping** parancsmag törli a megadott Azure-webhely-helyreállítás tárbesorolási megfeleltetését.

## PÉLDÁK

### 1. példa
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrStorageClassificationMapping -StorageClassificationMapping $StorageClassificationMapping
```

Elindítja a megadott tárbesorolási megfeleltetés törlését, és visszaadja a művelet nyomon követéséhez használt ASR-feladatot.
