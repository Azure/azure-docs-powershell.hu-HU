# Get-AzRedisCacheFirewallRule

## SYNOPSIS
A redis cache tűzfalszabályának beállítása

## SZINTAXIS

```
Get-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> [-RuleName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Ha meg van adva a **RuleName** paraméter, a **Get-AzRedisCacheFirewallRule** parancsmag részletes információkat kap a megadott tűzfalszabályról az Azure Redis cache-ben. Ha csak **a Név** van megadva, ez a művelet minden tűzfalszabályt elérhetővé fog tenni az adott Redis-gyorsítótárban.

## PÉLDÁK

### 1. példa: Egyetlen tűzfalszabály létrehozása
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32
```

Ez a parancs egy szabály nevű tűzfalszabályt kap a Mycache nevű redis gyorsítótárból.

### 2. példa: Az összes tűzfalszabály lekérte
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruletwo
        RuleName          : ruletwo
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.33
        EndIP             : 10.0.0.64
```

Ez a parancs az összes tűzfalszabályt a Mycache nevű redis gyorsítótárból kapja meg.
