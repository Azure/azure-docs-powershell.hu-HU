# Get-AzRedisCacheKey

## SYNOPSIS
Lekérte a redis-gyorsítótár hozzáférési kulcsait.

## SZINTAXIS

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzRedisCacheKey** parancsmag az Azure Redis Cache hozzáférési kulcsait kapja meg.

## PÉLDÁK

### 1. példa: A redis-gyorsítótár hozzáférési kulcsának lekérte
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Ez a parancs a MyCacheKey nevű hozzáférési kulcsokat kapja meg.
