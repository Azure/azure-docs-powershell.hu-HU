# Get-AzRedisCachePatchSchedule

## SYNOPSIS
Javításütemtervet kap.

## SZINTAXIS

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzRedisCachePatchSchedule** parancsmag javításütemtervet kap az Azure Redis Cache gyorsítótárában lévő gyorsítótárhoz.

## PÉLDÁK

### 1. példa: A javítás ütemezésének lekérte
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Ez a parancs a Javítás ütemezését a RedisCache06 nevű gyorsítótárból kapja.
