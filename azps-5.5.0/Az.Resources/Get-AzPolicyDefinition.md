## LEÍRÁS
A **Get-AzPolicyDefinition** parancsmag egy házirenddefiníciós gyűjteményt vagy egy név vagy azonosító által meghatározott házirenddefiníciót kap.

## PÉLDÁK

### 1. példa: Az összes házirenddefiníció lekérte
```
PS C:\> Get-AzPolicyDefinition
```

Ez a parancs az összes házirenddefiníciót beveszi.

### 2. példa: Házirenddefiníció lekérte az aktuális előfizetést név szerint
```
PS C:\> Get-AzPolicyDefinition -Name 'VMPolicyDefinition'
```

Ez a parancs a VMPolicyDefinition nevű házirend-definíciót az aktuális alapértelmezett előfizetésből kapja meg.

### 3. példa: Házirend-definíció lekérte a felügyeleti csoportból név szerint
```
PS C:\> Get-AzPolicyDefinition -Name 'VMPolicyDefinition' -ManagementGroupName 'Dept42'
```

Ez a parancs a VMPolicyDefinition nevű házirend-definíciót a Dept42 nevű felügyeleti csoportból kapja meg.

### 4. példa: Az összes beépített házirenddefiníció lekérte az előfizetésből
```
PS C:\> Get-AzPolicyDefinition -SubscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca' -Builtin
```

Ez a parancs a 3bf44b72-c631-427a-b8c8-53e2595398ca azonosítójú előfizetés összes beépített házirenddefinícióját megkapja.

### 5. példa: Házirend-definíciók lekérte egy adott kategóriából
```
PS C:\> Get-AzPolicyDefinition | where-object {$_.Properties.metadata.category -eq "Virtual Machine"}
```

Ez a parancs az összes házirenddefiníciót a "Virtuális gép" kategóriában kapja meg.
