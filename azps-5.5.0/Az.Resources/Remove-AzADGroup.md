# Remove-AzADGroup

## SYNOPSIS
Egy Active Directory-csoportot töröl.

## SZINTAXIS

### ObjectIdParameterSet (alapértelmezett)
```
Remove-AzADGroup -ObjectId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Remove-AzADGroup -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzADGroup -InputObject <PSADGroup> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Egy Active Directory-csoportot töröl.

## PÉLDÁK

### 1. példa: Csoport eltávolítása objektumazonosító alapján

```powershell
PS C:\> Remove-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Eltávolítja a "85F89C90-780E-4AA6-9F4F-6F268D322EEE" objektumazonosítót a bérlőből.

### 2. példa: Csoport eltávolítása pipázás használatával

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Remove-AzADGroup
```

A (85F89C90-780E-4AA6-9F4F-6F268D322EEE) objektumazonosítóval és az Remove-AzADGroup-nak a bérlői fiókból való eltávolításához a csoportot eltávolítócsövekkel együtt.
