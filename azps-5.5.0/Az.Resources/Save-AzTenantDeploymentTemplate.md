# Save-AzTenantDeploymentTemplate

## SYNOPSIS
Egy telepítősablont fájlba ment.

## SZINTAXIS

### SaveByDeploymentName (alapértelmezett)
```
Save-AzTenantDeploymentTemplate -DeploymentName <String> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SaveByDeploymentObject
```
Save-AzTenantDeploymentTemplate -DeploymentObject <PSDeployment> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Save-AzTenantDeploymentTemplate**  parancsmag JSON-fájlba menti a telepítősablont a bérlői webhely hatókörén üzemelő telepítéshez.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Save-AzTenantDeploymentTemplate -DeploymentName "TestDeployment"
```

Ez a parancs lekérte a telepítési sablont a TestDeploymentből, és JSON-fájlként menti azt az aktuális címtárba.

### 2. példa: Üzembe helyezés és sablon mentése
```
PS C:\>Get-AzTenantDeploymentTemplate -Name "RolesDeployment" | Save-AzTenantDeploymentTemplate
```

Ez a parancs a központi telepítés "RolesDeployment" parancsát a bérlő aktuális hatókörében kapja meg, és menti a sablonját.
