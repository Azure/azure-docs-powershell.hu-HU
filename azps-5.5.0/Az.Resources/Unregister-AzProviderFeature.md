# Unregister-AzProviderFeature

## SYNOPSIS
Az Azure-szolgáltatói szolgáltatások regisztrációjának a regisztrációját a fiókjában.

## SZINTAXIS

```
Unregister-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Unregister-AzProviderFeature** parancsmag megszüntet egy Azure-szolgáltatói funkciót a fiókjában.

## PÉLDÁK

### 1. példa: Funkció regisztrációjának visszaszava
```
PS C:\>Unregister-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Ezzel a fiókból nem regisztrálja a Microsoft.Network AllowApplicationSecurityGroups szolgáltatását.
