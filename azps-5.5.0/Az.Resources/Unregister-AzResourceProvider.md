# Unregister-AzResourceProvider

## SYNOPSIS
Erőforrás-szolgáltató regisztrációjának a visszaszava.

## SZINTAXIS

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Unregister-AzResourceProvider** parancsmag nem regisztrál egy Azure-erőforrás-szolgáltatót.

## PÉLDÁK

### 1. példa: Erőforrás-szolgáltató regisztrációjának megszűrése a ProviderNamespace szolgáltatóval

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

Ez a parancs a "Microsoft.support" erőforrásszolgáltatót nem regisztrálja.
