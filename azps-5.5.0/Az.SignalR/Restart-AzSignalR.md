# Restart-AzSignalR

## SYNOPSIS
Indítsa újra a SignalR-szolgáltatást.

## SZINTAXIS

### ResourceGroupParameterSet (alapértelmezett)
```
Restart-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Restart-AzSignalR -ResourceId <String> [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Restart-AzSignalR -InputObject <PSSignalRResource> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Indítsa újra a SignalR-szolgáltatást.

## PÉLDÁK

### Adott SignalR-szolgáltatás újraindítása
```powershell
PS C:\> Restart-AzSignalR -ResourceGroupName myResourceGroup -Name mysignalr1 -PassThru

True
```

Az alapértelmezett erőforráscsoportot a \` Set-AzDefault -ResourceGroupName myResourceGroup állíthatja \` be.
