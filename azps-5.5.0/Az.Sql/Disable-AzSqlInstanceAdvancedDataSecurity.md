# Disable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Letiltja a speciális adatbiztonságot egy felügyelt példányon.

## SZINTAXIS

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Disable-AzSqlInstanceAdvancedDataSecurity** parancsmag letiltja a speciális adatbiztonságot egy felügyelt példányon.

## PÉLDÁK

### 1. példa: A felügyelt példány speciális adatbiztonságának letiltása
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### 2. példa: A kiszolgáló speciális adatbiztonságának letiltása felügyelt példány típusú erőforrásból
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
