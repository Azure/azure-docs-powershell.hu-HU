# Get-AzSqlElasticPoolRecommendedAction

## SYNOPSIS
Egy vagy több javasolt műveletet kap egy Azure SQL rugalmassági készlet tanácsadóhoz.

## SZINTAXIS

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlElasticPoolRecomasticAction** parancsmag egy vagy több javasolt műveletet kap egy Azure SQL rugalmassági készlet tanácsadóhoz.
