# Get-AzSqlServerAdvisor

## SYNOPSIS
Egy vagy több tanácsadót kap egy Azure SQL Serverhez.

## SZINTAXIS

```
Get-AzSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlServerAdvisor** parancsmag egy vagy több Azure SQL Server-tanácsadót kap egy Azure SQL Serverhez.
