# Get-AzSqlSyncAgent

## SYNOPSIS
Információkat ad vissza az Azure SQL Sync-ügynökökről.

## SZINTAXIS

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlSyncAgent** parancsmag egy vagy több Azure SQL Sync-ügynök adatait adja vissza.
Adja meg a szinkronizálási ügynök nevét, hogy csak az adott szinkronizálási ügynök adatai legyenek láthatóak.
