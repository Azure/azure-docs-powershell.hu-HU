# Disable-AzStorageStaticWebsite

## SYNOPSIS
Tiltsa le az Azure Storage-fiók statikus webhelyét.

## SZINTAXIS

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Disable-AzStorageStaticWebsite** parancsmag letiltja az Azure Storage-fiók statikus webhelyét.

## PÉLDÁK

### 1. példa: Statikus webhely letiltása Azure Storage-fiókhoz
```
C:\PS>Disable-AzStorageStaticWebsite
```

Ez a parancs letiltja az Azure Storage-fiók statikus webhelyét.
