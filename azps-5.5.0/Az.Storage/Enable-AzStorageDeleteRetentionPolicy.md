# Enable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Engedélyezze a törlési adatmegőrzési házirendet az Azure Storage Blob szolgáltatáshoz.

## SZINTAXIS

```
Enable-AzStorageDeleteRetentionPolicy [-RetentionDays] <Int32> [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Enable-AzStorageDeleteRetentionPolicy parancsmag** törlési adatmegőrzési házirendet engedélyez az Azure Storage Blob szolgáltatásban.

## PÉLDÁK

### 1. példa: A Blob szolgáltatás törlési adatmegőrzési házirendének engedélyezése
```
C:\PS>Enable-AzStorageDeleteRetentionPolicy -RetentionDays 3
```

Ezzel a paranccsal törlési adatmegőrzési házirendet állíthat be a Blob szolgáltatásban, a törölt blobmegőrzési napokat pedig 3-ra állíthatja.
