# Get-AzRmStorageContainer

## SYNOPSIS
Tároló blobtárolók lekérte vagy listázza a tárolókat

## SZINTAXIS

### AccountName (alapértelmezett)
```
Get-AzRmStorageContainer [-ResourceGroupName] <String> [-StorageAccountName] <String> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzRmStorageContainer -StorageAccount <PSStorageAccount> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzRmStorageContainer** parancsmag lekérte vagy felsorolja a tároló blobtárolóit

## PÉLDÁK

### 1. példa: Tárterület blobtárolója tárfiók nevével és tárolónevével
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer"
```

Ez a parancs egy Tároló blobtárolót kap, amely tárfiók nevét és tárolónevét tárolja.

### 2. példa: Tárfiók összes tárterület blobtárolója
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
```

Ez a parancs felsorolja egy Tárfiók tárfiók nevét tartalmazó tárterület blobtárolóját.

### 3. példa: Tárterület blobtárolója tárfiók-objektummal és tárolónévvel.
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>Get-AzRmStorageContainer -StorageAccount $accountObject -ContainerName "myContainer"
```

Ez a parancs egy Tároló blobtárolót kap a Tárfiók-objektummal és a tároló nevével.
