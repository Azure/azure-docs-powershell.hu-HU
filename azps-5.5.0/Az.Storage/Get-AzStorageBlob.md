# Get-AzStorageBlob

## SYNOPSIS
Egy tárolóban lévő blobokat sorol fel.

## SZINTAXIS

### BlobName (alapértelmezett)
```
Get-AzStorageBlob [[-Blob] <String>] [-Container] <String> [-IncludeDeleted] [-MaxCount <Int32>]
 [-ContinuationToken <BlobContinuationToken>] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>]
 [<CommonParameters>]
```

### SingleBlobSnapshotTime
```
Get-AzStorageBlob [-Blob] <String> [-Container] <String> [-IncludeDeleted] -SnapshotTime <DateTimeOffset>
 [-MaxCount <Int32>] [-ContinuationToken <BlobContinuationToken>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

### SingleBlobVersionID
```
Get-AzStorageBlob [-Blob] <String> [-Container] <String> [-IncludeDeleted] -VersionId <String>
 [-MaxCount <Int32>] [-ContinuationToken <BlobContinuationToken>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

### BlobPrefix
```
Get-AzStorageBlob [-Prefix <String>] [-Container] <String> [-IncludeDeleted] [-IncludeVersion]
 [-MaxCount <Int32>] [-ContinuationToken <BlobContinuationToken>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzStorageBlob** parancsmag egy Azure-tárfiók adott tárolóján lévő blobokat sorolja fel.

## PÉLDÁK

### 1. példa: Blob lekérte a blob nevét
```
PS C:\>Get-AzStorageBlob -Container "ContainerName" -Blob blob*
```

Ez a parancs blobnevet és helyettesítő karaktereket használ a blob lekérthez.

### 2. példa: Blobok lekérte egy tárolóban a folyamat használatával
```
PS C:\>Get-AzStorageContainer -Name container* | Get-AzStorageBlob -IncludeDeleted

   Container Uri: https://storageaccountname.blob.core.windows.net/container1

Name                 BlobType  Length          ContentType                    LastModified         AccessTier SnapshotTime         IsDeleted 
----                 --------  ------          -----------                    ------------         ---------- ------------         --------- 
test1                BlockBlob 403116          application/octet-stream       2017-11-08 07:53:19Z            2017-11-08 08:19:32Z True      
test1                BlockBlob 403116          application/octet-stream       2017-11-08 09:00:29Z                                 True      
test2                BlockBlob 403116          application/octet-stream       2017-11-08 07:53:00Z                                 False
```

Ez a parancs a folyamat segítségével behoz minden blobot (a Törölt állapotban lévő blobokat is) egy tárolóba.

### 3. példa: Blobok lekért névelőtagja
```
PS C:\>Get-AzStorageBlob -Container "ContainerName" -Prefix "blob"
```

Ez a parancs egy névelőtagot használ a blobok leküldésekor.

### 4. példa: Több köteg blobos listájának létrehozása
```
PS C:\>$MaxReturn = 10000
PS C:\> $ContainerName = "abc"
PS C:\> $Total = 0
PS C:\> $Token = $Null
PS C:\> do
 {
     $Blobs = Get-AzStorageBlob -Container $ContainerName -MaxCount $MaxReturn  -ContinuationToken $Token
     $Total += $Blobs.Count
     if($Blobs.Length -le 0) { Break;}
     $Token = $Blobs[$blobs.Count -1].ContinuationToken;
 }
 While ($Token -ne $Null)
PS C:\> Echo "Total $Total blobs in container $ContainerName"
```

Ebben a példában *a MaxCount* és *a ContinuationToken paramétert* használva több kötegben listába soroljuk az Azure Storage blobokat.
Az első négy parancs értékeket rendel a példában használt változókhoz.
Az ötödik parancs egy **Olyan Do-While utasítást** ad meg, amely a **Get-AzStorageBlob** parancsmagot használja a blobok be szereznie.
Az utasítás tartalmazza a $Token változóban tárolt $Token tokent.
$Token a hurok futásakor megváltozik az érték.
További információért írja be a `Get-Help About_Do` .
Az utolsó parancs a **Visszhang paranccsal** jeleníti meg az összeget.

Egy nevet vagy egy névmintát ad meg, amely helyettesítő karakterekkel való keresésre használható.
Ha nincs megadva blobnév, a parancsmag felsorolja a megadott tároló összes blobját.
Ha a paraméterhez meg van adva egy érték, a parancsmag felsorolja az összes olyan blobot, amely megfelel ennek a paraméternek. Ez a paraméter a karakterlánc bármely helyében támogatja a helyettesítő karaktereket.
