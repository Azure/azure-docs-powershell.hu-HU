# Remove-AzStorageContainerStoredAccessPolicy

## SYNOPSIS
Eltávolít egy tárolt hozzáférési házirendet egy Azure-tárolóból.

## SZINTAXIS

```
Remove-AzStorageContainerStoredAccessPolicy [-Container] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStorageContainerStoredAccessPolicy** parancsmag eltávolít egy tárolt hozzáférési házirendet egy Azure-tárolóból.

## PÉLDÁK

### 1. példa: Tárolt hozzáférési házirend eltávolítása egy tárolóból
```
PS C:\>Remove-AzStorageContainerStoredAccessPolicy -Container "MyContainer" -Policy "Policy03"
```

Ez a parancs eltávolítja a Policy03 nevű hozzáférési házirendet a MyContainer nevű tárolt tárolóból.
