# Remove-AzStorageShareStoredAccessPolicy

## SYNOPSIS
Egy tárolt hozzáférési házirend eltávolítása egy tárterület-megosztásból.

## SZINTAXIS

```
Remove-AzStorageShareStoredAccessPolicy [-ShareName] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStorageShareStoredAccessPolicy** parancsmag eltávolít egy tárolt hozzáférési házirendet egy Azure Storage-megosztásból.

## PÉLDÁK

### 1. példa: Tárolt hozzáférési házirend eltávolítása Azure Storage-megosztásból
```
PS C:\>Remove-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Ez a parancs eltávolítja a GeneralPolicy nevű tárolt hozzáférési házirendet a ContosoShare szolgáltatásból.
