# Az.Subscription modul
## Leírás
A szakasz témakörei az Azure Resource Manager (ARM) keretrendszer Azure PowerShell-parancsmagját tartalmazzak. A parancsmagok a Microsoft.Azure.Commands.Subscription névterében létezik.

## Az.Subscription parancsmagok
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Előfizetési alias részleteinek lekérte

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Új alias és előfizetés létrehozása

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Az előfizetés aliasának törlése

### [Update-AzSubscription](Update-AzSubscription.md)
Azure-előfizetés frissítése
