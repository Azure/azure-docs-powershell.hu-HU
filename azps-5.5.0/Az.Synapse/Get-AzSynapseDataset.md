# Get-AzSynapseDataset

## SYNOPSIS
Információkat kap a munkaterületen található adatkészletekkel kapcsolatban.

## SZINTAXIS

### GetByName (alapértelmezett)
```
Get-AzSynapseDataset -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSynapseDataset** parancsmag információt kap a munkaterületen található adatkészletekkel kapcsolatban.
Ha megadja egy adatkészlet nevét, ez a parancsmag információt kap az adatkészletről.
Ha nem ad meg nevet, ez a parancsmag információt kap a munkaterületen található összes adatkészletről.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzSynapseDataset -WorkspaceName ContosoWorkspace
```

Ez a parancs információkat kap a ContosoWorkspace nevű munkaterületen található összes adatkészletről.

### 2. példa
```powershell
PS C:\> Get-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset
```

Ez a parancs információkat kap a ContosoDataset nevű adatkészletről a ContosoWorkspace nevű munkaterületen.
