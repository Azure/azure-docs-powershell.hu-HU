# Get-AzSynapseLinkedService

## SYNOPSIS
Információkat kap a munkaterületen található csatolt szolgáltatásokról.

## SZINTAXIS

### GetByName (alapértelmezett)
```
Get-AzSynapseLinkedService -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSynapseLinkedService** parancsmag információkat kap a munkaterületen található csatolt szolgáltatásokról.
Ha megadja egy csatolt szolgáltatás nevét, ez a parancsmag információt kap a hivatkozott szolgáltatásról.
Ha nem ad meg nevet, ez a parancsmag információt kap a munkaterületen található összes csatolt szolgáltatásról.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace
```

Ez a parancs információkat kap a ContosoWorkspace nevű munkaterület összes csatolt szolgáltatásának adatairól.

### 2. példa
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
```

Ez a parancs információkat kap a ContosoLinkedService nevű csatolt szolgáltatásról a ContosoWorkspace nevű munkaterületen.

### 3. példa
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseLinkedService -Name ContosoLinkedService
```

Ez a parancs információkat kap a ContosoLinkedService nevű csatolt szolgáltatásról a ContosoWorkspace nevű munkaterületről a folyamaton keresztül.
