# Add-AzWebAppTrafficRouting

## SYNOPSIS
Adjon hozzá egy útválasztási szabályt a slothoz.

## SZINTAXIS

```
Add-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Add-AzWebAppTrafficRouting** parancsmag útválasztási szabályt ad az Azure Web App slothoz.

## PÉLDÁK

### 1. példa útválasztási szabály hozzáadása a termelési forgalom 15%-ának átviteléhez Stg-tárolóhelyre
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

Ez a parancs egy útválasztási szabályt ad hozzá, amely a production traffice 15%-át az Stg-tárolóhelyre továbbítja

### 2. példa: Adjon hozzá egy útválasztási szabályt a termelési forgalom Stg-tárolóhelyre való átviteléhez 50%-tól 90%-ig növekményes módon.
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

Ez a parancs egy útválasztási szabályt ad hozzá a termelési adatforgalom Stg-tárolóhelyre való átviteléhez, növekményes módon 50%-tól 90%-ig.
