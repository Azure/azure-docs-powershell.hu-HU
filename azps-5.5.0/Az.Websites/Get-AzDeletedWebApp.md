# Get-AzDeletedWebApp

## SYNOPSIS
Törlődnek a webalkalmazások az előfizetésből.

## SZINTAXIS

```
Get-AzDeletedWebApp [[-ResourceGroupName] <String>] [[-Name] <String>] [[-Slot] <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzDeletedWebApp** parancsmag az előfizetés összes törölt webappját visszaadja. A törölt appok tetszés szerint szűrhetők erőforráscsoport, név és slot szerint. Több törölt app is lehet ugyanazokkal a névvel és erőforráscsoporttal. Az azonos nevű törölt appok megkülönböztetésére jelölje be a DeletionTime jelölőnégyzetet.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzDeletedWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Ez a parancs a ContosoSite nevű törölt appokat a Default-Web-WestUS erőforráscsoporthoz tartozóként kapja meg.
