# Get-AzWebAppSlotConfigName

## SYNOPSIS
A Web App Slot config-nevek listájának lekérte

## SZINTAXIS

### S1
```
Get-AzWebAppSlotConfigName [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzWebAppSlotConfigName [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzWebAppSlotConfigName** parancsmag beolvassa a jelenleg slot-beállításként megjelölt alkalmazásbeállítás- és kapcsolati karakterláncnevek listáját.

## PÉLDÁK

### 1. példa
```powershell
PS C:\>Get-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Ez a parancs a Default-Web-WestUS erőforráscsoporttal társított ContosoSite nevű webalkalmazásra vonatkozó alkalmazásbeállításokat és kapcsolati karakterláncokat kapja
