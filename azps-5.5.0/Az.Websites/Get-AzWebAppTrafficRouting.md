# Get-AzWebAppTrafficRouting

## SYNOPSIS
Routing Rule for the given Slot name.Get a routing Rule for the given Slot name.

## SZINTAXIS

```
Get-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RuleName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzWebAppTrafficRouting** parancsmag útválasztási szabálykonfigurációt kap egy Azure Web App Slotból.

## PÉLDÁK

### 1. példa: Az adott útválasztási szabály beolvasója webapphelyről
```powershell
PS C:\> Get-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite"  -RuleName 'Stg'
```

Ez a parancs útválasztási szabályt kap egy adott webapphelyhez.
