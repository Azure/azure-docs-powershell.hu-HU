# New-AzWebAppDatabaseBackupSetting

## SYNOPSIS

## SZINTAXIS

```
New-AzWebAppDatabaseBackupSetting [-Name] <String> [-DatabaseType] <String> [-ConnectionString] <String>
 [[-ConnectionStringName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzWebAppDatabaseBackupSetting** parancsmag létrehoz egy új Azure Web App biztonsági mentési beállítást.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> New-AzWebAppDatabaseBackupSetting -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -ConnectionString "MyConnectionString" -DatabaseType "SqlAzure"
```

Létrehoz egy SqlAzure típusú adatbázis-biztonsági mentési beállítást (kapcsolati karakterláncot) a Default-Web-WestUS erőforráscsoporton belül megadott ContosoWebApp alkalmazáshoz.

### 2. példa

A New-AzWebAppDatabaseBackupSetting parancsmag létrehoz egy új Azure Web App biztonsági mentési beállítást. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
New-AzWebAppDatabaseBackupSetting -ConnectionString 'MyConnectionString' -ConnectionStringName <String> -DatabaseType 'SqlAzure' -Name 'ContosoWebApp'
```
