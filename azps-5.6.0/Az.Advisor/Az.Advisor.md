# Az.Advisor modul
## Leírás
Az Azure Advisorral való interakcióra szolgáló parancsok.

## Az.Advisor parancsmagok
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Az Azure Advisor javaslatainak listáját tartalmazza.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Tiltsa le az Azure Advisor-ajánlást.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Engedélyezi az Azure Advisor javaslat(ak)t.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Szerezze be az Adott előfizetéshez vagy erőforráscsoporthoz az Azure Advisor-konfigurációkat.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Frissíti vagy létrehozza az Azure Advisor-konfigurációt.
