# Install-AzAksKubectl

## SYNOPSIS
Töltse le és telepítse a metszőt, a Kubernetes parancssori eszközt.

## SZINTAXIS

```
Install-AzAksKubectl [-Destination <String>] [-Version <String>] [-DownloadFromMirror] [-PassThru] [-AsJob]
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Töltse le és telepítse a metszőt, a Kubernetes parancssori eszközt.

## PÉLDÁK

### A kutectl legújabb verziójának letöltése és telepítése
```powershell
PS C:\> Install-AzAksKubectl -Version latest
```
