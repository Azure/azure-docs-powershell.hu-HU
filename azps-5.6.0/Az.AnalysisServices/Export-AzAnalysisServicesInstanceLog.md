# Export-AzAnalysisServicesInstanceLog

## SYNOPSIS
Napló exportálása az Analysis Services-kiszolgáló egy példányából az aktuálisan naplózott környezetben a Add-AzAnalysisServicesAccount szerint

## SZINTAXIS

```
Export-AzAnalysisServicesInstanceLog -OutputPath <String> [-Force] [-Instance] <String> [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Export-AzAnalysisServicesInstance parancsmag exportálja a naplót az Azure Analysis Services-kiszolgáló egy példányából a fájlba

## PÉLDÁK

### 1. példa
```
PS C:\>Export-AzAnalysisServicesInstanceLog -Instance testserver -OutputPath C:\path\to\log\testserver.log
```

Ez a parancs exportálja a naplót a kiszolgáló "testserver" kiszolgálóról a Add-AzAnalysisServicesAccount-parancsban megadott környezetből, és menti az OutputPath 'C:\path\to\log\testserver.log' mappában megadott fájlba.
