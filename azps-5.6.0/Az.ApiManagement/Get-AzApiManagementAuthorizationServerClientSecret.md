# Get-AzApiManagementAuthorizationServerClientSecret

## SYNOPSIS
Api Management authorization server client secret (API Management authorization server client secret) kap.

## SZINTAXIS

### ContextParameterSet (alapértelmezett)
```
Get-AzApiManagementAuthorizationServerClientSecret -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServerClientSecret [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApiManagementAuthorizationServerClientSecret** parancsmag az Azure API Management engedélyezési kiszolgálójának ügyféltitkját kapja meg.

## PÉLDÁK

### 1. példa: Egy megadott engedélyezési kiszolgáló ügyfélprogram-titkos kódjának lekérte azonosító alapján
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServerClientSecret -Context $ApiMgmtContext -ServerId "0123456789"
```

Ez a parancs megkapja a megadott engedélyezési kiszolgáló cient-titkos nevét.
