# Get-AzApiManagementSubscriptionKey

## SYNOPSIS
Előfizetési kulcsokat kap.

## SZINTAXIS

```
Get-AzApiManagementSubscriptionKey -Context <PsApiManagementContext> -SubscriptionId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApiManagementSubscriptionKey** parancsmag egy adott előfizetés kulcsait kapja meg.

## PÉLDÁK

### 1. példa: Előfizetési kulcsok lekérte egy megadott azonosítóval
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-East-US" -ServiceName "contoso"
PS C:\>Get-AzApiManagementSubscriptionKey -Context $apimContext -SubscriptionId "0123456789"

PrimaryKey        : 5e48532634114fe999a6979ce0d63a64
SecondaryKey      : 0a8efaf85a664aa0a412241015c7c8f6
```

Ez a parancs azonosító szerint kap egy előfizetést.
