# Publish-AzApiManagementTenantGitConfiguration

## SYNOPSIS
Módosításokat tesz közzé egy Git-ágból a konfigurációs adatbázisba.

## SZINTAXIS

```
Publish-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-ValidateOnly] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Publish-AzApiManagementTenantGitConfiguration parancsmag** közzéteszi a módosításokat egy Git-ágból a konfigurációs adatbázisba.
A Git-ágak módosításait közzététel nélkül is érvényesítheti.

## PÉLDÁK

### 1. példa: Git-módosítások telepítése
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Publish-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

Ez a parancs közzéteszi a módosításokat a megadott ágból a konfigurációs adatbázisba.

### 2. példa: Git-módosítások ellenőrzése
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Publish-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -ValidateOnly -PassThru
```

Ez a parancs ellenőrzi a Git-ág módosításait a konfigurációs adatbázissal szemben.
Nem tesz közzé módosításokat.
