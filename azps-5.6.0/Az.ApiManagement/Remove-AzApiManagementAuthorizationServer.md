# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
Eltávolít egy engedélyezési kiszolgálót.

## SZINTAXIS

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApiManagementAuthorizationServer** parancsmag eltávolítja az Azure API Management engedélyezési kiszolgálóját.

## PÉLDÁK

### 1. példa: Engedélyezési kiszolgáló eltávolítása
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Ez a parancs eltávolítja a megadott API Management Authorization Servert.
Mivel a *Force paraméter* meg van adva, nincs szükség megerősítésre.
