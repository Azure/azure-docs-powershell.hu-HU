# Get-AzAutomationDscCompilationJobOutput

## SYNOPSIS
Az Automatizálási DSC fordítási feladat naplózási adatfolyamát kapja meg.

## SZINTAXIS

```
Get-AzAutomationDscCompilationJobOutput [-Id] <Guid> [-Stream <CompilationJobStreamType>]
 [-StartTime <DateTimeOffset>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzAutomationDscCompilation JobOutput** parancsmag az APS kívánt állapotkonfigurációs (DSC) összeállítási feladatának adatfolyam-rekordjait kapja meg az Azure Automationben.

## PÉLDÁK

### 1. példa: A DSC összeállítási feladat naplóinak lekérte
```
PS C:\>$Jobs = Get-AzAutomationDscCompilationJob -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
PS C:\> $Jobs[0] | Get-AzAutomationDscCompilationJobOutput -Stream "Any"
```

Az első parancs a Contoso17 nevű automatizálási fiókban az összeállítási feladatokat a Get-AzAutomationDscCompilationJob parancsmag használatával kapja meg.
A parancs ezeket az objektumokat a $Jobs tárolja.
A második parancs a tömb első tagjára vonatkozó bármely adatfolyamhoz megkapja az $Jobs kimenetét.
