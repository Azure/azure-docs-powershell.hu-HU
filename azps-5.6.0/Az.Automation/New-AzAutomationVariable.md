# New-AzAutomationVariable

## SYNOPSIS
Automatizálási változót hoz létre.

## SZINTAXIS

```
New-AzAutomationVariable [-Name] <String> -Encrypted <Boolean> [-Description <String>] [-Value <Object>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzAutomationVariable** parancsmag létrehoz egy változót az Azure Automationben.
A változó titkosításához adja meg *a Titkosított paramétert.*
Létrehozás után a változó titkosított állapota nem módosítható.

## PÉLDÁK

### 1. példa: Egyszerű értékkel létrehozott változó
```
PS C:\>New-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -Encrypted $False -Value "My String" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs létrehoz egy StringVariable22 nevű változót, amely a Contoso17 nevű automatizálási fiókban egy karakterlánc-értéket ad meg.

### 2. példa: Változó létrehozása komplex értékkel
```
PS C:\>$VirtualMachine = Get-AzVM -ServiceName "VirtualMachine" -Name "VirtualMachine03"
PS C:\> New-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "ComplexVariable01" -Encrypted $False -Value $VirtualMachine -ResourceGroupName "ResourceGroup01"
```

Az első parancs egy virtuális gépet kap a Get-AzVM parancsmag használatával.
A parancs a $VirtualMachine tárolja.
A második parancs létrehoz egy ComplexVariable01 nevű változót a Contoso17 nevű automatizálási fiókban.
Ez a parancs egy komplex objektumot használ az értékében, ebben az esetben a virtuális $VirtualMachine.
