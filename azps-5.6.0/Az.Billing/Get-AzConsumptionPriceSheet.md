# Get-AzConsumptionPriceSheet

## SYNOPSIS
Szerezze be az előfizetés árlapját.

## SZINTAXIS

```
Get-AzConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
