# Get-AzBlueprintArtifact

## SYNOPSIS
Az összetevők lekérése a tervrajz-definícióból

## SZINTAXIS

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Az összetevők lekérése a tervrajz-definícióból Ha nincs megadva tervrajz-definíciós verzió, a program beolvassa a vázlatverziót. Abban az esetben, ha nincs piszkozat verzió, a legújabb közzétett tervdefiníciót ad vissza a program.
