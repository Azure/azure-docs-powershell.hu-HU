# Get-AzCognitiveServicesAccount

## SYNOPSIS
Fiókot kap.

## SZINTAXIS

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzCognitiveServicesAccount** parancsmag a ResourceGroupName paraméterrel megadott erőforráscsoportban kapja meg a kiépített *Kognitív szolgáltatások-fiókokat.*
Ha nem adja meg az *ResourceGroupName* paramétert, ez a parancsmag az aktuális előfizetés összes kognitív szolgáltatásfiókját megkapja.
