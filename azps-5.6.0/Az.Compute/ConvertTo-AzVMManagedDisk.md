# ConvertTo-AzVMManagedDisk

## SYNOPSIS
Blob-alapú lemezeket tároló virtuális gép átalakítása felügyelt lemezeket tároló virtuális gépké.

## SZINTAXIS

```
ConvertTo-AzVMManagedDisk [-ResourceGroupName] <String> [-VMName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **ConvertTo-AzVMManagedDisk** parancsmag egy blobalapú lemezeket tároló virtuális gépet felügyelt lemezeket tároló virtuális gépké konvertál.
A művelet meghozása előtt le kell állítani a virtuális gépet.

## PÉLDÁK

### 1. példa
```
PS C:\> ConvertTo-AzVMManagedDisk -ResourceGroupName 'ResourceGroup01' -VMName 'VM01'
```

Ez a parancs felügyelt lemezeket hoz létre az "ResourceGroup01" erőforráscsoport virtuális gép "VM01" nevű blobalapú lemezeiből.
