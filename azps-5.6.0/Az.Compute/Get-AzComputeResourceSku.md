# Get-AzComputeResourceSku

## SYNOPSIS
Az összes számítási erőforrás-skus felsorolása

## SZINTAXIS

```
Get-AzComputeResourceSku [[-Location] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Az összes számítási erőforrás-skus felsorolása

## PÉLDÁK

### 1. példa
```
PS C:\> Get-AzComputeResourceSku "westus";
```

Az összes számítási erőforrás-skus felsorolása az Amerikai Egyesült Térség nyugati részén
