# Get-AzVMImagePublisher

## SYNOPSIS
Beveszi a VMImage-közzétevőket.

## SZINTAXIS

```
Get-AzVMImagePublisher -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzVMImagePublisher** parancsmag beveszi a VMImage-közzétevőket.

## PÉLDÁK

### 1. példa: VMImage-közzétevők lekérte egy régióhoz
```
PS C:\> Get-AzVMImagePublisher -Location "Central US"
```

Ez a parancs lekérte a VMImage-példányok közzétevőit az Azure-profilján belül az Amerikai Egyesült Államok középső régiójában.
