# Get-AzVmss

## SYNOPSIS
A VMSS tulajdonságainak lekérte.

## SZINTAXIS

### DefaultParameter (alapértelmezett)
```
Get-AzVmss [[-ResourceGroupName] <String>] [[-VMScaleSetName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FriendMethod
```
Get-AzVmss [[-ResourceGroupName] <String>] [[-VMScaleSetName] <String>] [-InstanceView]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### OSUpgradeHistoryMethodParameter
```
Get-AzVmss [[-ResourceGroupName] <String>] [[-VMScaleSetName] <String>] [-OSUpgradeHistory]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzVmss** parancsmag egy virtuális gép méretarány-készletének (VMSS) modell- és példánynézetét kapja meg.
A modellnézet a virtuális gép méretaránykészletének felhasználó által megadott tulajdonságai.
A példánynézet a virtuális gép méretaránykészletének példányszintű állapota.
Adja meg *a InstanceView* paramétert úgy, hogy csak a virtuális gép méretaránykészletének példánynézetét adja meg.
