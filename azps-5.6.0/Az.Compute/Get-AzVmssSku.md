# Get-AzVmssSku

## SYNOPSIS
A VMSS-hez elérhető SKUs-okat kapja.

## SZINTAXIS

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzVmssSku** parancsmag megkapja a virtuálisgép-mérethalmazhoz (VMSS) elérhető termékváltozatokat.

## PÉLDÁK

### 1. példa: Az összes elérhető SKUs lekérte a VMSS-ről
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Ez a parancs a ContosoVMSS nevű VMSS-hez tartozó összes elérhető SKUs-t beveszi a ContosoGroup nevű erőforráscsoportba.
