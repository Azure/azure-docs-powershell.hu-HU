# Invoke-AzVMReimage

## SYNOPSIS
Egy Azure virtuális gép újraimizálása.

## SZINTAXIS

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Invoke-AzVMReimage** parancsmag újragondol egy Azure-virtuális gépet.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Ez a parancs újragondolja a VirtualMachine07 nevű virtuális gépet a ResourceGroup11-ben.
