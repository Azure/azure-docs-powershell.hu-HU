# Revoke-AzDiskAccess

## SYNOPSIS
Lemezhez való hozzáférés visszavonása.

## SZINTAXIS

```
Revoke-AzDiskAccess [-ResourceGroupName] <String> [-DiskName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Revoke-AzDiskAccess** parancsmag visszavonja a lemezhez való hozzáférést.

## PÉLDÁK

### 1. példa
```
PS C:\> Revoke-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

A "Disk01" lemezhez való hozzáférés visszavonása az "ResourceGroup01" erőforráscsoportban
