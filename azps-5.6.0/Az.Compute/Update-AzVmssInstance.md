# Update-AzVmssInstance

## SYNOPSIS
Elindítja a VMSS-példány manuális frissítését.

## SZINTAXIS

```
Update-AzVmssInstance [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-InstanceId] <String[]>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Update-AzVmssInstance parancsmag elindítja a virtuálisgép-mérethalmaz (VMSS) adott példányának manuális frissítését.
Ezt akkor használja a rendszer, ha a VMSS scale set frissítési házirend beállítása manuális.

## PÉLDÁK

### 1. példa: A VMSS-példány frissítésének kezdése
```
PS C:\> Update-AzVmssInstance -ResourceGroupName "Group011" -VMScaleSetName "VMScaleSet001" -InstanceId "0"
```

Ez a parancs elindítja a VMScaleSet001 nevű VMSS frissítését, amely 0 példányazonosítóval rendelkezik.
