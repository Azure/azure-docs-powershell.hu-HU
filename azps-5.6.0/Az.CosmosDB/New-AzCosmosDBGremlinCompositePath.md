# New-AzCosmosDBGremlinCompositePath

## SYNOPSIS
Létrehoz egy új, PSCompositePath típusú objektumot. A Set-AzCosmosDBGremlinGraph paraméterértékeként is át lehet adni.

## SZINTAXIS

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A Gremlin API CompositePath-jának megfelelő objektum.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
