# New-AzCosmosDBSqlUniqueKeyPolicy

## SYNOPSIS
Létrehoz egy új TárolóDB SqlUniqueKeyPolicy objektumot.

## SZINTAXIS

```
New-AzCosmosDBSqlUniqueKeyPolicy -UniqueKey <PSSqlUniqueKey[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzCosmosDBSqlUniqueKeyPolicy** parancsmag létrehoz egy új, PSSqlUniqueKeyPolicy típusú objektumot.

## PÉLDÁK

### 1. példa
```powershell
PS C:\>New-AzCosmosDBSqlUniqueKeyPolicy -UniqueKey {psUniqueKey1, psUniqueKey2}
```
