# Remove-AzDataFactoryDataset

## SYNOPSIS
Eltávolít egy adatkészletet az Azure Data Factoryból.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Remove-AzDataFactoryDataset [-Force] [-DataFactoryName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryDataset [-Force] [-DataFactory] <PSDataFactory> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataFactoryDataset** parancsmag eltávolít egy adatkészletet az Azure Data Factoryból.

## PÉLDÁK

### 1. példa: Adatkészlet eltávolítása
```
PS C:\>Remove-AzDataFactoryDataset -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -Name "DAWikiAggregatedData"
Confirm
Are you sure you want to remove dataset 'DAWikiAggregatedData' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
True
```

Ez a parancs eltávolítja a DAWikiAggregatedData nevű adatkészletet a WikiADF nevű adatüzemből.
A parancs egy számértéket ad $True.
