# Remove-AzDataFactoryLinkedService

## SYNOPSIS
Eltávolít egy csatolt szolgáltatást az Azure Data Factoryból.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Remove-AzDataFactoryLinkedService [-Force] [-DataFactoryName] <String> [-Name] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryLinkedService [-Force] [-DataFactory] <PSDataFactory> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataFactoryLinkedService** parancsmag eltávolít egy csatolt szolgáltatást az Azure Data Factoryból.

## PÉLDÁK

### 1. példa: Csatolt szolgáltatás eltávolítása
```
PS C:\>Remove-AzDataFactoryLinkedService -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -Name "LinkedServiceTest"
Confirm
Are you sure you want to remove linked service 'LinkedServiceTest' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Ez a parancs eltávolítja a LinkedServiceTest nevű csatolt szolgáltatást a WikiADF nevű adatüzemből.
Ez a parancs egy számértéket ad $True.
