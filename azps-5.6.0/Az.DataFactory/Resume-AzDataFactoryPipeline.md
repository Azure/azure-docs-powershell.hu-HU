# Resume-AzDataFactoryPipeline

## SYNOPSIS
Felfüggesztett folyamat folytatása a Data Factoryban.

## SZINTAXIS

### ByFactoryName (alapértelmezett)
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Resume-AzDataFactoryPipeline** parancsmag felfüggesztett folyamatot folytat az Azure Data Factoryban.

## PÉLDÁK

### 1. példa: Folyamat folytatása
```powershell
PS C:\>Resume-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to resume pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Ez a parancs folytatja a DPWikisample nevű folyamatot a WikiADF nevű adatüzemben.
A **Suspend-AzDataFactoryPipeline** parancsmagot használva függesztheti fel a folyamatot.
A parancs egy számértéket ad $True.

### 2. példa

Felfüggesztett folyamat folytatása a Data Factoryban. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
Resume-AzDataFactoryPipeline -DataFactory $DataFactory -Name 'DPWikisample'
```
