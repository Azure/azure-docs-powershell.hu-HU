# Remove-AzDataLakeAnalyticsComputePolicy

## SYNOPSIS
Egy megadott Azure Data Lake Analytics számítási házirend eltávolítása

## SZINTAXIS

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataLakeAnalyticsComputePolicy** eltávolít egy megadott Azure Data Lake Analytics számítási házirendet.

## PÉLDÁK

### 1. példa: Számítási házirend eltávolítása
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Ez a parancs eltávolítja a "contosoadla" fiókban a "myPolicy" nevű megadott számítási házirendet.
