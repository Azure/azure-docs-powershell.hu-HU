# Stop-AzDataLakeAnalyticsJob

## SYNOPSIS
Megszakít egy feladatot.

## SZINTAXIS

```
Stop-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Stop-AzDataLakeAnalytics Job** parancsmag megszakítja az Azure Data Lake Analytics-feladatot.

## PÉLDÁK

### 1. példa: Feladat lemondása
```
PS C:\>Stop-AzDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Ez a parancs megszakítja a feladatot a megadott azonosítóval.
