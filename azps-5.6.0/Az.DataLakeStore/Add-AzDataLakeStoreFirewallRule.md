# Add-AzDataLakeStoreFirewallRule

## SYNOPSIS
Tűzfalszabály hozzáadása a megadott Data Lake Store-fiókhoz.

## SZINTAXIS

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Add-AzDataLakeStoreFirewallRule parancsmag** hozzáad egy tűzfalszabályt a megadott Data Lake Store-fiókhoz.

## PÉLDÁK

### 1. példa: Új tűzfalszabály hozzáadása Data Lake Store-fiókhoz
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Ezzel létrehoz egy "MyRule" nevű új tűzfalszabályt a "ContosoADL" fiókban, 127.0.0.1-1-127.0.0.2 IP-tartománcával
