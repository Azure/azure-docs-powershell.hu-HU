# Remove-AzDataLakeStoreAccount

## SYNOPSIS
A Data Lake Store-fiók végleges törlése.

## SZINTAXIS

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataLakeStoreAccount** parancsmag véglegesen töröl egy Data Lake Store-fiókot.

## PÉLDÁK

### 1. példa: Adattavatár-fiók eltávolítása
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Ez a parancs eltávolítja a ContosoADL nevű fiókot az adattavas áruházból.
