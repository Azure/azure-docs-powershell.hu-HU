# Remove-AzDataLakeStoreTrustedIdProvider

## SYNOPSIS
A megadott megbízható identitásszolgáltató eltávolítása a megadott Data Lake Store-ból.

## SZINTAXIS

```
Remove-AzDataLakeStoreTrustedIdProvider [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDataLakeStoreTrustedIdProvider** parancsmag eltávolítja a megadott megbízható identitásszolgáltatót a megadott Data Lake Store-ból.

## PÉLDÁK

### 1. példa: Megbízható identitásszolgáltató eltávolítása.
```
PS C:\> Remove-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider
```

A "MyProvider" szolgáltató eltávolítása a "ContosoADL" fiókból
