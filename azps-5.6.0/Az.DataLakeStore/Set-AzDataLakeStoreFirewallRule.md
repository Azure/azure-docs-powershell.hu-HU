# Set-AzDataLakeStoreFirewallRule

## SYNOPSIS
Módosítja a megadott tűzfalszabályt a megadott Data Lake Store-ban.

## SZINTAXIS

```
Set-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzDataLakeStoreFirewallRule** parancsmag módosítja a megadott tűzfalszabályt a megadott Data Lake Store-ban.

## PÉLDÁK

### 1. példa: Tűzfalszabály kezdő és záró IP-tartományának frissítése
```
PS C:\> Set-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

A "ContosoADL" fiók "MyFirewallRule" tűzfalszabályának frissítése 127.0.0.1- 127.0.0.2 tartományra
