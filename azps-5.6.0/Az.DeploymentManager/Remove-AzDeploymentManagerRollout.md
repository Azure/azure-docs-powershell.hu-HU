# Remove-AzDeploymentManagerRollout

## SYNOPSIS
A bevezetés törlése.

## SZINTAXIS

### Interaktív (alapértelmezett)
```
Remove-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerRollout [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzDeploymentManagerRollout** parancsmag egy bevezetést töröl egy terminálállapotban.
Adja meg a bevezetést a neve és az erőforráscsoport neve szerint. Másik módon meg is használhatja a bevezetési objektumot vagy az Erőforrásazonosítót.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout
```

Ez a parancs törli a ContosoRollout nevű bevezetést a ContosoResourceGroup csoportban.

### 2. példa: Bevezetés törlése az erőforrás-azonosító használatával
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Ez a parancs törli a ContosoRollout nevű bevezetést a ContosoResourceGroup csoportban.

### 3. példa: Bevezetés törlése a bevezetési objektum használatával.
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Ez a parancs törli a névvel és erőforráscsoporttal egyező névvel és erőforráscsoportnévvel $rolloutObject nevét.
