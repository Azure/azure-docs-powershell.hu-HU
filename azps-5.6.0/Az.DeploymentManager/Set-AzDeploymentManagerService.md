# Set-AzDeploymentManagerService

## SYNOPSIS
Frissíti a szolgáltatást.

## SZINTAXIS

```
Set-AzDeploymentManagerService [-InputObject] <PSServiceResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzDeploymentManagerService parancsmag** frissíti a szolgáltatást a megadott szolgáltatásobjektummal.
A parancsmag a frissített szolgáltatásobjektumot adja vissza.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceObject
```

Ez a parancs frissíti azt a szolgáltatást, amelynek a neve, a szolgáltatás topológiája és az Erőforráscsoport neve megegyezik a $serviceObject, ServiceTopologyName és ResourceGroupName tulajdonságaival.
A szolgáltatás frissülni fog a $serviceObject.
