# Az.DevSpaces modul
## Leírás
Az Azure DevSpaces-vezérlővel való interakcióra alkalmazott parancsok.

## Az.DevSpaces parancsmagok
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Szerezze be vagy sorolja fel az Azure DevSpaces-vezérlőt.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Hozzon létre egy új Azure DevSpaces-vezérlőt.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
DevSpaces-vezérlő törlése

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Frissítse a DevSpaces-vezérlőt címkék hozzáadásához.
