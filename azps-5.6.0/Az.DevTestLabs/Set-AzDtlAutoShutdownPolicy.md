# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Egy laboratóriumi DevTest Labs automatikus leállítási házirendet állít be.

## SZINTAXIS

### Engedélyezés (alapértelmezett)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Letiltás
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzDtlAutoShutdownPolicy** parancsmag beállítja egy laboratórium automatikus leállítási házirendét, amely a nap adott időszakában automatikusan leállítja a laboratórium összes virtuális gépét.
A parancsmag a megadott erőforráscsoportot és a laboratórium nevét használja a házirend beállításhoz.

## PÉLDÁK

### 1. példa

Egy laboratóriumi DevTest Labs automatikus leállítási házirendet állít be. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoShutdownPolicy -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
