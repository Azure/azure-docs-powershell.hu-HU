# Disable-AzHDInsightMonitoring

## SYNOPSIS
A HDInsight-fürtben való figyelés letiltása és a kapcsolódó naplók az engedélyezés során megadott figyelési munkaterületre fognak áramni.

## SZINTAXIS

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Disable-AzHDInsightMonitoring** parancsmag letiltja a figyelés az Azure HDInsight-fürtben.

## PÉLDÁK

### 1. példa
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

A HDInsight-fürtben le lesz tiltva a figyelés, és a kapcsolódó naplók nem fognak tovább áramlni a figyelési munkaterületre.

### 2. példa
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

A HDInsight-fürtben le lesz tiltva a figyelés, és a kapcsolódó naplók nem fognak tovább áramlni a figyelési munkaterületre.
