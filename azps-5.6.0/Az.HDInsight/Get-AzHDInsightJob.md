# Get-AzHDInsightJob

## SYNOPSIS
Beolvassa a feladatok listáját egy fürtből, és fordított időrendi sorrendben listázza őket, vagy lekér egy adott feladatot.

## SZINTAXIS

```
Get-AzHDInsightJob [-ClusterName] <String> [-HttpCredential] <PSCredential> [[-JobId] <String>]
 [-NumOfJobs <Int32>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzHDInsight Jobs** parancsmag fordított időrendi sorrendben kapja meg egy adott Azure HDInsight-fürt legutóbbi feladatokat, a lista tetején pedig a legújabb feladat szerepel.
A JobId paraméter megadva *adott feladatot kap.*

## PÉLDÁK

### 1. példa: Legutóbbi feladatok lekérte egy adott Azure HDInsight-fürthöz
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

# Hive job details
PS C:\> $statusFolder = "tempStatusFolder/"
PS C:\> $query = "SHOW TABLES"

PS C:\> New-AzHDInsightHiveJobDefinition -StatusFolder $statusFolder `
            -Query $query `
        | Start-AzHDInsightJob -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
        | Get-AzHDInsightJob -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Ez a parancs a -hadoop-001 nevű fürt összes legutóbbi munkáját beveszi.
