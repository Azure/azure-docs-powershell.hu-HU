# Remove-AzHDInsightClusterAutoscaleConfiguration

## SYNOPSIS
A HDInsight-fürt automatikus skálázási konfigurációjának eltávolítása.

## SZINTAXIS

### RemoveByNameParameterSet (alapértelmezett)
```
Remove-AzHDInsightClusterAutoscaleConfiguration [[-ResourceGroupName] <String>] [-ClusterName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceIdParameterSet
```
Remove-AzHDInsightClusterAutoscaleConfiguration [-ResourceId] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObjectParameterSet
```
Remove-AzHDInsightClusterAutoscaleConfiguration [-InputObject] <AzureHDInsightCluster> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzHDInsightClusterAutoscaleConfiguration parancsmag** eltávolítja a HDInsight-fürt automatikus skálás konfigurációját.

## PÉLDÁK

### 1. példa: Távolítsa el a HDInsight-fürt automatikus skálázási konfigurációját.
```powershell
PS C:\> $clusterResourceGroup="group"
PS C:\> $clusterName="MyCluster"
PS C:\> Remove-AzHDInsightClusterAutoscaleConfiguration -ResourceGroupName $clusterResourceGroup -ClusterName $clusterName
```

Ez a parancs eltávolítja a HDInsight-fürt automatikus skálázási konfigurációját.
