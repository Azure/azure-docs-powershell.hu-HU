# Set-AzHDInsightGatewayCredential

## SYNOPSIS
Beállítja egy Azure HDInsight-fürt átjáró HTTP-hitelesítő adatait.

## SZINTAXIS

### SetByNameParameterSet (alapértelmezett)
```
Set-AzHDInsightGatewayCredential [-Name] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByInputObjectParameterSet
```
Set-AzHDInsightGatewayCredential [-HttpCredential] <PSCredential> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] -InputObject <AzureHDInsightCluster> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByResourceIdParameterSet
```
Set-AzHDInsightGatewayCredential [-HttpCredential] <PSCredential> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] -ResourceId <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzHDInsightGatewayCredential** parancsmag beállítja egy Azure HDInsight-fürt átjáró hitelesítő adatait.

## PÉLDÁK

### 1. példa
```powershell
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Set-AzHDInsightGatewayCredential `
            -ClusterName $clusterName `
            -HttpCredential $clusterCreds
```

Ez a parancs beállítja az átjáró hitelesítő adatait a "-hadoop-001" nevű fürthöz név paraméterkészlet szerint.

### 2. példa
```powershell
PS C:\> Set-AzHDInsightGatewayCredential `
            -ResourceId "/subscriptions/{subscriptionId}/resourceGroups/{resourceGroupName}/providers/Microsoft.HDInsight/clusters/your-hadoop-001" `
            -HttpCredential $clusterCreds
```

Ez a parancs a ResourceId paraméterkészlet által megadott" nevű fürt átjárói hitelesítő adatait állítja be.

### 3. példa
```powershell
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Get-AzHDInsightCluster -ClusterName $clusterName | Set-AzHDInsightGatewayCredential `
            -HttpCredential $clusterCreds
```

Ez a parancs az InputObject paraméterkészlettel beállítja a saját-hadoop-001 nevű fürt átjárói hitelesítő adatait.
