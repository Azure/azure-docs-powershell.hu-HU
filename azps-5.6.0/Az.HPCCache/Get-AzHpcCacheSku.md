# Get-AzHpcCacheSku

## SYNOPSIS
Az előfizetésben elérhető összes SKUs elérhetővé válik.

## SZINTAXIS

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzHpcCacheSku** parancsmag az előfizetésben elérhető termékváltozatok listáját adja vissza.
