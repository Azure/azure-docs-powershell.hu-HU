# Get-AzIotHubJob

## SYNOPSIS
Információt kap egy IotHub-feladatról.

## SZINTAXIS

```
Get-AzIotHubJob [-ResourceGroupName] <String> [-Name] <String> [[-JobId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Információkat kap az IotHub-feladatról.
Az IotHub-feladat akkor jön létre, amikor importálási vagy exportálási művelet inicializálódik a New-AzIotHubExportDevices vagy New-AzIotHubImportDevices paranccsal.
Felsorolhatja az összes feladatot, vagy szűrheti őket a feladatazonosító alapján.

## PÉLDÁK

### 1. példa az összes feladat felsorolásához
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

A myiothub nevű IotHub összes állását átveszi

### 2. példa: Adott feladat lekérte
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub" -JobId 3630fc31-4caa-43e8-a232-ea0577221cb2
```

Információkat kap a feladatról a "myiothub" nevű IotHub "3630fc31-4caa-43e8-a232-ea0577221cb2" azonosítóval
