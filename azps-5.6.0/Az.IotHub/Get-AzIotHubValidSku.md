# Get-AzIotHubValidSku

## SYNOPSIS
Minden olyan érvényes kódot bevesz, amelybe ez az IotHub át tud áttűnni.

## SZINTAXIS

```
Get-AzIotHubValidSku [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
Beveszi az összes olyan érvényes kódot, amelybe ez az IotHub át tud áttűnni.
Az IotHub nem tud áttűnni a szabad és a fizetős skus között, és fordítva. Ehhez törölnie és újból létre kell hoznia az iothubot.

## PÉLDÁK

### 1. példa Az érvényes skus lekérte
```
PS C:\> Get-AzIotHubValidSku -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

A "myiothub" nevű IotHub összes skusját tartalmazza
