# Invoke-AzIotHubQuery

## SYNOPSIS
Egy hatékony, SQL-hez hasonló nyelvet használva lekérdezheti az IoT-központot.

## SZINTAXIS

### ResourceSet (alapértelmezett)
```
Invoke-AzIotHubQuery [-ResourceGroupName] <String> [-IotHubName] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubQuery [-InputObject] <PSIotHub> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubQuery [-ResourceId] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az IoT-központban egy hatékony SQL-hez hasonló nyelven lekérdezheti az eszköz- és modullekérdezésekkel, a feladatokkal és az üzenetek átirányításával kapcsolatos információkat.
További https://docs.microsoft.com/azure/iot-hub/iot-hub-devguide-query-language információt itt is láthat.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices"
```

Lekérdezheti az összes eszköz eszközét egy Azure IoT-központban.

### 2. példa
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices.modules where devices.deviceId = 'myDevice1'" -Top 2
```

Lekérdezés a 2 modul legfelső adatokat egy céleszközön.
