# New-AzIotHubKey

## SYNOPSIS
Azure IoT-központ kulcs létrehozása

## SZINTAXIS

### ResourceSet (alapértelmezett)
```
New-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String> [-RenewKey] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
New-AzIotHubKey [-HubId] <String> [-KeyName] <String> [-RenewKey] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Azure IoT-központ kulcs létrehozása

## PÉLDÁK

### 1. példa: Az elsődleges kulcs újragenerálása
```
PS C:\> New-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "testKey" -RenewKey "primary"

KeyName     PrimaryKey                                      SecondaryKey                                        Rights
-------     ----------                                      ------------                                        ------

```

Az Azure iot-központ "testKey" engedélyezési házirendje által újragenerált elsődleges kulcs.

### 2. példa a billentyűk felcserélésére
```
PS C:\> New-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "testKey" -RenewKey "swap"

KeyName     PrimaryKey                                      SecondaryKey                                        Rights
-------     ----------                                      ------------                                        ------

```

Kulcsok felcserélése egy Azure iot-központ "testKey" engedélyezési házirendjére.
