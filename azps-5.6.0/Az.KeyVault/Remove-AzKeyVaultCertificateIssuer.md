# Remove-AzKeyVaultCertificateIssuer

## SYNOPSIS
Törli a tanúsítvány kibocsátóját egy kulcstárolóból.

## SZINTAXIS

### Alapértelmezett (alapértelmezett)
```
Remove-AzKeyVaultCertificateIssuer [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzKeyVaultCertificateIssuer [-InputObject] <PSKeyVaultCertificateIssuerIdentityItem> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzKeyVaultCertificateIssuer** parancsmag törli a tanúsítvány kibocsátóját egy kulcstárolóból.

## PÉLDÁK

### 1. példa: Tanúsítvány kibocsátójának eltávolítása
```powershell
PS C:\> Remove-AzKeyVaultCertificateIssuer -VaultName "ContosoKV01" -Name "TestIssuer01" -Force

AccountId           :
ApiKey              :
OrganizationDetails :
Name                : TestIssuer01
IssuerProvider      : test
VaultName           : ContosoKV01
```

Ez a parancs eltávolítja a TestIssuer01 nevű tanúsítvány kibocsátóját a ContosoKV01 kulcstárolóból.
