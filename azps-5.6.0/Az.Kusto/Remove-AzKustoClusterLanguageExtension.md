# Remove-AzKustoClusterLanguageExtension

## SYNOPSIS
Távolítsa el a KQL-lekérdezésekben futtatható nyelvi bővítmények listáját.

## SZINTAXIS

### RemoveExpanded (alapértelmezett)
```
Remove-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-Value <ILanguageExtension[]>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### RemoveViaIdentityExpanded
```
Remove-AzKustoClusterLanguageExtension -InputObject <IKustoIdentity> [-Value <ILanguageExtension[]>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## LEÍRÁS
Távolítsa el a KQL-lekérdezésekben futtatható nyelvi bővítmények listáját.

## PÉLDÁK

### 1. példa: Nyelvbővítmények listájának eltávolítása a fürtből
```powershell
PS C:\> Remove-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster -Value (@{Name="R"})
```

A fenti parancs eltávolítja a KQL-lekérdezésekben futtatható nyelvi bővítmények listáját.
