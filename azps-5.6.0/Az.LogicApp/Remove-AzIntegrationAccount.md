# Remove-AzIntegrationAccount

## SYNOPSIS
Eltávolít egy integrációs fiókot.

## SZINTAXIS

```
Remove-AzIntegrationAccount -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzIntegrationAccount** parancsmag eltávolít egy integrációs fiókot egy erőforráscsoportból.
Adja meg az integrációs fiók nevét és az erőforráscsoport nevét.
Ez a modul támogatja a dinamikus paramétereket.
Ha dinamikus paramétert használ, írja be a parancsba.
A dinamikus paraméterek nevének felfedezéséhez írjon be egy kötőjelet (-) a parancsmag neve után, majd a Tab billentyű többszöri lenyomása után lépkedhet a rendelkezésre álló paraméterek között.
Ha kihagy egy kötelező sablonparamétert, a parancsmag rákérdez az értékre.

## PÉLDÁK

### 1. példa: Integrációs fiók eltávolítása
```
PS C:\>Remove-AzIntegrationAccount -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -Force
```

Ez a parancs eltávolít egy IntegrationAccount31 nevű integrációs fiókot.
