# Remove-AzIntegrationAccountBatchConfiguration

## SYNOPSIS
Eltávolítja az integrációs fiók kötegkonfigurációját.

## SZINTAXIS

### ByIntegrationAccount (alapértelmezett)
```
Remove-AzIntegrationAccountBatchConfiguration -ResourceGroupName <String> -ParentName <String> -Name <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzIntegrationAccountBatchConfiguration -InputObject <PSIntegrationAccountBatchConfiguration> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzIntegrationAccountBatchConfiguration -ResourceId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzIntegrationAccountBatchConfiguration** parancsmag eltávolít egy kötegkonfigurációt egy integrációs fiókból.

## PÉLDÁK

### 1. példa: Köteg-konfiguráció eltávolítása paraméterek alapján
```powershell
PS C:\> Remove-AzIntegrationAccountBatchConfiguration -ResourceGroupName "sampleResourceGroup" -IntegrationAccountName "sampleIntegrationAccount" -BatchConfigurationName "sampleBatchConfig"
```

Eltávolítja a "sampleBatchConfig" nevű kötegkonfigurációt, amely a "sampleIntegrationAccount" integrációs fiókban található.
