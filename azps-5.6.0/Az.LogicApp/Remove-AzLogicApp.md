# Remove-AzLogicApp

## SYNOPSIS
Eltávolít egy logikai alkalmazást egy erőforráscsoportból.

## SZINTAXIS

```
Remove-AzLogicApp -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-Az LogicApp** parancsmag a Logic Apps funkcióval eltávolítja a logikai alkalmazásokat egy erőforráscsoportból.
Adja meg a logikai alkalmazást és az erőforráscsoportot.
Ez a modul támogatja a dinamikus paramétereket.
Ha dinamikus paramétert használ, írja be a parancsba.
A dinamikus paraméterek nevének felfedezéséhez írjon be egy kötőjelet (-) a parancsmag neve után, majd a Tab billentyű többszöri lenyomása után lépkedhet a rendelkezésre álló paraméterek között.
Ha kihagy egy kötelező sablonparamétert, a parancsmag rákérdez az értékre.

## PÉLDÁK

### 1. példa: Logikai alkalmazás eltávolítása
```
PS C:\>Remove-AzLogicApp -ResourceGroupName "ResourceGroup11" -Name "LogicApp03" -Force
```

Ez a parancs eltávolít egy logikai alkalmazást egy erőforráscsoportból.
A parancs tartalmazza az *Kényszerítés* paramétert, amely megakadályozza, hogy a parancs megerősítést kér.
