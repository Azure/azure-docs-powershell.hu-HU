# Update-AzMlCommitmentPlan

## SYNOPSIS
Frissíti egy meglévő kötelezettségvállalási terv típusú erőforrás tulajdonságait.

## SZINTAXIS

```
Update-AzMlCommitmentPlan -ResourceGroupName <String> -Name <String> -SkuName <String> -SkuTier <String>
 [-SkuCapacity <Int32>] [-Tag <Hashtable>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Frissíti a meglévő kötelezettségvállalási terv erőforrását. Ne feledje, hogy a kötelezettségvállalási terv legtöbb tulajdonsága nem módosítható, és nem módosítható. A módosítható tulajdonságok közé tartozik a termékváltozat (amely lehetővé teszi a kötelezettségvállalási terv áttelepítését egyik termékváltozatból a másikba) és a címkéket.

## PÉLDÁK

### 1. példa: Kötelezettségvállalási terv frissítése
```
Update-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName" -Tags @{'MyTagKey'='MyTagValue'}
```
