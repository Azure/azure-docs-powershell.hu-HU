# Az.ManagementPartner modul
## Leírás
Az Azure Resource Manager felügyeleti partner parancsmagja.

## Az.ManagementPartner parancsmagok
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
Az aktuális hitelesített felhasználó vagy egyszerű szolgáltatásnév Microsoft Partner Network(MPN)-azonosítóját kapja meg. 

### [New-AzManagementPartner](New-AzManagementPartner.md)
Egy Microsoft Partner Network(MPN) azonosítót társít az aktuálisan hitelesített felhasználóhoz vagy szolgáltatásnévhez.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
Törölje az aktuálisan hitelesített felhasználó vagy egyszerű szolgáltatásnév Microsoft Partner Network(MPN) azonosítóját.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
Frissíti a Microsoft Partner Network(MPN) azonosítóját a jelenlegi hitelesített felhasználó vagy egyszerű szolgáltatásnévhez.
