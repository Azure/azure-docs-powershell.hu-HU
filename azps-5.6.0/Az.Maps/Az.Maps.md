# Az.Maps modul
## Leírás
Ez a témakör az Azure Maps-parancsmagok súgóját mutatja be.

## Az.Maps parancsmagok
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
Beveszi a fiókot.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
Egy fiók API-kulcsait kapja meg.
Ezek a kulcsok azok a hitelesítési mechanizmusok, amelyek az Azure Maps későbbi hívása során használatosak.

### [New-AzMapsAccount](New-AzMapsAccount.md)
Létrehoz egy Azure Maps-fiókot.

### [New-AzMapsAccountKey](New-AzMapsAccountKey.md)
Újragenerálja a fiókkulcsot.

### [Remove-AzMapsAccount](Remove-AzMapsAccount.md)
Töröl egy Azure Maps-fiókot.
