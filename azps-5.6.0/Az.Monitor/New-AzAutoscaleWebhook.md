# New-AzAutoscaleWebhook

## SYNOPSIS
Automatikus méretezésű webhookot hoz létre.

## SZINTAXIS

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzAutoscaleWebhook** parancsmag létrehoz egy autoskála webhookot.

## PÉLDÁK

### 1. példa

Automatikus méretezésű webhookot hoz létre. (automatikusan generált)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
