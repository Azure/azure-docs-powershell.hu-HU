# Add-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Háttércímkészlet-konfigurációt ad hozzá a terheléselosztáshoz.

## SZINTAXIS

```
Add-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Add-AzLoadBalancerBackend** parancsmag hozzáad egy háttércímkészletet egy Azure load balancerhez.

## PÉLDÁK

### 1. példa: Háttércímkészlet-konfiguráció hozzáadása terheléselosztáshoz
```powershell
PS C:\>Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "myrg" | Add-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" | Set-AzLoadBalancer
```

Ez a parancs leküldi a MyLoadBalancer nevű terheléselosztási készletet, hozzáadja a BackendAddressPool02 nevű háttércímkészletet a MyLoadBalancer programhoz, majd a **Set-AzLoadBalancer** parancsmagot használja a MyLoadBalancer frissítéséhez.
