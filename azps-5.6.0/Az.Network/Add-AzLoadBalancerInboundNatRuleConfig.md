# Add-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Bejövő NAT-szabálykonfigurációt ad a terheléselosztáshoz.

## SZINTAXIS

### SetByResource (alapértelmezett)
```
Add-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfiguration <PSFrontendIPConfiguration>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceId
```
Add-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfigurationId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Add-AzLoadBalancerInboundNatRuleConfig** parancsmag egy bejövő hálózati címfordítási (NAT) szabálykonfigurációt ad egy Azure-terheléselegyenlítóhoz.

## PÉLDÁK

### 1. példa: Bejövő NAT-szabály konfigurációjának hozzáadása terheléselosztáshoz
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Add-AzLoadBalancerInboundNatRuleConfig -Name "NewNatRule" -FrontendIPConfiguration $slb.FrontendIpConfigurations[0] -Protocol "Tcp" -FrontendPort 3350 -BackendPort 3350 -EnableFloatingIP
PS C:\> $slb | Set-AzLoadBalancer
```

Az első parancs lekérte a MyloadBalancer nevű terheléselosztási törlőt, majd tárolja azt a $slb.
A második parancs a folyamat műveleti jelével adja át a $slb terheléselegyenlőt az **Add-AzLoadBalancerInboundNatRuleConfig** szolgáltatónak, amely bejövő NAT-szabálykonfigurációt ad a terheléselegyenlítóhoz.
Az utolsó parancs beállítja a konfigurációt a loadbalancerre, ha nem végzi el a Set-AzLoadBalancer műveletet, a módosítások nem lesznek érvényesek a loadbalancerre.
