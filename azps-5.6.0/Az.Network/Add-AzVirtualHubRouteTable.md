# Add-AzVirtualHubRouteTable

## SYNOPSIS
Virtuális központi útválasztási tábla típusú erőforrást hoz létre, amely a VirtualHub gyermeke.

## SZINTAXIS

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A virtuális központi útválasztási tábla erőforrása tartalmazza az útvonalak és kapcsolatok listáját, amelyekhez csatolható és használható a forgalom virtuális központban való útválasztásához.
