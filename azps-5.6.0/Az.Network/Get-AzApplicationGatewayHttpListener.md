# Get-AzApplicationGatewayHttpListener

## SYNOPSIS
Egy alkalmazás-átjáró HTTP- hallgatóját használja.

## SZINTAXIS

```
Get-AzApplicationGatewayHttpListener [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzApplicationGatewayHttpListener** parancsmag egy alkalmazás-átjáró HTTP- hallgatóját kapja meg.

## PÉLDÁK

### 1. példa: Adott HTTP-hallgató le kérése
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listener = Get-AzApplicationGatewayHttpListener -Name "Listener01" -ApplicationGateway $Appgw
```

Ez a parancs egy Listener01 nevű HTTP-hallgatót kap.

### 2. példa: HTTP-hallgatói listájának le kérése
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listeners = Get-AzApplicationGatewayHttpListener -ApplicationGateway $Appgw
```

Ez a parancs a HTTP-hallgatók listáját tartalmazza.
