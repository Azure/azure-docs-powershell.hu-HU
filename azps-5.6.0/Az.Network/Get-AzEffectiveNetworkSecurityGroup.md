# Get-AzEffectiveNetworkSecurityGroup

## SYNOPSIS
Egy hálózati felület hatékony hálózati biztonsági csoportját használja.

## SZINTAXIS

```
Get-AzEffectiveNetworkSecurityGroup -NetworkInterfaceName <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzEffectiveNetworkSecurityGroup** parancsmag a hálózati felületen alkalmazott hatékony hálózatbiztonsági csoportot adja vissza.

## PÉLDÁK

### 1. példa: A hatékony hálózatbiztonsági csoport kiépítése hálózati felületen
```
PS C:\>Get-AzEffectiveNetworkSecurityGroup -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "myResourceGroup"
```

Ez a parancs a MyNetworkInterface nevű hálózati felülettel társított összes hatékony hálózati biztonsági szabályt megkapja a myResourceGroup nevű erőforráscsoportban.
