# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Előlapi IP-konfigurációt kap a terheléselosztásban.

## SZINTAXIS

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzLoadBalancerFrontendIpConfig** parancsmag egy előlapi IP-konfigurációt vagy az előlapi IP-konfigurációk listáját kapja egy terheléselegyenítőben.

## PÉLDÁK

### 1. példa: Előlapi IP-konfiguráció beállítása terheléselosztásban
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Az első parancs lejátsa a MyLoadBalancer nevű terheléselosztási törlőt, majd tárolja azt a $slb.
A második parancs az adott terheléselosztáshoz társított előlapi IP-konfigurációt kapja meg.
