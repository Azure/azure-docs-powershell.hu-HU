# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Egy vagy több bejövő NAT-készlet-konfigurációt kap egy terheléselosztási eszköztől.

## SZINTAXIS

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzLoadBalancerInboundNatPoolConfig** parancsmag egy vagy több bejövő NAT-készlet-konfigurációt kap egy terheléselegyenlőtől.

## PÉLDÁK

### 1: Get
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
