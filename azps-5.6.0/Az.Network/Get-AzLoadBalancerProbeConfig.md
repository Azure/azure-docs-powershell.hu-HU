# Get-AzLoadBalancerProbeConfig

## SYNOPSIS
Konfigurációt kap a terheléselosztáshoz.

## SZINTAXIS

```
Get-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzLoadBalancerProbeConfig** parancsmag egy vagy több konfigurációt kap a terheléselegyenlők számára.

## PÉLDÁK

### 1. példa: Terheléselosztás konfigurációjának lekérte
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $slb
```

Az első parancs lejátsa a MyLoadBalancer nevű terheléselosztási törlőt, majd tárolja azt a $slb.
A második parancs lekérte a MyProbe nevű konfigurációs konfigurációt a rendszer terheléselosztási $slb.
