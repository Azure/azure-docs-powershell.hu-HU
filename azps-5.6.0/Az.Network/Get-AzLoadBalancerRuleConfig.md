# Get-AzLoadBalancerRuleConfig

## SYNOPSIS
A terheléselosztás szabálykonfigurációját kapja meg.

## SZINTAXIS

```
Get-AzLoadBalancerRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzLoadBalancerRuleConfig** parancsmag egy vagy több szabálykonfigurációt kap a terheléselegyenlők számára.

## PÉLDÁK

### 1. példa: Terheléselosztás szabálykonfigurációjának beállítása
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerRuleConfig -Name "MyLBrulename" -LoadBalancer $slb
```

Az első parancs lejátsa a MyLoadBalancer nevű terheléselosztási törlőt, majd tárolja azt a $slb.
A második parancs a myLName nevű társított szabálykonfigurációt a rendszer terheléselosztási $slb.
