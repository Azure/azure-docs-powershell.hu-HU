# New-AzApplicationGatewayFirewallExclusionConfig

## SYNOPSIS
Új kivételszabálylista létrehozása az alkalmazás-átjáróhoz

## SZINTAXIS

```
New-AzApplicationGatewayFirewallExclusionConfig -Variable <String> -Operator <String> -Selector <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
The **New-AzApplicationGatewayFirewallExclusionConfig cmdlet** a new exclusion rule list for application gateway waf.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $exclusion1 = New-AzApplicationGatewayFirewallExclusionConfig -Variable "RequestHeaderNames" -Operator "StartsWith" -Selector "xyz"
```

Ez a parancs létrehoz egy új kivételszabályt, amely felsorolja a RequestHeaderNames nevű változó és a StartsWith és a Selector nevű xyz nevű operátor konfigurációját. A kivétellista-konfigurációt a rendszer a $exclusion 1 fájlba menti.
