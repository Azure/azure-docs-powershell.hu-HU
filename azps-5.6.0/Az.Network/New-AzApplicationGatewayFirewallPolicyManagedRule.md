# New-AzApplicationGatewayFirewallPolicyManagedRule

## SYNOPSIS
Hozza létre a ManagedRules házirendet a tűzfal házirendhez.

## SZINTAXIS

```
New-AzApplicationGatewayFirewallPolicyManagedRule
 [-ManagedRuleSet <PSApplicationGatewayFirewallPolicyManagedRuleSet[]>]
 [-Exclusion <PSApplicationGatewayFirewallPolicyExclusion[]>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzApplicationGatewayFirewallPolicyManagedRule** létrehoz egy felügyelt szabályokat egy tűzfalszabályhoz.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallPolicyManagedRule -ManagedRuleSet $managedRuleSet -Exclusion $exclusion1,$exclusion2
```

A parancs létrehoz egy felügyelt szabálylistát a ManagedRuleSet listáról, $managedRuleSet egy kivétellistát pedig a következő bejegyzésekkel: $exclusion 1, $exclusion 2.
