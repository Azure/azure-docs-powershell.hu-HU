# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## SYNOPSIS
Létrehozza a RuleGroupOverride bejegyzést a ManagedRuleSets szolgáltatásban a tűzfal házirendhez.

## SZINTAXIS

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** létrehoz egy ruleGroupOverride bejegyzést egy felügyeltRuleSetben tűzfal-házirendhez.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Létrehoz egy RuleGroupOverride bejegyzést, amely a következő néven $ruleName és szabályok $rule 1, $rule 2. Ugyanazt rendeli a $overrideEntry
