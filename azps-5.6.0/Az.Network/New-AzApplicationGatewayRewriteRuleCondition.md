# New-AzApplicationGatewayRewriteRuleCondition

## SYNOPSIS
Hozzáad egy feltételt egy alkalmazás-átjáró RewriteRule-hez.

## SZINTAXIS

```
New-AzApplicationGatewayRewriteRuleCondition -Variable <String> [-Pattern <String>] [-IgnoreCase] [-Negate]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
**AzApplicationGatewayRewriteRuleCondition** parancsmag létrehoz egy újraírási szabályt egy Azure-alkalmazás-átjáróhoz.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $condition = New-AzApplicationGatewayRewriteRuleCondition -Variable "var_request_uri" -Pattern "http" -IgnoreCase
PS C:\> $condition

Variable   : var_request_uri
Pattern    : http
IgnoreCase : True
Negate     : False

PS C:\> $condition | Format-Table

Variable        Pattern IgnoreCase Negate
--------        ------- ---------- ------
var_request_uri http          True  False
```
Ez a parancs létrehoz egy feltételt egy újraírási szabályban, és az eredményt a következő nevű változóban $condition.
