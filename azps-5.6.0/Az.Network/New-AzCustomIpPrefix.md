# New-AzCustomIpPrefix

## SYNOPSIS
CustomIpPrefix erőforrás létrehozása

## SZINTAXIS

```
New-AzCustomIpPrefix -Name <String> -ResourceGroupName <String> -Location <String> -Cidr <String>
 [-Zone <String[]>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **New-AzCustomIpPrefix** parancsmag létrehoz egy CustomIpPrefix erőforrást.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> $myCustomIpPrefix = New-AzCustomIpPrefix -Name $prefixName -ResourceGroupName $rgName -Cidr 40.40.40.0/24 -Location westus
```

Ez a parancs létrehoz egy új CustomIpPrefix erőforrást $prefixName névvel az erőforráscsoportban $rgName 40.40.40.0/24-es cidr értékekkel a WestUsban
