# New-AzPrivateLinkServiceIpConfig

## SYNOPSIS
Hozzon létre egy privát hivatkozásszolgáltatás IP-konfigurációját.

## SZINTAXIS

```
New-AzPrivateLinkServiceIpConfig -Name <String> [-PrivateIpAddressVersion <String>]
 [-PrivateIpAddress <String>] [-PublicIpAddress <PSPublicIpAddress>]
 [-Subnet <PSSubnet>] [-Primary]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzPrivateLinkServiceIpConfig** parancsmag létrehoz egy privát hivatkozásszolgáltatás IP-konfigurációját. Ez a konfiguráció a privát végpontról bejövő forgalom forrás NAT-ingingjében használatos. 

## PÉLDÁK

### 1. példa
```powershell
New-AzPrivateLinkServiceIpConfig -Name $IpConfigurationName -PrivateIpAddress "10.0.0.5" -Primary
```

Ebben a példában egy privát hivatkozásszolgáltatás IP-konfigurációját hozhatja létre a memóriában.

### 2. példa

Hozzon létre egy privát hivatkozásszolgáltatás IP-konfigurációját. (automatikusan generált)

<!-- Aladdin Generated Example -->
```powershell
New-AzPrivateLinkServiceIpConfig -Name 'IP-Config' -PrivateIpAddress '10.0.0.5' -Subnet <PSSubnet>
```
