# New-AzPublicIpTag

## SYNOPSIS
IP-címkét hoz létre.

## SZINTAXIS

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzPublicIpTag** parancsmag létrehoz egy IP-címkét.

## PÉLDÁK

### 1. példa: Új IP-címke létrehozása
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

Ez a parancs létrehoz egy új IP-címkét a "FirstPartyUsage" címkével és a "/Sql" címkével. Ezt használja a publicIpAddress létrehozása ezekkel a konkrét címkékkel a kiosztáshoz.
