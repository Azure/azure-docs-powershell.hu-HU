# New-AzVirtualHubRoute

## SYNOPSIS
Létrehoz egy Azure Virtual Hub Route-objektumot.

## SZINTAXIS

```
New-AzVirtualHubRoute -AddressPrefix <String[]> -NextHopIpAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
Létrehoz egy Azure Virtual Hub Route-objektumot.

## PÉLDÁK

### 1. példa

```powershell
PS C:\> $route1 = New-AzVirtualHubRoute -AddressPrefix @("10.0.0.0/16", "11.0.0.0/16") -NextHopIpAddress "12.0.0.5"

AddressPrefixes            NextHopIpAddress
---------------            ----------------
{10.0.0.0/16, 11.0.0.0/16} 12.0.0.5
```

A fenti virtuális központi útvonalobjektumot hoz létre, amely része lehet a virtuális központi útvonaltáblának.

A virtuális központi útvonal egy memóriaobjektum, amely VirtualHubRouteTable-objektum létrehozásához használható.
