# Remove-AzApplicationGatewayHttpListenerCustomError

## SYNOPSIS
Eltávolít egy egyéni hibát egy alkalmazásátjáró http-hallgatója elől.

## SZINTAXIS

```
Remove-AzApplicationGatewayHttpListenerCustomError -StatusCode <String>
 -HttpListener <PSApplicationGatewayHttpListener> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApplicationGatewayCustomError** parancsmag eltávolít egy egyéni hibát egy alkalmazásátjáró http-hallgatója elől.

## PÉLDÁK

### 1. példa: Egyéni hiba eltávolítása a http-hallgatótól
```powershell
PS C:\> $updatedlistener = Remove-AzApplicationGatewayCustomError -HttpListener $listener01 -StatusCode HttpStatus502
```

Ez a parancs eltávolítja az 502-es http-állapotkódot a http$listener 01-es állapotkódból, és visszaadja a frissített hallgatót.
