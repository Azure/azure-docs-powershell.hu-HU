# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Eltávolít egy SSL-tanúsítványt egy Azure-alkalmazás-átjáróból.

## SZINTAXIS

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzApplicationGatewaySslCertificate** parancsmag eltávolít egy SSL-tanúsítványt egy Azure-alkalmazás átjáróból.

## PÉLDÁK

### 1. példa: SSL-tanúsítvány eltávolítása egy alkalmazás-átjáróból
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Az első parancs az ApplicationGateway01 nevű alkalmazás-átjárót kapja meg, és az eredményt a következő nevű változóban $AppGW.
A második parancs eltávolítja a Cert02 nevű SSL-tanúsítványt a $AppGW átjáróból.
