# Remove-AzDdosProtectionPlan

## SYNOPSIS
Eltávolít egy DDoS védelmi tervet.

## SZINTAXIS

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Remove-AzDdosProtectionPlan eltávolítja a DDoS védelmi tervet.

## PÉLDÁK

### 1. példa: Üres DDoS védelmi terv eltávolítása
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

Ebben az esetben eltávolítjuk a DDoS védelmi tervét a megadottak szerint.

### 2. példa: Virtuális hálózathoz társított DDoS védelmi terv eltávolítása
