# Remove-AzExpressRoutePort

## SYNOPSIS
Eltávolít egy ExpressRoutePortot.

## SZINTAXIS

### ResourceNameParameterSet (alapértelmezett)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzExpressRoutePort** parancsmag eltávolítja az ExpressRoutePortot.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Eltávolítja $PortName ExpressRoutePort-erőforrást $rg erőforráscsoportból az előfizetésében.

### 2. példa
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Eltávolítja az ExpressRoutePort erőforrást az InputObjectben.

### 3. példa
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Eltávolítja az ExpressRoutePort erőforrást a ResourceId $id.
