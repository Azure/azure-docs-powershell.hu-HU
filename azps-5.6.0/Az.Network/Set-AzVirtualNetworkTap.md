# Set-AzVirtualNetworkTap

## SYNOPSIS
Virtuális hálózati koppintás frissítése.

## SZINTAXIS

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzVirtualNetworkTap** parancsmag frissíti a virtuális hálózatra való rákoppintást.

## PÉLDÁK

### 1. példa: Virtuális hálózatra való koppintás konfigurálása
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

A parancs frissíti a Cél IpConfigurációt, és frissíti a virtuális hálózat koppintást.
Ha vannak rá vonatkozó koppintási konfigurációk, akkor a forrásforgalom nem indul el az új cél IP-konfigurációra a frissítés után.
