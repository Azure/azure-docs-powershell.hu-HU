# Test-AzDnsAvailability

## SYNOPSIS
Annak ellenőrzése, hogy használható-e tartománynév a cloudapp.azure.com zónában.

## SZINTAXIS

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
Annak ellenőrzése, hogy használható-e tartománynév a cloudapp.azure.com zónában.

## PÉLDÁK

### 1. példa: Ellenőrizze, contoso.westus.cloudapp.azure.com elérhető-e az alkalmazás.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```
