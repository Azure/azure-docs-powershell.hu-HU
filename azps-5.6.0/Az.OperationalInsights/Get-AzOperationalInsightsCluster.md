# Get-AzOperationalInsightsCluster

## SYNOPSIS
Get or list clusters

## SZINTAXIS

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
A "-ClusterName" meg nem adott, az erőforráscsoport alatti listacsoportokat, illetve a "-ClusterName" és az "ResourceGroupName" nem adott meg, az előfizetés alatt listába sorolt fürtcsoportokat.
