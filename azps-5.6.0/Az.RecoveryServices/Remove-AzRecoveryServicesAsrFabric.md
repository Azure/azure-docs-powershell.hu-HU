# Remove-AzRecoveryServicesAsrFabric

## SYNOPSIS
Törli a megadott Azure-webhely-helyreállítási hálót a helyreállítási szolgáltatások tárolóból.

## SZINTAXIS

```
Remove-AzRecoveryServicesAsrFabric -InputObject <ASRFabric> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzRecoveryServicesAsrFabric** parancsmag eltávolítja a megadott Azure-webhely-helyreállítási hálót a helyreállítási szolgáltatások tárolóból.

## PÉLDÁK

### 1. példa
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrFabric -Fabric $Fabric
```

Megkezdi a megadott anyag törlését, és visszaadja a művelet nyomon követéséhez használt ASR-feladatot.
