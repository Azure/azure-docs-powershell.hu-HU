# Remove-AzRecoveryServicesAsrServicesProvider

## SYNOPSIS
Törli/törli a megadott Azure-webhely-helyreállítási helyreállítási szolgáltatót a helyreállítási szolgáltatások tárolóból.

## SZINTAXIS

```
Remove-AzRecoveryServicesAsrServicesProvider -InputObject <ASRRecoveryServicesProvider> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzRecoveryServicesAsrServicesProvider** parancsmag eltávolítja a megadott Azure Site Recovery helyreállítási szolgáltatót a tárolóból.

## PÉLDÁK

### 1. példa
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrServicesProvider -ServicesProvider $ServicesProvider
```

Elindítja a megadott Azure-webhely-helyreállítási szolgáltató törlését/törlését, és visszaadja a művelet nyomon követéséhez használt ASR-feladatot.
