# New-AzRedisCachePatchSchedule

## SYNOPSIS
Javításütemtervet ad hozzá.

## SZINTAXIS

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **New-AzRedisCachePatchSchedule** parancsmag hozzáad egy javításütemtervet az Azure Redis Cache gyorsítótárában lévő gyorsítótárhoz.

## PÉLDÁK

### 1. példa: Javításütemterv létrehozása és hozzáadása gyorsítótárhoz
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Ez a parancs hozzáad egy javításütemtervet a RedisCache06 nevű gyorsítótárhoz.
Az Entries paraméter értékeként egy olyan parancsot használ, amely a **New-AzRedisCacheScheduleEntry** segítségével hoz létre ütemezést.
