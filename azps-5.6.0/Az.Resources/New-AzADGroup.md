# New-AzADGroup

## SYNOPSIS
Új Active Directory-csoportot hoz létre.

## SZINTAXIS

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Új Active Directory-csoportot hoz létre. Az alábbiakban a szükséges engedélyeket olvashatja:

- Azure Active Directory Graph
  - Directory.ReadWrite.All
- Microsoft Graph
  - Directory.ReadWrite.All
  - PrivilegedAccess.ReadWrite.AzureADGroup

## PÉLDÁK

### 1. példa: Új AD-csoport létrehozása

```powershell
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "MyGroupNick"
```

Létrehoz egy új AD-csoportot a "MyGroupDisplayName" névvel és a "MyGroupNick" e-mail becenévvel.
