# Get-AzSecurityPricing

## SYNOPSIS

Beveszi az Azure Defender csomagokat egy előfizetéshez az Azure Biztonsági központban.

## SZINTAXIS

### SubscriptionScope (alapértelmezett)

```powershell
Get-AzSecurityPricing [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource

```powershell
Get-AzSecurityPricing -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId

```powershell
Get-AzSecurityPricing -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS

Ezzel a parancsmagkal előfizetésenként megtekintheti az egyes Azure Defender-csomagokat.

Az Azure Defenderről és a rendelkezésre álló csomagokról további információt az Azure Defender – [Bevezetés szolgáltatásban talál.](https://docs.microsoft.com/azure/security-center/azure-defender)
