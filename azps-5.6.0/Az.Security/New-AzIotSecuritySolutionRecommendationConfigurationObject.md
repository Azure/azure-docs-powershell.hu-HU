# New-AzIotSecuritySolutionRecommendationConfigurationObject

## SYNOPSIS
Új ajánlott konfiguráció létrehozása iot biztonsági megoldáshoz

## SZINTAXIS

```
New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType <String> -Enabled <Boolean>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## LEÍRÁS
AzIotSecuritySolutionRecommendationConfigurationObject létrehoz egy új ajánlási konfigurációs objektumot az iot biztonsági megoldáshoz.
Ily módon a javaslat állapota konfigurálva van, akár engedélyezve van, akár le van tiltva.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> New-AzIotSecuritySolutionRecommendationConfigurationObject -RecommendationType "IoT_ACRAuthentication" -Enabled $false

RecommendationType: "IoT_ACRAuthentication"
Name: "Service prinicpal not used with ACR repository"
Status: "Disabled"
```

Új ajánlási konfiguráció létrehozása "IoT_ACRAuthentication" típushoz letiltható állapot beállítással
