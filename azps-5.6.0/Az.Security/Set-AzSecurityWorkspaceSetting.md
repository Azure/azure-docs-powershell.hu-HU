# Set-AzSecurityWorkspaceSetting

## SYNOPSIS
Frissíti az előfizetés munkaterületi beállításait.

## SZINTAXIS

```
Set-AzSecurityWorkspaceSetting -Name <String> -Scope <String> -WorkspaceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Frissíti az előfizetés munkaterületi beállításait.
A konfigurált munkaterület az ebben az előfizetésben lévő virtuális gépekre telepített Azure Log Analytics ügynök által gyűjtött biztonsági adatokat fogja tartalmazni.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Set-AzSecurityWorkspaceSetting -Name "default" -Scope "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869" -WorkspaceId  "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/resourcegroups/mainws/providers/microsoft.operationalinsights/workspaces/securityuserws"

Id                                                                                                         Name    WorkspaceId 
--                                                                                                         ----    ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/workspaceSettings/default default /...
```

A "securityuserws" munkaterületet úgy állítja be, hogy az Azure Log Analytics-ügynök által gyűjtött összes biztonsági adatot megtartsa.
