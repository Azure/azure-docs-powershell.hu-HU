# Add-AzSqlServerKeyVaultKey

## SYNOPSIS
Hozzáad egy kulcstárkulcsot egy SQL-kiszolgálóhoz.

## SZINTAXIS

```
Add-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Add-AzSqlServerKeyVaultKey parancsmag hozzáad egy kulcskulcsot a megadott SQL-kiszolgálóhoz.
A kiszolgálónak "get, wrapKey, unwrapKey" engedéllyel kell rendelkeznie a tárolóhoz.

## PÉLDÁK

### 1. példa: Kulcstár kulcs hozzáadása
```
PS C:\> Add-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Ezzel a paranccsal a "ContosoResourceGroup" erőforráscsoport "ContosoResourceGroup" erőforráscsoportjában a "ContosoServer" nevű SQL-kiszolgálóhoz hozzáadja a kulcstár kulcskulcsát " https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 azonosítóval.
