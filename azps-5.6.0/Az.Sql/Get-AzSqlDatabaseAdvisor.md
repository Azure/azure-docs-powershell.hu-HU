# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Egy vagy több tanácsadót kap egy Azure SQL-adatbázishoz.

## SZINTAXIS

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlDatabaseAdvisor** parancsmag egy vagy több Azure SQL-adatbázis tanácsadót kap egy Azure SQL-adatbázishoz.
