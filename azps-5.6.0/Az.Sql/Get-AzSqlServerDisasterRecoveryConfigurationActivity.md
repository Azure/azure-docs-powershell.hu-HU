# Get-AzSqlServerDisasterRecoveryConfigurationActivity

## SYNOPSIS
Egy adatbázis-kiszolgálói rendszer helyreállítási konfigurációjának tevékenységét kapja meg.

## SZINTAXIS

```
Get-AzSqlServerDisasterRecoveryConfigurationActivity [-ServerName] <String>
 -ServerDisasterRecoveryConfigurationName <String> [-OperationId <Guid>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlServerDisasterRecoveryConfigurationActivity** parancsmag egy SQL-adatbázis-kiszolgálói rendszer helyreállítási konfigurációjában kap tevékenységet.
