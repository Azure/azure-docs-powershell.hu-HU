# Get-AzSqlSyncSchema

## SYNOPSIS
Egy tag- vagy központi adatbázis szinkronizálási sémájáról ad információkat.

## SZINTAXIS

```
Get-AzSqlSyncSchema [-SyncGroupName] <String> [-SyncMemberName <String>] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSqlSyncSchema** parancsmag egy tag- vagy központi adatbázis szinkronizálási sémájáról ad információkat.

## PÉLDÁK

### 1. példa: A szinkronizálási séma lekérte egy központi adatbázishoz
```powershell
PS C:\>Get-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01"
Tables                     LastUpdateTime
------                     --------------
{dbo.Table_1, dbo.Table_2} 6/13/2017 10:03:44 AM
```

Ez a parancs a központi adatbázis szinkronizálási sémáját a szinkronizálási csoport syncGroup01 csoportjában kapja meg.

### 2. példa: A szinkronizálási séma lekérte egy központi adatbázist, és bontsa ki a táblákat
```powershell
PS C:\>Get-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01"  | select -ExpandProperty Tables
Columns    : {column1, column2}
ErrorId    : Schema_TableHasNoPrimaryKey
HasError   : True
Name       : dbo.Table_1
QuotedName : [dbo].[Table_1]

Columns    : {column2, column4}
ErrorId    : Schema_TableHasNoPrimaryKey
HasError   : True
Name       : dbo.Table_2
QuotedName : [dbo].[Table_2]
```

Ez a parancs begyakorlja a központi adatbázis szinkronizálási sémáját a szinkronizálási csoport syncGroup01 csoportjába, és kibővíti a Tables tulajdonságot.

### 3. példa: A tagadatbázis szinkronizálási sémája
```powershell
PS C:\>Get-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01" -SyncMemberName "syncMember01"
The schema payload is the same as Example 1.
```

Ez a parancs a tagadatbázis szinkronizálási sémáját a szinkronizálási tag syncMember01-ben kapja meg.
