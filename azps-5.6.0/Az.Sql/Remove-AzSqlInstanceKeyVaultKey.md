# Remove-AzSqlInstanceKeyVaultKey

## SYNOPSIS
Kulcsbillentyű eltávolítása egy SQL-kezelt példányból

## SZINTAXIS

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (alapértelmezett)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A Remove-AzSqlInstanceKeyVaultKey parancsmag eltávolítja a kulcstár kulcsát a megadott felügyelt példányból. Felhívjuk a figyelmét arra, hogy az SQL által felügyelt példánynak a kulcs tárolóhoz való engedélyei nem változnak. Az engedélyeket a Set-AzKeyVaultAccessPolicy segítségével módosíthatja. Ne feledje, hogy ez a parancsmag nem módosítja a kulcstárat. Ha el szeretne távolítani egy kulcsot a kulcstárból, használja az Remove-AzureKeyVaultKey kulcsot.
