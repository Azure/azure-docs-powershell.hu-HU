# Remove-AzSqlServerActiveDirectoryAdministrator

## SYNOPSIS
Eltávolít egy Azure AD-rendszergazdát az SQL Serverhez.

## SZINTAXIS

```
Remove-AzSqlServerActiveDirectoryAdministrator [-Force] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzSqlServerActiveDirectoryAdministrator** parancsmag eltávolítja az AzureSql Server Azure Active Directory -rendszergazdáját a jelenlegi előfizetésből.

## PÉLDÁK

### 1. példa: Rendszergazda eltávolítása
```
PS C:\>Remove-AzSqlServerActiveDirectoryAdministrator -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
Confirm

[Y] Yes [A] Yes to All [N] No [L] No to All [S] Suspend [?] Help (default is "Y"): Y 

ResourceGroupName ServerName DisplayName ObjectId 
----------------- ---------- ----------- -------- 
ResourceGroup01   Server01   DBAs        40b79501-b343-44ed-9ce7-da4c8cc7353b
```

Ez a parancs eltávolítja az Erőforráscsoport01 erőforráscsoporttal társított Kiszolgáló01 nevű kiszolgáló Azure AD-rendszergazdáját.
