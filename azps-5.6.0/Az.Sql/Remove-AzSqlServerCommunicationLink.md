# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
Törli a két kiszolgáló közötti rugalmas adatbázis-tranzakciók kommunikációs hivatkozását.

## SZINTAXIS

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzSqlServerCommunicationLink** parancsmag törli a kiszolgálók közötti kommunikációs hivatkozást az Azure SQL-adatbázisban rugalmas adatbázis-tranzakciókhoz.

## PÉLDÁK

### 1. példa: Kommunikációs hivatkozás törlése
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Ez a parancs törli a ContosoServer17 kiszolgálón a Link01 nevű kiszolgáló–kiszolgáló kommunikációs hivatkozást.
