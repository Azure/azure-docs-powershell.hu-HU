# Remove-AzSqlServerMSSupportAudit

## SYNOPSIS
Eltávolítja a Microsoft támogatási műveleteinek naplózási beállításait egy Azure SQL-kiszolgálóról.

## SZINTAXIS

### ServerParameterSet (alapértelmezett)
```
Remove-AzSqlServerMSSupportAudit [-ResourceGroupName] <String> [-ServerName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ServerObjectParameterSet
```
Remove-AzSqlServerMSSupportAudit -ServerObject <AzureSqlServerModel> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzSqlServerMSSupportAudit** parancsmag eltávolítja egy Azure SQL-kiszolgáló Microsoft támogatási műveleteinek naplózási beállításait.
A parancsmagot a *ResourceGroupName* és *a ServerName* paraméterrel használhatja a kiszolgáló azonosításához.

## PÉLDÁK

### 1. példa: A Microsoft támogatási műveleteinek naplózási beállításainak eltávolítása egy Azure SQL-kiszolgálóról
```powershell
PS C:\>Remove-AzSqlServerMSSupportAudit -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

### 2. példa: Az Azure SQL-kiszolgáló Microsoft által támogatott naplózási beállításainak eltávolítása folyamaton keresztül
```
PS C:\> Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Remove-AzSqlServerMSSupportAudit
```
