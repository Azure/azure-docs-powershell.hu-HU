# Resume-AzSqlDatabase

## SYNOPSIS
Sql Data Warehouse-adatbázis folytatása.

## SZINTAXIS

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Resume-AzSqlDatabase** parancsmag egy Azure SQL Data Warehouse-adatbázist folytat.

## PÉLDÁK

### 1. példa: Azure SQL Data Warehouse-adatbázis folytatása
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Ez a parancs egy felfüggesztett Azure SQL Data Warehouse-adatbázist folytat.
