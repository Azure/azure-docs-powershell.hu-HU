# Set-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Módosítja az adatbázis-kiszolgáló helyreállítási konfigurációját.

## SZINTAXIS

```
Set-AzSqlServerDisasterRecoveryConfiguration -VirtualEndpointName <String> [-Failover] [-AllowDataLoss]
 [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Set-AzSqlServerDisasterRecoveryConfiguration** parancsmag módosítja az SQL-adatbázis-kiszolgáló helyreállítási konfigurációját.
