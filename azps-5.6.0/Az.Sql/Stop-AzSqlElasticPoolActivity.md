# Stop-AzSqlElasticPoolActivity

## SYNOPSIS
Megszakítja a rugalmas készlet aszinkron frissítési műveletét.

## SZINTAXIS

```
Stop-AzSqlElasticPoolActivity [-PassThru] [-ServerName] <String> [-ElasticPoolName] <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Stop-AzSqlElasticPoolActivity** parancsmag megszakítja a rugalmas készlet aszinkron frissítési műveletét.

## PÉLDÁK
