# Suspend-AzSqlDatabase

## SYNOPSIS
Sql Data Warehouse-adatbázis felfüggesztése.

## SZINTAXIS

```
Suspend-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Suspend-AzSqlDatabase** parancsmag felfüggeszti az Azure SQL Data Warehouse-adatbázist.

## PÉLDÁK

### 1. példa: Azure SQL Data Warehouse-adatbázis felfüggesztése
```
PS C:\>Suspend-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Ez a parancs felfüggeszti egy aktív Azure SQL Data Warehouse-adatbázist.
