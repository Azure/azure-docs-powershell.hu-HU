# Disable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Tiltsa le az Azure Storage Blob szolgáltatás törlési adatmegőrzési házirendét.

## SZINTAXIS

```
Disable-AzStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Disable-AzStorageDeleteRetentionPolicy** parancsmag letiltja az Azure Storage Blob szolgáltatás törlési adatmegőrzési házirendét.

## PÉLDÁK

### 1. példa: A Blob szolgáltatás törlési adatmegőrzési házirendének letiltása
```
C:\PS>Disable-AzStorageDeleteRetentionPolicy
```

Ez a parancs letiltja a Blob szolgáltatás törlési adatmegőrzési házirendét.
