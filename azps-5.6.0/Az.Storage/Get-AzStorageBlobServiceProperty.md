# Get-AzStorageBlobServiceProperty

## SYNOPSIS
Az Azure Storage Blob-szolgáltatások szolgáltatástulajdonságokat kap.

## SZINTAXIS

### AccountName (alapértelmezett)
```
Get-AzStorageBlobServiceProperty [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzStorageBlobServiceProperty -StorageAccount <PSStorageAccount> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### BlobServicePropertiesResourceId
```
Get-AzStorageBlobServiceProperty [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzStorageBlobServiceProperty** parancsmag megkapja az Azure Storage Blob-szolgáltatások szolgáltatástulajdonságokat.

## PÉLDÁK

### 1. példa: Azure Storage Blob services tulajdonságának lekérte egy adott tárfiók
```powershell
PS C:\> Get-AzStorageBlobServiceProperty -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount"

StorageAccountName            : mystorageaccount
ResourceGroupName             : myresourcegroup
DefaultServiceVersion         : 
DeleteRetentionPolicy.Enabled : False
DeleteRetentionPolicy.Days    : 
RestorePolicy.Enabled         : 
RestorePolicy.Days            : 
ChangeFeed                    : True
IsVersioningEnabled           : True
```

Ez a parancs egy adott tárfiók Blob services tulajdonságát kapja meg.
