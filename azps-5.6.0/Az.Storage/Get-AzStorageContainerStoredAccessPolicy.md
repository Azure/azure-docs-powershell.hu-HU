# Get-AzStorageContainerStoredAccessPolicy

## SYNOPSIS
Egy Azure-tároló tárolt hozzáférési házirendet vagy házirendeket kap.

## SZINTAXIS

```
Get-AzStorageContainerStoredAccessPolicy [-Container] <String> [[-Policy] <String>]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzStorageContainerStoredAccessPolicy** parancsmag egy Azure-tároló tárolt hozzáférési házirendje vagy házirendjeit tartalmazza.

## PÉLDÁK

### 1. példa: Tárolt hozzáférési házirend lekérte egy tárolóban
```
PS C:\>Get-AzStorageContainerStoredAccessPolicy -Container "Container07" -Policy "Policy22"
```

Ez a parancs a Policy22 nevű hozzáférési házirendet a Container07 nevű tárolóban kapja meg.

### 2. példa: Az összes tárolt hozzáférési házirend lekérte egy tárolóba
```
PS C:\>Get-AzStorageContainerStoredAccessPolicy -Container "Container07"
```

Ez a parancs az összes hozzáférési házirendet a Container07 nevű tárolóban kapja meg.
