# Remove-AzStorageTableStoredAccessPolicy

## SYNOPSIS
Eltávolít egy tárolt hozzáférési házirendet egy Azure-tárolótáblából.

## SZINTAXIS

```
Remove-AzStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStorageTableStoredAccessPolicy** parancsmag eltávolít egy tárolt hozzáférési házirendet egy Azure-tárolótáblából.

## PÉLDÁK

### 1. példa: Tárolt hozzáférési házirend eltávolítása egy tárolótáblából
```
PS C:\>Remove-AzStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy05"
```

Ez a parancs eltávolítja a Policy05 nevű házirendet a MyTable nevű tárolótáblából.
