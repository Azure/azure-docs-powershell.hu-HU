# Remove-AzStreamAnalyticsInput

## SYNOPSIS
Adatbevitel törlése egy Stream Analytics-feladatból.

## SZINTAXIS

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
A **Remove-AzStreamAnalyticsInput** parancsmag aszinkron módon töröl egy bemenetet az Azure Stream Analytics-feladatából.

## PÉLDÁK

### 1. PÉLDA: Bemeneti adatfolyam eltávolítása egy feladatból
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Ez a parancs eltávolítja a bemeneti EventStreamet a StreamingStreamből.
