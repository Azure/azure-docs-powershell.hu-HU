# Get-AzSynapseSqlAdvancedDataSecurityPolicy

## SYNOPSIS
A munkaterület speciális adatbiztonsági házirendet kap.

## SZINTAXIS

### GetByNameParameterSet (alapértelmezett)
```
Get-AzSynapseSqlAdvancedDataSecurityPolicy [-ResourceGroupName <String>] -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObjectParameterSet
```
Get-AzSynapseSqlAdvancedDataSecurityPolicy -InputObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSqlAdvancedDataSecurityPolicy -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzSynapseSqlAdvancedDataSecurityPolicy** parancsmag beolvassa egy munkaterület speciális adatbiztonsági házirendjét.

## PÉLDÁK

### 1. példa
```powershell
PS C:\> Get-AzSynapseSqlAdvancedDataSecurityPolicy -WorkspaceName ContosoWorkspace
```

Ez a parancs a Speciális adatbiztonság parancsot kapja a ContosoWorkspace nevű munkaterületen.

### 2. példa
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace | Get-AzSynapseSqlAdvancedDataSecurityPolicy
```

Ez a parancs a Speciális adatbiztonság parancsot kapja a ContosoWorkspace nevű munkaterületen a folyamaton keresztül.
