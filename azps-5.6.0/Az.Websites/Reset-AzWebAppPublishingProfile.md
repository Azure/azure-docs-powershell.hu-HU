# Reset-AzWebAppPublishingProfile

## SYNOPSIS

## SZINTAXIS

### S1
```
Reset-AzWebAppPublishingProfile [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Reset-AzWebAppPublishingProfile [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Reset-AzWebAppPublishingProfile** parancsmag alaphelyzetbe állítja a megadott webapp közzétételi profilját.

## PÉLDÁK

### 1. példa

Az alábbi példa alaphelyzetbe állítja a MyResourceGroup erőforráscsoporthoz társított Web App IpRule közzétételi profilját.

```powershell <!-- Aladdin Generated Example --> 
Reset-AzWebAppPublishingProfile -Name IpRule -ResourceGroupName MyResourceGroup
```
