# Update-AzWebAppAccessRestrictionConfig

## SYNOPSIS
Frissíti egy Azure Web App SCM-webhelyére vonatkozó Főwebhely Hozzáférés restiction konfigját.

## SZINTAXIS

### InputValuesParameterSet (alapértelmezett)
```
Update-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String>
 [-ScmSiteUseMainSiteRestrictionConfig] [-SlotName <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzWebAppAccessRestrictionConfig [-PassThru] -InputObject <PSAccessRestrictionConfig>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## LEÍRÁS
Az **Update-AzWebAppAccessRestrictionConfig** parancsmag frissíti az Azure Web App hozzáférési korlátozási konfigurációját.

## PÉLDÁK

### 1. példa: Webalkalmazás SCM-webhelyének frissítése a főwebhely hozzáférési korlátozásait használva

Az alábbi példa frissíti az IpRule nevű webalkalmazást, amely a MyResourceGroup erőforráscsoporthoz tartozik, hogy az scm-webhelyen található fő webhely hozzáférési korlátozási konfigurációját használja.

```powershell <!-- Aladdin Generated Example --> 
Update-AzWebAppAccessRestrictionConfig -Name IpRule -ResourceGroupName MyResourceGroup -ScmSiteUseMainSiteRestrictionConfig
```
