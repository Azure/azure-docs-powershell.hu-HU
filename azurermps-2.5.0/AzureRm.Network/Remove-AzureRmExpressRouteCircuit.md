# Remove-AzureRmExpressRouteCircuit

## Áttekintés
ExpressRoute-áramkör eltávolítása.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmExpressRouteCircuit** parancsmag eltávolítja a ExpressRoute-áramkört.

## Példák

### Példa 1: ExpressRoute-áramkör törlése
```
Remove-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### 2. példa: ExpressRoute-áramkör törlése a csővezeték használatával
```
Get-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzureRmExpressRouteCircuit
```
