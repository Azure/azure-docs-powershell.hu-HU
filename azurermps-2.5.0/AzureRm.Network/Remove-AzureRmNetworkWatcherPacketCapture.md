# Remove-AzureRmNetworkWatcherPacketCapture

## Áttekintés
A csomag-rögzítő erőforrás eltávolítása.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### SetByResource (alapértelmezett)
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcher <PSNetworkWatcher> -PacketCaptureName <String>
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByName
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcherName <String> -ResourceGroupName <String>
 -PacketCaptureName <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A Remove-AzureRmNetworkWatcherPacketCapture eltávolítja a csomag-rögzítő erőforrást. A Remove-AzureRmNetworkWatcherPacketCapture hívása ajánlott Stop-AzureRmNetworkWatcherPacketCapture hívás előtt. Ha a csomag rögzítési munkamenete akkor fut, ha a Remove-AzureRmNetworkWatcherPacketCapture a csomagkapcsolt adatrögzítőt nem lehet menteni. Ha a munkamenet megszűnik az Eltávolítás előtt, a program nem távolítja el a rögzítési adatot tartalmazó. Cap fájlt. 

## Példák

### ---Példa 1: a csomag rögzítési munkamenetének eltávolítása –
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcher $networkWatcher -PacketCaptureName "PacketCaptureTest"
```

Ebben a példában eltávolítunk egy "PacketCaptureTest" nevű csomag-rögzítési munkamenetet.
