# Remove-AzureRmVirtualNetworkGateway

## Áttekintés
Virtuális hálózati átjáró törlése

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A virtuális hálózati átjáró az az objektum, amely az Azure-átjárót jelképezi.

A **Get-AzureRmVirtualNetworkGateway** parancsmag az Azure-átjáró objektumát adja eredményül a név és az erőforrás csoport neve alapján.

## Példák

### 1: virtuális hálózati átjáró törlése
```
Remove-AzureRmVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

Törli a virtuális hálózati átjáró objektumát "myGateway" néven a "myRG" erőforráscsoporthoz.

Megjegyzés: a **Remove-AzureRmVirtualNetworkGatewayConnection** parancsmag használatával először törölnie kell minden kapcsolatot a virtuális hálózati átjáróval.
