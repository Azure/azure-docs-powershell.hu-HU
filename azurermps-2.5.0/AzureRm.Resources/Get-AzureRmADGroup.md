# Get-AzureRmADGroup

## Áttekintés
Szűri az Active Directory-csoportokat.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### EmptyParameterSet (alapértelmezett)
```
Get-AzureRmADGroup [-ObjectId <Guid>] [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### SearchStringParameterSet
```
Get-AzureRmADGroup -DisplayNameStartsWith <String> [-DefaultProfile <IAzureContextContainer>]
 [-IncludeTotalCount] [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzureRmADGroup -DisplayName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Get-AzureRmADGroup -ObjectId <Guid> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

## Leírás
Szűri az Active Directory-csoportokat.

## Példák

### Példa 1 – az összes HIRDETÉSCSOPORT listázása
```
PS C:\> Get-AzureRmADGroup
```

A bérlői fiók összes HIRDETÉSCSOPORT-listája.

### Példa 2 – minden HIRDETÉSCSOPORT listázása a lapozással

```
PS C:\> Get-AzureRmADGroup -First 100
```

Az első 100-HIRDETÉSCSOPORTOK listája a bérlői fiókban.

### Példa 3 – HIRDETÉSCSOPORT-azonosító beolvasása

```
PS C:\> Get-AzureRmADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

A "85F89C90-780E-4AA6-9F4F-6F268D322EEE" azonosítójú HIRDETÉSCSOPORTOT kapja.

### Példa 4 – keresési karakterlánc szerint csoportosított listák

```
PS C:\> Get-AzureRmADGroup -SearchString Joe
```

Felsorolja az összes olyan HIRDETÉSCSOPORTOT, amelynek a megjelenítendő neve "Joe"-val kezdődik.
