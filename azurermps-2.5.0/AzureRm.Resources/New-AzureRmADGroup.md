# New-AzureRmADGroup

## Áttekintés
Új Active Directory-csoport létrehozása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
New-AzureRmADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Új Active Directory-csoport létrehozása

## Példák

### Példa 1 – új HIRDETÉSCSOPORT létrehozása

```
PS C:\> New-AzureRmADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Egy új HIRDETÉSCSOPORTOT hoz létre a "MyGroupDisplayName" névvel és a levelek becenevevel " myemail@domain.com ".
