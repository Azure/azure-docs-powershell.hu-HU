# Remove-AzureRmDeployment

## Áttekintés
A telepített példányok és a hozzájuk kapcsolódó műveletek eltávolítása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### RemoveByDeploymentName (alapértelmezett)
```
Remove-AzureRmDeployment [-Name] <String> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByDeploymentId
```
Remove-AzureRmDeployment -Id <String> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzureRmDeployment -InputObject <PSDeployment> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmDeployment** parancsmag az Azure-üzembe helyezi az előfizetéses hatókört, valamint bármely kapcsolódó műveletet.

## Példák

### 1. példa: a példány eltávolítása adott névvel
```
PS C:\>Remove-AzureRmDeployment -Name "RolesDeployment"
```

Ez a parancs eltávolítja a "RolesDeployment" telepítőt az aktuális előfizetési tartományon.

### 2. példa: központi üzembe állítás és eltávolítás
```
PS C:\>Get-AzureRmDeployment -Name "RolesDeployment" | Remove-AzureRmDeployment
```

A parancs a jelenlegi előfizetési tartomány "RolesDeployment" beállítását kapja, és eltávolítja azt.
