# Set-AzureRmApiManagementOpenIdConnectProvider

## Áttekintés
Egy OpenID csatlakoztatási szolgáltatót módosít.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Set-AzureRmApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 -OpenIdConnectProviderId <String> [-Name <String>] [-MetadataEndpointUri <String>] [-ClientId <String>]
 [-ClientSecret <String>] [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **set-AzureRmApiManagementOpenIdConnectProvider** parancsmag egy OpenID Connect-szolgáltatót módosít az Azure API-menedzsmentben.

## Példák

### Példa 1: a szolgáltató titkos ügyfelének módosítása
```
PS C:\>Set-AzureRmApiManagementOpenIdConnectProvider -Context $ApimContext -OpenIdConnectProviderId "OICProvicer01" -ClientSecret "q2w3e43r45" -PassThru
```

Ez a parancs módosítja az azonosító OICProvicer01 szolgáltatót.
A parancs a szolgáltatóhoz tartozó ügyfél-titkot adja meg.
