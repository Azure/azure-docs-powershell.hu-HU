# Remove-AzureRmAutomationAccount

## Áttekintés
Automatizálási fiók eltávolítása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmAutomationAccount** parancsmag az Azure automatizálási fiókokat távolítja el egy erőforráscsoport-csoportból.

Az automatizálási fiókokról további információt az New-AzureRmAutomationAccount parancsmag című témakörben találhat.

## Példák

### 1. példa: automatizálási fiók eltávolítása
```
PS C:\>Remove-AzureRmAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Ez a parancs eltávolítja a ContosoAutomationAccount nevű automatizálási fiókot a felhasználó érvényesítése kérése nélkül.
