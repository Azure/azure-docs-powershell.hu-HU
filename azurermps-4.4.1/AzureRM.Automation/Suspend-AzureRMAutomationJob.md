# Suspend-AzureRmAutomationJob

## Áttekintés
Egy automatizálási feladat felfüggesztése.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Suspend-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **felfüggesztés – AzureRmAutomationJob** parancsmag felfüggeszti az Azure automatizálási feladatot.
Adja meg a futó automatizálási feladatot.

Felfüggesztett feladat folytatásához használja az Resume-AzureRmAutomationJob parancsmagot.

## Példák

### 1. példa: feladat felfüggesztése
```
PS C:\>Suspend-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Ez a parancs felfüggeszti a megadott azonosítót tartalmazó feladatot.
