# Disable-AzureBatchJobSchedule

## Áttekintés
Letilt egy kötegelt feladatok ütemezését.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Disable-AzureBatchJobSchedule [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **disable-AzureBatchJobSchedule** parancsmag letiltja az Azure-alapú kötegelt feladatok ütemezését.
Ha letiltja az ütemtervet, a feladatok nem az adott ütemterv szerint jönnek létre.
Később is engedélyezheti a letiltott ütemtervet.

## Példák

### 1. példa: munkabeosztás letiltása
```
PS C:\>Disable-AzureBatchJobSchedule -Id "JobSchedule17" -BatchContext $Context
```

Ez a parancs letiltja a JobSchedule17 azonosító munkabeosztást.
A **Get-AzureRmBatchAccountKeys** parancsmaggal rendelhet környezetet a $Context változóhoz.
