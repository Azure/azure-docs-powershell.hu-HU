# Get-AzureRmCdnProfileSsoUrl

## Áttekintés
Beilleszti a CDN-profil egyszeri bejelentkezést tartalmazó URL-címét.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### Paraméterérték a mezők paramétereinek beállításához (alapértelmezett)
```
Get-AzureRmCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Az objektum paramétereinek beállítása paraméter
```
Get-AzureRmCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzureRmCdnProfileSsoUrl** parancsmag az Azure Content Delivery Network (CDN) profiljának egyszeri bejelentkezés URL-címét kapja meg.
Ez az URL lehetővé teszi a felhasználóknak, hogy Conntect egy kiegészítő portálon, és használják a CDN további funkcióit.
