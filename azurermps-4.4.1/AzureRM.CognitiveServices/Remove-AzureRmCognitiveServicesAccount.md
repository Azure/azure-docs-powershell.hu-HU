# Remove-AzureRmCognitiveServicesAccount

## Áttekintés
A kognitív Services-fiók törlése

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmCognitiveServicesAccount** parancsmag törli a megadott kognitív Services-fiókot.
