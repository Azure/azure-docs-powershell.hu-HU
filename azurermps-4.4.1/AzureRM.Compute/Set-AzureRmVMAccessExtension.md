# Set-AzureRmVMAccessExtension

## Áttekintés
A VMAccess bővítmény felvétele egy virtuális gépre.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Set-AzureRmVMAccessExtension [-UserName <String>] [-Password <String>] [-ResourceGroupName] <String>
 [-VMName] <String> [-Name <String>] [-TypeHandlerVersion <String>] [-Location <String>]
 [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **set-AzureRmVMAccessExtension** parancsmag a Virtual Machine Access (VMAccess) Virtual Machine bővítményt hozzáadja egy virtuális géphez.
A VMAccess alaphelyzetbe állíthatja a virtuális gép felhasználónevét és jelszavát.

## Példák

### Példa 1: VMAccess bővítmény felvétele
```
PS C:\> Set-AzureRmVMAccessExtension -ResourceGroupName "ResrouceGroup11" -Location "Central US" -VMName "VirtualMachine07" -Name "ContosoTest" -TypeHandlerVersion "2.0" -UserName "PFuller" -Password "Password"
```

Ez a parancs összead egy VMAccess-kiterjesztést a VirtualMachine07 nevű virtuális gépnek a ResrouceGroup11-ban.
A parancs a VMAccess név és típus kezelőjének verziószámát adja meg.
