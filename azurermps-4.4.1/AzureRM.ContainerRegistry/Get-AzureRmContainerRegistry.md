# Get-AzureRmContainerRegistry

## Áttekintés
Tároló-nyilvántartót kap.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### ResourceGroupParameterSet
```
Get-AzureRmContainerRegistry [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### RegistryNameParameterSet
```
Get-AzureRmContainerRegistry [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmContainerRegistry** parancsmag egy adott tároló-beállításjegyzéket vagy egy erőforráscsoport összes tároló-nyilvántartását, illetve az előfizetést kapja meg.
