# Remove-AzureRmDataLakeStoreAccount

## Áttekintés
Véglegesen törli az adattó-tároló fiókját.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmDataLakeStoreAccount** parancsmag véglegesen törli az adattó-tároló fiókját.

## Példák

### 1. példa: az adattó áruházbeli fiók eltávolítása
```
PS C:\>Remove-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Ez a parancs eltávolítja a ContosoADL nevű fiókot az adattó áruházból.
