# Remove-AzureRmLocalNetworkGateway

## Áttekintés
Helyi hálózati átjáró törlése

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A helyi hálózat átjárója a helyszíni VPN-eszközt jelképező objektum.

A **Remove-AzureRmLocalNetworkGateway** parancsmag a helyszíni átjáró név és erőforrás csoport neve alapján történő törlésére szolgáló objektumot törli.

## Példák

### 1: helyi hálózati átjáró törlése
```
Remove-AzureRmLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Törli a helyi hálózati átjáró objektumát "myLocalGW" néven a "myRG" erőforráscsoporthoz.

Megjegyzés: a **Remove-AzureRmVirtualNetworkGatewayConnection** parancsmag segítségével először törölnie kell minden kapcsolatot a helyi hálózati átjáróval.
