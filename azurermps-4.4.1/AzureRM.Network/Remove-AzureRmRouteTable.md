# Remove-AzureRmRouteTable

## Áttekintés
Egy útválasztási táblázat eltávolítása.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmRouteTable -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmRouteTable** parancsmag eltávolítja az Azure Route-táblázatot.

## Példák

### 1. példa: a Route Table eltávolítása
```
PS C:\>Remove-AzureRmRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Ez a parancs eltávolítja a RouteTable01 nevű útvonalkártya-táblázatot a ResourceGroup11 nevű erőforráscsoport-csoportból.
A parancsmag a táblázat eltávolítása előtt kéri a megerősítés megerősítését.
