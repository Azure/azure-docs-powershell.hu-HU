# New-AzureRmRecoveryServicesVault

## Áttekintés
Új helyreállítási szolgáltatások boltozatának létrehozása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
New-AzureRmRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzureRmRecoveryServicesVault** parancsmag új helyreállítási szolgáltatások boltozatát hozza létre.
