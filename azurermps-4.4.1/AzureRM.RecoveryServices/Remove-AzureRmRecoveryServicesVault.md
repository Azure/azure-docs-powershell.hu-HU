# Remove-AzureRmRecoveryServicesVault

## Áttekintés
Törli a helyreállítási szolgáltatások boltozatát.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmRecoveryServicesVault -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmRecoveryServicesVault** parancsmag törli a helyreállítási szolgáltatások boltozatát.
