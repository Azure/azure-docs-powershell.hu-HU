# Remove-AzureRmRedisCache

## Áttekintés
Eltávolítja a Redis-gyorsítótárat.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmRedisCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmRedisCache** parancsmag eltávolítja az Azure Redis-gyorsítótárát.

## Példák

### 1. példa: Redis-gyorsítótár eltávolítása és az eredmény visszaadása
```
PS C:\>Remove-AzureRmRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Ez a parancs eltávolítja a Redis-gyorsítótárat, és megjeleníti, hogy a művelet sikeres-e.

### 2. példa: Redis-gyorsítótár eltávolítása, és az eredmény nem jeleníthető meg
```
PS C:\>Remove-AzureRmRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Ez a parancs eltávolítja a Redis-gyorsítótárat.
Mivel a *PassThru* paraméter nincs megadva, a művelet eredménye nem jelenik meg.
