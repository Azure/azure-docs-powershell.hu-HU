# Remove-AzureRmServiceFabricNode

## Áttekintés
Csomópontok eltávolítása a megadott csomópont-típusból egy fürtből

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmServiceFabricNode [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 -NumberOfNodesToRemove <Int32> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmServiceFabricNode** használatával eltávolíthatja a csomópontokat egy adott csomópontból a fürtből. Az Eltávolítás csak akkor járjon le, ha az megfelel a fürtök állapotát jelző metrikának.

## Példák

### Példa 1
```
PS c:> Remove-AzureRmServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1' -NumberOfNodesToRemove 2
```

Ez a parancs eltávolítja a 2 csomópontot a "NT1" NodeType.
