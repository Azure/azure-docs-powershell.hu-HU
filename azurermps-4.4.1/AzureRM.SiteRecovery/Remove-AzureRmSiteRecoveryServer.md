# Remove-AzureRmSiteRecoveryServer

## Áttekintés
Eltávolítja a webhely-helyreállítási kiszolgálót az aktuális boltozatról.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmSiteRecoveryServer -Server <ASRServer> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmSiteRecoveryServer** parancsmag megszünteti az Azure-webhely helyreállítási kiszolgálóját, amely eltávolítja azt az aktuális boltozatról.
