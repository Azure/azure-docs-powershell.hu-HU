# Get-AzureRmSqlElasticPoolAdvisor

## Áttekintés
Egy vagy több tanácsadót kap egy Azure SQL elasztikus poolban.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlElasticPoolAdvisor** parancsmag egy vagy több Azure SQL elasztikus Pool-tanácsadót kap az Azure SQL rugalmas készletéhez.
