# Get-AzureRmSqlElasticPoolRecommendedAction

## Áttekintés
Egy vagy több javasolt műveletet kap egy Azure SQL elasztikus Pool tanácsadóban.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlElasticPoolRecommendedAction** parancsmag egy vagy több ajánlott műveletet kap egy Azure SQL elasztikus Pool tanácsadóban.
