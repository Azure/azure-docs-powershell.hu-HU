# Get-AzureRmSqlServerAdvisor

## Áttekintés
Egy vagy több tanácsadót kap egy Azure SQL Server-kiszolgálón.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlServerAdvisor** parancsmag egy vagy több Azure SQL Server-tanácsadót kap az Azure SQL Server rendszerhez.
