# Get-AzureRmSqlServerTransparentDataEncryptionProtector

## Áttekintés
Az áttetsző adattitkosítás (TDE) oltalmazójának beolvasása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az Get-AzureRmSqlServerTransparentDataEncryptionProtector parancsmag információkat kap a TDE-oltalmazóról.

## Példák

### --------------------------Példa 1: az átlátszó adattitkosító (TDE) fólia beolvasása--------------------------
```
PS C:\> Get-AzureRmSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Ez a parancs a TDE Protector nevű kiszolgálót kapja meg a ContosoServer nevű erőforráscsoport nevű ContosoResourceGroup.

ResourceGroupName kiszolgálónév – típus ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
