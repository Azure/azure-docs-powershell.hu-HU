# Start-AzureRmSqlSyncGroupSync

## Áttekintés
Szinkronizálási csoport szinkronizálását indítja el.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Start-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **Start-AzureRmSqlSyncGroupSync** parancsmag szinkronizálási csoport szinkronizálását indítja el.

## Példák

### Példa 1
```
PS C:\> Start-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Ez a parancs elindítja a szinkronizálást a szinkronizálás csoport mysg.
