# Stop-AzureRmSqlServerUpgrade

## Áttekintés
SQL-adatbázis kiszolgálójának frissítését leállítja.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Stop-AzureRmSqlServerUpgrade [-Force] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzureRmSqlServerUpgrade** parancsmag leállítja egy Azure SQL-adatbázis kiszolgálójának frissítését.

## Példák

### Példa 1: a kiszolgáló frissítésének leállítása
```
PS C:\>Stop-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server02"
```

Ez a parancs leállítja a ResourceGroup01 nevű Server02 hozzárendelt nevű kiszolgáló frissítésére vonatkozó kérést.
