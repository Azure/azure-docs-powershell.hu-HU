# Remove-AzureRmAutomationSchedule

## Áttekintés
Automatizálási ütemterv törlése

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmAutomationSchedule** parancsmag az Azure Automation szolgáltatásból törli az ütemtervet.

## Példák

### 1. példa: ütemterv eltávolítása
```
PS C:\>Remove-AzureRmAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs módosítja a Schedule01 nevű ütemterv leírását.
