# Disable-AzureRmBackupProtection

## Áttekintés
Letiltja a biztonsági másolat védett elemeinek védelmét.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Disable-AzureRmBackupProtection [-RemoveRecoveryPoints] [-Force] [-Item] <AzureRMBackupItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **disable-AzureRmBackupProtection** parancsmag letiltja az Azure Backup-védett elemek védelmét.
Ez a parancsmag leállítja az elemek rendszeres ütemezett biztonsági mentését.
Ez a parancsmag a biztonsági másolat meglévő helyreállítási pontját tudja törölni.
