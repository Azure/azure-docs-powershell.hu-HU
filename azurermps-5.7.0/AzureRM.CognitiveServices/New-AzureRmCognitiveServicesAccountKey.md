# New-AzureRmCognitiveServicesAccountKey

## Áttekintés
Visszahozza a fiók kulcsát.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
New-AzureRmCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <KeyName>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzureRmCognitiveServicesAccountKey** parancsmag regenerálta egy API-kulcsot a kognitív szolgáltatások fiókjához.
