# Restart-AzureRmVM

## Áttekintés
Azure virtuális gép újraindítása.

## Leírás
Az **Újraindítás – AzureRmVM** parancsmag újraindítja az Azure Virtual machinet.

## Példák

### 1. példa: virtuális gép újraindítása
```
PS C:\> Restart-AzureRmVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Ez a parancs elindítja a VirtualMachine07 nevű virtuális gépet a ResourceGroup11.
