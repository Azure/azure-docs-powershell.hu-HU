# Set-AzureRmSnapshotUpdateImageReference

## Áttekintés
A kép hivatkozásának tulajdonságait állítja be egy pillanatkép-frissítő objektumon.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Set-AzureRmSnapshotUpdateImageReference [-SnapshotUpdate] <SnapshotUpdate> [[-Id] <String>] [[-Lun] <Int32>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **set-AzureRmSnapshotUpdateImageReference** parancsmag egy pillanatkép-frissítő objektum képhivatkozási tulajdonságait állítja be.

## Példák

### Példa 1
```
PS C:\> $snapshotupdateconfig = New-AzureRmSnapshotUpdateConfig -SnapshotSizeGB 10 -AccountType PremiumLRS -OsType Windows -CreateOption FromImage;
PS C:\> $image = '/subscriptions/0000000-0000-0000-0000-000000000000/resourceGroups/ResourceGroup01/providers/Microsoft.Compute/images/TestImage123';        
PS C:\> $snapshotupdateconfig = Set-AzureRmSnapshotUpdateImageReference -Snapshot $snapshotupdateconfig -Id $image -Lun 0;
PS C:\> Update-AzureRmSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -SnapshotUpdate $snapshotupdateconfig;
```

Az első parancs létrehoz egy helyi Snapshot Update-objektumot, a 10GB méretet Premium_LRS tárterület-fiók típusában.  A Windows OS típust is beállítja.
A második parancs beállítja a képazonosítót és a 0 logikai egység számát a Snapshot Update objektumhoz.
Az utolsó parancs átveszi a Snapshot Update objektumát, és frissíti a "Snapshot01" nevű meglévő pillanatképet az "ResourceGroup01" erőforráscsoport csoportjában.
