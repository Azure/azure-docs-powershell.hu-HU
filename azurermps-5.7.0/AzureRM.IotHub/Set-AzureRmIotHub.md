# Set-AzureRmIotHub

## Áttekintés
Frissíti egy IotHub tulajdonságait.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### UpdateSku (alapértelmezett)
```
Set-AzureRmIotHub -ResourceGroupName <String> -Name <String> -SkuName <PSIotHubSku> [-Units <Int64>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UpdateEventHubEndpointProperties
```

 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]

 [-FileUploadNotificationMaxDeliveryCount <Int32>] -EnableFileUploadNotifications <Boolean>

```

## Leírás
Frissíti egy IotHub tulajdonságait.

## Példák

### Példa 1 a SKU frissítése
```
PS C:\> Set-AzureRmIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -SkuName S1 -Units 5
```

Frissítse a SKU-ot S1-re, és a "myiothub" nevű IotHub az 5 egységre

### 2. példa a eventhub tulajdonságainak frissítése
```
PS C:\> Set-AzureRmIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubRetentionTimeInDays 4
```

A "myiothub" nevű IotHub telemetriai-és operationsmonitoringevents-eseményeinek frissítése a napok és a 4 közötti adatmegőrzési idő között
