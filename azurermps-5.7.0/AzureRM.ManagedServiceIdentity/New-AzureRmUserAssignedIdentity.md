# New-AzureRmUserAssignedIdentity

## Áttekintés
Egy új felhasználó által kiosztott identitást hoz létre, vagy egy meglévő, felhasználó által kiosztott identitást frissít.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
New-AzureRmUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzureRmUserAssignedIdentity** parancsmag új felhasználó által kiosztott identitást hoz létre. Ha egy már meglévő identitással használja, frissítette az identitást.
Ha Azure Resource Manager címkét szeretne hozzáadni az identitáshoz, használja az Set-AzureRmResource parancsmagot.
