# New-AzureRmRedisCachePatchSchedule

## Áttekintés
Egy javítócsomag-ütemezést ad hozzá.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
New-AzureRmRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **New-AzureRmRedisCachePatchSchedule** parancsmag az Azure Redis-gyorsítótárban lévő gyorsítótárba helyezi a javítás ütemezését.

## Példák

### 1. példa: a javítás ütemezésének létrehozása és hozzáadása a gyorsítótárban
```
PS C:\>New-AzureRmRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzureRmRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

A parancs hozzáadja a javítás ütemezését a RedisCache06 nevű gyorsítótárhoz.
A Entries paraméter az a érték, amely a **New-AzureRmRedisCacheScheduleEntry-** t használja az ütemterv létrehozásához.
