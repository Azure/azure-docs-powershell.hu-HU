# Remove-AzureRmRelayHybridConnection

## Áttekintés
Eltávolítja a HybridConnection a megadott HybridConnection-névtérből.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmRelayHybridConnection** parancsmag eltávolítja a HybridConnection a megadott továbbító névtérből.

## Példák

### Példa 1
```
PS C:\> Remove-AzureRmRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Eltávolítja a HybridConnection `TestHybridConnection` a névtérből `TestNameSpace-Relay1` .
