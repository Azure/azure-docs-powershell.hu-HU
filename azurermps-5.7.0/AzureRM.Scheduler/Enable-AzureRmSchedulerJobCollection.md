# Enable-AzureRmSchedulerJobCollection

## Áttekintés
Begyűjtheti a beosztást.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Enable-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az **enable-AzureRmSchedulerJobCollection** parancsmag lehetővé teszi a begyűjtést az Azure schedulerben.
