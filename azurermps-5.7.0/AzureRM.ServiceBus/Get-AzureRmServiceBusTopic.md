# Get-AzureRmServiceBusTopic

## Áttekintés
A megadott szolgáltatási busz témakör leírását adja meg.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmServiceBusTopic** parancsmag a megadott szolgáltatási busz névterének leírását adja eredményül.
