# Start-AzureRmSiteRecoveryTestFailoverJob

## Áttekintés
Teszteli a feladatátvételt a webhely-helyreállítási védelem entitásban.

## Leírás
A **Start-AzureRmSiteRecoveryTestFailoverJob** parancsmag elindítja az Azure webhely-helyreállítási védelmi szervezet vagy helyreállítási terv feladatátvételét.
Ellenőrizheti, hogy a feladat sikeres volt-e az Get-AzureRmSiteRecoveryJob parancsmag használatával.
