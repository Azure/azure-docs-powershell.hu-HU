# Get-AzureRmSqlDatabaseAdvisor

## Áttekintés
Egy vagy több tanácsadót kap egy Azure SQL-adatbázishoz.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlDatabaseAdvisor** parancsmag egy vagy több Azure SQL-adatbázis-tanácsadót kap egy Azure SQL-adatbázishoz.
