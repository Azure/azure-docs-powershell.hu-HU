# Remove-AzureRmSqlSyncAgent

## Áttekintés
Azure SQL-szinkronizálási ügynök eltávolítása.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmSqlSyncAgent** parancsmag eltávolítja az Azure SQL szinkronizáló ügynököt.

## Példák

### 1. példa: a Szinkronizáló ügynök eltávolítása
```
PS C:\>Remove-AzureRmSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Ez a parancs eltávolítja a syncAgent01 nevű Azure SQL-szinkronizáló ügynököt.
