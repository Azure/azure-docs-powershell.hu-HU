# Azure. AnalysisServices modul
## Leírás
Ez a modul parancsokat tartalmaz az Azure Analysis Services-környezet műveleteinek végrehajtásához

## Azure. AnalysisServices parancsmagok
### [Add-AzureAnalysisServicesAccount](Add-AzureAnalysisServicesAccount.md)
Az Azure Analysis Services kiszolgálói parancsmag-kérelmekhez használt hitelesített fiók hozzáadása.

### [Exportálás – AzureAnalysisServicesInstanceLog](Export-AzureAnalysisServicesInstanceLog.md)
Napló exportálása az Analysis Services Server egy példányáról az aktuálisan bejelentkezett környezetben, az Add-AzureAnalysisServicesAccount parancsban megadott módon

### [Újraindítás – AzureAnalysisServicesInstance](Restart-AzureAnalysisServicesInstance.md)
Az Analysis Services Server példányának újraindítása az aktuálisan bejelentkezett környezetben, az Add-AzureAnalysisServicesAccount parancsban megadott módon

### [Szinkronizálás – AzureAnalysisServicesInstance](Sync-AzureAnalysisServicesInstance.md)
A megadott adatbázis szinkronizálása az Analysis Services-kiszolgáló meghatározott példányán az Add-AzureAnalysisServicesAccount parancsban megadott, a jelenleg bejelentkezett környezetben található összes lekérdezés scaleout-példányához
