# Get-AzureRmApiManagementApiRelease

## Áttekintés
Az API-kiadás beszerzése.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmApiManagementApiRelease -Context <PsApiManagementContext> -ApiId <String> [-ReleaseId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmApiManagementApiRelease** parancsmag egy vagy több kiadást kap az Azure API Management API-hoz.

ResourceGroupName : Api-Default-WestUS
