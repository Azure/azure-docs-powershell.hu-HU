# Get-AzureRmApiManagementApiRevision

## Áttekintés
Információt kap az API API-verziójáról.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmApiManagementApiRevision** PARANCSMAG az API összes korrektúrájának részleteit kapja meg.
