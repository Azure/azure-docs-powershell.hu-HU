# Get-AzureRmAutomationSourceControl

## Áttekintés
Beolvassa az Azure automatizálási vezérlők listáját.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### ByAll (alapértelmezett)
```
Get-AzureRmAutomationSourceControl [-SourceType <String>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzureRmAutomationSourceControl -Name <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Az Get-AzureRmAutomationSourceControl parancsmag automatizálási adatforrás-vezérlőket kap.
Ha egy konkrét forrás vezérlőt szeretne beszerezni, adja meg a nevét.

## Példák

### Példa 1
Ez a parancs egy VSTSNative nevű automatizálási vezérlőt kap a devAccount nevű fiókban.


```powershell
PS C:\> Get-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                           -AutomationAccountName "devAccount" `
                                           -Name "VSTSNative"
```
