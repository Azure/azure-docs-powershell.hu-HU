# Remove-AzureRmAutomationSourceControl

## Áttekintés
Az Azure automatizálási verziókövetés eltávolítása.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A Remove-AzureRmAutomationSourceControl parancsmag eltávolítja a verziókövetés az Azure automatizálási szolgáltatásból.

## Példák

### Példa 1
Ez a parancs eltávolítja a VSTSNative nevű automatizálási forrás vezérlőt a devAccount nevű fiókban.
Ez a parancs az *erő* paramétert adja meg. Ezért nem kéri a megerősítést.

```powershell
PS C:\> Remove-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
