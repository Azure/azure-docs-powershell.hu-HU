# Get-AzureRmBatchAccountKeys

## Áttekintés
Egy köteg fiók kulcsait kapja meg.

## SZINTAXISA

```
Get-AzureRmBatchAccountKeys [-AccountName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```
