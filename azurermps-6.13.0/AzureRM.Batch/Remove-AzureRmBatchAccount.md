# Remove-AzureRmBatchAccount

## Áttekintés
Egy köteg fiók eltávolítása.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmBatchAccount** parancsmag egy Azure-köteg-fiókot távolít el.
Ez a parancsmag arra kéri, hogy mielőtt eltávolítja a fiókot, kivéve ha az *erő* paramétert adja meg.

## Példák

### 1. példa: egy köteg fiók eltávolítása
```
PS C:\>Remove-AzureRmBatchAccount -AccountName "pfuller"
```

Ez a parancs eltávolítja a pfuller nevű köteg-fiókot.
Ez a parancs a fiók törlése előtt kéri a megerősítést.
