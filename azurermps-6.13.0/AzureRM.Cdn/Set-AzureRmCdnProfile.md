# Set-AzureRmCdnProfile

## Áttekintés
Egy CDN-profil frissítése

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Set-AzureRmCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **set-AzureRmCdnProfile** parancsmag egy Azure Content Delivery Network (CDN) profilt frissít.
