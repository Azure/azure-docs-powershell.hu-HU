# Set-AzureRmVMBootDiagnostics

## Áttekintés
A virtuális gép rendszerindítási diagnosztikai tulajdonságait módosítja.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### EnableBootDiagnostics
```
Set-AzureRmVMBootDiagnostics [-VM] <PSVirtualMachine> [-Enable] [-ResourceGroupName] <String>
 [[-StorageAccountName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### DisableBootDiagnostics
```
Set-AzureRmVMBootDiagnostics [-VM] <PSVirtualMachine> [-Disable] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **set-AzureRmVMBootDiagnostics** parancsmag a virtuális gép rendszerindítási diagnosztikai tulajdonságait módosítja.

## Példák

### 1. példa: a boot Diagnostics engedélyezése
```
PS C:\> $VM = Get-AzureRmVM -ResourceGroupName "ResourceGroup11" -Name "ContosoVM07"
PS C:\> Set-AzureRmVMBootDiagnostics -VM $VM -Enable -ResourceGroupName "ResourceGroup11" -StorageAccountName "DiagnosticStorage"
```

Az első parancs a **Get-AzureRmVM** segítségével a ContosoVM07 nevű virtuális gépet kapja.
A parancs a $VM változóban tárolja.
A második parancs engedélyezi a rendszerindítási diagnosztikát a virtuális gép számára a $VMban.
A diagnosztikai adatait a rendszer a megadott fiókban tárolja.
