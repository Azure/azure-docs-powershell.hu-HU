# Get-AzureRmConsumptionBudget

## Áttekintés
A költségvetések listájának beszerzése előfizetésben vagy erőforráscsoporthoz

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## Leírás
A Get-AzureRmConsumptionBudget parancsmag egy előfizetésben vagy egy erőforráscsoport listájában kapja meg a költségvetéseket.
