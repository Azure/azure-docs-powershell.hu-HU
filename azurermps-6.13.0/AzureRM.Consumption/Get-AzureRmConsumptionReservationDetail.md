# Get-AzureRmConsumptionReservationDetail

## Áttekintés
Foglalási adatok beszerzése a megadott Dátumtartomány alapján.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmConsumptionReservationDetail -StartDate <DateTime> -EndDate <DateTime> -ReservationOrderId <String>
 [-ReservationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmConsumptionReservationDetail** parancsmag foglalási adatokat kap a megadott dátumtartomány számára.
