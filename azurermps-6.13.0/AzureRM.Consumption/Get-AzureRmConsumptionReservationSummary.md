# Get-AzureRmConsumptionReservationSummary

## Áttekintés
Foglalási összesítések beszerzése napi vagy havi gabona esetén

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmConsumptionReservationSummary -Grain <String> -ReservationOrderId <String> [-ReservationId <String>]
 [-StartDate <DateTime>] [-EndDate <DateTime>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmConsumptionReservationSummay** parancsmag napi vagy havi gabona foglalási összefoglalókat kap.
