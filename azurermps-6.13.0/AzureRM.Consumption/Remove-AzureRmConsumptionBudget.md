# Remove-AzureRmConsumptionBudget

## Áttekintés
Költségvetés eltávolítása előfizetésből vagy erőforráscsoportből

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### Előfizetés (alapértelmezett)
```
Remove-AzureRmConsumptionBudget [-DefaultProfile <IAzureContextContainer>] -Name <String>
 [-ResourceGroupName <String>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Csővezeték
```
Remove-AzureRmConsumptionBudget [-DefaultProfile <IAzureContextContainer>] -InputObject <PSBudget> [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az Remove-AzureRmConsumptionBudget parancsmag egy előfizetésben vagy erőforráscsoportben távolítja el a költségvetést.

## Példák

### 1. példa: költségvetés eltávolítása költségvetés-névvel az előfizetési szinten
```powershell
PS C:\> Remove-AzureRmConsumptionBudget -Name PSBudget -PassThru
True
```

### 2. példa: költségvetés eltávolítása költségvetés-névvel az erőforrás csoport szintjén
```powershell
PS C:\> Remove-AzureRmConsumptionBudget -ResourceGroupName RGBudgets -Name PSBudgetRG -PassThru
True
```

### 3. példa: költségvetés eltávolítása a csővezetékek között az előfizetési szinten
```powershell
PS C:\> Get-AzureRmConsumptionBudget -Name PSBudget | Remove-AzureRmConsumptionBudget -PassThru
True
```
