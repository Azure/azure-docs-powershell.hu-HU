# Remove-AzureRmDataLakeStoreVirtualNetworkRule

## Áttekintés
A megadott virtuális hálózati szabály eltávolítása a megadott Data Lake Store-ból.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmDataLakeStoreVirtualNetworkRule** parancsmag eltávolítja a megadott virtuális hálózati szabályt a megadott Data Lake Store áruházban.

## Példák

### Példa 1
```powershell
PS C:\> Remove-AzureRmDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

A "myVNET" virtuális hálózati szabály eltávolítása a "frissíteni" fiókból
