# Set-AzureRmIoTDeviceProvisioningServiceCertificate

## Áttekintés
Ellenőrizzen egy Azure IoT hub-eszköz kiépítési szolgáltatási tanúsítványát.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### ResourceSet (alapértelmezett)
```
Set-AzureRmIoTDeviceProvisioningServiceCertificate [-ResourceGroupName] <String> [-Name] <String>
 [-CertificateName] <String> [-Etag] <String> [-Path] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Set-AzureRmIoTDeviceProvisioningServiceCertificate [-InputObject] <PSCertificateResponse> [-Path] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Set-AzureRmIoTDeviceProvisioningServiceCertificate [-ResourceId] <String> [-Etag] <String> [-Path] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Tanúsítvány ellenőrzéséhez a Generate-ellenőrzés-kód hívásával kapott ellenőrzési kódot tartalmazó ellenőrző tanúsítvány feltöltésével. Ez az utolsó lépés a birtoklási folyamat igazolásakor.
Az Azure IoT hub eszköz kiépítési szolgáltatásának HITELESÍTÉSSZOLGÁLTATÓI tanúsítványainak részletes ismertetése a következő témakörben található: https://docs.microsoft.com/en-us/azure/iot-dps/how-to-verify-certificates

