# Get-AzureRmApplicationGatewayCustomError

## Áttekintés
Egyéni hibák jelentkeznek egy alkalmazás-átjáróban.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmApplicationGatewayCustomError [-StatusCode <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmApplicationGatewayCustomError** parancsmag egyéni hibaüzeneteket kap egy alkalmazás-átjáróból.

## Példák

### Példa 1: egyéni hiba beolvasása egy alkalmazás-átjáróban
```powershell
PS C:\> $ce = Get-AzureRmApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

Ez a parancs a 502-ös http-állapotkód egyéni hibáját adja eredményül az Application Gateway $appgw.

### 2. példa: beolvassa az összes egyéni hiba listáját egy alkalmazás-átjáróban.
```powershell
PS C:\> $ces = Get-AzureRmApplicationGatewayCustomError -ApplicationGateway $appgw
```

Ez a parancs beolvassa és visszaadja az összes egyéni hiba listáját az Application Gateway $appgwból.
