# Get-AzureRmDdosProtectionPlan

## Áttekintés
Kap egy DDoS védelmi csomagot.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### GetByNameAndGroup
```
Get-AzureRmDdosProtectionPlan -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Lista
```
Get-AzureRmDdosProtectionPlan [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A Get-AzureRmDdosProtectionPlan parancsmag a DDoS védelmi csomagot kapja.

Ebben az esetben meg kell adnia a **ResourceGroupName** és a **Name** attribútumot is, amely megfelel az erőforráscsoport és a DDoS védelmi terv nevének.

Ebben az esetben csak az **ResourceGroupName** paramétert adjuk meg a DDoS védelmi csomagok listájának beszerzéséhez.
