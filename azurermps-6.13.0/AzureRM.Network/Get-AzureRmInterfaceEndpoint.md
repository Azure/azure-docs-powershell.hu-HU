# Get-AzureRmInterfaceEndpoint

## Áttekintés
A Get-AzureRmInterfaceEndpoint parancsmag a felület végpontját kapja.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### GetByNameParameterSet (alapértelmezett)
```
Get-AzureRmInterfaceEndpoint [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzureRmInterfaceEndpoint -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmInterfaceEndpoint** parancsmag a felület végpontját kapja.

## Példák

### Példa 1
```
$interfaceendpoint = Get-AzureRmInterfaceEndpoint -Name "InterfaceEndpoint1" -ResourceGroupName "ResourceGroup01"
```

Ez a parancs a InterfaceEndpoint1 nevű Interface végpontot kapja, amely a ResourceGroup01 nevű erőforráscsoport csoportjába tartozik, és a $interfaceendpoint változóban tárolja.

### 2. példa
```
$interfaceendpoint = Get-AzureRmInterfaceEndpoint -ResourceId "/subscriptions/sub1/resourceGroups/chsriniIEtest1/providers/Microsoft.Network/interfaceEndpoints/ie1.10"

```

Ez a parancs beilleszti az interface végpontot a resourceId/subscriptions/sub1/resourceGroups/chsriniIEtest1/providers/Microsoft.Network/interfaceEndpoints/ie1.10, és a $interfaceendpoint változóban tárolja.
