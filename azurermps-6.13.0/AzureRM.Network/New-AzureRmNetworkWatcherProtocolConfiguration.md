# New-AzureRmNetworkWatcherProtocolConfiguration

## Áttekintés
Új protokoll-konfigurációs objektum létrehozása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
New-AzureRmNetworkWatcherProtocolConfiguration -Protocol <String> [-Method <String>] [-Header <IDictionary>]
 [-ValidStatusCode <Int32[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
Az New-AzureRmNetworkWatcherProtocolConfiguration parancsmag új protokoll-konfigurációs objektumot hoz létre. Ezzel az objektummal korlátozhatja a protokoll Confiuration a connecitivity-ellenőrzés során a megadott feltételek teljesülése esetén.
