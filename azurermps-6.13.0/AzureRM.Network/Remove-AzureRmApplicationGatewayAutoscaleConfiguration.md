# Remove-AzureRmApplicationGatewayAutoscaleConfiguration

## Áttekintés
Az automatikus méretezési konfiguráció eltávolítása egy alkalmazás-átjáróról.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Remove-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmApplicationGatewayAutoscaleConfiguration** parancsmag egy meglévő alkalmazás-átjáróból távolítja el az automatikus méretezés konfigurációját.

## Példák

### Példa 1
```powershell
PS C:\> $gw = Get-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzureRmApplicationGateway -ApplicationGateway $gw
```

Az első parancs beolvassa az Application Gatewayt, és az $gw változóban tárolja.
A második parancs eltávolítja az automatikus méretezés konfigurációját az applicationg-átjáróról.
A harmadik parancs az Azure rendszerhez frissíti az Application Gatewayt.
