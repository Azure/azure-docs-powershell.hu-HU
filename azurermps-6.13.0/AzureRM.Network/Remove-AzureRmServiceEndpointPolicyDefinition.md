## SZINTAXISA

```
Remove-AzureRmServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmServiceEndpointPolicy** parancsmag eltávolítja a szolgáltatási végpontok házirendjét.

## Példák

### 1. példa: a szolgáltatás végpont-házirendjének eltávolítása név használatával
```
Remove-AzureRmServiceEndpointPolicyDefinition -Name "PolicyDef1"
```

Ez a parancs eltávolítja a szolgáltatás végpontjának házirendjét a név PolicyDef1
