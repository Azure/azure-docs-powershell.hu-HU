# Set-AzureRmVirtualNetworkTap

## Áttekintés
A virtuális hálózat cél állapotának beállítása koppintással.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Set-AzureRmVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **set-AzureRmVirtualNetworkTap** egy Azure Virtual Network cél állapotát állítja be.

## Példák

### Példa 1: virtuális hálózat beállítása koppintás
```
PS C:\>$vTap = Get-AzureRmVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzureRmVirtualNetworkTap -VirtualNetworkTap $vTap
```

A parancs frissíti a célhely IpConfiguration, és frissíti a virtuális hálózatot.
Ha vannak olyan koppintásos konfigurációk, amelyek hivatkoznak rá, akkor az összes forrás-forgalom nem fog megjelenni az új IP-címek konfigurációjának frissítése után.
