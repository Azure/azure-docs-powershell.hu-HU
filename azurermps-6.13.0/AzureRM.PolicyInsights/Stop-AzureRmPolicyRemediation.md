# Stop-AzureRmPolicyRemediation

## Áttekintés
Visszavonja a folyamatban lévő házirend-szervizelést.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### ByName (alapértelmezett)
```
Stop-AzureRmPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Stop-AzureRmPolicyRemediation -ResourceId <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Stop-AzureRmPolicyRemediation -InputObject <PSRemediation> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **stop-AzureRmPolicyRemediation** parancsmag megszünteti a folyamatban lévő házirend-szervizelést. Az aktív üzembe állítások törlődnek, és a rendszer nem hoz létre új központi telepítőket.

## Példák

### Példa 1: házirend-szervizelés megszüntetése az erőforráscsoport hatókörében
```
PS C:\> Stop-AzureRmPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

Ez a parancs a "myRG" erőforráscsoport "remediation1" nevű szervizelését törli.

### 2. példa: a kezelési csoport lemondása csővezetéken keresztüli szervizelésének lemondása
```
PS C:\> $remediation = Get-AzureRmPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Stop-AzureRmPolicyRemediation
```

Ez a parancs lemondja a "remediation1" nevű "mg1" felügyeleti csoportban található "" nevű szervizelést.
