# Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys

## Áttekintés
A megadott Access-kulcs visszaállítása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys [-ResourceGroupName] <String>
 [-WorkspaceCollectionName] <String> [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Leírás
A **reset-AzureRmPowerBIWorkspaceCollectionAccessKeys** parancsmag visszaállítja a megadott Access-kulcsot a Power bi-munkaterületi gyűjteményben.

## Példák

### 1. példa: az elsődleges hozzáférés kulcsának alaphelyzetbe állítása
```
PS C:\>Reset-AzureRmPowerBIWorkspaceCollectionAccessKeys -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Ez a parancs visszaállítja a megadott erőforráscsoport WCN11 nevű munkaterületi gyűjtemény elsődleges hozzáférési kulcsát.
