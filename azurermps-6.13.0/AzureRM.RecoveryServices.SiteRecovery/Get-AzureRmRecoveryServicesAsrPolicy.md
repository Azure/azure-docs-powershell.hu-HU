# Get-AzureRmRecoveryServicesAsrPolicy

## Áttekintés
ASR-replikációs házirendek jelentkeznek.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### Alapértelmezett (alapértelmezett)
```
Get-AzureRmRecoveryServicesAsrPolicy [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzureRmRecoveryServicesAsrPolicy -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzureRmRecoveryServicesAsrPolicy -FriendlyName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzureRmRecoveryServicesAsrPolicy** parancsmag a konfigurált Azure-webhelyek helyreállítási replikációs házirendjeinek vagy egy adott replikációs házirendnek a neve alapján kapja meg a listát.

## Példák

### Példa 1
```
PS C:\> $Policy = Get-AzureRmRecoveryServicesAsrPolicy
```

Retuns a replikációs házirendek listájával

### 2. példa
```
PS C:\>  Get-AzureRmRecoveryServicesAsrPolicy -Name abc
```

A Retuns replikációs házirendjének neve.
