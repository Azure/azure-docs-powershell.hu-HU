# Get-AzureRmRecoveryServicesBackupProperty

## Áttekintés
Biztonsági mentési tulajdonságok.

## SZINTAXISA

```
Get-AzureRmRecoveryServicesBackupProperty -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

A **Get-AzureRmRecoveryServicesBackupProperty** parancsmag biztonsági mentési tulajdonságokat kap a helyreállítási szolgáltatások boltozatához.
