### [Get-AzureRmSearchAdminKeyPair](Get-AzureRmSearchAdminKeyPair.md)
Megkapja az Azure Search szolgáltatás rendszergazdai kulcs párjait.

### [Get-AzureRmSearchQueryKey](Get-AzureRmSearchQueryKey.md)
Az Azure keresési szolgáltatás lekérdezési kulcsát (s) kapja meg.

### [Remove-AzureRmSearchQueryKey](Remove-AzureRmSearchQueryKey.md)
Távolítsa el a lekérdezési kulcsot az Azure Search szolgáltatásból.
