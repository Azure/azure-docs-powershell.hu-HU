# Start-AzureRmJitNetworkAccessPolicy

## Áttekintés
Ideiglenes hálózati hozzáférési kérést hív meg.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### ResourceGroupLevelResource (alapértelmezett)
```
Start-AzureRmJitNetworkAccessPolicy -ResourceGroupName <String> -Location <String> -Name <String>
 -VirtualMachine <PSSecurityJitNetworkAccessPolicyInitiateVirtualMachine[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Start-AzureRmJitNetworkAccessPolicy -VirtualMachine <PSSecurityJitNetworkAccessPolicyInitiateVirtualMachine[]>
 -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Start-AzureRmJitNetworkAccessPolicy -InputObject <PSSecurityJitNetworkAccessPolicyInitiateInputObject>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Ideiglenes hálózati hozzáférési kérést hív meg.
A kérés érvényesítése a beállított JIT-hálózat hozzáférési házirendjével történik, és ha a permittet, a felhasználó kérése szerint megnyílik egy hálózati kapcsolat.
A kérés a későbbi véleményezéshez loged válik, és a megadott időtartam túllépése után megszűnik.
