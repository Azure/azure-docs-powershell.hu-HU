# Get-AzureRmServiceBusQueue

## Áttekintés
A megadott szolgáltatás busz-várólistájának leírását adja meg.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A megadott szolgáltatás busz-várólistájának leírását adja meg.

A várólista leírását számítja ki.

### 2. példa
```
PS C:\> Get-AzureRmServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

A megadott névtérhez tartozó várólisták listáját adja vissza, alapértelmezés szerint az 100-várólisták lesznek visszaadva, ha több mint 100 várólistát ad vissza, kérjük, használja a-MaxCount paramétert.

### 3. példa
```
PS C:\> Get-AzureRmServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Az első 150-várólisták listáját adja meg az adott névtérhez.
