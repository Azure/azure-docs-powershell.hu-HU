# Get-AzureRmServiceBusTopic

## Áttekintés
A megadott szolgáltatási busz témakör leírását adja meg.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmServiceBusTopic** parancsmag a megadott szolgáltatási busz névterének leírását adja eredményül.

A megadott, a szolgáltatás busz névteréhez tartozó témakör leírását adja meg.

### 2. példa
```
PS C:\> Get-AzureRmServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

A szolgáltatás busz névteréhez tartozó témakörök listáját adja eredményül. Alapértelmezés szerint az 100-témaköröket a program visszaadja, ha több mint 100-témakört ad vissza, kérjük, használja a-MaxCount paramétert.

### 3. példa
```
PS C:\> Get-AzureRmServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Az első 150-témakörök listáját adja meg a szolgáltatás buszának névteréhez.
