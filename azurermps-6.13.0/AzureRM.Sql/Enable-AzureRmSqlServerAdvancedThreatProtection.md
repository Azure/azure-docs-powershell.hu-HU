# Enable-AzureRmSqlServerAdvancedThreatProtection

## Áttekintés
Lehetővé teszi a speciális veszélyforrások védelmét a kiszolgálón.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Enable-AzureRmSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
Az **enable-AzureRmSqlServerAdvancedThreatProtection** parancsmag lehetővé teszi a speciális veszélyforrások védelmét a kiszolgálón.

## Példák

### Példa 1 – a Server Advanced Threat Protection engedélyezése
```powershell
PS C:\>  Enable-AzureRmSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Példa 2 – a kiszolgáló speciális veszélyforrások elleni védelem engedélyezése a kiszolgálói erőforrásból
```powershell
PS C:\>  Get-AzureRmSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzureRmSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
