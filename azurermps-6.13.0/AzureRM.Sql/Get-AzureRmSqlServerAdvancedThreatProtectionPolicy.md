# Get-AzureRmSqlServerAdvancedThreatProtectionPolicy

## Áttekintés
Speciális veszélyforrások elleni védelemre vonatkozó házirendet kap a kiszolgálón.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlServerAdvancedThreatProtectionPolicy [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlServerAdvancedThreatProtectionPolicy** parancsmag a kiszolgáló speciális veszélyforrásokkal kapcsolatos védelmi házirendjét kapja meg.

## Példák

### Példa 1 – a Server Advanced Threat Protectiont kapja
```powershell
PS C:\>  Get-AzureRmSqlServerAdvancedThreatProtectionPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Példa: a Server Advanced Threat Protection a kiszolgálói erőforrásból
```powershell
PS C:\>  Get-AzureRmSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Get-AzureRmSqlServerAdvancedThreatProtectionPolicy

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
