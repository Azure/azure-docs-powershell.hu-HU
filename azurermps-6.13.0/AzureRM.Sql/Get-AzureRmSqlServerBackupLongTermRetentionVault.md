# Get-AzureRmSqlServerBackupLongTermRetentionVault

## Áttekintés
A kiszolgáló hosszú távú adatmegőrzési boltozatát kapja.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlServerBackupLongTermRetentionVault [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlServerBackupLongTermRetentionVault** parancsmag a kiszolgálóhoz regisztrált hosszú távú adatmegőrzési boltozatot kapja.
A boltozat a biztonsági másolatok adatainak tárolására szolgáló Azure Backup-erőforrás.
