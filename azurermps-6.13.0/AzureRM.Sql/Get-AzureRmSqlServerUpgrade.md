# Get-AzureRmSqlServerUpgrade

## Áttekintés
Az Azure SQL-adatbázis kiszolgálói frissítésének állapotát kapja meg.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmSqlServerUpgrade -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmSqlServerUpgrade** parancsmag az Azure SQL adatbázis-kiszolgáló frissítésének állapotát kapja meg.

## Példák

### Példa 1: frissítés állapotának beszerzése
```
PS C:\>Get-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Format-List
ResourceGroupName               : resourcegroup01
ServerName                      : server01
Status                          : Queued
```

Ez a parancs a Server01 nevű kiszolgálóról a ResourceGroup01 nevű erőforráscsoport frissítés állapotát kapja meg.
