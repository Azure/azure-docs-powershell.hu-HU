# Remove-AzureRmStorageContainer

## Áttekintés
Tároló blob-tároló eltávolítása

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

### AccountName (alapértelmezett)
```
Remove-AzureRmStorageContainer [-ResourceGroupName] <String> [-StorageAccountName] <String> -Name <String>
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Remove-AzureRmStorageContainer -Name <String> -StorageAccount <PSStorageAccount> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ContainerObject
```
Remove-AzureRmStorageContainer -InputObject <PSContainer> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **Remove-AzureRmStorageContainer** parancsmag eltávolítja a BLOB-tárolót

## Példák

### 1. példa: a tárolási blob-tároló eltávolítása a tárolási fiók nevével és a tároló nevével
```
PS C:\>Remove-AzureRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer"
```

Ez a parancs a tároló BLOB-tárolót eltávolítja a tárolási fiók nevével és a tároló nevével.

### 2. példa: a tárolási blob-tároló eltávolítása a Storage Account objektummal és a tároló nevével
```
PS C:\>$accountObject = Get-AzureRmStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>Remove-AzureRmStorageContainer -StorageAccount $accountObject -ContainerName "myContainer"
```

Ez a parancs eltávolítja a tároló BLOB-tárolót a Storage Account objektummal és a tároló nevével.

### 3. példa: az összes tárterületet tároló blob-tároló eltávolítása egy csővezetéktel
```
PS C:\>Get-AzureRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" | Remove-AzureRmStorageContainer -Force
```

Ez a parancs eltávolítja az összes tárterület-tárolót a tároló fiókból a csővezetéken.
