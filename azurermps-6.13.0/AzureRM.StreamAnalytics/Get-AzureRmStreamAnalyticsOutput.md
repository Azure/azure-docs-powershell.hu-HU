# Get-AzureRmStreamAnalyticsOutput

## Áttekintés
A megadott adatfolyam-munkafolyamatban vagy kimenetben meghatározott kimeneteket kapja meg.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SZINTAXISA

```
Get-AzureRmStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRmStreamAnalyticsOutput** parancsmag felsorolja a megadott adatfolyam-adatforrásban definiált összes kimenetet, illetve információt kap egy adott kimenetről.

## Példák

### 1. példa: adatok beszerzése a projekt kimenetéről
```
PS C:\>Get-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Ez a parancs információt ad eredményül a projekt StreamingJob meghatározott kimenetekről.

### 2. példa: információk kérése egy adott projekt kimenetéről
```
PS C:\>Get-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Ez a parancs információt ad eredményül a projekt StreamingJob definiált kimenetről.
