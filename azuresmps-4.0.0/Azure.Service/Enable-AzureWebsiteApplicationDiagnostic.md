# Enable-AzureWebsiteApplicationDiagnostic

## Áttekintés
Engedélyezi az alkalmazás-diagnosztikát az Azure-webhelyeken.

## SZINTAXISA

### FileParameterSet
```
Enable-AzureWebsiteApplicationDiagnostic [-PassThru] [-File] -LogLevel <LogEntryType> [-Name <String>]
 [-Slot <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### TableStorageParameterSet
```
Enable-AzureWebsiteApplicationDiagnostic [-PassThru] [-TableStorage] -LogLevel <LogEntryType>
 [-StorageAccountName <String>] [-StorageTableName <String>] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### BlobStorageParameterSet
```
Enable-AzureWebsiteApplicationDiagnostic [-PassThru] [-BlobStorage] -LogLevel <LogEntryType>
 [-StorageAccountName <String>] [-StorageBlobContainerName <String>] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

Engedélyezi az alkalmazások diagnosztikai szolgáltatását egy Azure-webhelyen, és lehetővé teszi a naplófájlok tárolását a fájlrendszerben vagy az Azure-tárterületen.

## Példák

### 1. példa: a diagnosztika engedélyezése a fájlrendszer használatával
```
PS C:\> Enable-AzureWebsiteApplicationDiagnostic -Name MyWebsite -File -LogLevel Verbose
```

Ez a példa lehetővé teszi az alkalmazások naplózását a fájlrendszerben bőbeszédű szintre.

### 2. példa: a naplózás engedélyezése az Azure Storage szolgáltatással
```
PS C:\> Enable-AzureWebsiteApplicationDiagnostic -Name MyWebsite -Storage -LogLevel Information -StorageAccountName myaccount
```

Ez a példa lehetővé teszi az alkalmazások naplózását a MyAccount nevű tárterület-fiókkal az adatok naplózási szinttel beállításával.
