# Get-AzureRemoteAppLocation

## Áttekintés
Az Azure RemoteApp által támogatott helyek listája.

## SZINTAXISA

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRemoteAppLocation** parancsmag felsorolja az Azure RemoteApp által támogatott helyeket.

## Példák

## PARAMÉTEREK

Ha nem ad meg profilt, a parancsmag a helyi alapértelmezett profilból olvassa be a szöveget.
