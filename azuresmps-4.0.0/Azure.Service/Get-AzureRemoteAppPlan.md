# Get-AzureRemoteAppPlan

## SYNOPSIS
Az Azure RemoteApp-csomagok listája.

## SZINTAXIS

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzureRemoteAppPlan parancsmag** felsorolja az Azure RemoteApp-csomagokat.

## PÉLDÁK

## PARAMETERS
