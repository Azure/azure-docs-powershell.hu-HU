# Get-AzureRemoteAppWorkspace

## Áttekintés
Információt keres az Azure RemoteApp-munkaterületről.

## SZINTAXISA

```
Get-AzureRemoteAppWorkspace [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
A **Get-AzureRemoteAppWorkspace** parancsmag információkat keres az Azure RemoteApp-munkaterületről.

## Példák

### 1. példa: a munkaterület adatainak beolvasása
```
PS C:\> Get-AzureRemoteAppWorkspace
ClientUrl                               EndUserFeedName
---------                               ---------------
https://www.remoteapp.windowsazure.com/ Contoso Work Applications
```

Ez a parancs információkat olvas be az Azure RemoteApp-munkaterületről.
