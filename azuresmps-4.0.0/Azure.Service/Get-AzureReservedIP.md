# Get-AzureReservedIP

## Áttekintés
Egy fenntartott IP-címet kap a nevével, vagy felsorolja az előfizetésben lévő összes lefoglalt IP-címet.

## SZINTAXISA

```
Get-AzureReservedIP [[-ReservedIPName] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Get-AzureReservedIP** parancsmag a lefoglalt IP-címet a neve alapján kapja meg, vagy felsorolja az előfizetésben szereplő összes lefoglalt IP-címet.

## Példák

### Példa 1: az összes fenntartott IP-cím beolvasása
```
PS C:\> Get-AzureReservedIP
```

Ez a parancs a lefoglalt IP-címeket kapja meg.

### 2. példa: fenntartott IP-cím beszerzése megadott névvel
```
PS C:\> Get-AzureReservedIP -ReservedIPName $IpName
```

Ez a parancs azt a fenntartott IP-címet kapja meg, amelynek nevét a $IpName változó adja meg.
