# Get-AzureSchedulerLocation

## Áttekintés
Elérhetővé válik az ütemező helye.

## SZINTAXISA

```
Get-AzureSchedulerLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

A **Get-AzureSchedulerLocation** parancsmag elérhetővé válik az ütemező helyekkel.

## Példák

### 1. példa: az elérhető ütemező helyének beszerzése
```
PS C:\> Get-AzureSchedulerLocation
```

A parancs elérhetővé válik az ütemező helyekkel.
