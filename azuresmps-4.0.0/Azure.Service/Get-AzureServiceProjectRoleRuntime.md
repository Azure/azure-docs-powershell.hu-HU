# Get-AzureServiceProjectRoleRuntime

## Áttekintés
A szerepkörben való telepítéshez elérhető futtatókörnyezetek beszerzése.

## SZINTAXISA

```
Get-AzureServiceProjectRoleRuntime [-Runtime <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

Elérhetővé válik a szerepkörben telepítendő futtatókörnyezetek.
