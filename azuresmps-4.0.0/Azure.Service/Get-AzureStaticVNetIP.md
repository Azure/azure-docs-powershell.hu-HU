# Get-AzureStaticVNetIP

## Áttekintés
A statikus VNet IP-címét egy virtuálisgép-objektumból kapja, ha van ilyen.

## SZINTAXISA

```
Get-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Get-AzureStaticVNetIP** parancsmag a statikus virtuális hálózat (VNet) IP-címének adatait egy virtuálisgép-objektumból kapja, ha van ilyen.
