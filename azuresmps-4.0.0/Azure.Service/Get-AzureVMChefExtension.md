# Get-AzureVMChefExtension

## Áttekintés
A séf által megadott kiterjesztést kapja egy virtuális gépen.

## SZINTAXISA

```
Get-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Get-AzureVMChefExtension** parancsmag beilleszti a Chef bővítményt egy virtuális gépre.

## Példák

### 1. példa: a szakács által megadott bővítmény beszerzése a megadott virtuális gépen
```
PS C:\> Get-AzureVMChefExtension -VM $VM;
```

Ez a parancs a megadott virtuális gépen a szakács által megadott kiterjesztést kapja.
