# Get-AzureVMDiagnosticsExtension

## Áttekintés
Az Azure Diagnostics bővítmény beállításait egy virtuális gépen kapja meg.

## SZINTAXISA

```
Get-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Get-AzureVMDiagnosticsExtension** parancsmag a Microsoft Azure Diagnostics bővítmény beállításait egy virtuális gépen kapja meg.

## Példák

### Példa 1: a diagnosztika bővítmény beszerzése virtuális gépre
```
PS C:\> Get-AzureVMDiagnosticsExtension -VM $VM
```

Ez a parancs a megadott virtuális gépre alkalmazott diagnosztikai bővítményt kapja meg a $VM változóban tárolt módon.
