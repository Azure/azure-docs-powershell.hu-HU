# Get-AzureVirtualNetworkGatewayKey

## Áttekintés
Egy Azure virtuális hálózati átjáró kulcsát kapja meg.

## SZINTAXISA

```
Get-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Leírás
A **Get-AzureVirtualNetworkGatewayKey** parancsmag az Azure virtuális hálózati átjáró kulcsát kapja meg.
