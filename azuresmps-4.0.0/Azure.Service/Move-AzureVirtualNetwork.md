# Move-AzureVirtualNetwork

## Áttekintés
Virtuális hálózat áttelepítése az Azure Resource Manager-kötegbe.

## SZINTAXISA

### ValidateMigrationParameterSet
```
Move-AzureVirtualNetwork [-VirtualNetworkName] <String> [-Validate] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### AbortMigrationParameterSet
```
Move-AzureVirtualNetwork [-VirtualNetworkName] <String> [-Abort] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### CommitMigrationParameterSet
```
Move-AzureVirtualNetwork [-VirtualNetworkName] <String> [-Commit] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### PrepareMigrationParameterSet
```
Move-AzureVirtualNetwork [-VirtualNetworkName] <String> [-Prepare] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Move-AzureVirtualNetwork** parancsmag a virtuális hálózatokat az Azure Resource Manager-kötegben lévő erőforráscsoporthoz telepíti át.

## Példák

### Példa 1: virtuális hálózati áttelepítés előkészítése
```
PS C:\> Move-AzureVirtualNetwork -Prepare -VirtualNetworkName "ContosoVNET"
```

Ez a parancs felkészíti a ContosoVNET nevű virtuális hálózatot az Azure Resource Manager-kötegbe való áttelepítésre.

### 2. példa: a virtuális hálózati áttelepítés indítása
```
PS C:\> Move-AzureVirtualNetwork -Commit -VirtualNetworkName "ContosoVNET"
```

Ez a parancs elindítja a ContosoVNET nevű virtuális hálózat áttelepítését az Azure Resource Manager-kötegbe.

### 3. példa: a virtuális hálózati áttelepítés érvényesítése
```
PS C:\> Move-AzureVirtualNetwork -Validate -VirtualNetworkName "ContosoVNET"
```

Ez a parancs ellenőrzi a ContosoVNET nevű virtuális hálózat áttelepítését az Azure Resource Manager-köteg számára.
