# New-AzureAclConfig

## Áttekintés
Üres ACL-konfigurációs objektumot hoz létre.

## SZINTAXISA

```
New-AzureAclConfig [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **New-AzureAclConfig** parancsmag üres hozzáférés-vezérlési lista (ACL) konfigurációs objektumot hoz létre.

## Példák

### Példa 1: ACL-konfigurációs objektum létrehozása
```
PS C:\> $Acl = New-AzureAclConfig
```

Ez a parancs létrehoz egy üres ACL-konfigurációs objektumot, majd a $Acl változóban tárolja.
