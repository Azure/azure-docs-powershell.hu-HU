# New-AzureAutomationAccount

## Áttekintés

Automatizálási fiókot hoz létre.

## SZINTAXISA

```
New-AzureAutomationAccount -Name <String> -Location <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

A **New-AzureAutomationAccount** parancsmag új fiókot hoz létre a Microsoft Azure automatizálásban.

## Példák

### Példa 1: automatizálási fiók létrehozása
```
PS C:\> New-AzureAutomationAccount -Name "MyAutomationAccount" -Location "East US"
```

Ez a parancs létrehoz egy MyAutomationAccount nevű automatizálási fiókot a kelet-amerikai régióban.
