# New-AzureAutomationCredential

## Áttekintés

Hitelesítő adatokat hoz létre az automatizálásban.

## SZINTAXISA

```
New-AzureAutomationCredential -Name <String> [-Description <String>] -Value <PSCredential>
 -AutomationAccountName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

A **New-AzureAutomationCredential** parancsmag **PSCredential** -objektumként hozza létre a hitelesítő adatait a Microsoft Azure automatizálásban.

## Példák

### Példa 1: hitelesítő adatok létrehozása
```
PS C:\> $user = "MyDomain\MyUser"
PS C:\> $pw = ConvertTo-SecureString "PassWord!" -AsPlainText -Force
PS C:\> $cred = New-Object -TypeName System.Management.Automation.PSCredential -ArgumentList $user, $pw
PS C:\> New-AzureAutomationCredential -AutomationAccountName "Contoso17" -Name "MyCredential" -Value $cred
```

Ezekkel a parancsokkal létrehozhatja a MyCredential nevű hitelesítő adatokat.
Először létrejön egy hitelesítőadat-objektum, amely tartalmazza a felhasználónevet és a jelszót.
Ezt követően a rendszer az utolsó parancsban használja az automatizálási hitelesítő adatok létrehozását.
