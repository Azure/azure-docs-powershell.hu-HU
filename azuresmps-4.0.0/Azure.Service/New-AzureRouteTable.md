# New-AzureRouteTable

## Áttekintés
Útválasztási táblázat létrehozása.

## SZINTAXISA

```
New-AzureRouteTable -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Leírás
A **New-AzureRouteTable** parancsmag egy útválasztási táblázatot hoz létre a megadott helyen.
Felhasználó által definiált útvonalakat adhat az útválasztási táblázathoz.
Az útválasztási táblázatot társíthatja a virtuális hálózat egy alhálózatához is.
