# Register-AzureAutomationScheduledRunbook

## Áttekintés

Runbook társít egy ütemtervhez.

## SZINTAXISA

### ByRunbookName (alapértelmezett)
```
Register-AzureAutomationScheduledRunbook -AutomationAccountName <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

### ByRunbookNameAndScheduleName
```
Register-AzureAutomationScheduledRunbook -RunbookName <String> -ScheduleName <String>
 [-Parameters <IDictionary>] -AutomationAccountName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

A **Register-AzureAutomationScheduledRunbook** parancsmag egy runbook társít egy ütemtervhez.
A runbook a *ScheduleName* paraméterrel megadott ütemezés alapján kezdődik.

## Példák

### 1. példa: runbook társítása ütemtervvel
```
PS C:\> Register-AzureAutomationScheduledRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ScheduleName "Sched01"
```

Ez a parancs a Runbk01 nevű runbook társítja a Contoso17 nevű Azure automatizálási fiók Sched01 nevű munkabeosztásával.
