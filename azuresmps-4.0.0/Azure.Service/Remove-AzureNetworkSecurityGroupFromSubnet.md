# Remove-AzureNetworkSecurityGroupFromSubnet

## Áttekintés
A hálózat biztonsági csoportjának kihagyása egy alhálózatról.

## SZINTAXISA

```
Remove-AzureNetworkSecurityGroupFromSubnet -Name <String> -VirtualNetworkName <String> -SubnetName <String>
 [-Force] [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureNetworkSecurityGroupFromSubnet** parancsmag egy alhálózatról egy Azure-hálózat biztonsági csoportját elhatárolja.
