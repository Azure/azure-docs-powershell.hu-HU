# Remove-AzureRemoteAppCollection

## Áttekintés
Azure RemoteApp-gyűjtemény eltávolítása.

## SZINTAXISA

```
Remove-AzureRemoteAppCollection [-CollectionName] <String> [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzureRemoteAppCollection** parancsmag eltávolítja az Azure RemoteApp-gyűjteményt.
