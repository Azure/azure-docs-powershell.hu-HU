# Remove-AzureRemoteAppTemplateImage

## Áttekintés
Azure RemoteApp-sablon törlése

## SZINTAXISA

```
Remove-AzureRemoteAppTemplateImage [-ImageName] <String> [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzureRemoteAppTemplateImage** parancsmag az Azure RemoteApp-sablon képét törli.
A sablonok képe csak akkor törölhető, ha az nem egy Azure RemoteApp-gyűjteményhez van társítva.

## Példák

### Példa 1: sablon törlése kép
```
PS C:\> Remove-AzureRemoteAppTemplateImage -ImageName "ContosoApps"
```

Ez a parancs törli a ContosoApps nevű sablon képét.
