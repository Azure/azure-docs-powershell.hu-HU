# Remove-AzureReservedIP

## Áttekintés
Eltávolítja a lefoglalt IP-címet a neve alapján.

## SZINTAXISA

```
Remove-AzureReservedIP [-ReservedIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureReservedIP** parancsmag a lefoglalt IP-címet a neve alapján távolítja el.

## Példák

### 1. példa: fenntartott IP-cím eltávolítása a nevével
```
PS C:\> Remove-AzureReservedIP -ReservedIPName $name
```

Ez a parancs eltávolítja a lefoglalt IP-címet a neve alapján.
