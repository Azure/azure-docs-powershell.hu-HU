# Remove-AzureRoute

## Áttekintés
Egy útvonal eltávolítása az útvonal-táblázatból.

## SZINTAXISA

```
Remove-AzureRoute -RouteName <String> [-Force] -RouteTable <IRouteTable> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Leírás
A **Remove-AzureRoute** parancsmag eltávolítja az útvonalat egy útvonal-táblázatból.

## Példák

### 1. példa: útvonal eltávolítása
```
PS C:\> Get-AzureRouteTable -Name "ApplianceRouteTable" | Remove-AzureRoute -RouteName "InternetRoute"
Confirm
Are you sure you want to remove the Route "InternetRoute" from Route Table "ApplianceRouteTable"?
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y

Routes                        Name                          Location                      Label
------                        ----                          --------                      -----
{approute}                    AppRT                         Central US                    Appliance Route Table
```

Ez a parancs a ApplianceRouteTable nevű útválasztási táblázatot kapja.
A parancs átadja az útválasztási táblázatot az aktuális parancsmagnak.
Az aktuális parancsmag eltávolítja a InternetRoute nevű útvonalat.
