# Remove-AzureService

## Áttekintés
Eltávolítja az aktuális felhő-szolgáltatást.

## SZINTAXISA

```
Remove-AzureService [-ServiceName <String>] [-Force] [-PassThru] [-DeleteAll] [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

A **Remove-AzureService** parancsmag leállítja és eltávolítja az aktuális Felhőbeli szolgáltatást, illetve a megadott szolgáltatáshoz és előfizetéshez tartozó szolgáltatást.
