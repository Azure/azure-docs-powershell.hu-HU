# Remove-AzureStorageAccount

## Áttekintés
Törli a megadott tárterület-fiókot egy előfizetésből.

## SZINTAXISA

```
Remove-AzureStorageAccount [-StorageAccountName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureStorageAccount** parancsmag az Azure-előfizetésből eltávolítja a fiókot.

## Példák

### 1. példa: tárterület-fiók eltávolítása
```
PS C:\> Remove-AzureStorageAccount -StorageAccountName "ContosoStore01"
```

Ez a parancs eltávolítja a ContosoStore01-tároló fiókot a megadott előfizetésből.
