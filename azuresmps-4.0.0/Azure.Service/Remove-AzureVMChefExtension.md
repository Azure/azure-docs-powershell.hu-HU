# Remove-AzureVMChefExtension

## Áttekintés
Eltávolítja a virtuális gépen alkalmazott séf-kiterjesztést.

## SZINTAXISA

```
Remove-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureVMChefExtension** parancsmag törli a virtuális gépen alkalmazott séf-bővítményt.

## Példák

### 1. példa: a kijelölt virtuális gépen alkalmazott Chef-kiterjesztés eltávolítása
```
PS C:\> Remove-AzureVMChefExtension -VM $VM;
```

Ez a parancs eltávolítja a virtuális gépen alkalmazott szakács-kiterjesztést.
