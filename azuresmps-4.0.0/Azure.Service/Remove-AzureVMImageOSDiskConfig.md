# Remove-AzureVMImageOSDiskConfig

## Áttekintés
Eltávolítja az operációs rendszer lemezének konfigurációját a DiskConfigSet objektumból.

## SZINTAXISA

```
Remove-AzureVMImageOSDiskConfig [-DiskConfig] <VirtualMachineImageDiskConfigSet>
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureVMImageOSDiskConfig** parancsmag eltávolítja az operációs rendszer lemezének konfigurációját a DiskConfigSet objektumból.

## Példák

### 1. példa: az operációs rendszer lemez-konfigurációjának eltávolítása a DiskConfigSet
```
PS C:\> $Disk = New-AzureDiskConfigSet
PS C:\> $Disk = Set-AzureOSDiskConfig -DiskConfig $Disk -HostCaching ReadWrite
PS C:\> Remove-AzureVMImageOSDiskConfig -DiskConfig $Disk
```

Ez a parancs eltávolítja az operációs rendszer lemezének konfigurációját a DiskConfigSet objektumból.
