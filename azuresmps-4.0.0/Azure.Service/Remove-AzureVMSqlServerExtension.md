# Remove-AzureVMSqlServerExtension

## Áttekintés
Azure Virtual Machine SQL Server-bővítmény eltávolítása egy virtuálisgép-objektumból.

## SZINTAXISA

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureVMSqlServerExtension** parancsmag egy virtuális Machine-objektumból távolítja el az Azure Virtual Machine SQL Server bővítményét.
