# Remove-AzureVirtualIP

## Áttekintés
Virtuális IP-cím törlése a felhőalapú szolgáltatásból.

## SZINTAXISA

```
Remove-AzureVirtualIP [-ServiceName] <String> [-VirtualIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **Remove-AzureVirtualIP** parancsmag egy virtuális IP-címet (VIP) töröl egy Azure-szolgáltatásból.
A művelet csak akkor sikerül, ha a virtuális IP-hez nincs végpont társítva.

## Példák

### 1. példa: virtuális IP-cím eltávolítása egy szolgáltatásból
```
PS C:\> Remove-AzureVirtualIP -VirtualIPName "Vip01" -ServiceName "ContosoService03"
```

Ez a parancs eltávolítja a virtuális IP-címet egy szolgáltatásból.
