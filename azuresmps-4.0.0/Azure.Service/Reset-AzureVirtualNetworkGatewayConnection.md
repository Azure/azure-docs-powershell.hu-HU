# Reset-AzureVirtualNetworkGatewayConnection

## Áttekintés
Virtuális hálózati átjáró kapcsolatának alaphelyzetbe állítása.

## SZINTAXISA

```
Reset-AzureVirtualNetworkGatewayConnection -GatewayId <String> -ConnectedEntityId <String>
 -RoutingWeight <Int32> [-SharedKey <String>] [-EnableBgp <Boolean>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Leírás
A **reset-AzureVirtualNetworkGatewayConnection** parancsmag visszaállítja a virtuális hálózati átjáró kapcsolatát.
