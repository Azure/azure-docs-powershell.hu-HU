# Restart-AzureSiteRecoveryJob

## Áttekintés
Webhely-helyreállítási feladat újraindítása.

## SZINTAXISA

### ByObject (alapértelmezett)
```
Restart-AzureSiteRecoveryJob -Job <ASRJob> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### ById
```
Restart-AzureSiteRecoveryJob -Id <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
Az **Újraindítás – AzureSiteRecoveryJob** parancsmag újraindítja az Azure webhely-helyreállítási feladatot.

State            : Failed

EndTime          : 10/6/2014 9:41:21 AM
AllowedActions   : {Cancel, Restart}

                   Starting initial replication...}

Ez a parancs elindítja a megadott azonosítót tartalmazó munkát.
