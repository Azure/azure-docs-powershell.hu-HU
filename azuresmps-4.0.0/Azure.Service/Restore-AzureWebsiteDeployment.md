# Restore-AzureWebsiteDeployment

## Áttekintés
Egy webhely előző telepített példányának újratelepítése az Azure-ban

## SZINTAXISA

```
Restore-AzureWebsiteDeployment [-CommitId <String>] [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

A **Restore-AzureWebsiteDeployment** parancsmag áttelepíti az Azure-webhely korábbi telepített példányát.
Ez a folyamat lecseréli az aktuális telepítést a kijelölt telepítéssel.

## Példák

### 1. példa: webhely újratelepítése
```
PS C:\> Restore-AzureWebsiteDeployment -Name "ContosoSite" -CommitId "f876543210"
```

Ez a parancs áttelepíti a ContosoSite nevű webhely azonosító f876543210 tartalmazó példányát.
