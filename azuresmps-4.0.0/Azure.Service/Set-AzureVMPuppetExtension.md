# Set-AzureVMPuppetExtension

## Áttekintés
Egy virtuális gép báb-kiterjesztését állítja be.

## SZINTAXISA

```
Set-AzureVMPuppetExtension [-PuppetMasterServer] <String> [[-Version] <String>] [-Disable]
 [[-ReferenceName] <String>] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
A **set-AzureVMPuppetExtension** parancsmag a virtuális gép báb-kiterjesztését állítja be.

## Példák

### Példa 1: a báb-kiterjesztés beállítása virtuális géphez
```
PS C:\> Set-AzureVMPuppetExtension -VM $VM
```

Ebben a példában a megadott virtuális gép báb-kiterjesztését állítja be a $VM változóban tárolt módon.
