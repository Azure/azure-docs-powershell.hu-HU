# Set-AzureVirtualNetworkGatewayKey

## Áttekintés
Egy Azure virtuális hálózati átjáró kulcsát állítja be.

## SZINTAXISA

```
Set-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> -SharedKey <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
A **set-AzureVirtualNetworkGatewayKey** parancsmag az Azure virtuális hálózati átjáró kulcsát állítja be.
