# Show-AzureWebsite

## Áttekintés
Megnyit egy böngészőt egy adott webhelyen.

## SZINTAXISA

```
Show-AzureWebsite [-Name <String>] [-Slot <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

A **show-AzureWebsite** parancsmag megnyit egy böngészőt egy adott webhelyen.
