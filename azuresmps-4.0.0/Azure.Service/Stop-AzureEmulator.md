# Stop-AzureEmulator

## Áttekintés
Leállítja a számítási emulátort.

## SZINTAXISA

```
Stop-AzureEmulator [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

A **stop-AzureEmulator** parancsmag leállítja az Azure számítási emulátort.
Az emulátorban jelenleg futó szolgáltatások törlődnek.
