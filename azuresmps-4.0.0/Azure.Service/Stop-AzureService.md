# Stop-AzureService

## Áttekintés
Leállítja az aktuálisan hosztolt szolgáltatást.

## SZINTAXISA

```
Stop-AzureService [-ServiceName <String>] [-Slot <String>] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Leírás
Ez a témakör a Microsoft Azure PowerShell modul 0.8.10 verziójában található parancsmagot ismerteti.
A használt modul verziójának beszerzéséhez az Azure PowerShell konzolon írja be a következőt: `(Get-Module -Name Azure).Version` .

A **stop-AzureService** parancsmag leállítja a Windows Azure által megadott bővítőhelyen a jelenlegi hosztolt szolgáltatást.
Ha nincs megadva bővítőhely, a parancsmag leállítja a szolgáltatást a termelési bővítőhelyen.
