# Stop-AzureWebsiteJob

## Áttekintés
Webfeladatot állít be egy webhelyhez.

## SZINTAXISA

```
Stop-AzureWebsiteJob -JobName <String> [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Leírás
A **stop-AzureWebsiteJob** parancsmag leállítja a webhelyek webfeladatát.

## Példák

### Példa 1: webfeladat leállítása a webhelyen
```
PS C:\> Stop-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob
```

Leállítja a MyWebJob nevű webes feladatot a MyWebSite.
