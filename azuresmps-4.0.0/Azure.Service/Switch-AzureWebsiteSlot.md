# Switch-AzureWebsiteSlot

## Áttekintés
Felcseréli egy másik bővítőhelyet tartalmazó webhelyhez tartozó termelési résidőt.

## SZINTAXISA

```
Switch-AzureWebsiteSlot [-Name <String>] [-Slot1 <String>] [-Slot2 <String>] [-Force]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A **switch-AzureWebsiteSlot** parancsmag egy másik bővítőhelyet tartalmazó webhely gyártási bővítőhelyét cseréli le.
Ez a funkció csak a két bővítőhelyet használó webhelyeken működik.

## Példák

### 1. példa: a webhely-tárolóhely váltása
```
PS C:\> Switch-AzureWebsiteSlot -Name MyWebsite
```

Váltson az Azure webhely MyWebsite biztonsági másolati nyílásába a termelési bővítőhelyen.
