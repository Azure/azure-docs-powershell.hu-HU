# Unpublish-AzureRemoteAppProgram

## Áttekintés
Azure RemoteApp program közzétételének visszavonása

## SZINTAXISA

```
Unpublish-AzureRemoteAppProgram [-CollectionName] <String> [[-Alias] <String[]>] [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
A Közzététel visszavonása **– AzureRemoteAppProgram** parancsmag visszavonja az Azure RemoteApp program közzétételét.
A program közzétételének megszüntetése után már nem érhető el az Azure RemoteApp-gyűjtemény felhasználói.
