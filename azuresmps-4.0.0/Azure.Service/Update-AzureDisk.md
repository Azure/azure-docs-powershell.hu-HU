# Update-AzureDisk

## Áttekintés
Az Azure Disk repositoryban módosítja a lemez feliratát.

## SZINTAXISA

### Átméretezése
```
Update-AzureDisk [-DiskName] <String> [[-Label] <String>] [-ResizedSizeInGB] <Int32>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### Átméretezés
```
Update-AzureDisk [-DiskName] <String> [-Label] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Leírás
Az **Update-AzureDisk** parancsmag az aktuális Azure-előfizetés lemezéhez társított címkét módosítja.

## Példák

### Példa 1: a lemez címkéjének módosítása
```
PS C:\> Update-AzureDisk ?DiskName "ContosoOSDisk" -Label "DoNotUse"
```

Ez a parancs megváltoztatja a ContosoOSDisk nevű lemez DoNotUse.
