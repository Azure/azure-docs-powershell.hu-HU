# Update-AzureWebsiteRepository

## Áttekintés
Frissíti egy helyi git-tárház távoli tárházait az összes bővítőhelyen.

## SZINTAXISA

```
Update-AzureWebsiteRepository [-Name <String>] -PublishingUsername <String> [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Leírás
Az **Update-AzureWebsiteRepository** parancsmag egy helyi git tárház távoli tárházait frissíti a résidők számára.

## Példák

### 1. példa: reupdate website Remote tárolók
```
PS C:\> Update-AzureWebsiteRepository -Name MyWebsite
```

Frissíti a helyi git-tárház távoli tárházait a webhelyek MyWebsite összes bővítőhelyéhez.
