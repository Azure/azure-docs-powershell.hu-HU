# AZS. Gallery. admin modul
## Leírás
A AzureStack Gallery Operator Module előzetes verziója, amely a piactéren a gyűjtemény elemeinek kezeléséhez nyújt funkciókat.

## AZS. Gallery. felügyeleti parancsmagok
### [Add-AzsGalleryItem](Add-AzsGalleryItem.md)
Feltölti a szolgáltató gyűjteményének elemét a tárolóba.

### [Get-AzsGalleryItem](Get-AzsGalleryItem.md)
A gyűjtemény felsorolja a gyűjtemény elemeit.

### [Remove-AzsGalleryItem](Remove-AzsGalleryItem.md)
Adott gyűjtemény törlése
