# Get-AzsStorageContainerMigrationStatus

## Áttekintés
Egy tároló áttelepítési feladatának állapotát számítja ki.

## SZINTAXISA

```
Get-AzsStorageContainerMigrationStatus [-FarmName] <String> [-JobId] <String> [[-ResourceGroupName] <String>]
 [<CommonParameters>]
```

## Leírás
Egy tároló áttelepítési feladatának állapotát számítja ki.
