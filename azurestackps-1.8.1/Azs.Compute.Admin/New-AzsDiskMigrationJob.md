# New-AzsDiskMigrationJob

## Áttekintés
Felügyelt áttelepítési feladatot indít át a távtárolású lemezek áttelepítéséhez a megadott célhelyre.

## SZINTAXISA

```
New-AzsDiskMigrationJob [-Disks] <Disk[]> [-TargetShare] <String> [[-Location] <String>] [-Name] <String>
 [<CommonParameters>]
```

## Leírás
Lemezvezérlő-áttelepítési feladat létrehozása

## Példák

### --------------------------PÉLDA 1--------------------------
```
New-AzsDiskMigrationJob -Name "MyMigrationJob" -Disks $disks -location local -TargetShare "\\SU1FileServer.azurestack.local\SU1_ObjStore"
```

Felügyelt áttelepítési feladat indítása az első 20 lemezen
