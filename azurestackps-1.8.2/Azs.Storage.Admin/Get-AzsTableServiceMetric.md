# Get-AzsTableServiceMetric

## Áttekintés
A Table Service metrikájának listáját számítja ki.

## SZINTAXISA

```
Get-AzsTableServiceMetric [-FarmName] <String> [-ResourceGroupName <String>] [-Skip <Int32>] [-Top <Int32>]
 [<CommonParameters>]
```

## Leírás
A Table Service metrikájának listáját számítja ki.
