# Get-AzsOffer

## Áttekintés
A legfelső szintű szolgáltató nyilvános ajánlatok listájának beszerzése.

## SZINTAXISA

```
Get-AzsOffer [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Leírás
A legfelső szintű szolgáltató nyilvános ajánlatok listájának beszerzése.
