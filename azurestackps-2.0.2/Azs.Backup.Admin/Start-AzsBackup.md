# Start-AzsBackup

## Áttekintés
Adott helyről készítsen biztonsági másolatot.

## SZINTAXISA

### Létrehozás (alapértelmezett)
```
Start-AzsBackup [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentity
```
Start-AzsBackup -InputObject <IBackupAdminIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## Leírás
Adott helyről készítsen biztonsági másolatot.

## Példák

### 1. példa: azurestack biztonsági másolatának indítása
```powershell
PS C:\>Start-AzsBackup

```

Nyisson meg egy Azure-halom biztonsági mentést.
