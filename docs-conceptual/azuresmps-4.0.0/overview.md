# <a name="overview-of-the-azure-powershell-service-management-module"></a>Az Azure PowerShell Service Management modul áttekintése

Az Azure Service Management a klasszikus üzemi modell. Ez a modul lehetővé teszi azon klasszikus Azure-környezetek használatát, amelyek még nem lettek átalakítva az Azure Resource Managerhez. Az új környezeteknek Azure Resource Manager-metódusokat kell használniuk. A parancsmagok használatához telepítenie és konfigurálnia kell az Azure PowerShellt, és össze kell kapcsolnia a fiókjával. További információk: [Az Azure PowerShell telepítése és konfigurálása](install-azure-ps.md).
